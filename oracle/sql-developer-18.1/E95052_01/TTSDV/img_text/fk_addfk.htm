The figure shows the information for creating a foreign key on a TimesTen table. The connection navigator is displayed. The TimesTen table is right clicked and shows the following choices: Edit, Open, Export, Table, Column, Constraint, Index, Privileges, Aging, and Statistics. The Constraint option is selected and shows the following choices: Drop, Add Primary Key, Change Primary Key Parameters, and Add Foreign Key. The Add Foreign Key option is selected.
