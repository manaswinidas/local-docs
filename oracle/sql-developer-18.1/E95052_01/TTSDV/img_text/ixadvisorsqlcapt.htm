This figure shows the Index Advisor Configuration. The dialog has a drop-down list for the Capture Mode. The options are Prepare SQL and Execute SQL. Below the drop-down list is a check box that enables you to include optimizer hints for an Oracle BI server. At the bottom of the dialog are the Start and Cancel buttons.
