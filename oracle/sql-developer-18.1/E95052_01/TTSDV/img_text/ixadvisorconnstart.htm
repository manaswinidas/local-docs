This figure shows the Index Advisor - Connection Level Capture dialog. There are three tabs at the top: Configuration, Index Recommendations (this option is not available), and SQL. The Configuration tab is selected. Below the tabs is a drop-down list for the Capture Mode. The selected option for the Capture Mode is Prepare SQL. Below the drop-down list is a field and Browse button to specify a workload script. Below the Workload script field is a check box that enables you to include optimizer hints for an Oracle BI server. At the bottom of the dialog are the Start and Close buttons.
