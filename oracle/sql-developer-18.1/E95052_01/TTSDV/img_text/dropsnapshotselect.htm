This figure shows the Drop TimesTen Metrics Snapshot(s) dialog. There are two tabs at the top: Prompts and SQL. The Prompts tab is selected. Below the tabs is a Select all check box. The properties of the available snapshots are presented in a table format and include: Selector, Snapshot (which shows the ID of the snapshot), Timestamp, and Capture Level. At the bottom of the dialog are the Apply and Cancel buttons.
