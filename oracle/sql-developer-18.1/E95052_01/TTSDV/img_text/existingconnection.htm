This figure shows the information for an existing TimesTen connection. For the connection name, there is a box below the connection name with connection options. The existing connection is right clicked and shows the following choices: Connect (which is not available), Disconnect, Reconnect (which is not available), Schema Browser, Rename Connection, Delete. There are TimesTen specific options at the bottom. These options are: Start/Stop Cache Agent, Start/Stop Replication Agent, Set Cache Administration User, Change Usage-based Aging Attributes, Open SQL Worksheet, TimesTen Performance Metrics, Index Advisor, and Properties.
