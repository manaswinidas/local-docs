This figure shows the Index Advisor - Database Capture dialog. There are two tabs at the top: Control and Index Recommendations. The Index Recommendations tab is selected. Below the tabs is a Select All button. Below the Select All button is a table with the index recommendations. Below the table are the Create Selected Indexes and Save Indexes buttons. At the bottom of the dialog are the Start, Stop (this option is not available), and Close buttons.
