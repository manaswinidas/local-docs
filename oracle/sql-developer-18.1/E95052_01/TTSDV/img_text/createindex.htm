This figure shows the Create Index dialog. In the top left of the dialog are the Schema drop-down list and the Name text field. Below the Name field are the Properties tab and DDL tab. The Properties tab is the default selection. Below the tabs are the Schema drop-down list and the Table drop-down list. Below the Table field is the Type field. Below the type field are 3 choices for the index: Non-unique, Unique, Bitmap. There is a circle to the left of the index choices. You click in these circles to choose the index. Below the index type is a section where you enter the column name or expression for the index. To the right is the Order field. In the bottom, right of the dialog are the OK and Cancel buttons.
