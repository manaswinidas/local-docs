This figure shows the column information for a table. The Columns tab is the first tab in the figure. There are tabs located to the right of the columns tab. Below the tabs are icons. Below the icons is the data for the columns of the table. The data is grouped into columns. The headers for the columns are Column Name, Data Type, Nullable, Data Default, Column ID, and Primary Key. Below the headers are rows of data for each column of the table.
