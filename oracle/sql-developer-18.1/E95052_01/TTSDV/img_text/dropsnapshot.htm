The figure shows the information for editing the configuration of the TimesTen performance metrics. The connection navigator is displayed. The database is right clicked and shows the following choices: Connect (this option is not available), Disconnect, Reconnect (this option is not available), Schema Browser, Rename Connection, Delete, Add to Folder, Remove from Folder (this option is not available), Apply Filter, Clear Filter (this option is not available), Start/Stop Cache Agent, Start/Stop Replication Agent, Set Cache Administration User, Change Usage-Based Aging Attributes, Open SQL Worksheet, TimesTen Performance Metrics, and Properties. The TimesTen Performance Metrics option is selected and shows the following choices: Create Snapshot, Report, Edit Configuration, and Drop Snapshot. The Drop Snapshot option is selected.
