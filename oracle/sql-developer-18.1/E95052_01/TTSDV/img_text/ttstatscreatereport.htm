This figure shows the Create TimesTen Metrics Report dialog. The properties of the available snapshots are presented in a table format and include: Selector, Snapshot (which shows the ID of the snapshot), Timestamp, and Capture Level. In the middle right of the dialog is a Generate report button. Below this button is the report that you generated. At the bottom of the dialog are the Save and Close buttons.
