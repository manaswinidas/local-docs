This figure shows the Run PL/SQL dialog. In the top left of the dialog there is a Target section that lists the target. In the top right of the dialog there is a Parameters section. The parameters section describes the parameters of the PL/SQL block: Parameter, Data Type, Mode and Input Value. In the middle of the dialog there is PL/SQL Block section that shows the PL/SQL block. Below the PL/SQL Block section there are buttons: Save File, From File, and Reset. At the bottom of the dialog are the OK and Cancel buttons.
