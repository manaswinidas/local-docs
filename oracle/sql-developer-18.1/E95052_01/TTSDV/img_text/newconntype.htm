This figure shows the New Connection dialog. In the top middle of the dialog are fields where you enter information to create a new database connection. In the middle of the dialog there are the following tabs: Oracle, TimesTen, Access. The TimesTen tab is selected. Below the tabs are a Connection Type drop-down list and an Autocommit check box. The connection type drop-down list is expanded and shows the following options: DSN, Client/Server, and Advanced. After you enter the information, in the bottom right of the page there are five buttons: Save, Clear, Test, Connect, and Cancel. If you click the Connect button, you create a new database connection. There is also a button named Test for you to test the database connection.
