This figure shows the Load from Oracle Database Tables dialog. There are two panes: a schema browser and a table that lists selected Oracle database tables. The Oracle database schema is expanded. At the bottom of the dialog there is the Index Creation Policy drop-down list. The Index Creation Policy drop-down list is expanded and shows the following choices: Copy all indexes from peer Oracle database table, Copy unique indexes from peer Oracle database table, and Do not copy indexes from peer Oracle database table. The Copy all indexes from peer Oracle database table option is selected.
