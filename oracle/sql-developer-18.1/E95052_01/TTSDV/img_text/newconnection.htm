This figure shows the Connections pane in the main SQL Developer window. There are two nodes, Connections and Cloud Connections. The Connections node is right clicked and shows the following choices: New Connection, Import Connections, Export Connections (which is not available), Create Local Connections, and Disconnect Connections. You can also get a context menu for Cloud Connections.
