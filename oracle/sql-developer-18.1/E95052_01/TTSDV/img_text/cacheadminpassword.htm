This figure shows the Setting the cache administration user dialog. There are two tabs at the top left of the dialog: Prompts and SQL. The Prompts tab enables you to specify the Cache administrator user ID and cache administrator user password. The SQL tab shows the SQL used to specify the cache administration user. At the bottom of the dialog are tabs: Apply and Cancel. Choose Apply to set the cache administration user.
