This figure shows Oracle SQL Developer as a client application that executes SQL and PL/SQL with the Oracle Database. Oracle Data Miner is one component of Oracle SQL Developer. The ODMRSYS repository in the database stores metadata and APIs that support Oracle Data Miner operations and administration. The Oracle Data Miner repository supports multiple users. Oracle Data Miner uses the services of Oracle Data Mining, Oracle XML DB, Oracle Scheduler, and Oracle Text within Oracle Database Enterprise Edition.
