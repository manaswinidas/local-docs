This example of the Upgrade Older Repository error window indicates that the repository has an older version that is not compatible with this application, and prompts you to choose Yes or No if you want to upgrade the repository. It also warns you that all sessions with the ODMRUSER role will be disconnected if you select Yes. It also displays the client version, the repository version , and the upgrade version.
