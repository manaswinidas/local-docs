开发可拖动的小程序

通过指定draggable可以将参数拖动到浏览器之外，并动态转换为Java Web Start应用程序。可以通过按Alt键和鼠标左键并拖动鼠标来拖动Java小程序。开始拖动操作后，小程序将从其父容器中删除（ Applet要么JApplet ），并放置在新的未修饰的顶层窗口中（ Frame要么JFrame ）。拖动的小程序旁边会显示一个小的浮动关闭按钮。单击浮动的关闭按钮后，小程序将重新放置在浏览器中。从此以后，可以从浏览器中拖出的Java小程序将称为可拖动小程序 。

您可以通过以下方式自定义可拖动小程序的行为：

  • 您可以更改用于将小程序拖动到浏览器外部的按键和鼠标按钮顺序。
  • 您可以添加桌面快捷方式，该快捷方式可用于在浏览器外部启动应用程序。
  • 您可以定义将小程序拖到浏览器之外后应如何关闭。

以下各节描述如何实现和自定义可拖动小程序。的MenuChooserApplet类用于演示可拖动小程序的开发和部署。打开AppletPage.html在浏览器中查看新页面上的菜单选择器小程序。

注意：如果看不到小程序正在运行，则至少需要安装Java SE Development Kit（JDK）6 update 10发行版。
注意：如果看不到示例正在运行，则可能需要在浏览器中启用JavaScript解释器，以便Deployment Toolkit脚本能够正常运行。

启用拖动小程序的功能

您可以通过以下方式启用拖动小程序的功能： draggable参数true部署小程序时，如以下代码片段所示：

<script src="https://www.java.com/js/deployJava.js"></script>
<script>
    var attributes = { code:'MenuChooserApplet', width:900, height:300 };
    var parameters = { jnlp_href: 'draggableapplet.jnlp', draggable: 'true' };
    deployJava.runApplet(attributes, parameters, '1.6');
</script>

更改用于拖动小程序的按键和鼠标按钮顺序

您可以通过实现以下操作来更改用于拖动小程序的按键和鼠标按钮顺序isAppletDragStart方法。在以下代码片段中，可以通过按鼠标左键并拖动鼠标来拖动小程序：

 public boolean isAppletDragStart(MouseEvent e) {
        if(e.getID() == MouseEvent.MOUSE_DRAGGED) {
            return true;
        } else {
            return false;
        }
    }

当Applet与浏览器断开连接时，启用桌面快捷方式的添加

如果用户在将小应用程序拖动到页面之外后关闭浏览器窗口或导航离开该页面，则该小应用程序被称为与浏览器断开连接 。当小程序与浏览器断开连接时，可以为小程序创建桌面快捷方式。桌面快捷方式可用于在浏览器外部启动应用程序。要启用桌面快捷方式的创建，请添加offline-allowed和shortcut小程序的Java网络启动协议（JNLP）文件中的标签。

<information>
    <!-- ... -->
    <offline-allowed />
    <shortcut online="false">
        <desktop />
    </shortcut>
</information>
注意：根据用户Java控制面板中“快捷方式创建”的设置，在创建快捷方式之前，可能会提示用户进行确认。

定义应如何关闭小程序

您可以定义如何关闭小程序。例如，您的Swing小程序可以有一个JButton关闭小程序，而不是依靠默认的浮动“关闭”按钮。

Java插件软件为applet提供了ActionListener类。这个实例ActionListener类，也称为关闭侦听器 ，可用于修改小程序的默认关闭行为。

要定义应如何关闭小程序，请实施setAppletCloseListener和appletRestored小程序中的方法。

在以下代码段中， MenuChooserApplet类接收关闭的侦听器，并将其传递给的实例MenuItemChooser类：

MenuItemChooser display = null;
// ...
display = new MenuItemChooser();
// ...
public void setAppletCloseListener(ActionListener cl) {
    display.setCloseListener(cl);
}

public void appletRestored() {
    display.setCloseListener(null);
}

的MenuItemChooser类负责控制小程序的用户界面。的MenuItemChooser类定义一个JButton标记为“关闭”。当用户单击关闭按钮时，将执行以下代码：

private void close() {
    // invoke actionPerformed of closeListener received
    // from the Java Plug-in software.
    if (closeListener != null) {
        closeListener.actionPerformed(null);
    }
}

请求和自定义Applet装饰

从Java SE 7发行版开始，在部署小程序时，您可以指定拖动的小程序窗口应使用默认或自定义窗口标题进行装饰。

要启用拖动小程序的窗口装饰，请指定java_decorated_frame参数值为"true" 。要启用自定义窗口标题，请指定java_applet_title参数也。此参数的值应该是窗口标题的文本。

<script src="https://www.java.com/js/deployJava.js"></script>
<script>
    var attributes =
      { code:'SomeDraggableApplet', width:100, height:100 };
    var parameters =
      { jnlp_href: 'somedraggableapplet.jnlp', 
          java_decorated_frame: 'true',
          java_applet_title: 'A Custom Title'   
      };
    deployJava.runApplet(attributes, parameters, '1.7');
</script>

的java_decorated_frame和java_applet_title也可以在applet的JNLP文件中指定参数，如以下代码片段所示：

<applet-desc main-class="SayHello" name="main test" height="150" width="300">
    <param name="java_decorated_frame" value="true" />
    <param name="java_applet_title" value="" />
</applet-desc>
