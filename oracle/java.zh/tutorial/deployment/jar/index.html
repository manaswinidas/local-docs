课程：将程序打包为JAR文件

Java™存档（JAR）文件格式使您可以将多个文件捆绑为一个存档文件。通常，JAR文件包含与小程序和应用程序关联的类文件和辅助资源。

JAR文件格式具有许多优点：

  • 安全性 ：您可以对JAR文件的内容进行数字签名。然后，识别您签名的用户可以选择授予您的软件安全性特权，而该特权本来是不会的。
  • 减少下载时间 ：如果您的applet捆绑在JAR文件中，则可以通过一次HTTP事务将applet的类文件和相关资源下载到浏览器，而无需为每个文件打开新的连接。
  • 压缩 ：JAR格式允许您压缩文件以进行有效存储。
  • 扩展的打包 ：扩展框架提供了一种可以向Java核心平台添加功能的方法，并且JAR文件格式定义了扩展的打包。通过使用JAR文件格式，您还可以将软件转换为扩展名。
  • 程序包密封 ：可以选择密封JAR文件中存储的程序包，以便程序包可以实现版本一致性。将包密封在JAR文件中意味着必须在同一JAR文件中找到该包中定义的所有类。
  • 软件包版本控制 ：JAR文件可以保存有关其包含的文件的数据，例如供应商和版本信息。
  • 可移植性 ：处理JAR文件的机制是Java平台核心API的标准部分。
