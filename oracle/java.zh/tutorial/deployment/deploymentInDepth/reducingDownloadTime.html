减少下载时间

当用户尝试访问富Internet应用程序（RIA）时，将从网站上下载它们。（RIA可以在初次下载后缓存，以提高性能）。下载RIA所需的时间取决于RIA的JAR文件的大小。较大的JAR文件需要更长的下载时间。

您可以通过应用以下技术来减少RIA的下载时间：

  • 使用以下命令压缩RIA的JAR文件pack200工具。
  • 从Java网络启动协议（JNLP）文件和JavaScript文件中删除不必要的空格。
  • 优化图像和动画。

以下步骤描述了如何为签名的RIA创建和部署压缩的JAR文件。

  1. 使用规范化JAR文件--repack选项。

    此步骤确保启动RIA时，安全证书和JAR文件将通过验证检查。

    pack200 --repack DynamicTreeDemo.jar
  2. 签名标准化的JAR文件。 
    jarsigner -keystore myKeyStore DynamicTreeDemo.jar me
    哪里myKeyStore是密钥库的名称，并且me是密钥库的别名。
  3. 打包签名的JAR文件
    pack200 DynamicTreeDemo.jar.pack.gz DynamicTreeDemo.jar
  4. 设置jnlp.packEnabled财产true在RIA的JNLP文件中。 
    <resources>    
    <j2se version="1.6+"
        href="http://java.sun.com/products/autodl/j2se"
              max-heap-size="128m" />
    <jar href="DynamicTreeDemo.jar"
        main="true"/>
    <property name="jnlp.packEnabled"
        value="true"/>
    <!-- ... -->
</resources>

当。。。的时候jnlp.packEnabled属性是在JNLP文件中设置的，Java插件软件会使用以下命令查找压缩的JAR文件： .pack.gz扩展名（例如， DynamicTreeDemo.jar.pack.gz ）。如果找到，则Java Plug-in软件会自动解压缩并加载JAR文件。如果一个文件带有.pack.gz找不到扩展名，那么Java Plug-in软件会尝试加载常规JAR文件（例如， DynamicTreeDemo.jar ）。

注意：您需要在Web服务器上部署RIA，以测试jnlp.packEnabled属性。
