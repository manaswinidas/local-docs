添加外部库

自包含的应用程序包括应用程序需要运行的所有内容。如果您的应用程序需要外部库，则可以将该库添加到该应用程序的捆绑软件中。可以通过不同方式添加库。

协会演示中介绍的文件使用文件关联下载的Groovy库作为构建过程的一部分。该库放置在/lib应用程序项目中的目录。然后将此目录复制到/dist从中生成自包含应用程序捆绑包的目录。

下面的代码-pre-init任务build.xml该文件显示了如何下载该库：

<!-- download and copy groovy library -->
<copy toFile="lib/groovy-all-2.3.8.jar">
    <resources>
      <url url="http://central.maven.org/maven2/org/codehaus/groovy/groovy-all/2.3.8/groovy-all-2.3.8.jar"/>
    </resources>
</copy>

看到build.xml以获得完整的构建代码。

您可以从“独立应用程序示例”中下载文件关联演示的源文件。
