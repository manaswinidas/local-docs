创建CelsiusConverter GUI

本节说明如何使用NetBeans IDE创建应用程序的GUI。当您将每个组件从“组件面板”拖动到“设计区域”时，IDE会自动生成适当的源代码。

步骤1：设定标题

首先，设置应用程序标题JFrame只需单击“ Celsius Converter”即可JFrame在检查器中：

选择JFrame

然后，使用属性编辑器设置其标题：

设定标题

您可以通过双击title属性并直接输入新文本来设置标题，或通过单击按钮，然后在提供的字段中输入标题。或者，作为快捷方式，您可以单击JFrame检查器，然后直接输入其新文本，而无需使用属性编辑器。

步骤2：添加一个JTextField

接下来，拖动JTextField从选用板到设计区域的左上角。当您接近左上角时，GUI生成器会提供视觉提示（虚线），以提示适当的间距。使用这些提示作为指导，删除JTextField进入窗口的左上角，如下所示：

添加一个JTextField

您可能会想删除默认文本“ JTextField1”，但现在就保留它。我们将在本课程的稍后部分替换它，因为我们将对每个组件进行最终调整。有关此组件的更多信息，请参见如何使用文本字段 。

步骤3：添加JLabel

接下来，拖动JLabel进入设计区域。将其放在JTextField ，再次注意视觉提示是否暗示适当的间距。确保此组件的文本库与JTextField 。IDE提供的视觉提示应易于确定。

有关此组件的更多信息，请参见如何使用标签 。

步骤4：添加JButton

接下来，拖动JButton从面板中将其定位到左侧和下方JTextField 。同样，视觉提示有助于将其引导到位。

添加一个JButton
