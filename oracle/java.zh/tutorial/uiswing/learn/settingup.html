设置CelsiusConverter项目

如果您过去曾经使用过NetBeans IDE，则本节的大部分内容看起来会很熟悉，因为大多数项目的初始步骤都是相似的。不过，以下步骤描述了特定于此应用程序的设置，因此请务必严格遵循它们。

步骤1：创建一个新项目

要创建一个新项目，请启动NetBeans IDE，然后从“文件”菜单中选择“新建项目”：

创建一个新项目

每个命令的键盘快捷键出现在每个菜单项的最右侧。NetBeans IDE的外观在各个平台上可能有所不同，但是功能将保持不变。

步骤2：选择“常规”->“ Java应用程序”

接下来，从类别列中选择常规，并从项目列中选择Java应用程序：

步骤3：设定专案名称

现在输入“ CelsiusConverterProject”作为项目名称。您可以将“项目位置”和“项目文件夹”字段设置为默认值，或单击“浏览”按钮以选择系统上的备用位置。
