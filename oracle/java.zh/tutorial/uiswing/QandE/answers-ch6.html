答案：执行自定义绘画

问题

问题1 用什么方法定义JComponent油漆组件的内部？
答案1： paintComponent方法绘制组件的内部（其包含的任何子代除外）。

问题2。 下列哪个代码片段绘制了100x100像素的矩形（填充或未填充）？

一种。g.fillRect(x, y, 100, 100)
b。g.fillRect(x, y, 99, 99)
C。g.drawRect(x, y, 100, 100)
d。 b和c
e。 a和c

答案2：的drawRect方法在指定矩形的正下方绘制线条。因此，要获得100x100的矩形，您需要指定宽度和高度99， drawRect 。对于fillXxx另一方面，您可以精确指定所需的宽度和高度，因此g.fillRect(x, y, 100, 100)产生100x100的矩形。


问题3。 您将使用什么代码使组件以50％的透明度使用背景色执行下一个绘画操作？
答案三：

g2d.setComposite(AlphaComposite.getInstance(AlphaComposite.SRC_OVER, 0.5f));

练习题

练习1。 使用标准边框和自定义组件绘制，实现一个组件，该组件的首选大小为250x100，默认情况下是不透明的，具有5像素的黑色边框，并在屏幕上绘制“ X”（使用5像素粗线）前景色，如下图所示。

答案1：请参阅XMarksTheSpot.java ，您可以使用运行ComponentDisplayer.java 。

这是设置边框的代码XMarksTheSpot零件：

setBorder(BorderFactory.createMatteBorder(5,5,5,5, Color.BLACK));

这是来自paintComponent的方法XMarksTheSpot ：

Graphics2D g2 = (Graphics2D)g;
Insets insets = getInsets();
g2.setStroke(new BasicStroke(5.0f));
g2.draw(new Line2D.Double(insets.left,
                          insets.top,
                          getWidth() - insets.right,
                          getHeight() - insets.bottom));
g2.draw(new Line2D.Double(insets.left,
                          getHeight() - insets.bottom,
                          getWidth() - insets.right,
                          insets.top));

答案2：请参阅SquareIcon.java ，您可以使用经过修改的ButtonDemo.java 。

这是设置图标的代码：

Icon middleButtonIcon = new SquareIcon();
...
b2 = new JButton("Middle button", middleButtonIcon);

这是SquareIcon的三种方法的实现Icon接口：

public void paintIcon(Component c, Graphics g,
                      int x, int y) {
    if (c.isEnabled()) {
        g.setColor(Color.RED);
    } else {
        g.setColor(Color.GRAY);
    }

    g.fillRect(x, y, SIZE, SIZE);
}

public int getIconWidth() {
    return SIZE;
}

public int getIconHeight() {
    return SIZE;
}


练习3。 实现一个边框，该边框在其组件的顶部一直画一个红色的15像素高的条纹。通过用边界替换您在练习1中创建的组件上的边界来测试该边界。结果应如下图所示。 答案3：请参阅StripeBorder.java ，您可以使用运行ComponentDisplayer.java和XMarksTheSpot.java （请确保取消注释将边框设置为StripeBorder ）。这是StripeBorder的三种方法的实现Border接口：

public void paintBorder(Component c, Graphics g,
                        int x, int y,
                        int width, int height) {
    g.setColor(Color.RED);
    g.fillRect(x, y, c.getWidth(), HEIGHT);
}

public Insets getBorderInsets(Component c) {
    return new Insets(HEIGHT, 0, 0, 0);
}

public boolean isBorderOpaque() {
    return true;
}
