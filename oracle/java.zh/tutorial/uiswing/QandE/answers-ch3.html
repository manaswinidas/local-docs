答案：使用Swing组件

使用本课中的信息以及组件的操作方法部分可帮助您完成这些问题和练习。

问题

问题1：找到最适合以下每个需求的组件。写下组件的通用名称（例如“框架”），然后在线查找组件的操作方法页面。

问题1a ：让用户选择颜色的组件。
答案1a ： 颜色选择器

问题1b：显示图标但对用户单击不响应的组件。
答案1b ： 标签

问题1c：一个看起来像按钮的组件，当按下该组件时，会弹出一个菜单项供用户选择。
答案1c ： 无法编辑的组合框

问题1d：一个看起来像框架的容器，但是（通常与其他类似的容器一起）出现在真实框架中。
答案1d ： 内部框架

问题1e：一个使用户可以确定两个组件如何共享有限空间的容器。
答案1e ： 拆分窗格

问题2：您使用哪种方法将菜单栏添加到顶级容器（例如JFrame ？
答案2 ：setJMenuBar

问题3：您使用哪种方法为顶级容器（例如a）指定默认按钮JFrame要么JDialog ？
答案3 ： JRootPane的setDefaultButton方法。（您可以使用来获取顶级容器的根窗格getRootPane方法定义RootPaneContainer接口，每个顶级容器都实现）。

问题4：您使用哪种方法启用和禁用组件，例如JButton s？它在什么类别中定义？
答案4 ： setEnabled ，在Component类

问题5a：使用哪个Swing组件ListSelectionModel ？[ 提示：每个接口和类的规范顶部的“使用”链接带您到一个页面，显示在API中引用该接口或类的位置。]
答案5a ： JList和JTable

问题5b：这些组件是否使用任何其他模型来处理组件状态的其他方面？如果是这样，请列出其他型号的类型。
答案5b ： JList也使用了ListModel ，其中包含列表的数据。 JTable使用一个TableModel保存其数据和TableColumnModel管理表的列。

问题6：哪种类型的模型可以容纳文本组件的内容？
答案6 ：Document

练习题

练习1。 使用如下所示的GUI实现程序。将main方法放在名为MyDemo1 。

答案1 ：参见MyDemo1.java 。这是添加粗体，斜体文本的代码：

JLabel label = new JLabel("My Demo");
frame.getContentPane().add(BorderLayout.CENTER, label);
label.setFont(label.getFont().deriveFont(Font.ITALIC | Font.BOLD));
label.setHorizontalAlignment(JLabel.CENTER)

练习2 复制一份MyDemo1.java命名MyDemo2.java 。将菜单栏添加到MyDemo2 。
答案2 ：请参阅MyDemo2.java 。菜单栏可以用以下代码实现：

JMenu menu = new JMenu("Menu");
JMenuBar mb = new JMenuBar();
mb.add(menu);
frame.setJMenuBar(mb);

练习3。 复制MyDemo1.java至MyDemo3.java 。添加按钮（ JButton ） 至MyDemo3.java 。将其设置为默认按钮。
答案3 ：请参阅MyDemo3.java 。这是添加按钮并将其设置为默认按钮的代码：

JButton b = new JButton("A button");
frame.getContentPane().add(BorderLayout.PAGE_END, b);
frame.getRootPane().setDefaultButton(b);
