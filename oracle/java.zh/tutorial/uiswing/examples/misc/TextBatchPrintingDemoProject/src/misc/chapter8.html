第八章

女王的槌球地面

一棵大玫瑰树站在花园的入口附近：玫瑰长在上面，是白色的，但那里有三个园丁，忙着把它们涂成红色。爱丽丝认为这是一件非常奇怪的事情，她走近了看他们，正当他们走到他们身边时，她听到一个人说：“看，五个！别那样把油漆泼在我身上！'

“我忍不住了。”五人闷闷不乐地说。 ”“七个人慢跑了我的肘。'

七个在上面抬头说：'是的，五个！总是怪别人！'

你最好别说话！'五。``我听说女王只在昨天才说你应该被斩首！'

'做什么的？”第一个讲话的人说。

``那不关你的事 ，两个！'七说。

是的，这是他的事！五岁的孩子说，“我会告诉他的-这是为了带给厨师郁金香根而不是洋葱。'

七个人从他的笔刷上甩下来，刚开始“好吧，所有不公正的事情”-当他的眼睛有机会落在爱丽丝站着看着他们的那一刻时，他突然注视着自己：其他人也环顾四周，所有他们低下头。

“你能告诉我吗，”爱丽丝有些胆怯地说道，“为什么要画那些玫瑰？'

五点和七点什么也没说，但看着二点。两个人低声说道：“小姐，为什么事实是这样，这里应该是一棵红色的玫瑰树，而我们却误以为是白色的。如果女王想找出原因，我们都应该切断头脑。因此，小姐，您看到了，我们正在竭尽全力，在她来之前-'此时此刻，焦急地望着花园的五个人喊着“女王！皇后！”，三个园丁立刻将自己平放在脸上。有很多脚步声，爱丽丝转过身，渴望见见女王。

首先是十名携带俱乐部的士兵。他们的形状都像三个园丁，长而扁平，手脚在角落。这些都被装饰满了钻石，像士兵一样，分别走了两个和两个。这些王室的孩子们来了。那里有十个，小伙伴们成对地欢快地跳起来，成对地跳舞：他们都被点缀着心。接下来是来宾，主要是国王和皇后，其中的爱丽丝认出了白兔子：它匆匆紧张地说话，对所说的一切微笑着，并没有注意到她。然后跟随红心大战，将国王的王冠戴在深红色天鹅绒坐垫上。最后，在这场盛大的游行中，出现了“国王与皇后”。

爱丽丝很怀疑她是否应该像三位园丁一样躺在她的脸上，但是她不记得曾经在游行中听到过这样的规则。她说：“此外，游行的用处是什么，如果人们都躺在他们的脸上，使他们看不见的话？因此，她站着不动，等待着。

当游行队伍来到爱丽丝对面时，他们都停下来看着她，女王大声说：“这是谁？她对红心大侠说了那句话，红心只鞠了一躬，笑了笑。

'白痴！女王说，不耐烦地把头扔了。然后，转向爱丽丝，她继续说道：“孩子，你叫什么名字？'

“我叫爱丽丝，请Ma下。”爱丽丝很客气地说。但她对自己补充说：“毕竟，它们只是一副纸牌。我不用怕他们！'

“他们是谁？女王说，指着躺在玫瑰树上的三个园丁。因为，你看，因为他们躺在他们的脸上，并且背上的图案与其余的背包相同，她无法分辨他们是园丁，军人，朝臣还是自己的三个孩子。

'我怎么知道？爱丽丝对自己的勇气感到惊讶。“这与我无关。'

女王怒不可遏地变成了深红色，在瞪着她一会儿像野兽一样后，尖叫着说：“甩开她的头！关-'

'废话！爱丽丝大声而坚定地说道，女王沉默了。

国王把手放在她的手臂上，胆怯地说：“考虑，亲爱的：她只是个孩子！'

女王怒气冲冲地离开他，对him夫说：“把他们翻过来！'

Knave非常小心地用一只脚这样做。

'起床！女王用一种尖锐而响亮的声音说，三个园丁立刻跳了起来，开始向国王，女王，王室子女以及其他所有人鞠躬。

'别说了！女王大叫。'你让我头晕。“然后转身对玫瑰树，她接着说，”你一直在这里做什么？'

“请您Ma下。”两人以一种非常谦逊的语气说，他跪下时说道，“我们正在努力-”

'我懂了！女王说，他一直在检查玫瑰。'开着头！游行队伍继续前进，其中三名士兵留在后面，处死了不幸的园丁，后者跑向爱丽丝寻求保护。

'你不会被斩首！爱丽丝说，她把它们放到了一个大花盆里，那花盆就站在附近。三名士兵徘徊了一两分钟，寻找他们，然后悄悄地向其他士兵进发。

'他们的头掉了吗？女王大喊。

“请他们s下，如果it下！士兵们大声回应。

'那就对了！女王大喊。'你会打槌球吗？'

士兵们保持沉默，看着爱丽丝，因为这个问题显然是对她的。

'是！爱丽丝大喊。

“那就来！女王大吼，爱丽丝参加游行，非常想知道接下来会发生什么。

``-今天天气很好！”她一边胆怯的声音说。她正走过白色兔子，后者焦急地窥视着她的脸。

“很好，”爱丽丝说：“-公爵夫人在哪里？'

嘘！嘘！”兔子低沉而仓促地说道。他说话时焦急地看着肩膀，然后tip起脚尖，嘴巴靠近她的耳朵，低声说：“她正在被处决。'

'做什么的？爱丽丝说。

“您是说“真可惜！“？兔子问。

“不，我没有，”爱丽丝说：“我不认为这很可惜。我说：“为什么？”

“她打了女王的耳朵-”兔子开始说。爱丽丝发出一点笑声。'哦，安静！兔子吓得小声说。女王会听到你的声音！你看，她来得晚了，女王说-”

'去你的地方！女王用雷声喊道，人们开始四面八方奔跑，互相撞倒。但是，他们在一两分钟内就安定下来，比赛开始了。爱丽丝认为她一生中从未见过如此奇特的槌球。到处都是山脊和犁沟。球是活的刺猬，木槌是火烈鸟，士兵们不得不加倍抬起双手，站起来制作拱门。

爱丽丝起初发现的主要困难是管理火烈鸟：她成功地将其身体藏在手臂下，足够舒适地收起，双腿垂下，但总的来说，就像她的脖子很好地伸直了一样，想要刺猬的头，它会转过身，抬头看着她的脸，以一种迷惑的表情使她忍不住大笑起来：当她低下头去时重新开始时，非常令人激动的是，发现刺猬已经展开，并且正在爬行：除了所有这些以外，无论她想把刺猬送往何处，通常都有一条山脊或犁沟，并且，因为倍增的士兵总是起身前往地面上的其他地方，所以爱丽丝很快得出结论，那确实是一场非常困难的比赛。

玩家们一次都玩而无需等待轮流，一直吵架，并为刺猬而战。女王在很短的时间内就充满了激情，走来走去，高喊着头！”或“抬起头！大约每分钟一次。

爱丽丝开始感到非常不安：可以肯定的是，她还没有与女王发生争执，但是她知道这可能会在任何时候发生，然后，然后，她想，“对我会有什么影响？他们非常喜欢在这里斩首人头。伟大的奇迹是，还有任何一个还活着！'

她正在寻找某种逃生途径，想知道是否能逃脱而不会被看见，这是因为她注意到空中出现了一个奇怪的现象：起初她非常困惑，但是看了一两分钟后，她发现她笑着对自己说：“那是柴郡猫：现在我要找人说话。'

'你过得怎么样？猫说，只要它有足够的嘴可以说话。

爱丽丝等到眼睛出现，然后点了点头。她想，“对它说话毫无用处，直到它的耳朵来了，或者至少有一只耳朵来了。”再过一分钟，整个脑袋出现了，然后爱丽丝放下了火烈鸟，开始记下这场比赛，感到很高兴她能有人听她讲话。猫似乎认为眼前已经有足够的东西了，没有更多的东西出现了。

爱丽丝开始用一种抱怨的语气说：“我认为他们的表现完全不公平，他们都争吵得如此可怕，以至于听不到自己的声音-而且他们似乎并没有特别的规则。 ;至少，如果有的话，没有人会照看他们-而且你不知道所有活着的东西到底有多令人困惑；例如，有一个拱门，我必须穿过它，然后在地面的另一端走来走去-我刚才应该对女王的刺猬进行槌球，只有在看到我的来袭时才跑掉！'

'你喜欢女王吗？猫低声说。

“一点也不，”爱丽丝说：“她太极端了-”就在这时，她注意到女王就在她身后，听着：所以她继续说道，“-很可能会获胜，完成比赛几乎不值得。'

女王微笑着继续前进。

“你是谁说话？国王说，上去爱丽丝，好奇地看着猫的头。

爱丽丝说：“这是我的一个朋友-柴郡猫（Cheshire Cat），请允许我介绍一下。'

国王说：“我一点都不喜欢它的外观。”但是，如果喜欢的话，它可能会吻我的手。'

猫说：“我不想。”

国王说：“不要轻描淡写，也不要那样看着我！讲话时，他落后于爱丽丝。

爱丽丝说：“猫也许看着国王。”``我已经在某本书中阅读过，但我不记得在哪里。'

“好吧，必须将它清除掉。”国王非常坚定地说道，然后他打电话给当时正在过去的女王，“亲爱的！我希望你能把这只猫删除！'

女王只有一种解决所有大小困难的方法。“他的头开了！她说，甚至没有四处张望。

国王热切地说道，“我自己去拿the子手。”他赶紧走了。

爱丽丝想她也应该回去看看比赛的进行情况，因为她听到远处女王的声音，热情地尖叫着。她已经听到了三位因错过回合而被处决的球员的判决，她根本不喜欢事物的外观，因为比赛太混乱了，她甚至不知道是否轮到她了。于是她去寻找刺猬。

刺猬正在与另一只刺猬进行战斗，这对爱丽丝来说是一个很好的机会，让他们一只和另一只门球拍：唯一的困难是，她的火烈鸟被带到了花园的另一边，爱丽丝可以在那里看到它试图以一种无助的方式飞到一棵树上。

当她抓到火烈鸟并带回火烈鸟时，战斗结束了，两只刺猬都不在视线内：“但这没什么大不了的，”爱丽丝想，“因为所有的拱门都从这边走了地面。因此，她把它藏在手臂下，以免它再次逃脱，然后回去和她的朋友再谈一点。

回到柴郡猫（Cheshire Cat）时，她惊讶地发现周围聚集了一大群人：the子手，国王和王后之间发生了争执，他们都立刻讲话，而其余的人很沉默，看起来很不舒服。

爱丽丝出现的那一刻，三个人都向她求婚，以解决这个问题，尽管他们都一次讲话，但她还是重复了自己的论点，但她发现确实很难弄清他们所说的话。

刽子手的理由是，你不能切断头部，除非是从把它割下体：他以前从未做这样的事情，他不打算开始在他生命的时间。

国王的论点是，任何有头脑的人都可以被斩首，而且你不要胡说八道。

女王的论点是，如果不花任何时间对此事做任何事情，她就会让所有人都被处决。（这是最后一句话，使整个党派显得如此严肃和焦虑。）

爱丽丝想不出其他话要说，但“它属于公爵夫人：你最好向她询问。'

女王对the子手说：“她在监狱里，在这里抓她。'the子手像箭一样飞了出去。

猫走了那一刻，猫的头就开始消失了，到他回到公爵夫人的时候，它已经完全消失了。因此，国王和the子手四处奔走寻找，而其他党派则回到比赛中。
