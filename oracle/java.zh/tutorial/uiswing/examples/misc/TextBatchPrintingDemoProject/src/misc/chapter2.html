第二章

眼泪池

'越来越好奇！爱丽丝哭了（她非常惊讶，以至于此刻她完全忘记了如何说一口流利的英语）； “现在，我像有史以来最大的望远镜一样向外开放！再见，脚！”（因为当她低头看着脚时，它们似乎几乎看不见了，它们离得太远了）。“哦，我那可怜的小脚丫，我想知道，亲爱的，谁现在会穿上你的鞋子和长筒袜？我确定我不会！我要为自己烦恼还差得远。 -但是我必须对他们好，”爱丽丝想，“否则他们不会走我想走的路！让我看看：每个圣诞节我都会给他们一双新靴子。'

她继续自己计划如何管理它。她想：“他们必须经过航空母舰。” “以及将礼物寄到自己的脚下看起来会多么有趣！方向看起来多么奇怪！

爱丽丝的右脚，ESQ。

心跳

在防御者附近，

（爱丽丝的爱）。

哦，亲爱的，我在胡说八道！'

就在这时，她的头撞到了大厅的屋顶上：事实上，她现在已经超过了9英尺高，她立刻拿起了小金钥匙，匆匆赶到花园的门前。

可怜的爱丽丝！她一边躺一边一边用一只眼睛直望花园，这是她所能做的。但是通行比以往任何时候都更加无望：她坐下来，再次哭了起来。

“你应该为自己感到羞耻，”爱丽丝说，“像你这样的伟大女孩，”（她可能会这么说），“继续这样哭！停止这一刻，我告诉你！但是，她还是一样，继续流下加仑的眼泪，直到周围环绕着一个大水池，大约四英寸深，到达大厅的一半。

过了一会儿，她听到远处传来一阵脚掌的，啪声，然后她急忙干dried了眼睛，看看即将发生什么。是只白兔子回来，穿着华丽，一只手戴着一副白色小孩子的手套，另一只手则戴着一扇大风扇：他急忙小跑，小步自言自语，“哦！公爵夫人，公爵夫人！哦! 如果我一直等她，她会不会野蛮！爱丽丝感到非常绝望，她随时准备寻求任何人的帮助。因此，当兔子靠近她的时候，她以一种怯而低沉的声音开始说：“先生，请您来-”兔子猛烈地起身，放下了白手套和风扇，并艰难地向黑暗中走去。尽他所能。

爱丽丝拿起风扇和手套，由于大厅非常热，她一直在扇着扇子说话：“亲爱的，亲爱的！今天的一切真奇怪！昨天一切照常进行。我想知道我是否在夜晚被改变了？让我想想：今天早上起床时我还是一样吗？我几乎以为我会记得有些不同。但是如果我不一样，那么下一个问题是：我到底是谁？啊， 那真是个大难题！“然后她开始考虑所有她认识的与她同龄的孩子，看看她是否可以换个孩子。

她说：“我确定我不是艾达，因为她的头发穿了这么长的发lets，而我的头发根本没有发ring。我确定我不能成为Mabel，因为我知道各种各样的事情，她，哦！她知道的很少！此外， 她是她， 我是我，而且-亲爱的，这一切令人困惑！如果我知道我以前知道的所有事情，我会尝试。让我看看：四乘五是十二，四乘六是十三，四乘七是-亲爱的！我永远也不会达到这个速度！但是，乘法表并不表示：让我们尝试一下地理。伦敦是巴黎的首都，巴黎是罗马和罗马的首都。不， 这是完全错误的，我敢肯定！我一定是换了Mabel！我会试着说“小孩子怎么办 ”，然后她就像在讲课一样在膝盖上交叉了双手，开始重复一遍，但是她的声音听起来沙哑而奇怪，但这些话没有来与过去一样：-

小鳄鱼怎么办

改善他闪亮的尾巴，

倒入尼罗河的水

在每一个黄金尺度上！


``他似乎很高兴地笑着，

爪子如何整齐地张开，

并欢迎小鱼

轻轻地笑着下巴！'

可怜的爱丽丝说：“我确定那是不正确的话。”继续前进时，她的眼睛又充满了泪水。 ，并且几乎没有玩具可玩，哦！有太多的教训要学习！不，我已经决定了。如果我是Mabel，我会留在这里！他们低着头说：“再来，亲爱的！“我只会抬头说：“那我是谁？先告诉我，然后，如果我喜欢成为那个人，我会上来：如果不喜欢，我会一直待在这里直到我成为其他人。”但是，亲爱的！爱丽丝喊着，以眼泪突然爆发，“我真希望他们会把他们的头了！我这样很厌倦了独自一人在这里！'

正如她所说的那样，她低头看着自己的手，很惊讶地看到她在说话时戴上了Rabbit的白色小小孩手套之一。“我怎么能这样做呢？' 她想。”我必须再次变小。她起身去桌子旁用它测量自己，发现，她几乎可以猜到，她现在高约2英尺，并且正在迅速萎缩：她很快发现造成这种情况的原因是她所握住的风扇，于是为了避免完全缩水，她急忙放下了风扇。

``那是一个狭窄的逃生！爱丽丝说，突然的变化使很多人感到恐惧，但很高兴发现自己仍然存在。 '现在去花园！然后她全速跑回小门：但是，!！小门又关上了，小金钥匙像以前一样躺在玻璃桌上，“事情比以往任何时候都糟，”这个可怜的孩子想，“因为我从来没有像现在这么小，从来没有！我声明这太糟糕了！'

当她说出这些话时，她的脚滑了下来，又过了一会儿，飞溅起来！她在盐水中下巴。她对自己说：“她的第一个想法是，她以某种方式掉入海中，那样的话，我可以乘铁路回去了。”（爱丽丝一生中曾经去过海边，并得出了一个普遍的结论，那就是，无论您去英格兰海岸的哪个地方，都会在海里发现许多沐浴机，有些孩子用木铲挖沙子，然后是一排住宿房，在它们后面是火车站。）但是，她很快发现自己在9英尺高的眼泪中流泪。

``我希望我没有那么哭！爱丽丝游动时说，试图找到自己的出路。“我想，现在我将为此被淹死在自己的眼泪中！可以肯定，这将是一件奇怪的事！但是，今天一切都很奇怪。'

就在这时，她听到游泳池附近溅出了一些东西，然后她游得近些才知道那是什么：起初，她以为一定是海象或河马，但后来她想起自己现在很小了。很快发现，只有一只老鼠像她一样滑了进去。

爱丽丝想，“现在和这只老鼠说话会有用吗？”这里的一切都太遥不可及，以至于我很可能会说：无论如何，尝试没有任何危害。”于是她开始说：“哦，老鼠，你知道从这个水池出来的路吗？我非常厌倦在这里游泳，老鼠啊！'（爱丽丝认为这一定是对老鼠说话的正确方法：她以前从未做过这样的事情，但她记得自己在哥哥的拉丁文法文中看到过，“老鼠-老鼠-老鼠- -鼠标-鼠标！'）老鼠相当好奇地看着她，对她来说似乎是用一只小眼睛眨眨眼，但它什么也没说。

爱丽丝想：“也许它听不懂英语。” “我敢说这是一只法国老鼠，与征服者威廉一起过来。'（因为，爱丽丝以其所有的历史知识，并没有很清楚的认识，多年前发生了什么事。）所以她又开始说：'哦，聊天吧？”，这是她法语课本中的第一句话。老鼠突然跳出水面，似乎吓得全身发抖。'哦，对不起！爱丽丝仓促地哭了，害怕她伤害了这只可怜的动物的感情。'我完全忘记了你不喜欢猫。'

'不像猫！老鼠用尖利而热情的声音喊道。“ 你会喜欢猫，如果你是我？'

“好吧，也许吧，”爱丽丝用一种舒缓的语气说：“不要为此生气。但是我希望我能向您展示我们的猫Dinah：如果您只看到猫，我想您会看中猫的。她是如此可爱的安静的事物，”爱丽丝在游泳池里懒洋洋地游来游去，爱丽丝继续说着自己的话，“而她坐在火炉旁，舔着爪子，洗了洗脸，真是太好了。护理是一件柔软的事情-而且她是捉老鼠的好手-噢，求求您了！爱丽丝再次哭泣，这次是老鼠满脑子都是疯了，她确定必须真的得罪了。``如果你不想的话，我们不会再谈论她了。'

'我们确实！老鼠哭了，老鼠发抖到尾巴的尾巴。'好像我会谈论这样的话题！我们一家人一直讨厌猫：讨厌，低矮，庸俗的东西！不要再让我听到这个名字了！'

``我真的不会！爱丽丝急切地想改变话题。“您-您喜欢狗吗？“老鼠没有回答，所以爱丽丝急切地说道：“我们家附近有一只很好的小狗，我想带你看看！你知道，有一只眼睛明亮的小猎犬，有着长长的棕色卷发！它会在您扔东西时取回东西，它会坐起来乞讨晚餐，还有各种各样的东西-我不记得其中的一半-它属于农夫，您知道，他说这很有用，价值一百英镑！他说它杀死了所有老鼠，而且-亲爱的！”爱丽丝悲哀地喊道，“恐怕我又冒犯了它！'因为老鼠正在尽力向她游去，并且在游动时在水池中引起很大的骚动。

于是她轻声叫道，“亲爱的老鼠！再回来一次，如果您不喜欢猫或狗，我们也不会谈论它们！“当老鼠听到声音时，它转过身，慢慢向她游去：脸色苍白（充满激情，爱丽丝想着），它颤抖的声音说道：“让我们到岸边，然后我告诉你我的历史，你会明白为什么我讨厌猫和狗。'

到了该去的时候了，因为游泳池里挤满了掉进去的鸟和动物：有只鸭子和渡渡鸟，一只鹦鹉和一只小鹰，还有其他几种好奇的生物。爱丽丝一路领先，整个队伍游到岸边。
