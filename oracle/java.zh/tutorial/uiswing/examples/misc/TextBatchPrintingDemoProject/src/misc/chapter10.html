龙虾四重奏

“谢谢你，”爱丽丝说，“这很有趣。我以前从未对白鲸一无所知。'

“我从没想过，”爱丽丝说。'为什么？'

“这是靴子和鞋子。”狮ry非常庄严地回答。

爱丽丝完全不解。'靴子和鞋子！她以一种令人惊讶的语气重复道。

“为什么， 你的鞋子是干什么的？狮ry说。'我是说，是什么让它们如此闪亮？'

爱丽丝低头看着他们，在她回答之前稍作考虑。”我相信，它们已经变黑了。'

“海底的靴子和鞋子，”狮ry以低沉的声音继续说道，“白白地做完了。现在你知道了。'

他们是由什么制成的？爱丽丝好奇地问。

狮phon当然不耐烦地回答：“鞋底和鳗鱼，任何虾都可以告诉你的。”'

真的吗？爱丽丝惊讶地说道。

“当然不是，”模拟乌龟说：“为什么，如果有一条鱼来找我 ，告诉我他要去旅行，我应该说：“用什么海豚？”

“我的意思是我说的，”嘲笑的龟嘲讽地说。狮ry队又说：“来，让我们听听您的一些冒险经历。'

爱丽丝有些胆怯地说：“我可以告诉你我的冒险经历-从今天早上开始。”但是回到昨天是没有用的，因为那时我是另一个人。'

'不，不！狮first先冒险了，”鹰头狮不耐烦地说道：“探险花费了如此可怕的时间。'
