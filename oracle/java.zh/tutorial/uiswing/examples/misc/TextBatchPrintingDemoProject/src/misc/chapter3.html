第三章

核心种族和长篇故事

他们的确是一个看起来很酷的派对，聚集在河岸上—羽毛drag乱的鸟儿，紧贴着毛皮的动物，全都湿淋湿，交叉且不舒服。

当然，第一个问题是如何再次变干：他们对此进行了咨询，几分钟后，爱丽丝很自然地发现自己与他们交谈，就好像她一生都认识他们一样。的确，她对劳瑞（Lory）争论了很长时间，劳瑞（Lory）最终变得阴郁，只能说：“我比你大，必须知道得更多”。爱丽丝不知道自己多大年纪就不允许这样做，而且由于劳瑞（Lory）积极拒绝透露其年龄，因此无话可说了。

最后，似乎是其中一位权威人物的老鼠叫道：“你们大家坐下，听我说！ 我很快会让你足够干！他们全都一下子坐了下来，中间是老鼠。爱丽丝焦急地注视着它，因为她觉得如果不尽快变干她会感冒重感冒。

“啊！老鼠以一种重要的姿态说道，“大家都准备好了吗？这是我所知道的最干燥的事情。如果您愿意，请保持沉默！“征服者威廉受到教皇的青睐，很快就被要求领导的英国人屈服，并且最近已经习惯了篡夺和征服。埃德温和莫尔卡，即梅西娅和诺森比亚的伯爵-” '

'啊！洛瑞颤抖着说道。

'请再说一遍！老鼠皱着眉头说，但很客气：“你说吗？'

'不是我！洛里急忙说。

“我以为你做到了，”老鼠说。'-我继续。 “梅西娅和诺森比亚的伯爵埃德温和莫尔卡尔向他宣布：甚至坎特伯雷爱国大主教斯蒂格德也认为这是可取的-”

'发现了什么 ？鸭子说。

“找到了它 ，”老鼠相当不满意地回答：“当然，您知道“它”的含义。'

鸭子说：“当我发现一件东西时，我知道“它”的含义就足够了。”它通常是青蛙或蠕虫。问题是，大主教找到了什么？'

老鼠没有注意到这个问题，但是急忙继续说，'-发现与埃德加·阿特林（Edgar Atheling）见面并向他提供皇冠是明智的。威廉一开始的举止是温和的。但是他的诺曼底人的无礼-“亲爱的，你现在过得怎么样？继续，转身向爱丽丝讲话。

爱丽丝用忧郁的语气说：“像从前一样潮湿”：“这似乎并没有使我干燥。'

“在那种情况下，”渡渡鸟庄严地说道，“我动议会议休会，以立即采取更有力的补救措施-”

'说英语！”伊格特说。“我不知道这些长词一半的含义，而且，我也不相信你也这样做！”“鹰鹰”弯下腰掩饰着微笑：其他一些鸟儿发出嘶哑的声音。

渡渡鸟怒气冲冲地说：“我要说的是，让我们干的最好的事情就是考克斯种族。'

'什么是核心小组竞赛？爱丽丝说；不是说她想知道什么，而是渡渡鸟停了一下，好像觉得有人应该说话，而且似乎没人愿意说什么。

“为什么”，渡渡鸟说，“解释它的最好方法就是这样做。'（而且，您可能想亲自尝试一下，在某个冬天的日子里，我会告诉您渡渡鸟是如何做到的。）

首先，它以某种圆圈的形式划出了一个赛马场（“确切的形状无关紧要”，然后说），然后所有的聚会都沿着赛道在这里和那里放置。没有“一，二，三和远”，但他们在喜欢的时候就开始跑步，而在喜欢的时候就停下来了，所以不容易知道比赛何时结束。但是，当他们跑步了半个小时左右，又又变得非常干燥时，渡渡鸟突然喊道：“比赛结束了！”他们都围着它拥挤，气喘吁吁地问，“但是谁赢了？'

渡渡鸟不加思索就无法回答这个问题，它用一个手指压在额头上很长时间（您通常在他的照片中看到莎士比亚的位置），而其余的则等待着。安静。最后，渡渡鸟说：“ 每个人都赢了， 所有人都必须有奖品。'

“但是谁来给奖品呢？相当多的声音问。

“为什么， 她 ，当然。”渡渡鸟用一根手指指着爱丽丝说。整个聚会立刻拥挤了她，以一种困惑的方式喊道：“奖品！奖品！'

爱丽丝不知道该怎么办，在绝望中，她把手伸到口袋里，掏出一盒饼干（幸运的是盐水没进去），并把它们递给了奖品。整个过程中只有一个。

老鼠说：“但是她自己必须得奖。”

“当然，”渡渡鸟很严肃地回答。'你还有什么东西在口袋里？他继续说，转向爱丽丝。

“只有一个顶针，”爱丽丝悲伤地说道。

“把它交给这里。”渡渡鸟说。

然后他们都再次拥挤她，而渡渡鸟郑重地提出了顶针，说：“我们请您接受这种优雅的顶针”；当这简短的演讲结束后，所有人都为之欢呼。

爱丽丝认为整件事很荒谬，但他们看上去都很严肃，以至于她都不敢笑。而且，由于她想不出要说的话，她只是鞠了一躬，拿了顶针，看上去很庄重。

接下来的事情是吃这种羊杂：这引起了一些噪音和混乱，因为大鸟抱怨它们不能品尝它们的味道，而小鸟则cho咽了，不得不拍打在后面。然而，终于结束了，他们又坐了下来，恳求老鼠告诉他们更多事情。

爱丽丝说：“你答应告诉我你的历史，你知道为什么，以及为什么讨厌你-C和D，”她小声说道，半是害怕它会再次冒犯。

'我的故事是一个漫长而可悲的故事！老鼠说，转向爱丽丝，叹了口气。

“那肯定是一条长长的尾巴。”爱丽丝惊讶地低头看着老鼠的尾巴。 “但是为什么你称它为悲伤？'而且当老鼠在说话时，她一直对此感到困惑，所以她对这个故事的想法是这样的：

“弗里对老鼠说，他在屋子里遇见了他，“让我们俩都去法律：我将起诉您 。没事做。”老鼠对老兄说：“亲爱的先生，这样的审判，没有陪审团或法官，将会浪费我们的呼吸。”狡猾的老愤怒说：“我将作为法官，陪审团。我将全力以赴，并谴责你至死。”

'你不参加！'老鼠严厉地对爱丽丝说。'你在想什么？'

“对不起，”爱丽丝谦卑地说：“我想你已经到第五弯了？'

'我没有 ！'老鼠非常愤怒地喊道。

'一个结！爱丽丝一直准备让自己变得有用，并焦急地看着她。'哦，请让我帮助您撤消它！'

“我不会做任何事情，”老鼠起身走开。'你通过胡说八道侮辱我！'

``我不是那个意思！恳求可怜的爱丽丝。“但是你很容易生气，你知道！'

鼠标只在回应中咆哮。

'请回来并完成您的故事！爱丽丝追赶它;其他人都参加了合唱，“是的，请这样做！'但是鼠标只是不耐烦地摇了摇头，然后走得更快。

``可惜不会留下来！劳瑞（Lory）视而不见，叹了口气。老蟹趁机对女儿说：“啊，亲爱的！让这是一个教训，你永远不会失去你的脾气！”“妈，别说了！”年轻的螃蟹有点生气地说道。“您足够尝试牡蛎的耐心！'

``我希望我在这里有我们的迪娜，我知道我做到了！爱丽丝大声说，没有特别针对任何人。她很快就会把它取回来！'

“迪娜是谁，如果我敢冒险问这个问题？洛里说。

爱丽丝急切地回答，因为她一直准备谈论自己的宠物：“黛娜是我们的猫。她是捉住你想不到的老鼠的资本！哦，我希望你能在飞鸟之后见到她！为什么，她一看就会吃一只小鸟！'

这次讲话引起了党内的极大轰动。有些鸟立刻赶了出去：一只老喜Mag开始非常小心地把自己包裹起来，说：“我真的一定要回家；我必须回家。夜空中不适合我的喉咙！”，金丝雀用颤抖的声音向孩子们喊道，“亲爱的，快走吧！现在是时候您都在床上了！'他们以各种借口全部离开，而爱丽丝很快就离开了。

“我希望我没有提到黛娜！她用忧郁的口气对自己说。“在这里似乎没有人喜欢她，而且我敢肯定，她是世界上最好的猫！哦，亲爱的黛娜！我想知道我是否还会再见到你！'可怜的爱丽丝在这里再次哭泣，因为她感到非常孤独和沮丧。过了一会儿，她再次听到远处的脚步声，她急切地抬起头，一半希望老鼠改变了主意，然后回来完成他的故事。
