第一章

放下兔子的洞

爱丽丝开始厌倦了她的姐姐在银行旁坐着，无所事事：她曾一两次偷窥妹妹正在读的书，但里面没有照片或谈话，“还有什么？是爱丽丝（Alice）说的，没有书籍或对话的书籍的使用吗？'

因此，她一直在考虑自己的想法（由于天气炎热，这使她感到非常困倦和愚蠢）（尽其所能），制造菊花链的乐趣是否值得起床挑选雏菊的麻烦，突然，一只粉红色的眼睛的白兔子跑到她身边。

没什么那么特别了不起的。也没有爱丽丝觉得这样很闪开听到兔子自言自语地说，“哦，亲爱的！噢亲爱的！我会迟到！”（当她后来考虑时，她想到她应该对此感到奇怪，但当时一切似乎很自然）；但是当兔子实际上从背心口袋里拿出一只手表，看了看，然后匆匆忙忙时，爱丽丝开始站起脚来，因为它闪过她的脑海，她从未见过有背心的兔子。口袋里，或者是要拿出的手表，好奇地燃烧着，她跑过去穿过田野，幸运的是，正好是时候看到它从树篱下的一个大兔子洞里弹出了。

再过一会儿，爱丽丝紧追不舍，再也没有考虑过要再次走出去。

兔子洞像隧道一样一直往前走了一段时间，然后突然下沉，以至于爱丽丝突然间没有时间考虑停下脚步，直到她发现自己跌倒了很深的井。

井是很深，还是跌得很慢，因为当她下山时，她有很多时间去看她，想知道接下来会发生什么。首先，她试图低头看清楚自己要去的东西，但是太黑了，看不见任何东西。然后她看了看井的侧面，发现它们充满了橱柜和书架。她在这里到那里看到地图和图片挂在钉子上。她过去时，从一个架子上取了一个罐子。它被贴上了“橘子果酱”的标签，但令她非常失望的是它是空的：她不喜欢把罐子扔掉，因为担心会杀死某人，所以当她掉下来时设法把它放进一个柜子里。

'好！“爱丽丝自言自语，”在经历了这样的跌倒之后，我不会再跌倒！他们都会在家里以为我多么勇敢！为什么，即使我掉下了房顶，我也不会说什么！'（很可能是真的。）

下来，下来，下来。秋季永远不会结束！``我想知道这次我跌了多少英里？她大声说。``我一定要到地球中心附近的某个地方。让我看看：这将是大约四千英里了，我think--”（为你瞧，爱丽丝已经了解到这种几件事情在她的教室课，虽然这不是炫耀的好机会她的知识，因为没有人听她的话，还是一个好习惯，再说一遍。”'-是的，那是正确的距离-但是我想知道我要去的经度或纬度是什么？'（爱丽丝不知道纬度是什么，或者经度是什么，但是认为它们是好话。）

现在她又开始了。“我不知道如果我倒下的权利，通过地球！头朝下走路的人似乎会觉得这很有趣！我认为，“反感”-（她很高兴， 这次没有人听，因为它听起来完全不正确）–-但我不得不问他们这个国家的名字是什么是，你知道。夫人，请问，这是新西兰还是澳大利亚？``（她说话时试图屈膝-当你从空中摔下来时，幻想屈膝 ！您认为您可以管理吗？）“还有一个无知的小女孩，她会想我问我！不，永远不会问：也许我会看到它写在某个地方。'

下来，下来，下来。没有其他事情可做，所以爱丽丝很快又开始讲话。迪纳今晚会非常想念我，我想！'（黛娜是猫。）``我希望他们能在茶歇时记得她的牛奶碟。亲爱的黛娜！我希望你和我在一起！恐怕没有老鼠在空中，但是您可能会捉到一只蝙蝠，这就像老鼠一样。但是，我想知道猫吃蝙蝠吗？“在这里，爱丽丝开始变得困倦，然后以一种梦幻般的方式对自己说：“猫吃蝙蝠吗？猫吃蝙蝠吗？有时，“蝙蝠吃猫吗？因为，因为她不能回答任何一个问题，所以用什么方式表达都没关系。她感到自己正在打do睡，刚刚开始梦见自己与黛娜（Dinah）手拉手散步，对她非常认真地说道：'现在，黛娜，告诉我真相：您曾经吃过蝙蝠吗？'突然之间，重击！扑通！下来，她碰到了一堆树枝和干树叶，秋天就结束了。

爱丽丝没有受到什么伤害，她立刻跳了起来。她抬起头，但头顶全是黑漆漆的。在她又走了一段很长的路要走之前，“白兔”仍在视线中，匆匆驶下。没什么可迷失的：爱丽丝像风一样走了，正好及时听到它说，转弯时说：“哦，我的耳朵和胡须，有多晚了！'当她转过弯时，她紧紧靠近它的后面，但是兔子再也看不见了：她发现自己在一个长长的矮小的大厅里，屋顶上悬挂着一排灯照亮了大厅。

大厅四周有门，但都锁着。当爱丽丝从一侧到另一侧一直走到尽头时，她试着走到每扇门的中间，可悲地走下中间，想知道自己将如何再次走出去。

突然，她来到了一张三足的桌子，桌子都是用实心玻璃制成的。除了一个很小的金钥匙，上面什么都没有。爱丽丝首先想到的是它可能属于大厅的一扇门。可惜！锁太大或钥匙太小，但无论如何都不会打开它们。但是，在第二回合中，她碰到了一条以前未曾注意到的低帘子，在它的后面是一扇约15英寸高的小门：她尝试了锁中的小金钥匙，令她高兴的是它装好了！

爱丽丝打开门，发现它通向一条很小的通道，比老鼠洞还大：她跪下，沿着通道望向您所见过的最美丽的花园。她多么渴望离开那个黑暗的大厅，在鲜艳的花朵和凉爽的喷泉的床间四处游荡，但她甚至无法从门口抬起头来。 “即使我的头都过不去，”可怜的爱丽丝想，“那将是非常小的使用没有我的肩膀上。哦，我希望我能像望远镜一样闭嘴！如果我只知道如何开始，我想可以。“因为，您看到最近发生了太多事情，以至于爱丽丝开始认为很少有事情确实是不可能的。

等候在小门旁似乎没有任何用处，所以她回到桌子旁，希望自己能在桌子上找到另一个钥匙，或者至少希望有一本规则可以使人们像望远镜一样闭嘴：这一次，她发现上面放了一个小瓶子（“以前肯定不在这里，”爱丽丝说），在瓶子的脖子上是一个纸质标签，上面写着“ DRINK ME”字样的大写字母精美。

这一切都非常好地说“喝我”，但聪明的小爱丽丝不会做，在赶时间。她说：“不，我先看，看看它是否被标记为” 毒药 ”。因为她曾经读过一些关于谁怎样被烧伤，被野兽和其他不愉快的事情吃掉的孩子，都是因为他们不会记得不错的小故事，简单的规则自己的朋友曾告诉他们：比如，一个火钳如果持有时间过长，将会烫伤您；而且如果您用刀将手指切得很深，通常会流血；她从来没有忘记过，如果您从标有“ 有毒 ”的瓶子中大量喝酒，几乎可以肯定的是，迟早会与您不同意。

但是，这瓶酒没有标记为“毒药”，因此爱丽丝冒险品尝它，发现它非常好吃（事实上，它具有樱桃味，奶油蛋cherry，菠萝，烤火鸡的混合风味，太妃糖和黄油烤面包），她很快就完成了。

'多么奇怪的感觉！爱丽丝说； ”我必须像望远镜一样闭嘴。'

的确如此：她现在只有十英寸高，当她想到自己正好适合通过小门进入那个可爱的花园时，她的脸就变得明亮了。但是，首先，她等待了几分钟，看她是否会进一步收缩：对此她感到有点紧张；爱丽丝对自己说：“因为它可能会结束，在我出去的时候，就像一支蜡烛。我不知道那会是什么样子？'她试着幻想蜡烛吹完后的火焰是什么样的，因为她不记得曾经见过这样的事情。

过了一会儿，发现什么都没发生，她决定立即进花园。但是，可怜的爱丽丝a！当她到达门口时，发现自己忘记了那把小金钥匙，当她回到桌旁找时，她发现自己不可能伸手去拿：她可以很清楚地透过玻璃杯看到它，于是她试着她尽力爬上桌子的一只脚，但那太滑了。当她厌倦了自己的尝试时，可怜的小东西坐下来哭了。

“来吧，那样哭是没有用的！爱丽丝敏锐地对自己说。 ”我建议您离开这一分钟！她通常给自己很好的建议（尽管她很少遵循），有时她会严厉地责骂自己，以至于眼泪涌入。而且，一旦她想起自己用自己的耳朵在门球比赛中作弊就试图用自己的耳朵给自己打耳洞，因为这个好奇的孩子非常喜欢假装两个人。可怜的爱丽丝想，“但现在假装是两个人了！为什么，我几乎没有足够的人去做一个受人尊敬的人！'

很快，她的眼睛落在桌子底下的一个小玻璃盒上：她打开它，在里面发现了一个很小的蛋糕，在蛋糕上用葡萄干漂亮地标了“ EAT ME”字样。爱丽丝说：“好吧，我会吃掉它，如果它使我变得更大，我就能找到关键了。如果它使我变得越来越小，我可以在门下爬行；因此，无论哪种方式，我都会进入花园，而且我不在乎会发生什么！'

她吃了一点，焦急地对自己说：“哪种方式？哪一条路？”，将手放在头顶上，以感觉它的生长方向，她惊讶地发现自己的大小保持不变：可以肯定的是，这通常发生在一个人吃蛋糕的时候，但爱丽丝却是如此除了过分发生事情之外，别无他法，生活以一种普通的方式进行下去似乎很愚蠢和愚蠢。

于是她开始工作，很快就完成了蛋糕。
