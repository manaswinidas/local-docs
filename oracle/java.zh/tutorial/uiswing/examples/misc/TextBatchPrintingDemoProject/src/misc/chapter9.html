第九章

模拟龟的故事

“亲爱的老东西，你不能认为我很高兴再次见到你！公爵夫人说，当她亲切地将手臂塞进爱丽丝的怀里时，他们一起走了出去。

爱丽丝很高兴找到她这种令人愉快的脾气，并自以为是，当他们在厨房相遇时，也许正是胡椒使她如此野蛮。

“当我是一个公爵夫人，”她对自己说，（不是在一个非常有希望的语气虽然），“我不会在我的所有厨房的任何辣椒。如果没有，汤做得很好-也许总是胡椒使人脾气暴躁，”她继续说道，很高兴发现了一种新的法则，“醋使它们变酸-甘菊使它们变酸”苦涩和大麦糖之类的东西使孩子脾气暴躁。我只希望人们知道：然后，他们不会那么小气，你知道-”

这一次她已经完全忘记了公爵夫人，当她听到自己的声音靠近耳朵时，有些吃惊。亲爱的，你在想什么，这让你忘了说话。我现在不能告诉你那是什么寓意，但我会记住一点。'

“也许还没有，”爱丽丝冒险地说。

'T，tu，孩子！”公爵夫人说。“只有你能找到一切，一切都有道义。”她说话时紧紧地向爱丽丝的身边靠近。

爱丽丝并不太喜欢靠近她：首先，因为公爵夫人非常丑陋；其次，因为她恰好是将下巴靠在爱丽丝肩膀上的合适身高，所以下巴很不舒服。但是，她不喜欢无礼，因此尽其所能。

她说，通过保持一段对话，游戏现在进展得更好。

“是的，”公爵夫人说：“而那的寓意是-”哦，'是爱，'是爱，这使世界运转！”

“有人说，”爱丽丝小声说，“这是每个人关注自己的生意来完成的！'

'呃，好吧！这意味着同样的事情，“公爵夫人说，使劲儿把尖下巴到爱丽丝的肩膀，她补充说，”和道德的那就是- ‘带感关怀，声音会照顾自己。’ '

``她对在事物中发现道德有多么高兴！爱丽丝心想。

“我敢说你想知道为什么我不把手臂放在腰间，”公爵夫人停顿了一会后说：“原因是，我对你的火烈鸟的脾气很怀疑。我可以尝试实验吗？'

“ 他可能会咬人，”爱丽丝谨慎地回答，根本不急于进行实验。

“非常正确，”公爵夫人说：“火烈鸟和芥末都咬人。这样做的寓意是-“羽毛鸟聚在一起”。

爱丽丝评论说：“只有芥末不是鸟。”

公爵夫人说：“像往常一样，对了。你摆东西的明确方式！'

“我想这是一种矿物，”爱丽丝说。

“当然是，”公爵夫人说，他似乎准备同意爱丽丝所说的一切。 '附近有一个大芥末矿。而这的寓意是-“我的越多，你的越少。”

'哦，我知道！爱丽丝喊道，他没有参加最后一句话，“这是一种蔬菜。它看起来不像一个，但确实如此。'

“我完全同意你的看法，”公爵夫人说。 “而这的寓意是-“做你想做的事”-或者，如果你想更简单地说-“永远不要想像自己不是别人，而是别人可能会看到的东西曾经或可能曾经不是，除非您曾经经历过，否则在他们看来却并非如此。”

爱丽丝很礼貌地说道：“我想我应该更好地理解这一点，如果我把它写下来的话，但我不能完全按照你说的去做。'

“如果我选择了，那我什么也没说。”公爵夫人高兴地说道。

爱丽丝说：“求求你不要再这么说了。”

'哦，别说麻烦了！”公爵夫人说。“我把你刚才所说的一切都当做礼物。'

'一种便宜的礼物！爱丽丝想。``我很高兴他们不给那样的生日礼物！“但她不敢大声说出来。

再次思考？”公爵夫人问道，又掏出了她那尖锐的小下巴。

“我有权思考，”爱丽丝敏锐地说，因为她开始有点担心。

公爵夫人说：“差不多就对了，就像猪必须飞一样。还有他们 - '

但是在这里，令爱丽丝大吃一惊的是，即使在她最喜欢的单词“道德”的中间，公爵夫人的声音也消失了，与她联系在一起的手臂开始颤抖。爱丽丝抬起头，女王站在他们面前，双臂交叉着，像雷暴一样皱着眉头。

A下，美好的一天！公爵夫人低沉而微弱的声音开始了。

“现在，我给你一个合理的警告。”女王在讲话时踩在地上喊道。 ”“您或您的头都必须掉头，而这几乎没有时间！随您选择！'

公爵夫人选择了她，不久就走了。

女王对爱丽丝说：“让我们继续比赛。”爱丽丝吓得不敢说一句话，但慢慢地跟着她回到了槌球场。

其他客人利用了女王的缺席，并在阴凉处休息：但是，当他们看到她的那一刻，他们急忙回到比赛中，女王只是在说一会儿的延误将使他们丧命。

在他们玩皇后的所有时间里，女王从未与其他玩家争吵过，大喊着“开枪！”或“抬起头！'她所判刑的那些人被士兵拘留，他们当然不得不离开拱门来做到这一点，以便在半小时左右结束时，没有拱门了，除了金，女王和爱丽丝被拘留并处以死刑。

皇后气喘吁吁地离开了，对爱丽丝说：“你看到过模拟龟了吗？'

“不，”爱丽丝说。``我什至不知道模拟乌龟是什么。'

女王说：“这就是制作乌龟汤的东西。”

爱丽丝说：“我从未见过或听说过一个。”

女王说：“那就来吧，他会告诉你他的历史。”

当他们一起走时，爱丽丝听见国王对公司低声说：“你们都被赦免了。''来吧， 那是一件好事！她对自己说，因为她对女王下令处决的次数感到非常不满。

他们很快遇到了狮ry，在阳光下睡着了。（ 如果您不知道狮ry是什么，请看图片。）'起来，懒惰的东西！女王说：“带着这位年轻女士去看龟龟，听听他的历史。我必须回去看看我下达了一些死刑命令。她走开了，只剩下爱丽丝和狮ry。爱丽丝不太喜欢这种生物的外观，但是总的来说，她认为和它呆在一起就像跟随那个野蛮的女王一样安全：所以她等待了。

狮ry坐了起来，揉了揉眼睛：然后看着女王，直到她看不见：然后咯咯笑了。'真有趣！狮ry说，一半来自爱丽丝。

“有什么好玩的？爱丽丝说。

“为什么， 她 ，”狮ry说。``这完全是她的幻想：你永远不会处决任何人。来吧！'

所有人都说：“加油！“在这里，”爱丽丝缓慢地走了过去，想着：“我一生中从未受到如此命令，从来没有！'

他们走了很远才看到远处的模拟龟，悲伤而孤独地坐在一块小小的岩石壁架上，当他们走近时，爱丽丝听见他在叹息，仿佛他的心碎了。她深深地可怜他。'他的悲伤是什么？她问格里芬，格里芬几乎和以前一样用同样的话回答，“这完全是他的幻想：你没有悲伤。来吧！'

于是他们走上了嘲笑的海龟，海龟满眼含泪地看着他们，但什么也没说。

狮ry说：“这位年轻女士，她想知道你的历史。'

“我告诉她，”嘲讽的乌龟用一种深沉而空洞的语气说：“你们两个坐下，直到我说完一句话都不要说。'

于是他们坐下，几分钟没有人说话。爱丽丝想，“我不知道他是如何完成甚至 ，如果他不开始。但是她耐心地等待着。

“一次，”嘲笑的海龟最后说道，“我是真正的海龟。'

这些话之后是很长的沉默，只有偶尔的“ Hjckrrh！来自狮ry，还有不断沉重的模拟海龟抽泣声。爱丽丝几乎要站起来说：“先生，谢谢您的有趣的故事。”但是她不由自主地想必还会有更多的东西来，所以她坐着不说话。

“小时候，”模拟龟终于平静下来了，尽管时不时抽泣着，“我们还是在海里上学了。主人是一只老乌龟-我们曾经叫他乌龟-

“如果他不是一个人，你为什么叫他乌龟？爱丽丝问。

“我们叫他乌龟是因为他教了我们，”小乌龟生气地说：“真的，你很呆板！'

狮phon补充说：“你应该为提出这样一个简单的问题而感到羞耻。”然后他们俩都静静地坐着，看着可怜的爱丽丝，后者已经准备好沉入地下。最后，狮ry对模拟龟说：“继续前进，老家伙！不要整日忙于此！”他接着说：

“是的，我们在海里上学了，尽管你可能不相信-”

``我从未说过我没有！”爱丽丝打断了。

“你做到了。”小海龟说。

'握住你的舌头！”狮ry补充道，爱丽丝才可以再次讲话。模拟龟继续前进。

``我们受过最好的教育-事实上，我们每天都上学-''

爱丽丝说：“ 我也去过一所日校。” '您不必为此感到骄傲。'

“还有其他？”模拟龟有些不安地问。

“是的，”爱丽丝说，“我们学了法语和音乐。'

“洗吗？”模拟龟说。

'当然不是！”爱丽丝愤慨地说。

'啊！那时你的学校就不是一个好学校，”嘲笑的龟龟松了一口气。“现在，在我们的账单末尾，他们有“额外的法国，音乐和洗衣服 ”。

爱丽丝说：“你本来不是很想要的。” '生活在海底。'

“我付不起学。”模拟龟叹了口气。``我只参加常规课程。'

'那是什么？”爱丽丝问。

“当然，起步和挣扎，”模拟龟回答。然后是算术的不同分支-野心，分心，丑化和嘲弄。'

“我从未听说过“丑化”，”爱丽丝大胆地说。'它是什么？'

狮ry惊讶地抬起两只爪子。'什么！从未听说过丑陋！”它喊道。“我想你知道要美化什么吗？'

“是的，”爱丽丝怀疑地说道：“这意味着-做-任何事-更漂亮。'

“那么，”狮ry继续说，“如果你不知道要丑化什么，那你就是一个简单的人。'

爱丽丝并没有受到鼓励去问更多的问题，于是她转向模拟龟，说：“你还学什么？'

“好了，有个谜团，”嘲笑的龟在回答他拍打的东西时说：“-古老而又现代的谜团，与海洋学有关：然后画图的人-画图大师是一个古老的鳗鱼，每周一次： 他教了我们在线圈中绘制，拉伸和晕倒。'

“ 那是什么样的？爱丽丝说。

“好吧，我不能亲自给你看。”模拟龟说：“我太僵硬了。而狮ry从来没有学过。'

狮time说：“没时间。”但是我去了古典大师。他是个老螃蟹， 他是。'

“我从来没有去过他，”模拟龟叹了口气：“他们曾经说过，他教过笑和悲伤。'

狮ry叹了口气说：“所以他做到了，所以他做到了。”两个生物都将脸庞藏在爪子中。

“您每天上多少课？爱丽丝急着改变话题说。

模拟乌龟说：“第一天十个小时，第二天九点，依此类推。'

'一个很好奇的计划！爱丽丝大叫。

格里芬评论说：“这就是他们被称为课程的原因，因为它们每天都在减少。'

对于爱丽丝来说，这是一个相当新的主意，她在下一个评论之前考虑了一下。“那么第十一天一定是放假吗？'

“当然是，”模拟龟说。

“您在第十二届如何管理？爱丽丝热切地继续。

狮ry以非常坚定的语气打断了他的话：“上课就足够了。”'
