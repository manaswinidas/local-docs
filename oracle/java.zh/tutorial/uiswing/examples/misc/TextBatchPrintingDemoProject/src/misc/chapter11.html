首先，她梦见自己是小爱丽丝，然后再次用小手紧紧抓住膝盖，明亮的急切眼睛注视着她-她可以听见声音的语调，看到那奇怪的折腾。她的头保持不停飘动的头发， 这种头发总是会进入她的眼睛-直到她听着或似乎在听时，她周围的整个地方都变得活着，充满了她妹妹梦strange以求的怪异生物。

白兔匆匆驶过时，长长的草丛在脚下沙沙作响-惊恐的老鼠从附近的水池中溅出-当三月野兔和他的朋友们分享他们永不休止的饭菜时，她可以听到茶杯的嘎嘎声，女王发出刺耳的声音命令她不幸的客人被处决-越来越多的猪宝宝在公爵夫人的膝盖上打喷嚏，而盘子和盘子掉在了地上-再次响起了狮，的尖叫声，蜥蜴的尖叫声板岩铅笔和被抑制的豚鼠的窒息声充斥着空气，与悲惨的模拟龟的远处的抽泣声混杂在一起。

所以她闭着眼睛坐在那里，一半的人相信自己在仙境中，尽管她知道自己只能再次打开它们，所有的一切都会变成沉闷的现实-草只会在风中沙沙作响，而游泳池则荡漾着挥舞着芦苇-嘎嘎作响的茶杯将变成叮叮当当的铃铛，女王的刺耳的叫喊声响彻牧羊犬的声音-婴儿的打喷嚏，狮ry的尖叫声，以及所有其他奇怪的声音将改变（她知道）忙碌的农场里混乱的喧闹声，而远处放低牲口将代替嘲笑的乌龟。

最后，她向自己描绘了自己的同一个妹妹如何在以后成为自己的成年女性。以及她将如何在整个成熟的岁月中保持童年时代纯朴而充满爱心的心态；以及如何与其他小孩子们聚在一起，使他们的眼睛明亮并渴望看到许多奇怪的故事，甚至是梦想她很早以前就对《仙境》感兴趣：她会以他们所有的简单悲伤感到如何，并在他们所有的简单快乐中找到快乐，回想起自己的童年和快乐的夏日。
