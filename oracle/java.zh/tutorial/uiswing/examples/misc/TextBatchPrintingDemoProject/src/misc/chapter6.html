第六章

猪和胡椒

一两分钟，她站着看房子，想知道下一步该怎么做，当时一个穿着制服的男仆突然从树林中跑出来-（她认为他是一个穿着制服的男仆，因为他在穿着制服;否则，判断她只凭他的脸就叫他一条鱼）-然后用指关节大声敲门。它是由另一个穿着制服的侍者打开的，他的脸圆圆的，大眼睛像青蛙。爱丽丝（Alice）注意到，两个步兵的头发都是粉末状，卷曲在他们的头顶。她感到非常好奇，想知道这是怎么回事，于是走了一点路就走了。

Fish-Footman首先从他的胳膊下制作了一封几乎和他一样大的好信，然后他将这封信移交给了另一位，并以庄重的口吻说：“给公爵夫人。女王发出打槌球的邀请。'青蛙-脚夫以同样庄重的语气重复了一下，只是稍微改变了单词的顺序，'来自女王。邀请公爵夫人打槌球。'

然后他们俩低下了头，卷发纠缠在一起。

爱丽丝对此大笑，以至于因为害怕他们听到她而不得不跑回树林。当她下次偷看时，鱼夫走了，另一个坐在门旁的地上，愚蠢地凝视着天空。

爱丽丝胆怯地走到门前，敲了敲门。

步行者说，“敲门没有任何用处，这有两个原因。首先，因为我和你在门的同一侧；其次，因为他们在内部发出如此大的声音，所以没人会听到你的声音。”“当然，里面有一种最特别的声音-持续不断的and叫和打喷嚏，时不时地发生巨大的碰撞，就好像碟子或水壶被打碎了一样。

“那么请，”爱丽丝说，“我要怎么进去？'

“我们敲门可能有些道理，”如果我们之间有门，那步兵便不理她地继续说道。例如，如果您在里面 ，您可能会敲门，而我可以让您出去，您知道。'他一直在讲话，一直仰望天空，而这个爱丽丝绝对不文明。“但是也许他无能为力，”她对自己说。他的眼睛非常接近他的头顶。但无论如何，他可能会回答问题。——我要怎么进入？她大声重复道。

夫特曼说：“我坐在这里，直到明天……”

这时，房子的门打开了，一大块盘子正对着步兵的脑袋掠过一眼：它只是擦过他的鼻子，在他身后的一棵树上摔成碎片。

“-或第二天，也许吧，”步兵以相同的语气继续说着，仿佛什么也没发生。

“我怎么进去？”爱丽丝再次大声问。

' 你要进去吗？步兵说。“这是第一个问题，你知道。'

毫无疑问，只有爱丽丝不喜欢这样被告知。“这真是可怕的，”所有生物争论的方式，她对自己喃喃自语。足以使一个人疯狂！'

步兵似乎认为这是一个很好的机会来重复他的言论，但要有所不同。他说：“我将坐在这里，断断续续，日复一日。'

“但是我该怎么办？爱丽丝说。

“随便什么，”步兵说，然后开始吹口哨。

“哦，和他说话毫无用处，”爱丽丝拼命地说：“他真是愚蠢！她打开门走进去。

门直接通向一个大厨房，厨房的一端到另一端都充满了烟雾：公爵夫人坐在中间的三足凳子上，照顾着一个婴儿。厨师靠在炉火上，搅拌着一个大锅，锅里似乎装满了汤。

“那汤里肯定有太多胡椒粉！爱丽丝自言自语，也可以打喷嚏。

肯定有太多的空气。甚至公爵夫人也偶尔打喷嚏。至于婴儿，它不停地停顿着打喷嚏和how叫。厨房里唯一没有打喷嚏的东西是厨师，还有一只大猫坐在炉床上，从耳朵到耳朵都笑着。

“请你告诉我，”爱丽丝有些胆怯地说道，因为她不确定她首先说话是否有礼貌，“为什么你的猫会这样咧嘴？'

公爵夫人说：“这是柴郡的猫，这就是原因。猪！'

她说出最后一句话时，突然感到暴力，以至于爱丽丝跳了起来。但是她又看了一遍，那是给婴儿的，而不是给她的，所以她鼓起勇气，继续说下去：

“我不知道柴郡的猫总是笑嘻嘻。实际上，我不知道猫会笑。'

“他们都可以，”公爵夫人说。 ”，而大部分时间都是。'

“我不知道有什么事可做，”爱丽丝非常有礼貌地说道，感到很高兴参加对话。

“你不太了解，”公爵夫人说。 ”这是事实。'

爱丽丝根本不喜欢这句话的基调，并认为引入其他话题也是可以的。当她试图固定时，厨师从火上取下了大锅汤，立即开始工作，将所有东西都扔到公爵夫人和婴儿身上。然后洗锅，盘子和盘子。公爵夫人即使打中她也没有注意到他们。婴儿已经哭了很多，以至于很难说打击是否伤害到婴儿。

'哦， 请介意您在做什么！爱丽丝哭泣，在恐怖的痛苦中跳来跳去。“哦，他的鼻子很珍贵 ”；当一个不寻常的大锅靠近它时，几乎把它拿走了。

公爵夫人嘶哑地咆哮着说：“如果每个人都想着自己的事，世界将比现在更快地达成协议。'

爱丽丝说：“这不是优势，”爱丽丝很高兴有机会展示自己的一些知识。只是想想白天和黑夜将完成什么工作！您会看到地球绕轴旋转需要二十四个小时-''

“说斧头，”公爵夫人说，“砍下她的头！'

爱丽丝相当焦虑地看了看厨师，看她是否打算接受暗示。但是厨师正在忙着搅拌汤，似乎没有在听，于是她又继续说：“我想二十四个小时。还是十二点？一世 - '

“哦，别打扰我 ，”公爵夫人说。 ``我永远无法遵守数字！然后她又开始给孩子喂奶，在给孩子唱催眠曲的同时，在每行结尾处剧烈晃动：

'和你的小男孩粗话，

打喷嚏打他：

他只会惹恼他

因为他知道这很有趣。'

合唱

（厨师和婴儿参加的）：-

'哇！哇！哇！'

当公爵夫人演唱这首歌的第二节时，她一直在剧烈地上下颠簸着扔婴儿，可怜的小东西大叫起来，以致爱丽丝几乎听不到这句话：

``我对我的男孩严厉讲话，

他打喷嚏时我打了他。

因为他可以彻底享受

他喜欢的时候胡椒粉！'

合唱

'哇！哇！哇！'

'这里！如果愿意，可以稍作护理！公爵夫人对爱丽丝说着，一边说着婴儿一边向她扑来。“我必须去准备和女王打槌球了，”她赶紧走出房间。厨师在她外出时向她扔了一个平底锅，但只是想念她。

爱丽丝难以置信地抓住了婴儿，因为它是一个奇怪的形状的小动物，并向各个方向伸出胳膊和腿，“就像海星一样，”爱丽丝想。那个可怜的小东西抓住她时像个蒸汽机一样着，不断加倍并再次伸直，所以一，两分钟之内，她就竭尽全力了。

一旦她确定了正确的护理方法（即将其扭曲成一个结，然后紧紧握住它的右耳和左脚，以防止其解体），把它带到露天。爱丽丝想， 如果我不把这个孩子带走，他们一定会在一两天之内杀死它：把它抛在后面不是谋杀吗？她大声说出了最后一句话，小东西咕gr了一声（到了这时候它不再打喷嚏了。“别咕gr，”爱丽丝说。 '那根本不是表达自己的正确方法。'

婴儿再次咕gr一声，爱丽丝焦急地望着它的脸，看到底是怎么回事。毫无疑问，它的鼻子非常弯曲，更像是鼻子，而不是真正的鼻子。婴儿的眼睛也变得很小：爱丽丝完全不喜欢这东西的外观。“但是也许只是在哭泣，”她想着，再次看着它的眼睛，看是否有眼泪。

不，没有眼泪。“亲爱的，如果你要变成猪，”爱丽丝认真地说，“与你无关。现在注意！可怜的小东西又抽泣了（或咕，一声，无法说出哪一个），他们沉默了片刻。

爱丽丝刚刚开始对自己说：“现在，当我把它带回家时，我该怎么办？当它再次发出强烈的咕unt声时，她有些惊恐地低头看着它的脸。这次没有任何错误：它既不比猪多，也不比猪小，她觉得如果将它进一步携带，那将是非常荒谬的。

因此，她放下了这个小生物，看到它悄悄溜进树林里时，感到很欣慰。她对自己说：“如果它长大了，它本来可以养成一个可怕的丑陋的孩子：但是我认为它可以养成一头漂亮的猪。“然后她开始考虑自己认识的其他孩子，这些孩子可能会像猪一样做得很好，只是对自己说：“如果只知道正确的方式来改变它们-”当她看到柴郡时有点惊讶猫坐在几码远的树的树枝上。

猫只在看到爱丽丝时笑了。她想，它看起来很自然，但它的爪子仍然很长，牙齿很多，因此她认为应该对此予以尊重。

“ Cheshire Puss，”她怯怯地开始，因为她根本不知道它是否想要这个名字：但是，它笑得更宽了一点。“来吧，到目前为止很高兴，”爱丽丝想，然后继续。``请您告诉我，我应该从这走哪条路？'

猫说：“这取决于您想去的地方。”

“我不在乎-”爱丽丝说。

“那走哪条路都没关系，”猫说。

“-只要我到某处 ，”爱丽丝补充说。

“哦，您一定会这样做的，”猫咪说，“只要走了足够长的时间。'

爱丽丝认为这是不能否认的，所以她尝试了另一个问题。'什么样的人住在这里？'

猫说着，朝那个方向，挥舞着右爪子，住着一个帽匠；朝那个方向，挥舞着另一只爪子，住着了三月野兔。随便问一个，他们都疯了。'

爱丽丝评论说：“但是我不想和疯子呆在一起。”

“哦，你无能为力，”猫说：“我们都在这里生气。我生气了。你疯了。'

'你怎么知道我生气了？爱丽丝说。

“你一定是，”猫说，“否则你就不会来这里。'

爱丽丝根本没有证明这一点。但是，她继续说：“你怎么知道你生气了？'

猫说：“开始，狗不是疯子。你同意吗？'

“我想是的，”爱丽丝说。

“那么，”猫继续说道，“你看，狗生气时会咆哮，高兴时会摇尾巴。现在，我高兴时咆哮，生气时摇尾巴。因此我很生气。'

爱丽丝说：“我叫它发出呼pur声，而不是咆哮。”

“随便叫什么，”猫说。 “你今天和女王一起打槌球吗？'

爱丽丝说：“我非常喜欢它，但是我还没有被邀请。'

“你会在那儿见到我的。”猫说，消失了。

爱丽丝对此并不感到惊讶，她已经习惯了这种奇怪的事情。当她看着它曾经去过的地方时，突然又出现了。

“再见了，婴儿怎么了？猫说。 ``我几乎忘了问。'

“它变成了猪，”爱丽丝轻声说，就像它以自然的方式回来一样。

“我以为会，”猫说，然后消失了。

爱丽丝稍等了一会儿，一半希望再次看到它，但它没有出现。一两分钟后，她朝据说三月野兔生活的方向走去。她对自己说：“我以前见过仇敌。” ``三月野兔将是最有趣的，也许因为这是五月，所以它不会疯狂-至少不会像三月那样疯狂。就像她说的那样，她抬起头，又有只猫坐在树的树枝上。

“你是说猪还是无花果？猫说。

“我说猪，”爱丽丝回答。 ”而且我希望您不要突然出现并消失：您使自己很头晕。'

“好吧，”猫说。这次它消失得很慢，从尾巴的末端开始，到咧嘴结束，直到剩下的部分消失了一段时间。

'好！爱丽丝想，我经常见到没有笑容的猫。 '但是没有猫的笑容！这是我一生中看到的最奇怪的事情！'

在看到三月野兔的房子之前，她并没有走太远：她认为那一定是正确的房子，因为烟囱的形状像耳朵，屋顶是用毛皮盖起来的。这么大的房子，她不喜欢走近一点，直到她又吃了一点左手蘑菇，并将自己抬高到大约两英尺高时：即使如此，她还是胆怯地朝它走去，对自己说``毕竟应该疯了！我几乎希望我能去见帽匠！'
