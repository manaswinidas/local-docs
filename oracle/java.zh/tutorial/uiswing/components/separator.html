如何使用分隔符

的JSeparator类提供水平或垂直分隔线或空白区域。它最常用于菜单和工具栏。实际上，您甚至可以在不知道分隔符的情况下使用分隔符JSeparator之所以存在类，是因为菜单和工具栏提供了方便的方法，这些方法可以创建和添加为其容器定制的分隔符。分隔符与边框有些相似，不同之处在于分隔符是真正的组件，因此它们被绘制在容器内，而不是围绕特定组件的边缘。

这是具有三个分隔符的菜单图片，用于将菜单分为四组：

将菜单项和分隔符添加到菜单的代码非常简单，具体如下：

menu.add(menuItem1);
menu.add(menuItem2);
menu.add(menuItem3);
menu.addSeparator();
menu.add(rbMenuItem1);
menu.add(rbMenuItem2);
menu.addSeparator();
menu.add(cbMenuItem1);
menu.add(cbMenuItem2);
menu.addSeparator();
menu.add(submenu);

向工具栏添加分隔符的过程与此类似。您可以在菜单和工具栏的操作方法部分中找到解释的完整代码。如果您想进一步控制菜单和工具栏中的分隔符，则可以直接使用JSeparator实现它们的子类： JPopupMenu。分隔符和JToolBar。分隔符 。特别是， JToolBar.Separator具有用于指定分隔符大小的API。

使用JSeparator

您可以使用JSeparator直接在任何容器中提供分隔线。下图显示了一个GUI，在带有Fire的按钮右侧有一个分隔符。

分隔符几乎没有API，并且只要记住一件事就非常容易使用：在大多数实现中，垂直分隔符的首选高度为0，而水平分隔符的首选宽度为0。这意味着分隔符是不可见的，除非您设置其首选大小或将其置于布局管理器的控制下，例如BorderLayout要么BoxLayout将其拉伸以填充其可用显示区域。

垂直分隔符确实有点宽（水平分隔符有点高），所以您应该看到分隔符所在的空间。但是，除非宽度和高度都不为零，否则不会绘制实际的分界线。

以下代码段显示ListDemo如何将包含垂直分隔符的面板组合在一起。您可以在以下位置找到ListDemo的完整源代码。 ListDemo.java 。

JPanel buttonPane = new JPanel();
buttonPane.setLayout(new BoxLayout(buttonPane,
                                   BoxLayout.LINE_AXIS));
buttonPane.add(fireButton);
buttonPane.add(Box.createHorizontalStrut(5));
buttonPane.add(new JSeparator(SwingConstants.VERTICAL));
buttonPane.add(Box.createHorizontalStrut(5));
buttonPane.add(employeeName);
buttonPane.add(hireButton);
buttonPane.setBorder(BorderFactory.createEmptyBorder(5,5,5,5));

如代码所示，按钮，分隔符和文本字段都共享一个容器- JPanel使用从左到右的盒子布局的实例。多亏了布局管理器（以及分隔符具有不受限制的最大尺寸的事实），分隔符会自动制作成与其可用显示区域一样高。

在前面的代码中，水平支柱是不可见的组件，用于在分隔符周围提供空间。一个5像素的空白边框在面板周围提供了缓冲，还可以防止分隔符一直延伸到其上方的组件以及其下方的窗口边缘。

这是另一个使用分隔符的GUI的图片，这次是在一组控件和显示区域之间放置一条分隔线。

您可以在示例索引中找到代码。这是设置分隔符容器的代码：

JPanel panel = new JPanel(new BorderLayout());
...
panel.setBorder(BorderFactory.createEmptyBorder(
                        GAP/2, //top
                        0,     //left
                        GAP/2, //bottom
                        0));   //right
panel.add(new JSeparator(JSeparator.VERTICAL),
          BorderLayout.LINE_START);
panel.add(addressDisplay,
          BorderLayout.CENTER);

与上一个示例一样，面板使用空边框，因此分隔符不会一直延伸到其容器的边缘。将分隔符放在打印机最左侧的区域BorderLayout受控的容器使分隔符与容器中心的地址显示组件一样高。有关边框布局如何工作的详细信息，请参见如何使用BorderLayout 。

分隔符API

使用分隔符的API最少，因为它们没有内容并且不会响应用户输入。

创建和初始化分隔符
构造函数或方法 目的
无效的addSeparator（）
void addSeparator（Dimension）
（在JToolBar ）		 将工具栏分隔符（在大多数（如果不是全部）外观上不可见）附加到工具栏的当前末端。可选参数指定分隔符的大小。此方法的无参数版本使用默认大小的分隔符，该分隔符由当前外观决定。
无效的addSeparator（）
void insertSeparator（int）
（在JMenu ）		 在菜单中放置分隔符。的addSeparator方法将分隔符放在菜单的当前末尾。的insertSeparator方法将分隔符插入菜单中的指定位置。
无效的addSeparator（）
（在JPopupMenu ）		 在弹出菜单的当前末尾放置一个分隔符。
JSeparator（）
JSeparator（int）		 创建一个分隔符。如果不指定参数，则分隔符为水平分隔符。参数可以是SwingConstants.HORIZONTAL要么SwingConstants.VERTICAL 。
setOrientation（int）
int getOrientation（）
（在JSeparator ）		 获取或设置分隔符的方向，可以是SwingConstants.HORIZONTAL要么SwingConstants.VERTICAL 。
JToolBar。分隔器（）
JToolBar。分离器（尺寸）		 创建用于工具栏的分隔符。可选参数指定分隔符的大小。
setSeparatorSize（Dimension）
（在JToolBar.Separator ）		 指定分隔符的尺寸。更具体地说，指定Dimension用作分隔符的最小，首选和最大尺寸。
JPopupMenu。分隔器（） 创建用于菜单的分隔符。
