如何使用按钮，复选框和单选按钮

要创建按钮，您可以实例化从AbstractButton类。下表显示了Swing定义的AbstractButton您可能要使用的子类：

JButton 通用按钮。 如何使用Common Button API和如何使用JButton功能

首先，本节介绍了基本的按钮API AbstractButton定义-因此所有Swing按钮都具有相同之处。接下来，它描述了少量的API JButton添加到AbstractButton 。之后，本节将向您展示如何使用专用的API来实现复选框和单选按钮。

  2. 单击左按钮。
    它禁用中间按钮（以及它本身，因为它不再有用了），并启用了右键。
  3. 单击右键。
    它启用中键和左键，并禁用自身。

作为ButtonDemo如示例所示，Swing按钮可以同时显示文本和图像。在ButtonDemo ，每个按钮相对于其图像在不同位置具有其文本。每个按钮文本中带下划线的字母显示每个按钮的助记符 （键盘替代）。在大多数外观上，用户可以通过按Alt键和助记符来单击按钮。例如，Alt-M将单击ButtonDemo中的“中间”按钮。

禁用按钮后，外观会自动生成按钮的禁用外观。但是，您可以提供一个图像来代替普通图像。例如，您可以提供左右按钮中使用的图像的灰色版本。

如何实现事件处理取决于您使用的按钮类型以及使用方式。通常，您实现一个动作侦听器 ，每次用户单击该按钮时都会收到通知。对于复选框 ，通常使用项目侦听器 ，该侦听器在选中或取消选中复选框时会得到通知。

以下是来自的代码ButtonDemo.java在上一个示例中创建按钮并对按钮单击做出反应。粗体代码是如果按钮没有图像则将保留的代码。

//In initialization code:
    ImageIcon leftButtonIcon = createImageIcon("images/right.gif");
    ImageIcon middleButtonIcon = createImageIcon("images/middle.gif");
    ImageIcon rightButtonIcon = createImageIcon("images/left.gif");

    b1 = new JButton("Disable middle button", leftButtonIcon);
    b1.setVerticalTextPosition(AbstractButton.CENTER);
    b1.setHorizontalTextPosition(AbstractButton.LEADING); //aka LEFT, for left-to-right locales
    b1.setMnemonic(KeyEvent.VK_D);
    b1.setActionCommand("disable");

    b2 = new JButton("Middle button", middleButtonIcon);
    b2.setVerticalTextPosition(AbstractButton.BOTTOM);
    b2.setHorizontalTextPosition(AbstractButton.CENTER);
    b2.setMnemonic(KeyEvent.VK_M);

    b3 = new JButton("Enable middle button", rightButtonIcon);
    //Use the default text position of CENTER, TRAILING (RIGHT).
    b3.setMnemonic(KeyEvent.VK_E);
    b3.setActionCommand("enable");
    b3.setEnabled(false);

    //Listen for actions on buttons 1 and 3.
    b1.addActionListener(this);
    b3.addActionListener(this);

    b1.setToolTipText("Click this button to disable "
                      + "the middle button.");
    b2.setToolTipText("This middle button does nothing "
                      + "when you click it.");
    b3.setToolTipText("Click this button to enable the "
                      + "middle button.");
    ...
}

public void actionPerformed(ActionEvent e) {
    if ("disable".equals(e.getActionCommand())) {
        b2.setEnabled(false);
        b1.setEnabled(false);
        b3.setEnabled(true);
    } else {
        b2.setEnabled(true);
        b1.setEnabled(true);
        b3.setEnabled(false);
    }
}

如何使用JButton功能

普通按钮— JButton对象-功能比AbstractButton课程提供：您可以制作一个JButton是默认按钮。

顶级容器中最多有一个按钮可以是默认按钮。默认按钮通常具有突出显示的外观，并且只要顶级容器具有键盘焦点并且用户按下Return或Enter键，就会单击该按钮。这是在ListDialog示例中实现的对话框的图片，其中“ 设置”按钮是默认按钮：

您可以通过调用setDefaultButton顶级容器的根窗格上的方法。这是设置默认按钮的代码ListDialog例：

//In the constructor for a JDialog subclass:
getRootPane().setDefaultButton(setButton);

默认按钮功能的确切实现取决于外观。例如，在Windows外观中，默认按钮将更改为具有焦点的按钮，因此按Enter键将单击焦点按钮。当没有按钮获得焦点时，您最初指定为默认按钮的按钮将再次变为默认按钮。

如何使用复选框

的JCheckBox该类提供对复选框按钮的支持。您也可以将复选框放在菜单中 ，使用JCheckBoxMenuItem类。因为JCheckBox和JCheckBoxMenuItem继承自AbstractButton ，Swing复选框具有所有通常的按钮特征，如本节前面所述。例如，您可以指定要在复选框中使用的图像。

复选框类似于单选按钮，但按照惯例，它们的选择模型不同。可以选择组中的任意数量的复选框-无，全部或全部。另一方面，一组单选按钮只能选择一个按钮。

如何使用单选按钮

单选按钮是一组按钮，按照惯例，一次只能选择一个按钮。Swing版本支持带有JRadioButton和ButtonGroup类。要将单选按钮放入菜单 ，请使用JRadioButtonMenuItem类。显示多个选择的其他方法是组合框和列表 。单选按钮的外观类似于复选框 ，但是按照惯例，复选框对一次可以选择多少项没有限制。

因为JRadioButton继承自AbstractButton ，如本节前面所述，Swing单选按钮具有所有通常的按钮特征。例如，您可以指定单选按钮中显示的图像。

尝试这个： 
  1. 单击启动按钮，以使用Java™Web Start （ 下载JDK 7或更高版本 ）运行RadioButton演示。另外，要自己编译并运行示例，请参考示例索引 。

  2. 单击“ 狗”按钮或按Alt-d。
    “ 狗”按钮变为选中状态，从而使“ 鸟”按钮变为未选中状态。图片从鸟变成狗。该应用程序具有一个动作侦听器，可侦听所有单选按钮。每次动作侦听器收到事件时，应用程序都会显示刚刚单击的单选按钮的图片。

每次用户单击单选按钮（即使已选择它），该按钮都会触发一个action事件 。也会发生一个或两个项目事件 -一个事件来自刚刚选择的按钮，另一个事件来自失去选择的按钮（如果有）。通常，您使用动作侦听器处理单选按钮的单击。

对于每组单选按钮，您需要创建一个ButtonGroup实例，并向其中添加每个单选按钮。的ButtonGroup当用户在组中选择另一个按钮时，它会取消选择先前选择的按钮。

通常，您应该初始化一组单选按钮，以便选择一个。但是，API没有强制执行此规则-一组单选按钮不能进行初始选择。一旦用户做出选择，则从此开始仅选择一个按钮。

JToggleButton（）		 创建一个JToggleButton实例，类似于JButton ，但有两种状态。通常，您使用JRadioButton要么JCheckBox而不是直接实例化JToggleButton ，但是JToggleButton当您不希望使用典型的单选按钮或复选框外观时，此功能很有用。字符串参数指定切换按钮应显示的文本（如果有）。同样， Icon参数指定应使用的图像。将布尔参数指定为true初始化要选择的切换按钮。如果不存在布尔参数，或者false ，然后最初未选择切换按钮。
