如何使用文字栏位

文本字段是一个基本的文本控件，使用户能够键入少量文本。当用户指示文本输入完成时（通常通过按Enter键），文本字段将触发一个动作事件 。如果您需要从用户那里获得多行输入，请使用文本区域 。

Swing API为组件提供了几个类，这些类可以是多种文本字段，也可以包括文本字段。

JTextField 本节内容：基本文本字段。
JFormattedTextField 一种JTextField子类，可让您指定用户可以输入的合法字符集。请参阅如何使用格式化的文本字段 。
JPasswordField 一种JTextField子类，不显示用户键入的字符。请参阅如何使用密码字段 。
JComboBox 可以进行编辑，并提供一个字符串菜单供您选择。请参阅如何使用组合框 。
JSpinner 将格式化的文本字段与几个小按钮结合在一起，使用户能够选择上一个或下一个可用值。请参阅如何使用微调框 。

下面的示例显示一个基本的文本字段和一个文本区域。文本字段是可编辑的。文本区域不可编辑。当用户在文本字段中按Enter键时，程序会将文本字段的内容复制到文本区域，然后选择该文本字段中的所有文本。

单击启动按钮以使用Java™Web Start （ 下载JDK 7或更高版本 ）运行TextDemo。另外，要自己编译并运行示例，请参考示例索引 。


您可以在以下位置找到该程序的完整代码TextDemo.java 。以下代码创建并设置文本字段：

textField = new JTextField(20);

传递给的整数参数JTextField构造函数20在示例中，指示字段中的列数。该数字与字段的当前字体提供的度量一起使用，以计算字段的首选宽度。它不限制用户可以输入的字符数。为此，可以使用格式化的文本字段或文档侦听器，如“ 文本组件功能”中所述 。

注意： 

我们建议您为每个文本字段指定列数。如果未指定列数或首选大小，则每当文本更改时，字段的首选大小都会更改，这可能会导致不必要的布局更新。

下一行代码注册一个TextDemo对象作为文本字段的动作侦听器。

textField.addActionListener(this);

的actionPerformed方法处理文本字段中的操作事件：

private final static String newline = "\n";
...
public void actionPerformed(ActionEvent evt) {
    String text = textField.getText();
    textArea.append(text + newline);
    textField.selectAll();
}

注意使用JTextField的getText检索文本字段当前包含的文本的方法。此方法返回的文本不包含触发操作事件的Enter键的换行符。

您已经了解了如何使用基本文本字段。因为JTextField类继承自JTextComponent在课堂上，文本字段非常灵活，几乎可以按照您喜欢的任何方式进行自定义。例如，您可以添加文档侦听器或文档过滤器，以便在文本更改时得到通知，并且在过滤器的情况下，您可以相应地修改文本字段。有关文本组件的信息可以在“ 文本组件功能”中找到。自定义之前JTextField ，但是，请确保其他基于文本字段的组件不会为您完成这项工作。

通常，文本字段与描述文本字段的标签配对。有关创建这些对的信息，请参见使用文本字段的示例 。

另一个示例：TextFieldDemo

的TextFieldDemo该示例介绍了一个文本字段和一个文本区域。您可以在以下位置找到该程序的完整代码TextFieldDemo.java 。

在文本字段中键入字符时，程序将在文本区域中搜索键入的文本。如果找到该条目，它将突出显示。如果程序找不到该条目，则文本字段的背景变为粉红色。文本区域下方的状态栏会显示一条消息，指示是否找到文本。退出键用于开始新的搜索或完成当前搜索。这是一张照片TextFieldDemo应用。

单击启动按钮，使用Java™Web Start来运行TextFieldDemo（ 下载JDK 7或更高版本 ）。另外，要自己编译并运行示例，请参考示例索引 。


为了突出显示文本，本示例使用了荧光笔和绘画工具。下面的代码为文本区域创建并设置荧光笔和绘画工具。

final Highlighter hilit;
final Highlighter.HighlightPainter painter;
...
hilit = new DefaultHighlighter();
painter = new DefaultHighlighter.DefaultHighlightPainter(HILIT_COLOR);
textArea.setHighlighter(hilit);

此代码将文档侦听器添加到文本字段的文档中。

entry.getDocument().addDocumentListener(this);

文件监听者insertUpdate和removeUpdate方法调用search方法，该方法不仅可以在文本区域中执行搜索，还可以处理突出显示。以下代码突出显示找到的文本，将插入号设置为找到的匹配项的末尾，设置文本字段的默认背景，并在状态栏中显示一条消息。

hilit.addHighlight(index, end, painter);
textArea.setCaretPosition(end);
entry.setBackground(entryBg);
message("'" + s + "' found. Press ESC to end search");

状态栏是一个JLabel宾语。以下代码显示了message方法已实现。

private JLabel status;
...
void message(String msg) {
    status.setText(msg);
}

如果文本区域中没有匹配项，则以下代码将文本字段的背景更改为粉红色，并显示正确的信息消息。

entry.setBackground(ERROR_COLOR);
message("'" + s + "' not found. Press ESC to start a new search");

的CancelAction类负责按以下方式处理Escape键。

   class CancelAction extends AbstractAction {
       public void actionPerformed(ActionEvent ev) {
               hilit.removeAllHighlights();
               entry.setText("");
               entry.setBackground(entryBg);
           }
   }

文字栏位API

下表列出了常用的JTextField构造函数和方法。您可能会调用的其他方法在JTextComponent类。请参阅Text Component API 。

您也可以在继承自文本字段其他祖先的文本字段上调用方法，例如setPreferredSize ， setForeground ， setBackground ， setFont ， 等等。有关常用继承方法的表，请参见JComponent类 。
