如何使用选项卡式窗格

随着JTabbedPane类，您可以有多个组件（例如panel ）共享同一空间。用户通过选择与所需组件相对应的选项卡来选择要查看的组件。如果您想要没有选项卡界面的类似功能，则可以使用卡片布局而不是选项卡式窗格。

创建选项卡式窗格

要创建选项卡式窗格，请实例化JTabbedPane ，创建您希望显示的组件，然后使用addTab方法。

下图介绍了一个名为TabbedPaneDemo具有带四个选项卡的选项卡式窗格。

尝试这个： 
  1. 单击启动按钮，以使用Java™Web Start （ 下载JDK 7或更高版本 ）运行TabbedPaneDemo。另外，要自己编译并运行示例，请参考示例索引 。
  2. 将光标放在选项卡上。
    出现与选项卡关联的工具提示。为方便起见，可以在将组件添加到选项卡式窗格中时指定工具提示文本。
  3. 通过单击选择一个选项卡。
    选项卡式窗格显示与选项卡相对应的组件。
  4. 通过输入助记符选择一个选项卡。
    例如，在Java外观中，您可以通过键入Alt-3选择标记为“ Tab 3”的选项卡。
  5. 在可滚动的标签之间导航。
    本示例提供了可滚动的选项卡。通过移动对话框的左边界或右边界来调整对话框的大小，以使选项卡不适合对话框。滚动箭头出现在选项卡旁边。
    单击箭头查看隐藏的选项卡之一。
    请注意，单击箭头只会显示隐藏的选项卡。它不会选择新的选项卡。

作为TabbedPaneDemo如示例所示，选项卡可以具有工具提示和助记符，并且可以同时显示文本和图像。

标签窗格的代码

以下代码来自TabbedPaneDemo.java在上一个示例中创建选项卡式窗格。请注意，不需要事件处理代码。的JTabbedPane object为您处理鼠标和键盘事件。

JTabbedPane tabbedPane = new JTabbedPane();
ImageIcon icon = createImageIcon("images/middle.gif");

JComponent panel1 = makeTextPanel("Panel #1");
tabbedPane.addTab("Tab 1", icon, panel1,
                  "Does nothing");
tabbedPane.setMnemonicAt(0, KeyEvent.VK_1);

JComponent panel2 = makeTextPanel("Panel #2");
tabbedPane.addTab("Tab 2", icon, panel2,
                  "Does twice as much nothing");
tabbedPane.setMnemonicAt(1, KeyEvent.VK_2);

JComponent panel3 = makeTextPanel("Panel #3");
tabbedPane.addTab("Tab 3", icon, panel3,
                  "Still does nothing");
tabbedPane.setMnemonicAt(2, KeyEvent.VK_3);

JComponent panel4 = makeTextPanel(
        "Panel #4 (has a preferred size of 410 x 50).");
panel4.setPreferredSize(new Dimension(410, 50));
tabbedPane.addTab("Tab 4", icon, panel4,
                      "Does nothing at all");
tabbedPane.setMnemonicAt(3, KeyEvent.VK_4);

如前面的代码所示， addTab方法处理在选项卡式窗格中设置选项卡的大部分工作。的addTab方法有几种形式，但是它们都使用字符串标题和选项卡上要显示的组件。（可选）您可以指定图标和工具提示字符串。文本或图标（或两者）可以为空。创建标签的另一种方法是使用insertTab方法，该方法可让您指定要添加的标签的索引。请注意addTab方法不允许在此步骤中指定索引。

切换到特定选项卡

使用GUI可以通过三种方式切换到特定选项卡。

  1. 使用鼠标。 要切换到特定选项卡，用户可以用鼠标单击它。
  2. 使用键盘箭头。 当。。。的时候JTabbedPane对象具有焦点，可以使用键盘箭头在选项卡之间切换。
  3. 使用键助记符。 的setMnemonicAt方法允许用户使用键盘切换到特定选项卡。例如， setMnemonicAt(3, KeyEvent.VK_4)使“ 4”成为第四个选项卡的助记符（位于索引3，因为索引以0开头）；按Alt-4会出现第四个选项卡的组件。助记符通常在选项卡标题中使用一个字符，然后该字符会自动加下划线。

要以编程方式切换到特定标签，请使用setSelectedIndex或者setSelectedComponent方法。

标签中的首选大小

当构建要添加到选项卡式窗格的组件时，请记住，无论选项卡式窗格的哪个子项可见，每个子项都会获得相同的空间来显示自己。选项卡式窗格的首选大小恰好足以在其首选高度显示其最高子级，并在首选宽度显示其最宽的子级。同样，选项卡式窗格的最小大小取决于其所有子项的最大最小宽度和高度。

在里面TabbedPaneDemo例如，第四面板具有比其他面板更大的优选宽度和高度。因此，选项卡式窗格的首选大小恰好足以以其首选大小显示第四面板。假定选项卡式窗格处于其首选的大小，每个面板将获得完全相同的空间量-410像素宽和50高。如果您不了解如何使用首选尺寸，请参阅布局管理的工作原理。

删除标签

以下代码来自ButtonTabComponent.java从选项卡式窗格中删除一个选项卡。请注意，事件处理代码是必需的。由于每个标签都包含一个JButton对象，您必须附加一个ActionListener关闭按钮。当用户单击按钮时， actionPerformed方法确定其所属标签的索引，并删除相应的标签。

public void actionPerformed(ActionEvent e) {
    int i = pane.indexOfTabComponent(ButtonTabComponent.this);
    if (i != -1) {
    pane.remove(i);
    }
}

将标题赋予自定义标签

下面的代码取自ButtonTabComponent.java ，显示自定义选项卡组件如何从原始选项卡式窗格选项卡中获取标题。

JLabel label = new JLabel(title) {
    public String getText() {
        int i = pane.indexOfTabComponent(ButtonTabComponent.this);
        if (i != -1) {
            return pane.getTitleAt(i);
        }
        return null;
    }
};
