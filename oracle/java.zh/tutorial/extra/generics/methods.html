通用方法

考虑编写一种方法，该方法采用一个对象数组和一个集合，并将该数组中的所有对象放入集合中。这是第一次尝试：

static void fromArrayToCollection(Object[] a, Collection<?> c) {
    for (Object o : a) { 
        c.add(o); // compile-time error
    }
}

到目前为止，您将学会避免初学者尝试使用的错误。 Collection
