“你好，世界！“用于Solaris OS和Linux

现在是时候编写您的第一个应用程序了！这些详细说明适用于Solaris OS和Linux用户。有关其他平台的说明，请参见“ Hello World！”用于Microsoft Windows和“ Hello World！“对于NetBeans IDE 。

如果您在此页面上的说明上遇到问题，请查阅“ 常见问题（及其解决方案）” 。

清单

要编写第一个程序，您需要：

  1. Java SE开发套件8（JDK 8）

    您现在可以下载Solaris OS或Linux版本 。（确保下载JDK 而不是 JRE。）请查阅安装说明 。

  2. 文字编辑器

    在此示例中，我们将使用Pico，该编辑器可用于许多基于UNIX的平台。如果您使用其他文本编辑器（例如， vi要么emacs 。

这两个都是编写第一个应用程序所需要的。

创建您的第一个应用程序

您的第一个申请， HelloWorldApp ，只会显示问候语“ Hello world！”。要创建此程序，您将：

  • 创建一个源文件

    源文件包含用Java编程语言编写的代码，您和其他程序员可以理解。您可以使用任何文本编辑器来创建和编辑源文件。

  • 将源文件编译成.class文件

    Java编程语言编译器 （ javac ）获取您的源文件，并将其文本转换为Java虚拟机可以理解的指令。其中包含的说明.class文件被称为字节码 。

  • 运行程序

    Java应用程序启动器工具 （ java ）使用Java虚拟机来运行您的应用程序。

创建一个源文件

要创建源文件，您有两个选择：

  • 您可以保存文件HelloWorldApp.java在您的计算机上，并避免大量输入。然后，您可以直接编译源文件 。

  • 或者，您可以使用以下（更长的）说明。

首先，打开一个外壳或“终端”窗口。

一个新的终端窗口。

首次显示提示时， 当前目录通常是主目录 。您可以随时输入以下内容将当前目录更改为您的主目录cd在提示符下，然后按Return键 。

您创建的源文件应保存在单独的目录中。您可以使用以下命令创建目录mkdir 。例如，创建目录examples/java在/ tmp目录中，使用以下命令：

cd /tmp
mkdir examples
cd examples
mkdir java

要将当前目录更改为该新目录，请输入：

cd /tmp/examples/java

现在，您可以开始创建源文件了。

通过键入启动Pico编辑器pico在提示符下，然后按Return键 。如果系统响应消息pico: command not found ，则很可能无法使用Pico。请咨询系统管理员以获取更多信息，或使用其他编辑器。

启动Pico时，它将显示一个新的空白缓冲区 。这是您将在其中键入代码的区域。

在新缓冲区中键入以下代码：

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

输入时请小心

注意：完全按照所示键入所有代码，命令和文件名。两者都是编译器（ javac ）和启动器（ java ） 区分大小写 ，因此您必须始终使用大写字母。

HelloWorldApp是不一样的helloworldapp 。

将代码保存在名称为HelloWorldApp.java 。在Pico编辑器中，您可以通过按Ctrl-O ，然后在底部看到提示的方式来执行此操作File Name to write: ，输入您要在其中创建文件的目录，然后输入HelloWorldApp.java 。例如，如果您想保存HelloWorldApp.java在目录中/tmp/examples/java ，然后输入/tmp/examples/java/HelloWorldApp.java然后按回车键。

您可以键入Ctrl-X退出Pico。

将源文件编译为.class文件

调出另一个外壳窗口。要编译源文件，请将当前目录更改为文件所在的目录。例如，如果您的源目录是/tmp/examples/java ，在提示符下键入以下命令，然后按Return键：

cd /tmp/examples/java

如果输入pwd在提示符下，您应该看到当前目录，在本示例中该目录已更改为/tmp/examples/java 。

如果输入ls在提示符下，您应该看到您的文件。

结果ls命令，显示.java源文件。

现在可以编译源文件了。在提示符下，键入以下命令，然后按Return键。

javac HelloWorldApp.java

编译器已生成一个字节码文件， HelloWorldApp.class 。在提示符下，键入ls查看生成的新文件： 下图 。

结果ls命令，显示生成的.class文件。

现在您有了一个.class文件，您可以运行您的程序。

如果您在此步骤的说明中遇到问题，请查阅“ 常见问题（及其解决方案）” 。

运行程序

在同一目录中，在提示符下输入：

java HelloWorldApp

下图显示了您现在应该看到的内容。

输出显示“ Hello World！屏幕上。

恭喜你！您的程序有效！

如果您在此步骤的说明中遇到问题，请查阅“ 常见问题（及其解决方案）” 。
