课程：仔细研究“ Hello World！申请

现在您已经看到了“ Hello World！应用程序（甚至可能编译并运行它），您可能想知道它是如何工作的。这里再次是它的代码：


class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

“ Hello World！“应用程序由三个主要部分组成： 源代码注释 ， 该HelloWorldApp类的定义 ，并在main方法 。以下说明将使您对代码有基本的了解，但是更深层次的含义只有在您阅读完本教程的其余部分后，才会变得显而易见。

源代码注释

下面的粗体文字定义的“Hello World的评论 ！”应用：

/**
 * The HelloWorldApp class implements an application that
 * simply prints "Hello World!" to standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

注释被编译器忽略，但对其他程序员有用。Java编程语言支持三种注释：

/* text */
编译器忽略所有内容/*至*/ 。
/** documentation */
这表示文档注释（简称doc comment ）。编译器会忽略这种注释，就像它会忽略使用/*和*/ 。的javadoc准备自动生成的文档时，该工具会使用文档注释。有关更多信息javadoc ，请参阅Javadoc™工具文档 。
// text
编译器忽略所有内容//到行尾

的HelloWorldApp类定义

以下粗体字开始了“ Hello World！”的类定义块。”应用：

/**
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); // Display the string.
    }
}

如上所示，类定义的最基本形式是：

class name {
    . . .
}

关键字class开始名为的类的类定义name ，并且每个类的代码都显示在上方用粗体标记的大括号之间。第2章概述了所有类，第4章详细讨论了类。现在就足以知道每个应用程序都以类定义开头。

的main方法

以下粗体字开始定义main方法：

/**
 * The HelloWorldApp class implements an application that
 * simply displays "Hello World!" to the standard output.
 */
class HelloWorldApp {
    public static void main(String[] args) {
        System.out.println("Hello World!"); //Display the string.
    }
}

在Java编程语言中，每个应用程序都必须包含一个main其签名为的方法：

public static void main(String[] args)

修饰符public和static可以以任何顺序写（ public static要么static public ），但约定是使用public static如上图所示。您可以为参数指定任意名称，但是大多数程序员都选择“ args”或“ argv”。

的main方法类似于main在C和C ++中起作用；它是应用程序的入口点，随后将调用程序所需的所有其他方法。

的main方法接受一个参数：类型为元素的数组String 。

public static void main(String[] args)

该数组是运行时系统将信息传递到您的应用程序的机制。例如：

java MyApp arg1 arg2

数组中的每个字符串都称为命令行参数 。命令行参数使用户无需重新编译即可影响应用程序的操作。例如，排序程序可能允许用户使用以下命令行参数指定数据以降序排序：

-descending

“ Hello World！应用程序会忽略其命令行参数，但是您应该知道这样的参数确实存在。

最后一行：

System.out.println("Hello World!");

使用System核心库中的类打印“ Hello World！”消息发送到标准输出。该库的某些部分（也称为“应用程序编程接口”或“ API”）将在本教程的其余部分中进行讨论。
