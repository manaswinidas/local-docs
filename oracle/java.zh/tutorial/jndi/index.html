Trail：Java命名和目录接口

此跟踪描述了用于访问目录和命名服务的API JNDI™（Java命名和目录接口）。在这里，您将了解基本的命名和目录服务，以及如何使用JNDI编写简单的应用程序来使用这些服务。最受欢迎的目录服务LDAP用于说明使用JNDI访问目录服务。

命名和目录概念从此处开始以概述命名和目录概念。

JNDI概述为您提供JNDI的概述 ，其体系结构和包装。

软件设置描述了设置运行本教程中描述的示例以及任何其他JNDI应用程序所需的环境所涉及的说明和步骤。

命名和目录操作描述各种命名和目录操作，并通过大量使用JNDI访问命名/目录服务的示例进行演示。

LDAP用户的高级主题LDAP用户的专门课程。它讨论了将JNDI建模为LDAP API，如何执行LDAP身份验证，SSL以及在生产环境中管理连接。

访问目录中的对象向您展示如何将应用程序与目录集成，以便您可以在目录中存储Java对象以及从目录中检索Java对象。

JDK 5.0和JDK 6中的功能介绍JDK 5.0和JDK 6中可用的JNDI和LDAP Service Provider中的功能。

注意：本教程资源基于https://docs.oracle.com/javase/jndi/tutorial/上的独立JNDI教程。独立的JNDI教程最近在Java 2 SDK Standard Edition v 1.4.2下更新，提供了有关JNDI的全面介绍，但不再受支持。本教程摘录了独立教程的基础知识，并包括在Java Platform Standard Edition 5.0和6版本中添加到JNDI的功能。

较旧的JNDI教程保留为docs.oracle.com上的存档。
