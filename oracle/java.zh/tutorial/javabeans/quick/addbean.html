使用第三方Bean

几乎所有代码都可以打包为bean。到目前为止，您所看到的Bean都是可视Bean，但是Bean可以提供功能而没有可见的组件。

JavaBeans的强大之处在于，您可以使用软件组件而无需编写它们或了解它们的实现。

该页面描述了如何将JavaBean添加到应用程序并利用其功能。

将Bean添加到NetBeans面板

下载an example JavaBean component, BumperSticker 。Bean作为JAR文件分发。将文件保存在计算机上的某个位置。 BumperSticker是图形组件，它公开了一种方法， go() ，即会启动动画。

添加BumperSticker在NetBeans面板上，从NetBeans菜单中选择“ 工具”>“面板”>“ Swing / AWT组件 ”。

单击“ 从JAR添加...”按钮。NetBeans要求您找到包含要添加到面板中的bean的JAR文件。找到您刚刚下载的文件，然后单击“ 下一步” 。

NetBeans在JAR文件中显示了这些类的列表。选择您希望添加到调色板中的那些。在这种情况下，选择BumperSticker ，然后单击Next 。

最后，NetBeans需要知道调色板的哪一部分将接收新bean。选择Beans ，然后单击Finish 。

单击关闭以使“ 调色板管理器”窗口消失。现在看一下调色板。 BumperSticker在Beans部分中。

使用新的JavaBean

继续拖动BumperSticker从调色板中移到您的表单中。

您可以使用BumperSticker实例，就像您要与其他任何Bean一起使用一样。要查看实际效果，请将另一个按钮拖出到表单中。此按钮将启动BumperSticker的动画。

将按钮连接到BumperSticker Bean，就像您已经将第一个按钮连接到文本字段一样。

  1. 首先单击“ 连接模式”按钮。
  2. 单击第二个按钮。NetBeans为其提供了红色轮廓。
  3. 点击BumperSticker零件。弹出连接向导 。
  4. 单击操作旁边的+ ，然后选择actionPerformed 。单击下一步> 。
  5. 选择Method Call ，然后从列表中选择go（） 。点击完成 。

如果不确定任何步骤，请查看“ 为应用程序布线” 。这里的过程非常相似。

再次运行该应用程序。当您单击第二个按钮时， BumperSticker组件使心脏的颜色动起来。

同样，请注意您是如何在不编写任何代码的情况下生成功能正常的应用程序的。
