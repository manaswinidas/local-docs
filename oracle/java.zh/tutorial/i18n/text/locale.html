执行与语言环境无关的比较

整理规则定义字符串的排序顺序。这些规则因地区而异，因为各种自然语言对单词的排序方式不同。您可以使用由Collator类以与语言环境无关的方式对字符串进行排序。

实例化Collator类调用getInstance方法。通常，您创建一个Collator为默认Locale ，如以下示例所示：

Collator myDefaultCollator = Collator.getInstance();

您还可以指定一个特定的Locale当您创建一个Collator ， 如下：

Collator myFrenchCollator = Collator.getInstance(Locale.FRENCH);

的getInstance方法返回一个RuleBasedCollator ，它是的一个具体子类Collator 。的RuleBasedCollator包含一组规则，这些规则确定您指定的语言环境的字符串的排序顺序。这些规则是为每个语言环境预定义的。因为规则被封装在RuleBasedCollator ，您的程序将不需要特殊的例程来处理排序规则随语言而变化的方式。

您调用Collator.compare用于执行与语言环境无关的字符串比较的方法。的compare当第一个字符串参数小于，等于或大于第二个字符串参数时，该方法返回一个小于，等于或大于零的整数。下表包含一些示例调用Collator.compare ：

例 返回值 说明
myCollator.compare("abc", "def") -1 "abc"小于“ def”
myCollator.compare("rtf", "rtf") 0 两个字符串相等
myCollator.compare("xyz", "abc") 1 “ xyz”大于“ abc”

您使用compare执行排序操作时的方法。该示例程序称为CollatorDemo使用compare对英语和法语单词数组进行排序的方法。此程序显示了使用两个不同的排序规则对相同的单词列表进行排序时会发生什么：

Collator fr_FRCollator = Collator.getInstance(new Locale("fr","FR"));
Collator en_USCollator = Collator.getInstance(new Locale("en","US"));

排序的方法称为sortStrings ，可以与任何Collator 。请注意sortStrings方法调用compare方法：

public static void sortStrings(Collator collator, String[] words) {
    String tmp;
    for (int i = 0; i < words.length; i++) {
        for (int j = i + 1; j < words.length; j++) { 
            if (collator.compare(words[i], words[j]) > 0) {
                tmp = words[i];
                words[i] = words[j];
                words[j] = tmp;
            }
        }
    }
}

英文Collator将单词排序如下：

peach
péché
pêche
sin

根据法语的排序规则，前面的列表顺序错误。在法语中，péché应该在排序列表中跟随pêche。法国人Collator正确地对单词数组进行排序，如下所示：

peach
pêche
péché
sin
