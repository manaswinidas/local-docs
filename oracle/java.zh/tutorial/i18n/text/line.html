线边界

格式化文本或执行换行的应用程序必须找到潜在的换行符。您可以找到这些换行符或边界BreakIterator用getLineInstance方法：

BreakIterator lineIterator =
    BreakIterator.getLineInstance(currentLocale);

这个BreakIterator确定字符串中文本可以中断以在下一行继续的位置。被检测到的位置BreakIterator是潜在的换行符。屏幕上显示的实际换行符可能不同。

以下两个示例使用markBoundaries的方法BreakIteratorDemo.java显示由a检测到的线边界BreakIterator 。的markBoundaries方法通过在目标字符串下方打印插入符号（^）来指示线边界。

根据一个BreakIterator ，则在一系列空白字符（空格，制表符，换行符）终止之后会出现行边界。在以下示例中，请注意，您可以在检测到的任何边界处断开该行：

She stopped.  She said, "Hello there," and then went on.
^   ^         ^   ^     ^      ^     ^ ^   ^    ^    ^  ^

连字符后也可能立即发生断行：

There are twenty-four hours in a day.
^     ^   ^      ^    ^     ^  ^ ^   ^

下一个示例使用称为的方法将一长串文本分成固定长度的行formatLines 。此方法使用BreakIterator找到潜在的换行符。的formatLines该方法简短，简单，并且要感谢BreakIterator ，与语言环境无关。这是源代码：

static void formatLines(
    String target, int maxLength,
    Locale currentLocale) {

    BreakIterator boundary = BreakIterator.
        getLineInstance(currentLocale);
    boundary.setText(target);
    int start = boundary.first();
    int end = boundary.next();
    int lineLength = 0;

    while (end != BreakIterator.DONE) {
        String word = target.substring(start,end);
        lineLength = lineLength + word.length();
        if (lineLength >= maxLength) {
            System.out.println();
            lineLength = word.length();
        }
        System.out.print(word);
        start = end;
        end = boundary.next();
    }
}

的BreakIteratorDemo程序调用formatLines方法如下：

String moreText =
    "She said, \"Hello there,\" and then " +
    "went on down the street. When she stopped " +
    "to look at the fur coats in a shop + "
    "window, her dog growled. \"Sorry Jake,\" " +
    "she said. \"I didn't know you would take " +
    "it personally.\"";

formatLines(moreText, 30, currentLocale);

此调用的输出为formatLines是：

She said, "Hello there," and
then went on down the
street. When she stopped to
look at the fur coats in a
shop window, her dog
growled. "Sorry Jake," she
said. "I didn't know you
would take it personally."
