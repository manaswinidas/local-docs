分支语句

的break声明

的break语句有两种形式：标记和未标记。您在之前的讨论中看到了无标签的表单switch声明。您也可以使用未标记的break终止for ， while ， 要么do-while循环，如下所示BreakDemo程序：

class BreakDemo {
    public static void main(String[] args) {

        int[] arrayOfInts = 
            { 32, 87, 3, 589,
              12, 1076, 2000,
              8, 622, 127 };
        int searchfor = 12;

        int i;
        boolean foundIt = false;

        for (i = 0; i < arrayOfInts.length; i++) {
            if (arrayOfInts[i] == searchfor) {
                foundIt = true;
                break;
            }
        }

        if (foundIt) {
            System.out.println("Found " + searchfor + " at index " + i);
        } else {
            System.out.println(searchfor + " not in the array");
        }
    }
}

该程序在数组中搜索数字12。的break粗体显示的语句终止for找到该值时循环。然后，控制流转移到语句之后for环。该程序的输出为：

Found 12 at index 4

未贴标签break语句终止最里面的switch ， for ， while ， 要么do-while声明，但带有标签break终止外部语句。以下程序， BreakWithLabelDemo ，与之前的程序类似，但使用嵌套for循环搜索二维数组中的值。找到该值后， break终止外部for循环（标记为“搜索”）：


class BreakWithLabelDemo {
    public static void main(String[] args) {

        int[][] arrayOfInts = { 
            { 32, 87, 3, 589 },
            { 12, 1076, 2000, 8 },
            { 622, 127, 77, 955 }
        };
        int searchfor = 12;

        int i;
        int j = 0;
        boolean foundIt = false;

    search:
        for (i = 0; i < arrayOfInts.length; i++) {
            for (j = 0; j < arrayOfInts[i].length;
                 j++) {
                if (arrayOfInts[i][j] == searchfor) {
                    foundIt = true;
                    break search;
                }
            }
        }

        if (foundIt) {
            System.out.println("Found " + searchfor + " at " + i + ", " + j);
        } else {
            System.out.println(searchfor + " not in the array");
        }
    }
}

这是程序的输出。

Found 12 at 1, 0

的break语句终止带标签的语句；它不会将控制流转移到标签上。控制流紧接在标记（终止）的语句之后转移到该语句。

的continue声明

的continue语句跳过当前的迭代for ， while ， 要么do-while环。无标签的表单会跳到最内层循环的主体的末尾，并评估boolean控制循环的表达式。以下程序， ContinueDemo ，逐步执行String ，计算字母“ p”的出现次数。如果当前字符不是ap，则continue语句跳过循环的其余部分，并继续执行下一个字符。如果它是 “ p”，则程序将增加字母数。


class ContinueDemo {
    public static void main(String[] args) {

        String searchMe = "peter piper picked a " + "peck of pickled peppers";
        int max = searchMe.length();
        int numPs = 0;

        for (int i = 0; i < max; i++) {
            // interested only in p's
            if (searchMe.charAt(i) != 'p')
                continue;

            // process p's
            numPs++;
        }
        System.out.println("Found " + numPs + " p's in the string.");
    }
}

这是该程序的输出：

Found 9 p's in the string.

要更清楚地看到此效果，请尝试删除continue语句并重新编译。当您再次运行该程序时，计数将是错误的，表示它找到的是35 p而不是9。

有标签continue语句跳过标有给定标签的外部循环的当前迭代。以下示例程序， ContinueWithLabelDemo ，使用嵌套循环在另一个字符串中搜索子字符串。需要两个嵌套循环：一个循环遍历子字符串，另一个循环遍历要搜索的字符串。以下程序， ContinueWithLabelDemo ，使用标记形式的Continue继续跳过外循环中的迭代。


class ContinueWithLabelDemo {
    public static void main(String[] args) {

        String searchMe = "Look for a substring in me";
        String substring = "sub";
        boolean foundIt = false;

        int max = searchMe.length() - 
                  substring.length();

    test:
        for (int i = 0; i <= max; i++) {
            int n = substring.length();
            int j = i;
            int k = 0;
            while (n-- != 0) {
                if (searchMe.charAt(j++) != substring.charAt(k++)) {
                    continue test;
                }
            }
            foundIt = true;
                break test;
        }
        System.out.println(foundIt ? "Found it" : "Didn't find it");
    }
}

这是该程序的输出。

Found it

的return声明

最后一个分支语句是return声明。的return语句从当前方法退出，控制流返回到调用该方法的位置。的return语句有两种形式：一种返回值，另一种不返回值。要返回值，只需将值（或计算该值的表达式）放在return关键词。

return ++count;

返回值的数据类型必须与方法声明的返回值的类型匹配。声明方法时void ，使用的形式return没有返回值。

return;
