问题和练习的答案：嵌套类

问题

  1. 问题 ：程序Problem.java无法编译。您需要做什么才能使其编译？为什么？

    答案 ：删除static在声明的前面Inner类。静态内部类无权访问外部类的实例字段。看到ProblemSolved.java 。

  2. 将Java API文档用于Box类（在javax.swing包）以帮助您回答以下问题。

    1. 问题 ：静态嵌套类做什么Box限定？

      答 ：Box.Filler

    2. 问题 ：内层是做什么的Box限定？

      答 ：Box.AccessibleBox

    3. 问题 ：什么是超类？ Box的内部阶级？

      答 ：[java.awt.]Container.AccessibleAWTContainer

    4. 问题 ：哪个Box您可以在任何类中使用的嵌套类吗？

      答 ：Box.Filler

    5. 问题 ：如何创建的实例Box的Filler类？

      答 ：new Box.Filler(minDimension, prefDimension, maxDimension)

练习题

  1. 练习 ：获取文件Class1.java 。编译并运行Class1 。输出是什么？

    答 ：InnerClass1: getString invoked.
    InnerClass1: getAnotherString invoked.

  2. 练习 ：以下练习涉及修改班级DataStructure.java ，“ 内部类示例 ”部分对此进行了讨论。

    1. 定义一个名为print(DataStructureIterator iterator) 。使用类的实例调用此方法EvenIterator使其执行与方法相同的功能printEven 。

      提示 ：如果在命令行中指定这些语句，则不会编译这些语句main方法：

      DataStructure ds = new DataStructure();
ds.print(new EvenIterator());

      编译器在遇到表达式时会生成错误消息“无法从静态上下文引用的非静态变量” new EvenIterator() 。班级EvenIterator是内部的非静态类。这意味着您可以创建一个实例EvenIterator仅在外部类的实例中， DataStructure 。

      您可以在中定义方法DataStructure创建并返回一个新的实例EvenIterator 。

    2. 调用方法print(DataStructureIterator iterator)这样就可以打印出具有奇数索引值的元素。使用匿名类作为方法的参数，而不是接口的实例DataStructureIterator 。

      提示 ：您无法访问私人成员SIZE和arrayOfInts课外DataStructure ，这意味着您无法从外部定义的匿名类访问这些私有成员DataStructure 。

      您可以定义访问私有成员的方法SIZE和arrayOfInts然后在您的匿名课程中使用它们。

    3. 定义一个名为print(java.util.Function iterator)执行与print(DataStructureIterator iterator) 。使用lambda表达式调用此方法以打印具有偶数索引值的元素。再次使用lambda表达式调用此方法以打印具有奇数索引值的元素。

      提示 ：在此print方法，您可以遍历数组中包含的元素arrayOfInts与一个for表达。对于每个索引值，调用方法function.apply 。如果此方法为特定索引值返回一个真值，则打印该索引值中包含的元素。

      调用这个print打印具有偶数索引值的元素的方法，可以指定一个实现该方法的lambda表达式Boolean Function.apply(Integer t) 。这个lambda表达式需要一个Integer参数（索引）并返回一个Boolean值（ Boolean.TRUE如果索引值是偶数Boolean.FALSE除此以外）。

    4. 定义两个方法，以便这些语句打印具有偶数索引值的元素，然后打印具有奇数索引值的元素：

      DataStructure ds = new DataStructure()
// ...
ds.print(DataStructure::isEvenIndex);
ds.print(DataStructure::isOddIndex);

      提示 ：在类中创建两个方法DataStructure命名isEvenIndex和isOddIndex与抽象方法具有相同的参数列表和返回类型Boolean Function.apply(Integer t) 。这意味着这些方法需要一个Integer参数（索引）并返回一个Boolean值。

    答 ：查看文件DataStructure.java 。
