问题和练习的答案：对象

问题

  1. 问题 ：以下程序有什么问题？

    public class SomethingIsWrong {
    public static void main(String[] args) {
        Rectangle myRect;
        myRect.width = 40;
        myRect.height = 50;
        System.out.println("myRect's area is " + myRect.area());
    }
}

    答 ：该代码永远不会创建一个Rectangle宾语。使用此简单程序，编译器会生成错误。但是，在更现实的情况下， myRect可能被初始化为null在一个地方，例如在构造函数中，然后再使用。在这种情况下，程序将编译得很好，但会生成一个NullPointerException在运行时。

  2. 问题 ：以下代码创建一个数组和一个字符串对象。代码执行后，存在多少对这些对象的引用？这两个对象都可以进行垃圾收集吗？

    ...
String[] students = new String[10];
String studentName = "Peter Smith";
students[0] = studentName;
studentName = null;
...

    答 ：有一种参考students数组，该数组对字符串有一个引用Peter Smith 。这两个对象都没有资格进行垃圾回收。数组students不符合垃圾回收的条件，因为它对对象有一个引用studentName即使已为该对象分配了值null 。物体studentName不符合条件，因为students[0]仍然指它。

  3. 问题 ：程序如何销毁它创建的对象？

    答 ：程序不会显式销毁对象。程序可以将对对象的所有引用设置为null这样就可以进行垃圾收集了。但是该程序实际上并没有销毁对象。

练习题

  1. 练习 ：修复名为SomethingIsWrong如问题1所示。

    答案 ：见SomethingIsRight ：

    
public class SomethingIsRight {
    public static void main(String[] args) {
        Rectangle myRect = new Rectangle();
        myRect.width = 40;
        myRect.height = 50;
        System.out.println("myRect's area is " + myRect.area());
    }
}

  2. 练习 ：鉴于以下课程，称为NumberHolder ，编写一些代码来创建类的实例，初始化其两个成员变量，然后显示每个成员变量的值。

    
public class NumberHolder {
    public int anInt;
    public float aFloat;
}

    答案 ：见NumberHolderDisplay ：

    
public class NumberHolderDisplay {
    public static void main(String[] args) {
	NumberHolder aNumberHolder = new NumberHolder();
	aNumberHolder.anInt = 1;
	aNumberHolder.aFloat = 2.3f;
	System.out.println(aNumberHolder.anInt);
	System.out.println(aNumberHolder.aFloat);
    }
}
