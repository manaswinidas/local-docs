格式化数字打印输出

之前您看到了print和println将字符串打印到标准输出的方法（ System.out ）。由于所有数字都可以转换为字符串（如本课程后面所述），因此您可以使用这些方法打印出字符串和数字的任意混合形式。但是，Java编程语言还有其他方法，允许您在包含数字时对打印输出进行更多控制。

printf和format方法

的java.io包装包括PrintStream类，具有两个可用于替换的格式化方法print和println 。这些方法format和printf ，彼此相等。熟悉的System.out你一直在使用的恰好是PrintStream对象，因此您可以调用PrintStream方法System.out 。因此，您可以使用format要么printf您代码中以前使用过的任何地方print要么println 。例如，

System.out.format(.....);

这两个的语法java.io.PrintStream方法是一样的：

public PrintStream format(String format, Object... args)

哪里format是一个字符串，指定要使用的格式，并且args是要使用该格式打印的变量的列表。一个简单的例子是

System.out.format("The value of " + "the float variable is " +
     "%f, while the value of the " + "integer variable is %d, " +
     "and the string is %s", floatVar, intVar, stringVar);

第一个参数， format ，是一个格式字符串，用于指定第二个参数中对象的方式， args ，将被格式化。格式字符串包含纯文本以及格式说明符 ，这些格式说明符是格式化以下参数的特殊字符： Object... args 。（符号Object... args称为varargs ，这意味着参数的数量可能会有所不同。）

格式说明符以百分号（％）开头，以Converter结束。转换器是一个字符，指示要格式化的参数的类型。在百分号（％）和转换器之间，您可以具有可选的标志和说明符。有许多转换器，标志和说明符，在java.util.Formatter

这是一个基本示例：

int i = 461012;
System.out.format("The value of i is: %d%n", i);

的%d指定单个变量是十进制整数。的%n是与平台无关的换行符。输出为：

The value of i is: 461012

的printf和format方法是重载的。每个版本都有以下语法：

public PrintStream format(Locale l, String format, Object... args)

例如，要在法语系统中打印数字（用逗号代替浮点数的英语表示中的小数点），可以使用：

System.out.format(Locale.FRANCE,
    "The value of the float " + "variable is %f, while the " +
    "value of the integer variable " + "is %d, and the string is %s%n", 
    floatVar, intVar, stringVar);

以下程序显示了您可以使用的某些格式format 。输出显示在嵌入式注释中的双引号中：

import java.util.Calendar;
import java.util.Locale;

public class TestFormat {
    
    public static void main(String[] args) {
      long n = 461012;
      System.out.format("%d%n", n);      //  -->  "461012"
      System.out.format("%08d%n", n);    //  -->  "00461012"
      System.out.format("%+8d%n", n);    //  -->  " +461012"
      System.out.format("%,8d%n", n);    // -->  " 461,012"
      System.out.format("%+,8d%n%n", n); //  -->  "+461,012"
      
      double pi = Math.PI;

      System.out.format("%f%n", pi);       // -->  "3.141593"
      System.out.format("%.3f%n", pi);     // -->  "3.142"
      System.out.format("%10.3f%n", pi);   // -->  "     3.142"
      System.out.format("%-10.3f%n", pi);  // -->  "3.142"
      System.out.format(Locale.FRANCE,
                        "%-10.4f%n%n", pi); // -->  "3,1416"

      Calendar c = Calendar.getInstance();
      System.out.format("%tB %te, %tY%n", c, c, c); // -->  "May 29, 2006"

      System.out.format("%tl:%tM %tp%n", c, c, c);  // -->  "2:34 am"

      System.out.format("%tD%n", c);    // -->  "05/29/06"
    }
}

DecimalFormat类

您可以使用java.text.DecimalFormat类，用于控制前导和尾随零，前缀和后缀，分组（千位）分隔符和十进制分隔符的显示。 DecimalFormat在数字格式方面提供了很大的灵活性，但是它会使您的代码更复杂。

以下示例创建了一个DecimalFormat宾语， myFormatter ，通过将模式字符串传递给DecimalFormat构造函数。的format()方法， DecimalFormat继承自NumberFormat ，然后由调用myFormatter -它接受double值作为参数，并以字符串形式返回格式化的数字：

这是一个示例程序，说明了DecimalFormat ：


import java.text.*;

public class DecimalFormatDemo {

   static public void customFormat(String pattern, double value ) {
      DecimalFormat myFormatter = new DecimalFormat(pattern);
      String output = myFormatter.format(value);
      System.out.println(value + "  " + pattern + "  " + output);
   }

   static public void main(String[] args) {

      customFormat("###,###.###", 123456.789);
      customFormat("###.##", 123456.789);
      customFormat("000000.000", 123.78);
      customFormat("$###,###.###", 12345.67);  
   }
}

输出为：

123456.789  ###,###.###  123,456.789
123456.789  ###.##  123456.79
123.78  000000.000  000123.780
12345.67  $###,###.###  $12,345.67
