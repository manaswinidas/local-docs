问题和练习的答案：数字

问题

  1. 使用API文档查找以下问题的答案：

    1. 问题：什么Integer您可以用来转换int转换成以十六进制表示数字的字符串？例如，哪种方法将整数65转换为字符串“ 41”？

      回答： toHexString

    2. 问题：什么Integer您将使用哪种方法将以5为基数的字符串转换为等效字符串int ？例如，如何将字符串“ 230”转换为整数值65？显示用于完成此任务的代码。

      回答： valueOf 。这是如何做：

      String base5String = "230";
int result = Integer.valueOf(base5String, 5);

    3. 问题：您可以使用哪种Double方法来检测浮点数是否具有特殊值Not Number（ NaN ）？

      回答： isNaN

  2. 问题：以下表达式的值是什么，为什么？

    Integer.valueOf(1).equals(Long.valueOf(1))

    答：错。两个对象（ Integer和Long ）有不同的类型。

练习题

  1. 练习：改变MaxVariablesDemo显示最小值而不是最大值。您可以删除与变量相关的所有代码aChar和aBoolean 。输出是什么？

    答案：见MinVariablesDemo 。这是输出：

    The smallest byte value is -128
The smallest short value is -32768
The smallest integer value is -2147483648
The smallest long value is -9223372036854775808
The smallest float value is 1.4E-45
The smallest double value is 4.9E-324

  2. 练习：创建一个程序，该程序从命令行读取未指定数量的整数参数，并将它们加在一起。例如，假设您输入以下内容：

           java Adder 1 3 2 10

    该程序应显示16然后退出。如果用户仅输入一个参数，则程序应显示一条错误消息。您可以根据您的程序ValueOfDemo 。

    答案：见Adder 。

  3. 练习：创建一个与上一个程序相似的程序，但有以下区别：

    • 而不是读取整数参数，而是读取浮点参数。
    • 它使用小数点右边的两位数字显示参数的总和。

    例如，假设您输入以下内容：

    java FPAdder 1 1e2 3.0 4.754

    该程序将显示108.75 。根据您的语言环境，小数点可能是逗号（ , ），而不是句点（ . ）。

    答案：见FPAdder 。
