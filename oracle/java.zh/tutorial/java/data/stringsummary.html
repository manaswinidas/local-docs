字符和字符串摘要

在大多数情况下，如果您使用单个字符值，则将使用基元char类型。但是，有时您需要将char用作对象，例如，将其用作期望对象的方法参数。Java编程语言提供了一个包装器类，用于“包装” char在一个Character为此目的。类型的对象Character包含一个类型为char 。这个Character类还提供了许多用于处理字符的有用的类（即静态）方法。

字符串是字符序列，在Java编程中广泛使用。在Java编程语言中，字符串是对象。的String该类具有60多个方法和13个构造函数。

最常见的是，您使用以下语句创建字符串

String s = "Hello world!";

而不是使用其中之一String构造函数。

的String类有许多查找和检索子字符串的方法；然后可以使用+串联运算符。

的String类还包括许多实用程序方法，其中split() ， toLowerCase() ， toUpperCase()和valueOf() 。在将用户输入的字符串转换为数字时，后一种方法必不可少。的Number子类还具有将字符串转换为数字的方法，反之亦然。

除了String上课，还有一个StringBuilder类。与...合作StringBuilder对象有时比使用字符串更有效。的StringBuilder类提供了一些对字符串有用的方法，其中包括reverse() 。但是，一般而言， String课堂上有各种各样的方法。
