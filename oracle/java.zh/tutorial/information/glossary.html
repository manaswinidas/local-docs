Java（TM）平台版本
Java平台“版本”是Java平台的最终版本，该版本提供了广泛市场领域所需的功能。

一个版本包含两种API集：（i）“核心软件包”，对于给定平台版本的所有实现都是必不可少的；以及（ii）“可选软件包”，它们可用于给定平台版本，并且在兼容的实现中可能受支持。

Java平台有3个不同的版本：

* Java 2平台企业版：
针对企业的Java平台版本，用于开发，部署和管理以多层服务器为中心的应用程序。

* Java 2平台标准版：
这是Java平台的版本，支持跨平台通用应用程序的开发，部署和管理。

* Java 2平台微型版：
Java平台的版本，针对小型，独立或可连接的消费类和嵌入式设备，以实现应用程序的开发，部署和管理，这些应用程序可以从智能卡通过移动设备和机顶盒扩展到常规计算设备。

Java（TM）远程方法调用（RMI）
从Java™程序到Java程序的分布式对象模型，其中可以用Java编程语言编写的远程对象的方法可以从其他Java虚拟机* （可能在不同的主机上）调用。

Java开发工具包（JDK™）的子集，供仅希望重新分发运行时环境的最终用户和开发人员使用。Java运行时环境由Java虚拟机* ，Java核心类和支持文件组成。
