  • Linux上的CD问题：

    某些Linux系统使用文件系统自动挂载CD，该文件系统会强制所有文件名都使用小写。这将破坏浏览器到具有大小写混合名称的HTML文件的链接，也将阻止示例的编译。

    如果您有此问题，请尝试使用UDF文件系统手动安装CD。去做这个：

    • 成为超级用户。
    • 卸载CD：
      umount cd_device_path
    • 使用UDF文件系统重新安装CD：
      mount -t udf cd_device_path cd_mount_point
