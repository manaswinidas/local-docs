2014年3月18日-

本教程的该版本对应于JDK 8版本。有关此版本中新功能的更多信息，请参见JDK 8的新增功能，其中包括：

  • Lambda表达式使您可以将功能视为方法参数，或将代码视为数据。Lambda表达式使您可以更紧凑地表示单方法接口（称为功能接口）的实例。

  • 默认方法是接口中具有实现的方法。它们使新功能可以添加到库的接口中，并确保与为这些接口的较旧版本编写的代码二进制兼容。

  • 聚合操作使您可以在元素流上执行功能样式的操作，尤其是对集合的批量操作，例如顺序或并行map-reduce转换。

  • 使用日期时间 API，您可以表示日期和时间并处理日期和时间值。它们支持国际标准化组织（ISO）日历系统以及其他常用的全球日历。

  • 类型注释可以与可插拔类型系统一起使用，以改进类型检查， 重复注释可以将相同注释应用于声明或类型使用。

除了修正错误和错误外，此更新还包括以下内容：

  • 安全跟踪已进行了重组；在Java Applets课程中查找有关applet和Java Web Start应用程序的安全性信息。

  • 的Permissions无论安全级别设置如何，都需要JAR文件清单属性。请参阅使用清单属性增强安全性 。

  • JDBC基础课程已更新，因此与Java DB 10.10（JDK 8当前支持的最新版本）兼容。这包括添加DERBY是有效的选项PARAMETER STYLE用于将参数传递到存储过程的过程元素。请参阅使用SQL脚本或JDBC API在Java DB中创建存储过程 。
