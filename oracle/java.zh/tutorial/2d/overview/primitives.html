几何图元

Java 2D API提供了一组有用的标准形状，例如点，线，矩形，圆弧，椭圆和曲线。定义常见几何图元的最重要包是java.awt.geom包。任意形状可以由直的几何图元的组合表示。

的Shape界面表示具有轮廓和内部的几何形状。此接口提供了一组用于描述和检查二维几何对象的通用方法，并支持曲线段和多个子形状。的Graphics类仅支持直线段。的Shape界面可以支持曲线段。

有关如何绘制和填充形状的更多详细信息，请参见“ 使用几何”课程。

点数

的Point2D类定义一个表示（x，y）坐标空间中位置的点。Java 2D API中的术语“点”与像素不同。点没有面积，不包含颜色且无法渲染。

点用于创建其他形状。的Point2D该类还包括一种用于计算两个点之间的距离的方法。

线数

的Line2D class是代表一行的抽象类。一条线的坐标可以检索为双精度。的Line2D该类包括几种设置线端点的方法。

矩形形状

的Rectangle2D ， RoundRectangle2D ， Arc2D和Ellipse2D原语全部来自RectangularShape类。此类定义用于Shape可以由矩形边界框描述的对象。一个的几何RectangularShape可以从完全包围对象轮廓的矩形中推断对象Shape 。

任意形状

的GeneralPath类使您可以通过沿形状边界指定一系列位置来构造任意形状。这些位置可以通过线段，二次曲线或三次方（贝塞尔曲线）连接。可以使用三个线段和三次曲线来创建以下形状。有关此形状的实现的更多信息，请参见抚摸和填充图形基元 。

地区

随着Area类中，您可以对任意两个执行布尔运算，例如并集，交集和减法Shape对象。这种技术通常称为构造区域几何 ，可让您快速创建复杂的Shape对象，而不必描述每个线段或曲线。
