描边和填充图形基元

您已经知道如何创建不同的几何图元和更复杂的形状。本课教如何在图形中添加一些颜色和花式轮廓，并表示填充和抚摸：

  • 填充 –用单色，颜色渐变或纹理图案对形状的内部进行绘画的过程
  • 描边 –是应用描边宽度，线条样式和颜色属性绘制形状轮廓的过程

要将花式线型和填充图案应用于几何图元，请更改Graphics2D渲染之前的上下文。例如，通过创建一个适当的点画一条虚线Stroke宾语。将此笔划添加到Graphics2D渲染线之前的上下文调用setStroke方法。同样，您将渐变填充应用于Shape通过创建一个对象GradientPaint对象并将其添加到Graphics2D上下文。

以下代码行通过填充和描边上下文丰富了几何图元：

// draw RoundRectangle2D.Double

final static float dash1[] = {10.0f};
    final static BasicStroke dashed =
        new BasicStroke(1.0f,
                        BasicStroke.CAP_BUTT,
                        BasicStroke.JOIN_MITER,
                        10.0f, dash1, 0.0f);
g2.setStroke(dashed);
g2.draw(new RoundRectangle2D.Double(x, y,
                                   rectWidth,
                                   rectHeight,
                                   10, 10));


// fill Ellipse2D.Double
redtowhite = new GradientPaint(0,0,color.RED,100, 0,color.WHITE);
g2.setPaint(redtowhite);
g2.fill (new Ellipse2D.Double(0, 0, 100, 50));

定义花式线型和填充图案

使用Java 2D Stroke和Paint类中，您可以定义花式线条样式和填充图案。

线型

线条样式是由Graphics2D渲染上下文。要设置笔触属性，请创建一个BasicStroke对象并将其传递给Graphics2DsetStroke方法。

一种BasicStroke对象包含有关线宽，连接样式，端盖样式和破折号样式的信息。在以下情况下使用此信息： Shape用draw方法。

线宽是垂直于其轨迹测量的线的粗细。线宽指定为float以用户坐标单位表示的值，大约等于使用默认转换时的1/72英寸。

连接样式是在两个线段相交处应用的装饰。 BasicStroke支持以下三种联接样式：

JOIN_BEVEL

JOIN_MITER

JOIN_ROUND

端盖样式是在线段结束处应用的装饰。 BasicStroke支持以下三种端盖样式：

CAP_BUTT

CAP_ROUND

CAP_SQUARE

破折号样式定义了沿着线的长度应用的不透明和透明部分的图案。破折号样式由破折号数组和破折号阶段定义。破折号阵列定义了破折号图案。数组中的交替元素表示虚线长度和以用户坐标为单位的虚线之间的间距的长度。元素0代表第一个破折号，元素1代表第一个空格，依此类推。虚线阶段是虚线样式的偏移量，也以用户坐标单位指定。虚线阶段指示虚线图案的哪一部分应用于行的开头。

填充图案是由Graphics2D渲染上下文。要设置paint属性，请创建一个对象实例，以实现Paint接口并将其传递到Graphics2DsetPaint方法。
