绘制几何图元

Java 2D API提供了几个类，这些类定义了常见的几何对象，例如点，线，曲线和矩形。这些几何类别是java.awt.geom包。
的PathIterator接口定义用于从路径检索元素的方法。
的Shape界面提供了一组用于描述和检查几何路径对象的方法。该接口由GeneralPath类和其他几何类。

本节中表示的所有示例均使用java.awt.geom打包，然后使用Graphics2D类。首先，您需要获得Graphics2D对象，例如通过投射Graphics的参数paint()方法。

public void paint (Graphics g) {
    Graphics2D g2 = (Graphics2D) g;
    ...
}

点

的Point类创建一个表示（x，y） 坐标空间中位置的点。子类Point2D.Float和Point2D.Double提供相应的float和double精度以存储点的坐标。

//Create Point2D.Double
Point2D.Double point = new Point2D.Double(x, y);

要创建坐标为0,0的点，请使用默认构造函数， Point2D.Double() 。
您可以使用setLocation设置点位置的方法如下：

  • setLocation(double x, double y) –将点定义坐标的位置设置为双精度值。
  • setLocation(Point2D p) –使用另一个点的坐标设置点的位置。

另外， Point2D类提供了一些方法来计算当前点和具有给定坐标的点之间的距离，或者两个点之间的距离。

线

的Line2D class表示（x，y）坐标空间中的线段。的Line2D. Float和Line2D.Double子类以浮点和双精度指定行。例如：

// draw Line2D.Double
g2.draw(new Line2D.Double(x1, y1, x2, y2));

本课程包括setLine()定义线端点的方法。
或者，可以通过使用构造函数来指定行的端点Line2D.Float类如下：

  • Line2D.Float(float X1, float Y1, float X2, float Y2)
  • Line2D.Float(Point2D p1, Point2D p2)

在中使用Stroke对象Graphics2D类，用于定义直线路径的笔触。

曲线

的java.awt.geom包使您可以创建二次曲线或三次曲线段。

二次曲线段

的QuadCurve2D类实现Shape接口。此类表示（x，y）坐标空间中的二次参数曲线段。的QuadCurve2D.Float和QuadCurve2D.Double子类指定浮点和双精度的二次曲线。

一些setCurve方法用于指定曲线的两个端点和一个控制点，可以通过其他点的坐标以及使用给定的数组直接定义其坐标。
一个非常有用的方法setCurve(QuadCurve2D) ，用与提供的曲线相同的端点和控制点设置二次曲线。例如：

// create new QuadCurve2D.Float
QuadCurve2D q = new QuadCurve2D.Float();
// draw QuadCurve2D.Float with set coordinates
q.setCurve(x1, y1, ctrlx, ctrly, x2, y2);
g2.draw(q);

三次曲线段

的CubicCurve2D该类还实现了Shape接口。此类表示（x，y）坐标空间中的三次参数曲线段。 CubicCurve2D.Float和CubicCurve2D.Double子类指定浮点和双精度的三次曲线。

的CubicCurve2D类具有将曲线设置为QuadraticCurve2D类，除了第二个控制点。例如：

// create new CubicCurve2D.Double
CubicCurve2D c = new CubicCurve2D.Double();
// draw CubicCurve2D.Double with set coordinates
c.setCurve(x1, y1, ctrlx1,
           ctrly1, ctrlx2, ctrly2, x2, y2);
g2.draw(c);

长方形

指定以下示例中表示的图元的类扩展了RectangularShape类，它实现了Shape界面并添加了一些自己的方法。

这些方法使您可以获取有关形状的位置和大小的信息，可以检查矩形的中心点并设置形状的边界。

的Rectangle2D类表示由位置（x，y）和尺寸（wxh）定义的矩形。的Rectangle2D.Float和Rectangle2D.Double子类以float和double精度指定一个矩形。例如：

// draw Rectangle2D.Double
g2.draw(new Rectangle2D.Double(x, y,
                               rectwidth,
                               rectheight));

的RoundRectangle2D class表示具有圆角的矩形，该矩形由位置（x，y），尺寸（wxh）以及拐角弧的宽度和高度定义。的RoundRectangle2D.Float和RoundRectangle2D.Double子类以float和double精度指定一个圆形矩形。

圆角矩形使用以下参数指定：

  • 地点
  • 宽度
  • 高度
  • 角弧宽度
  • 角弧高度

设置零件的位置，大小和圆弧RoundRectangle2D对象，使用方法setRoundRect(double a, double y, double w, double h, double arcWidth, double arcHeight) 。例如：

// draw RoundRectangle2D.Double
g2.draw(new RoundRectangle2D.Double(x, y,
                                   rectwidth,
                                   rectheight,
                                   10, 10));

椭圆

的Ellipse2D类表示由边界矩形定义的椭圆。的Ellipse2D.Float和Ellipse2D.Double子类以float和double精度指定椭圆。

椭圆完全由位置，宽度和高度定义。例如：

// draw Ellipse2D.Double
g2.draw(new Ellipse2D.Double(x, y,
                             rectwidth,
                             rectheight));

弧

要绘制一个椭圆，请使用Arc2D类。此类表示由边界矩形，起始角度，角度范围和闭合类型定义的弧。的Arc2D.Float和Arc2D.Double子类以float和double精度指定椭圆。

的Arc2D类定义了以下三种类型的弧，由此类中的相应常数表示：OPEN，PIE和CHORD。

几种方法可以设置圆弧的大小和参数：

  • 直接通过坐标
  • 由提供Point2D和Dimension2D
  • 通过复制现有的Arc2D

另外，您可以使用setArcByCenter从圆心指定圆弧的方法，该圆弧由其坐标和半径给出。

// draw Arc2D.Double
g2.draw(new Arc2D.Double(x, y,
                         rectwidth,
                         rectheight,
                         90, 135,
                         Arc2D.OPEN));

的ShapesDemo2D.java代码示例包含所有描述的几何图元的实现。有关本节中表示的类和方法的更多信息，请参见java.awt.geom规格。
