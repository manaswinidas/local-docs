裁剪绘图区域

任何Shape object可用作剪切路径，用于限制将要渲染的绘图区域的一部分。剪切路径是Graphics2D上下文设置clip属性，您调用Graphics2D.setClip并传递Shape定义您要使用的剪切路径。您可以通过调用clip方法并传入另一个Shape ;片段设置为当前片段和指定片段的交点Shape 。

示例：ClipImage

本示例为剪切路径设置了动画，以显示图像的不同部分。

注意：如果看不到小程序正在运行，则至少需要安装Java SE Development Kit（JDK）7版本。

ClipImage.java包含此applet的完整代码。小程序要求clouds.jpg图像文件。

剪切路径由椭圆和矩形的交点定义，椭圆和矩形的尺寸是随机设置的。椭圆传递给setClip方法，然后clip调用以将剪切路径设置为椭圆和矩形的交点。

private Ellipse2D ellipse = new Ellipse2D.Float();
private Rectangle2D rect = new Rectangle2D.Float();
...
ellipse.setFrame(x, y, ew, eh);
g2.setClip(ellipse);
rect.setRect(x+5, y+5, ew-10, eh-10);
g2.clip(rect);

示例：星空

也可以从文本字符串创建剪切区域。以下示例创建了一个TextLayout与字符串“星夜” 。然后，它得到了TextLayout 。的TextLayout.getOutline方法返回一个Shape对象和一个Rectangle从此边界创建Shape宾语。边界包含布局可以绘制的所有像素。图形上下文中的颜色设置为蓝色，并绘制轮廓形状，如以下图像和代码片段所示。

FontRenderContext frc = g2.getFontRenderContext();
Font f = new Font("Helvetica", 1, w/10);
String s = new String("The Starry Night");
TextLayout textTl = new TextLayout(s, f, frc);
AffineTransform transform = new AffineTransform();
Shape outline = textTl.getOutline(null);
Rectangle r = outline.getBounds();
transform = g2.getTransform();
transform.translate(w/2-(r.width/2), h/2+(r.height/2));
g2.transform(transform);
g2.setColor(Color.blue);
g2.draw(outline);

接下来，使用Shape从创建的对象getOutline 。的starry.gif该图片是梵高（Van Gogh）的著名绘画《星夜》（Starry Night） ，从该图片的左下角开始绘制到该剪贴区域中Rectangle宾语。

g2.setClip(outline);
g2.drawImage(img, r.x, r.y, r.width, r.height, this);

注意：如果看不到小程序正在运行，则至少需要安装Java SE Development Kit（JDK）7版本。

Starry.java包含此程序的完整代码。此小程序需要Starry.gif图像文件。
