转换形状，文本和图像

您可以在Graphics2D渲染时移动，旋转，缩放和剪切图形图元的上下文。转换属性由的实例定义AffineTransform类。仿射变换是诸如平移，旋转，缩放或剪切之类的变换，其中，即使在变换之后，平行线仍保持平行。

的Graphics2D类提供了几种更改transform属性的方法。您可以构造一个新的AffineTransform并更改Graphics2D通过调用转换属性transform 。

AffineTransform定义以下工厂方法以使构造新转换更加容易：

  • getRotateInstance
  • getScaleInstance
  • getShearInstance
  • getTranslateInstance

或者，您可以使用Graphics2D修改当前变换的变换方法。当您调用以下便捷方法之一时，生成的变换将与当前变换连接在一起，并在渲染期间应用：

  • rotate —以弧度指定旋转角度
  • scale —在x和y方向上指定比例因子
  • shear —在x和y方向上指定剪切因子
  • translate —在x和y方向上指定平移偏移量

您还可以构造一个AffineTransform直接调用对象，并通过调用transform方法。

的drawImage方法也被重载以允许您指定一个AffineTransform应用于渲染时的图像。调用时指定转换drawImage不影响Graphics2D变换属性。

从“变换”菜单中选择一个变换后，该变换将串联到AffineTransformat ：

public void setTrans(int transIndex) {
    // Sets the AffineTransform.
    switch ( transIndex ) {
    case 0 :
        at.setToIdentity();
        at.translate(w/2, h/2);
        break;
    case 1 :
        at.rotate(Math.toRadians(45));
        break;
    case 2 :
        at.scale(0.5, 0.5);
        break;
    case 3 :
        at.shear(0.5, 0.0);
        break;
    }
}

在显示与菜单选项相对应的形状之前，应用程序首先从菜单中检索当前变换。 Graphics2D宾语：

AffineTransform saveXform = g2.getTransform();

此转换将恢复为Graphics2D渲染后。

检索当前转换后，另一个AffineTransform ， toCenterAt将创建，使形状呈现在面板的中心。的atAffineTransform串联到toCenterAt ：

AffineTransform toCenterAt = new AffineTransform();
toCenterAt.concatenate(at);
toCenterAt.translate(-(r.width/2), -(r.height/2));

的toCenterAt转换串联到Graphics2D用transform方法：

g2.transform(toCenterAt);

渲染完成后，将使用setTransform方法：

g2.setTransform(saveXform);
注意：切勿使用setTransform将坐标变换连接到现有变换的方法。的setTransform方法将覆盖Graphics2D对象的当前转换，可能由于其他原因（例如在窗口中放置Swing和轻型组件）可能需要此转换。使用以下步骤执行转换：
  1. 使用getTransform当前转换的方法。
  2. 使用transform ， translate ， scale ， shear ， 要么rotate连接转换。
  3. 执行渲染。
  4. 使用还原原始转换setTransform方法。
