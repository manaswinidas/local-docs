字体概念

本节向您介绍Font类，它支持详细字体信息的规范以及复杂印刷功能的使用。

一种Font object表示系统上可用字体集中的字体实例。常见字体的示例包括Helvetica Bold和Courier Bold Italic。三个名称与Font对象：其逻辑名称，姓氏和字体名称：

  • 一种Font对象的逻辑名是映射到物理字体的名称，该物理字体是系统上可用的特定字体之一。指定一个Font在Java中，使用字体名称而不是逻辑名称。您可以从Font通过调用getName方法。要获取映射到系统上可用的特定字体的逻辑名称的列表，请调用java.awt。GraphicsEnvironment.getAvailableFontFamilyNames方法。

    有关更多信息，请参见物理和逻辑字体 。

  • 一种Font object的家族名称是字体家族的名称，该字体家族确定了多个面孔（例如Helvetica）上的印刷设计。通过检索姓氏getFamily方法。

  • 一种Font对象的字体名称是指系统上安装的实际字体。这是指定字体时应使用的名称。它通常仅称为字体名称 。通过调用检索字体名称getFontName 。要确定系统上可用的字体，请致电java.awt.GraphicsEnvironment.getAllFonts方法。

您可以访问有关Font通过getAttributes方法。一种Font对象的属性包括其名称，大小，变换和字体特征（例如权重和姿势）。

一种LineMetrics对象封装了与Font ，例如上升，下降和领先：

  • 上升是从基线到上升线的距离。该距离代表大写字母的典型高度，但是某些字符可能会在上升行上方延伸。
  • 下降是从基线到下降线的距离。大多数字符的最低点将位于下降范围内，但某些字符可能会在下降线下方延伸。
  • 领先是从下降线底部到下一行顶部的推荐距离。
