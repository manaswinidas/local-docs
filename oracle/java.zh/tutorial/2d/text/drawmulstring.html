绘制多行文本

如果您有一段样式的文本想要适合特定的宽度，则可以使用LineBreakMeasurer类。此类使样式化的文本可以分成几行，以使它们适合特定的视觉效果。每行都以TextLayout对象，表示不可更改的样式字符数据。但是，此类也允许访问布局信息。的getAscent和getDescent方法TextLayout返回有关用于在组件中放置线条的字体的信息。文本存储为AttributedCharacterIterator对象，以便字体和磅值属性可以与文本一起存储。

以下小程序使用以下命令在组件中放置样式文本的段落LineBreakMeasurer ， TextLayout和AttributedCharacterIterator 。

注意：如果看不到小程序正在运行，则至少需要安装Java SE Development Kit（JDK）7版本。

此小程序的完整代码在LineBreakSample.java 。

以下代码使用字符串创建一个迭代器vanGogh 。检索迭代器的开始和结束，并获取一个新的LineBreakMeasurer是从迭代器创建的。

    AttributedCharacterIterator paragraph = vanGogh.getIterator();
    paragraphStart = paragraph.getBeginIndex();
    paragraphEnd = paragraph.getEndIndex();
    FontRenderContext frc = g2d.getFontRenderContext();
    lineMeasurer = new LineBreakMeasurer(paragraph, frc);

窗口的大小用于确定该行应在何处中断。也TextLayout为段落中的每一行创建一个对象。

// Set break width to width of Component.
float breakWidth = (float)getSize().width;
float drawPosY = 0;
// Set position to the index of the first
// character in the paragraph.
lineMeasurer.setPosition(paragraphStart);

// Get lines from until the entire paragraph
// has been displayed.
while (lineMeasurer.getPosition() < paragraphEnd) {

    TextLayout layout = lineMeasurer.nextLayout(breakWidth);

    // Compute pen x position. If the paragraph
    // is right-to-left we will align the
    // TextLayouts to the right edge of the panel.
    float drawPosX = layout.isLeftToRight()
        ? 0 : breakWidth - layout.getAdvance();

    // Move y-coordinate by the ascent of the
    // layout.
    drawPosY += layout.getAscent();

    // Draw the TextLayout at (drawPosX,drawPosY).
    layout.draw(g2d, drawPosX, drawPosY);

    // Move y-coordinate in preparation for next
    // layout.
    drawPosY += layout.getDescent() + layout.getLeading();
}

的TextLayout类不是经常由应用程序直接创建的。但是，当应用程序需要直接处理在样式中的特定位置应用了样式（文本属性）的文本时，此类非常有用。例如，要绘制一个以斜体显示的单词，应用程序将需要执行测量并为每个子字符串设置字体。如果文本是双向的，则正确执行此任务并不容易。创建一个TextLayout来自的对象AttributedString对象为您处理此问题。有关更多信息，请查阅Java SE规范。 TextLayout 。
