双端队列接口

通常发音为deck ，双端队列是双端队列。双端队列是元素的线性集合，支持在两个端点处元素的插入和移除。的Deque接口是比两者都丰富的抽象数据类型Stack和Queue因为它同时实现了堆栈和队列。的Deque接口，定义访问元素两端的元素的方法Deque实例。提供了用于插入，删除和检查元素的方法。预定义的类，例如ArrayDeque和LinkedList实施Deque接口。

请注意Deque接口既可以用作后入先出堆栈，又可以用作先进先出队列。给出的方法Deque界面分为三个部分：

插入

的addfirst和offerFirst方法在开始时插入元素Deque实例。方法addLast和offerLast在末尾插入元素Deque实例。当容量Deque实例受到限制，首选方法是offerFirst和offerLast因为addFirst如果已满，可能无法引发异常。

去掉

的removeFirst和pollFirst方法从的开头删除元素Deque实例。的removeLast和pollLast方法从最后删除元素。方法pollFirst和pollLast返回null如果Deque为空，而方法removeFirst和removeLast如果Deque实例为空。

取回

方法getFirst和peekFirst检索的第一个元素Deque实例。这些方法不会从Deque实例。同样，方法getLast和peekLast检索最后一个元素。方法getFirst和getLast如果deque实例为空，而方法peekFirst和peekLast返回NULL 。

除了这些基本方法以外，您还可以插入，删除和检查Deque例如， Deque接口还具有一些预定义的方法。其中之一是removeFirstOccurence ，如果指定元素存在于指定元素中，则此方法将其删除Deque实例。如果元素不存在，则Deque实例保持不变。另一种类似的方法是removeLastOccurence ;此方法删除指定元素最后出现的位置Deque实例。这些方法的返回类型为boolean ，他们回来了true如果元素存在于Deque实例。
