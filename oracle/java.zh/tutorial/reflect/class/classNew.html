检索类对象

所有反射操作的入口点是java.lang.Class 。除了java.lang.reflect.ReflectPermission ，没有任何课程java.lang.reflect有公共构造函数。要进入这些类，必须在上调用适当的方法Class 。有几种方法可以获取Class取决于代码是否可以访问对象，类的名称，类型或现有的对象Class 。

Object.getClass（）

如果对象的实例可用，则获取其实例的最简单方法Class是要调用Object.getClass() 。当然，这仅适用于全部继承自的引用类型Object 。以下是一些示例。

Class c = "foo".getClass();

返回Class对于String

Class c = System.console().getClass();

有一个与虚拟机关联的唯一控制台，由虚拟机返回。 static方法System.console() 。传回的值getClass()是个Class对应于java.io.Console 。

enum E { A, B }
Class c = A.getClass();

A是枚举的一个实例E ;从而getClass()返回Class对应于枚举类型E 。

byte[] bytes = new byte[1024];
Class c = bytes.getClass();

由于数组是Objects ，也可以调用getClass()在数组的实例上。返回的Class对应于具有组件类型的数组byte 。

import java.util.HashSet;
import java.util.Set;

Set<String> s = new HashSet<String>();
Class c = s.getClass();

在这种情况下， java.util.Set是类型对象的接口java.util.HashSet 。传回的值getClass()是对应于的类别java.util.HashSet 。

.class语法

如果类型可用，但没有实例，则可以获取Class通过附加".class"类型的名称。这也是获得Class用于原始类型。

boolean b;
Class c = b.getClass();   // compile-time error

Class c = boolean.class;  // correct

注意声明boolean.getClass()会产生编译时错误，因为boolean是基本类型，无法取消引用。的.class语法返回Class对应的类型boolean 。

Class c = java.io.PrintStream.class;

变量c将是Class对应的类型java.io.PrintStream 。

Class c = int[][][].class;

的.class语法可用于检索Class对应于给定类型的多维数组。

Class.forName（）

如果一个类的全名可用，则有可能获得相应的名称。 Class使用静态方法Class.forName() 。这不能用于基本类型。数组类名称的语法由Class.getName() 。此语法适用于引用和原始类型。

Class c = Class.forName("com.duke.MyLocaleServiceProvider");

该语句将从给定的完全限定名称创建一个类。

Class cDoubleArray = Class.forName("[D");

Class cStringArray = Class.forName("[[Ljava.lang.String;");

变量cDoubleArray将包含Class对应于原始类型的数组double （即与double[].class ）。的cStringArray变量将包含Class对应于的二维数组String （即与String[][].class ）。

原始类型包装程序的TYPE字段

的.class语法是一种更方便且首选的获取方式Class对于原始类型；但是还有另一种方式来获取Class 。每个原始类型和void在具有包装器类java.lang用于将原始类型装箱到引用类型。每个包装器类都包含一个名为TYPE等于Class对于要包装的原始类型。

Class c = Double.TYPE;

有一个班java.lang.Double用于包装原始类型double每当Object是必须的。的价值Double.TYPE与...相同double.class 。

Class c = Void.TYPE;

Void.TYPE等同于void.class 。

返回类的方法

有几种反射API可以返回类，但是只有在以下情况下才可以访问它们： Class已经直接或间接获得。

Class.getSuperclass()
返回给定类的超类。 
Class c = javax.swing.JButton.class.getSuperclass();
的超一流javax.swing.JButton是javax.swing.AbstractButton 。
Class.getClasses()
返回属于该类成员的所有公共类，接口和枚举，包括继承的成员。 
Class<?>[] c = Character.class.getClasses();
Character包含两个成员类Character.Subset和Character.UnicodeBlock 。
Class.getDeclaredClasses()
返回所有类接口，以及在该类中显式声明的枚举。 
Class<?>[] c = Character.class.getDeclaredClasses();
Character包含两个公共成员类Character.Subset和Character.UnicodeBlock和一个私人班级Character.CharacterCache 。
Class.getDeclaringClass()
java.lang.reflect.Field.getDeclaringClass()
java.lang.reflect.Method.getDeclaringClass()
java.lang.reflect.Constructor.getDeclaringClass()
返回Class在其中宣布了这些成员。 匿名类声明将没有声明类，但将具有封闭类。 
import java.lang.reflect.Field;

Field f = System.class.getField("out");
Class c = f.getDeclaringClass();
场out在中声明System 。 
public class MyClass {
    static Object o = new Object() {
        public void m() {} 
    };
    static Class<c> = o.getClass().getEnclosingClass();
}
由定义的匿名类的声明类o是null 。
Class.getEnclosingClass()
返回该类的直接封闭类。 
Class c = Thread.State.class().getEnclosingClass();
枚举的封闭类Thread.State是Thread 。 
public class MyClass {
    static Object o = new Object() { 
        public void m() {} 
    };
    static Class<c> = o.getClass().getEnclosingClass();
}
定义的匿名类o被封闭MyClass 。
