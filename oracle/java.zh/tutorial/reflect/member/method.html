方法

一种方法包含可以被调用的可执行代码。方法是继承的，并且在非反射代码行为中，编译器会强制执行诸如重载，重写和隐藏之类的行为。相反，反射代码使方法选择可以限于特定类而不考虑其超类。可以访问超类方法，但是可以确定它们的声明类。没有反射就不可能通过程序发现，这是许多细微错误的来源。

的java.lang.reflect.Method类提供API，以访问有关方法的修饰符，返回类型，参数，注释和引发的异常的信息。它也可用于调用方法。以下各节介绍了这些主题：

  • 获取方法类型信息显示了如何枚举在类中声明的方法并获取类型信息
  • 获取方法参数的名称显示了如何检索方法或构造函数的参数的名称和其他信息
  • 检索和解析方法修饰符描述了如何访问和解码修饰符以及与方法相关的其他信息
  • 调用方法说明了如何执行方法并获得其返回值
  • 故障排除涵盖了查找或调用方法时遇到的常见错误
