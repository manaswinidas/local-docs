创建一个URL

创建一个最简单的方法URL对象来自String代表人类可读的网址形式。这通常是其他人用于URL的形式。在Java程序中，您可以使用String包含此文本以创建一个URL宾语：

URL myURL = new URL("http://example.com/");

的URL上面创建的对象表示绝对URL 。绝对URL包含访问相关资源所需的所有信息。您也可以创建URL来自相对URL地址的对象。

创建一个相对于另一个的URL

相对URL仅包含足够的信息以到达相对于另一个URL（或在另一个URL中）的资源。

相对URL规范通常在HTML文件中使用。例如，假设您编写了一个名为JoesHomePage.html 。在此页面中，是指向其他页面的链接， PicturesOfMe.html和MyKids.html ，它们位于同一台计算机上，并且位于同一目录中JoesHomePage.html 。链接到PicturesOfMe.html和MyKids.html从JoesHomePage.html可以指定为文件名，如下所示：

<a href="PicturesOfMe.html">Pictures of Me</a>
<a href="MyKids.html">Pictures of My Kids</a>

这些URL地址是相对URL 。也就是说，URL是相对于其中包含的文件指定的- JoesHomePage.html 。

在Java程序中，您可以创建一个URL相对网址规范中的对象。例如，假设您在站点上知道两个URL example.com ：

http://example.com/pages/page1.html
http://example.com/pages/page2.html

您可以建立URL这些页面相对于它们的公共基本URL的对象： http://example.com/pages/像这样：

URL myURL = new URL("http://example.com/pages/");
URL page1URL = new URL(myURL, "page1.html");
URL page2URL = new URL(myURL, "page2.html");

此代码段使用URL构造函数，可让您创建一个URL来自另一个的对象URL对象（基础）和相对URL规范。该构造函数的一般形式为：

URL(URL baseURL, String relativeURL)

第一个参数是URL指定新对象基础的对象URL 。第二个参数是String指定相对于基础的其余资源名称。如果baseURL为null，则此构造函数将relativeURL像绝对URL规范。相反，如果relativeURL是绝对URL规范，则构造函数将忽略baseURL 。

此构造函数对于创建URL文件中命名锚（也称为引用）的对象。例如，假设page1.html文件具有名为的锚BOTTOM在文件底部。您可以使用相对网址构造函数创建一个URL像这样的对象：

URL page1BottomURL = new URL(page1URL,"#BOTTOM");

其他URL构造函数

的URL类提供了两个额外的构造函数来创建一个URL宾语。当您使用URL（例如HTTP URL）等URL的主机名，文件名，端口号和引用组件时，这些构造函数非常有用。当您没有包含完整URL规范的String，但是您确实知道URL的各种组成部分时，这两个构造函数很有用。

例如，假设您设计一个类似于文件浏览面板的网络浏览面板，该面板允许用户选择协议，主机名，端口号和文件名。您可以构造一个URL从面板的组件。第一个构造函数创建一个URL协议，主机名和文件名中的对象。以下代码段创建了一个URL到page1.html档案在example.com现场：

new URL("http", "example.com", "/pages/page1.html");

这相当于

new URL("http://example.com/pages/page1.html");

第一个参数是协议，第二个参数是主机名，最后一个参数是文件的路径名。请注意，文件名开头包含一个正斜杠。这表明文件名是从主机的根目录指定的。

决赛URL构造函数将端口号添加到先前构造函数使用的参数列表中：

URL gamelan = new URL("http", "example.com", 80, "pages/page1.html");

这创建了一个URL以下URL的对象：

http://example.com:80/pages/page1.html

如果构造一个URL使用这些构造函数之一的对象，您可以获得一个String通过使用包含完整的URL地址URL对象的toString方法或等效方法toExternalForm方法。

具有特殊字符的URL地址

某些URL地址包含特殊字符，例如空格字符。像这样：

http://example.com/hello world/

为了使这些字符合法，需要在将它们传递给URL构造函数之前对其进行编码。

URL url = new URL("http://example.com/hello%20world");

在此示例中，对特殊字符进行编码很容易，因为只有一个字符需要编码，但是对于具有多个字符的URL地址，或者如果不确定在编写代码时需要访问哪些URL地址，您可以使用的多参数构造函数java.net.URI类自动为您处理编码。

URI uri = new URI("http", "example.com", "/hello world/", "");

然后将URI转换为URL。

URL url = uri.toURL();

MalformedURLException

四个URL构造函数抛出MalformedURLException如果构造函数的参数引用了null或未知协议。通常，您希望通过将URL构造函数语句嵌入到try / catch对，像这样：

try {
    URL myURL = new URL(...);
} 
catch (MalformedURLException e) {
    // exception handler code here
    // ...
}

有关处理异常的信息，请参见异常 。

注意： 

URL s是“一次写入”对象。创建完一个URL对象，您不能更改其任何属性（协议，主机名，文件名或端口号）。
