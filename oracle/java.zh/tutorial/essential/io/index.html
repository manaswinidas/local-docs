课程：基本I / O

本课涵盖用于基本I / O的Java平台类。首先关注I / O流 ，这是一个功能强大的概念，可大大简化I / O操作。该课程还讨论了序列化，序列化使程序可以将整个对象写到流中，然后再次读取它们。然后，本课程着眼于文件I / O和文件系统操作，包括随机访问文件。

本课程涵盖的大多数课程I/O Streams部分在java.io包。本课程涵盖的大多数课程File I/O部分在java.nio.file包。

I / O流

  • 字节流处理原始二进制数据的I / O。
  • 字符流处理字符数据的I / O，自动处理与本地字符集之间的转换。
  • 缓冲流通过减少对本机API的调用次数来优化输入和输出。
  • 扫描和格式化允许程序读取和写入格式化的文本。
  • 命令行中的I / O描述了标准流和控制台对象。
  • 数据流处理原始数据类型的二进制I / O和String价值观。
  • 对象流处理对象的二进制I / O。

文件I / O（以NIO.2为特色）

  • 什么是路径？ 检查文件系统上路径的概念。
  • 路径类介绍了java.nio.file包。
  • 路径操作着眼于Path处理语法操作的类。
  • 文件操作介绍了许多文件I / O方法共有的概念。
  • 检查文件或目录显示了如何检查文件的存在及其可访问性级别。
  • 删除文件或目录 。
  • 复制文件或目录 。
  • 移动文件或目录 。
  • 管理元数据说明了如何读取和设置文件属性。
  • 读取，写入和创建文件显示了读取和写入文件的流和通道方法。
  • 随机访问文件显示了如何以非顺序方式读取或写入文件。
  • 创建和读取目录涵盖特定于目录的API，例如如何列出目录内容。
  • 链接（符号链接或其他链接）涵盖特定于符号链接和硬链接的问题。
  • 遍历文件树演示了如何递归访问文件树中的每个文件和目录。
  • 查找文件显示了如何使用模式匹配来搜索文件。
  • 监视目录中的更改显示了如何使用监视服务来检测在一个或多个目录中添加，删除或更新的文件。
  • 其他有用的方法涵盖了本课程中其他地方无法使用的重要API。
  • 旧版文件I / O代码显示了如何利用Path功能，如果您有使用java.io.File类。表映射java.io.File API java.nio.file提供了API。
