介绍

什么是正则表达式？

正则表达式是一种基于一组字符串中每个字符串共有的共同特征来描述一组字符串的方法。它们可用于搜索，编辑或处理文本和数据。您必须学习一种特定的语法来创建正则表达式-超出了Java编程语言的常规语法。正则表达式的复杂度各不相同，但是一旦您了解了它们的构造基础，便可以破译（或创建）任何正则表达式。

本教程讲解了支持的正则表达式语法java.util.regex API，并提供了几个工作示例来说明各种对象之间的交互方式。在正则表达式世界中，有很多不同的样式可供选择，例如grep，Perl，Tcl，Python，PHP和awk。中的正则表达式语法java.util.regex API与Perl中最相似。

此包中的正则表达式如何表示？

的java.util.regex软件包主要包括三个类： Pattern ， Matcher和PatternSyntaxException 。

  • 一种Pattern object是正则表达式的编译表示。的Pattern该类不提供公共构造函数。要创建模式，您必须首先调用其模式之一public static compile方法，然后将返回一个Pattern宾语。这些方法接受正则表达式作为第一个参数。本教程的前几课将教您所需的语法。
  • 一种Matcher object是解释模式并针对输入字符串执行匹配操作的引擎。如Pattern类， Matcher没有定义公共构造函数。您获得一个Matcher通过调用对象matcher上的方法Pattern宾语。
  • 一种PatternSyntaxException object是未经检查的异常，表示正则表达式模式中的语法错误。
