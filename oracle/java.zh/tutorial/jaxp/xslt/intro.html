引入XSL，XSLT和XPath

可扩展样式表语言（XSL）具有三个主要子组件：

XSL-FO

格式对象标准。迄今为止最大的子组件，该标准提供了用于描述字体大小，页面布局以及对象呈现的其他方面的机制。JAXP并未涵盖此子组件，本教程也未包含此子组件。

XSLT

这是转换语言，它使您可以定义从XML到其他格式的转换。例如，您可以使用XSLT生成HTML或其他XML结构。您甚至可以用它来生成纯文本或将信息放入其他文档格式中。（正如您将在从任意数据结构生成XML中看到的那样，一个聪明的应用程序也可以将其压入服务以操纵非XML数据）。
