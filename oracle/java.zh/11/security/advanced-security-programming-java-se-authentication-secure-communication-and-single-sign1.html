Java SE身份验证，安全通信和单点登录中的高级安全性编程

Java SE提供了丰富的API和功能集，用于开发安全的Java应用程序和服务。此处列出的练习将帮助您使用Java SE GSS API来构建对用户进行身份验证的应用程序，以与其他应用程序和服务进行安全通信，并帮助您在Kerberos环境中配置应用程序以实现单点登录。此外，您还将学习如何在Kerberos环境中使用更强大的加密算法，以及如何使用Java GSS机制（例如SPNEGO）来保护关联。

设置您的开发环境

在进行第一个练习之前，请按照以下步骤设置开发环境：

  1. 安装和设置Solaris机器（ 练习7：在Kerberos环境中配置为使用更强的加密算法来确保通信安全 ）。
  2. 使用练习使用的帐户在Solaris计算机上配置Kerberos服务器。请参阅附录A：设置Kerberos帐户 。
  3. 在Solaris计算机上设置密钥分发中心（KDC），然后启动Kerberos服务器。
  4. 在客户端计算机上设置Kerberos配置。
  5. 设置JDK环境：

    • 设置JAVA_HOME环境变量以指向JDK安装目录

    • 地点%JAVA_HOME%\bin （Windows）或$JAVA_HOME/bin （Solaris / Linux）中PATH环境变量。
