JSSE示例代码故障排除

人们在使用JSSE时遇到的最常见问题之一是，当JSSE收到进行信任决策的机制所未知的证书时。如果收到未知证书，则信任机制将引发异常，表明证书不受信任。确保使用正确的信任库，并且正确安装和配置了JSSE。

如果您使用的是“ localhost”或“ duke”凭据，请确保已正确指定了samplecacerts文件的位置，否则您的应用程序将无法运行。（有关更多信息，请参见示例信任库 。）

SSL调试机制可用于调查此类信任问题。有关此主题的更多信息，请参见实施文档。
