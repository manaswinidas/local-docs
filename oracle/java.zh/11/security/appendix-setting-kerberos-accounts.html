附录A：设置Kerberos帐户

Kerberos帐户在密钥分发中心（KDC）上设置。Kerberos数据库中的每个条目都包含一个Kerberos主体。您应该为将要运行服务器的计算机创建基于主机的主体（例如，“ host / j1hol-001”）和用于访问服务器的客户端主体（例如，“ test”）。

对于Solaris，请参考以下文档，了解如何设置Kerberos主体。

  • 如何在Windows Server 2003中创建Active Directory服务器

练习假定操作系统已配置为使用正确的Kerberos服务器。此配置通常需要管理特权。如果无法配置操作系统，则可以将Kerberos配置文件与java通过使用命令-Djava.security.krb5.conf选项。这是一个示例，说明如何从练习中调用其中一个命令来使用krb5.conf配置文件。 

% java -Djava.security.auth.login.config=jaas-krb5.conf\
  -Djava.security.krb5.conf=krb5.conf Jaas client
