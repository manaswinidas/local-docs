说明HTTPS连接的示例代码

有两种主要的API，用于通过JSSE访问安全通信。一种方法是通过套接字级别的API，该API可用于任意安全的通信，如SSLSocketClient.java ， SSLSocketClientWithTunneling.java和SSLSocketClientWithClientAuth.java示例（带有和不带有运行ClassFileServer中描述的示例）。

第二种方法（通常是更简单的方法）是通过标准Java URL API。您可以使用HTTPS URL协议或使用java.net的方案与启用SSL的Web服务器进行安全通信。URL类。

许多常见的浏览器都实现了对HTTPS URL方案的支持，从而无需访问JSSE随附的套接字级API即可访问安全的通信。一个示例URL是https://www.verisign.com 。

HTTPS URL实施的信任和密钥管理是特定于环境的。JSSE实现提供HTTPS URL实现。要使用其他HTTPS协议实现，请设置java.protocol.handler.pkgs系统属性；请参见如何指定java.lang。系统属性 。

运行URLReader

这个例子URLReader.java说明了使用URL访问安全站点上的资源。默认情况下，此示例连接到www.verisign.com ，但可以进行调整以连接到ClassFileServer.java 。为此，需要修改URL以指向正确的地址。如果服务器证书中的主机名与URL的主机名不匹配，则可能还需要更新服务器的证书或提供自定义HostNameVerifier （请参阅HttpsURLConnection ）。

注意：

如果您位于防火墙后面，则可能需要设置https.proxyHost和https.proxyPort系统属性以正确指定代理。

用法

java URLReader

URLReader.java

import java.net.*;
import java.io.*;

/*
 * This example illustrates using a URL to access resources
 * on a secure site.
 *
 * If you are running inside a firewall, please also set the following
 * Java system properties to the appropriate value:
 *
 *   https.proxyHost = <secure proxy server hostname>
 *   https.proxyPort = <secure proxy server port>
 *
 */

public class URLReader {
    public static void main(String[] args) throws Exception {
        URL verisign = new URL("https://www.verisign.com/");
        BufferedReader in = new BufferedReader(
                                new InputStreamReader(
                                verisign.openStream()));

        String inputLine;

        while ((inputLine = in.readLine()) != null)
            System.out.println(inputLine);

        in.close();
    }
}

运行URLReaderWithOptions

这个例子非常类似于URLReaader.java ，但是它使您可以通过主要方法参数而不是as来设置系统属性-D Java运行时环境的选项。

用法

java URLReaderWithOptions [-h proxyhost] [-p proxyport] [-k protocolhandlerpkgs] [-c ciphersarray]

  • proxyHost ：安全代理服务器主机名（ https.proxyHost ）
  • proxyPort ：安全代理服务器端口（ https.proxyPort ）
  • protocolhandlerpkgs ：用管道分隔（ | ）协议处理程序列表（ java.protocol.handler.pkgs ）
  • ciphersarray ：将密码套件作为逗号分隔的列表（ https.cipherSuites ）

注意：

多个协议处理程序可以包含在protocolhandlerpkgs参数为列表，其中各项之间用竖线分隔。多个SSL密码套件名称可以包含在ciphersarray参数为列表，其中各项之间用逗号分隔。可能的密码套件名称与密码套件名称返回的密码套件名称相同。 SSLSocket.getSupportedCipherSuites()方法。套件名称取自SSL和TLS协议规范。

你需要一个protocolhandlerpkgs仅当您要使用Oracle提供的默认HTTPS协议处理程序实现以外的其他参数时，才需要使用参数。

如果要在防火墙后面运行示例代码，则必须包括代理主机和代理端口的参数。此外，您可以包括要启用的密码套件的列表。

这是一个跑步的例子URLReaderWithOptions并在端口8080上指定代理主机“ webproxy”：

java URLReaderWithOptions -h webproxy -p 8080

URLReaderWithOptions.java

import java.net.*;
import java.io.*;

/*
 * Using a URL to access resources on a secure site.
 *
 * You can optionally set the following command line options:
 *
 *     -h <secure proxy server hostname>
 *     -p <secure proxy server port>
 *     -k <| separated list of protocol handlers>
 *     -c <enabled cipher suites as a comma separated list>
 *
 */

public class URLReaderWithOptions {
    public static void main(String[] args) throws Exception {

        System.out.println("USAGE: java URLReaderWithOptions " +
            "[-h proxyhost] [-p proxyport] [-k protocolhandlerpkgs] " +
            "[-c ciphersarray]");

        // initialize system properties
        char option = 'd';
        for (int i = 0; i < args.length; i++) {
            System.out.println(option+": "+args[i]);
            switch(option) {
            case 'h':
                System.setProperty("https.proxyHost", args[i]);
                option = 'd';
                break;
            case 'p':
                System.setProperty("https.proxyPort", args[i]);
                option = 'd';
                break;
            case 'k':
                System.setProperty("java.protocol.handler.pkgs", args[i]);
                option = 'd';
                break;
            case 'c':
                System.setProperty("https.cipherSuites", args[i]);
                option = 'd';
                break;
            default:
                // get the next option
                if (args[i].startsWith("-")) {
                    option = args[i].charAt(1);
                }
            }
        }

        URL verisign = new URL("https://www.verisign.com/");
        BufferedReader in = new BufferedReader(
                                new InputStreamReader(
                                verisign.openStream()));

        String inputLine;

        while ((inputLine = in.readLine()) != null)
            System.out.println(inputLine);

        in.close();
    }
}
