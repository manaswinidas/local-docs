该图由标记如下的框组成：

  • 密钥：包含一个标有“秘密密钥”的框
  • 纯文本
  • 这两个盒子彼此相连：
    • 密码（AES）：第1盒，共2盒
    • 加密
  • 算法参数
  • 密文
  • 这两个盒子彼此相连：
    • 密码（AES）：第2盒，共2盒
    • 解密
  • 纯文本

带标记的箭头将这些框连接起来：

  • update() ：从纯文本到密码（AES）＃1
  • doFinal() ：从纯文本到加密
  • update() ：从密文到密文（AES）＃2
  • doFinal() ：从密文到解密

未标记的箭头将这些框连接起来：

  • 从密钥到密码（AES）＃1和密码（AES）＃2（箭头分支）
  • 从密码（AES）＃1到算法参数
  • 从算法参数到解密框的密码（AES）隔离专区
  • 从密码（AES）＃2到纯文本
  • 从解密到纯文本
