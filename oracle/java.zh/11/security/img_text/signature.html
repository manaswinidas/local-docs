附图说明签名类

该图由标记如下的框组成：

  • 由密钥对生成器生成：包含一个标记为私钥/公钥的框
  • 数据
  • 这两个盒子彼此相连：
    • 签名（SHA256withRSA）：第1盒，共2盒
    • 标志
  • 签名字节
  • 这两个盒子彼此相连：
    • 签名（SHA256withRSA）：2之2
    • 校验
  • 是/否

带有标签的箭头连接以下框：

  • update() ：从数据到签名（SHA256withRSA）＃1
  • update() ：从数据到签名（SHA256withRSA）＃2
  • sign() ：从签名（SHA256withRSA）＃1到签名字节
  • verify() ：从签名字节数据到签名（SHA256withRSA）＃2

未标记的箭头将这些框连接起来：

  • 从由密钥对生成器生成到签名（SHA256withRSA）＃1和签名（SHA256withRSA）＃2（箭头分支）
  • 从签名（SHA256withRSA）＃2到是/否
