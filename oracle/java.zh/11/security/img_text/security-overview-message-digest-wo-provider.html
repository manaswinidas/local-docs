图描述从特定提供者请求SHA-256消息摘要实现

该图像由代表以下内容的五个圆柱组成：

  • 应用

  • 提供者框架

  • ProviderA ：消息摘要实现SHA384和SHA512

  • ProviderB ：消息摘要实现SHA256和SHA384

  • ProviderC ：消息摘要实现SHA256和SHA512

代表呼叫的箭头MessageDigest.getInstance("SHA-256")从“应用”筒开始，并按指示顺序通过以下筒：

  1. 提供者框架

  2. ProviderA

  3. ProviderB

  4. 提供者框架

箭头在应用圆柱体处结束。致电MessageDigest.getInstance("SHA-256")已从返回SHA-256消息摘要实现ProviderB 。
