图说明KeyFactory类

该图由标记如下的框组成：

  • 关键规格
  • 密钥工厂（RSA）：在此标记了两个框
  • 私钥
  • 公钥
  • 关键规格

带标记的箭头将这些框连接起来：

  • generatePrivate() ：从密钥工厂（RSA）＃1到私钥
  • generatePublic() ：从密钥工厂（RSA）＃1到公钥
  • getKeySpec() ：从密钥工厂（RSA）＃2到密钥规格

未标记的箭头将这些框连接起来：

  • 从密钥规格到密钥工厂（RSA）＃1
  • 从密钥到密钥工厂（RSA）＃2
