DTLS握手期间的状态机

此图像说明了DTLS握手状态和DTLS握手期间的不同消息。在确定握手状态之前，请执行以下步骤：

  • 创建SSL / TLS SSLEngines
  • 创建缓冲区
  • 设置服务器模式的客户端
  • 开始握手
  • 设置最大片段大小

此图像说明了一些可能的握手状态。“ 了解SSLEngine操作状态 ”部分将更详细地描述这些状态：

  • NEED_TASK
  • NEED_WRAP
  • NEED_UNWRAP
  • NEED_UNWRAP_AGAIN
  • FINISHED
