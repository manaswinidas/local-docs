图说明Mac类

该图由标记如下的框组成：

  • 数据：两个框标有此标签
  • MAC（HmacSHA256）：两个框标记为
  • 共享密钥
  • 信号摘要哈希：两个框标记为
  • 如果数据相同，则哈希相同

带标记的箭头将这些框连接起来：

  • update() ：从数据＃1到MAC（HmacSHA256）＃1
  • doFinal() ：从数据＃1到MAC（HmacSHA256）＃1
  • update() ：从数据＃2到MAC（HmacSHA256）＃2
  • doFinal() ：从数据＃2到MAC（HmacSHA256）＃2

未标记的箭头将这些框连接起来：

  • 从共享密钥到MAC（HmacSHA256）（两个单独的箭头）
  • 从MAC（HmacSHA256）＃1到签名摘要哈希＃1
  • 从MAC（HmacSHA256）＃2到签名摘要哈希＃2
  • 从带符号的摘要哈希＃1到如果数据相同，则哈希相同
  • 从签名摘要哈希＃2到如果数据相同，则哈希相同
