图说明KeyAgreement类

该图由虚线分为两半。上半部分标记为Alice，下半部分标记为Bob。该图由标记如下的框组成：

  • 这些盒子在爱丽丝的一半中：
    • 密钥：此框包含一个标有“爱丽丝的私人”的框
    • 密钥：此框包含一个标记为Bob's Public的框
    • 关键协议（DH）
    • 字节：此框有另一个标签，字节
  • 这些盒子在鲍勃的一半里：
    • 密钥：此框包含一个标记为Bob's Private的框
    • 密钥：此框包含一个标有“爱丽丝的公共”的框
    • 关键协议（DH）
    • 字节：此框有另一个标签，字节

带标记的箭头将这些框连接起来：

  • 这些箭头在爱丽丝的一半中：
    • init() ：从爱丽丝的私钥到密钥协议（DH）
    • doPhase() ：从Bob的公钥到密钥协议（DH）
    • generateSecret() ：从密钥协议（DH）到字节
  • 这些箭头在鲍勃的一半中：
    • init() ：从Bob的私钥到密钥协议（DH）
    • doPhase() ：从爱丽丝的公钥到密钥协议（DH）
    • generateSecret() ：从密钥协议（DH）到字节

双头箭头将爱丽丝和鲍勃的两半中的“字节”框连接在一起。此箭头标记为“应该相同”。

  • init() ：从密钥长度到密钥生成器（AES）
  • init() ：从AlgorithmParameterSpec到密钥生成器（AES）
  • generateKey() ：从密钥生成器（AES）到密钥对
