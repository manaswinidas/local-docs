前言

本指南总结了Java SE平台的国际化API和功能。

听众

本指南适用于希望设计应用程序的Java程序员，这样他们可以在不进行工程更改的情况下适应各种语言和地区。

文档可访问性

有关Oracle对可访问性承诺的信息，请访问Oracle可访问性计划网站， 网址为http://www.oracle.com/pls/topic/lookup?ctx=acc&id=docacc 。

获得Oracle支持

购买了支持的Oracle客户可以通过My Oracle Support获得电子支持。有关信息，请访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=info或访问http://www.oracle.com/pls/topic/lookup?ctx=acc&id=trs听力受损。

约定

本文档中使用以下文本约定：

惯例 含义

黑体字

粗体类型表示与操作或以文本或词汇表定义的术语相关联的图形用户界面元素。

斜体

斜体类型表示要为其提供特定值的书名，重点或占位符变量。

monospace

等宽字体类型表示段落中的命令，URL，示例中的代码，屏幕上显示的文本或您输入的文本。
