前言

本文档介绍了Java Access Bridge，Java Accessibility API（JAAPI）和Java Accessibility Utilities，它们使您可以创建可访问的应用程序。

听众

本文档适用于希望创建残疾人可以访问的Java应用程序的开发人员。可访问的Java应用程序与辅助技术兼容，例如屏幕阅读器，屏幕放大镜，语音识别系统和可刷新的盲文显示器。

约定

本文档中使用以下文本约定：

惯例 含义

黑体字

粗体类型表示与操作或以文本或词汇表定义的术语相关联的图形用户界面元素。

斜体

斜体类型表示要为其提供特定值的书名，重点或占位符变量。

monospace

等宽字体类型表示段落中的命令，URL，示例中的代码，屏幕上显示的文本或您输入的文本。
