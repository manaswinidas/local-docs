Java Access Bridge体系结构描述

该图描述了Java Access Bridge和Java Accessibility Utilities组件如何交互。

此图表示带有标记框的组件和子组件。它包含三个主要组件：

  • Java运行时

  • 本机辅助技术

  • Windows的Java Access Bridge

Windows的Java Runtime和Java Access Bridge框相互重叠。位于这两个主要组件相互重叠的位置的任何子组件都属于这两个组件。

Java运行时组件包含以下组件：

  • Java应用程序，包含组件UI工具包，其中包含组件Java可访问性API（JAAPI）

  • Java可访问性实用程序

Windows的Java Access Bridge组件包含以下组件：

  • 运行时DLL

  • Windows DLL

  • Windows的Java可访问性实用程序和Java Access Bridge

  • 本机辅助技术和Windows DLL
