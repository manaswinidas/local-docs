3 Nashorn和JavaFX

本节介绍如何使用Nashorn引擎解释的脚本创建和运行JavaFX应用程序。假定您熟悉JavaFX。

话题

  • 用Nashorn解释JavaFX脚本应用程序
  • 使用Nashorn简化JavaFX脚本应用程序
  • Nashorn脚本对象

用Nashorn解释JavaFX脚本应用程序

您可以使用Nashorn来解释JavaFX脚本应用程序jjs用命令-fx选项。例如，以下命令调用Nashorn来解释JavaFXscript.js文件：

jjs -fx --module-path /SOMEDIR/javafx-sdk-11/lib --add-modules javafx.controls JavaFXscript.js

注意：

  • 您必须显式添加JavaFX模块才能将脚本作为JavaFX应用程序启动。如果您不添加它们，则jjs命令显示一条消息并退出：

    jjs -fx JavaFXscript.js

    JavaFX is not available.

  • JavaFX模块包括javafx.base ， javafx.controls ， javafx.fxml ， javafx.graphics ， javafx.media ， javafx.swing和javafx.web 。请确保在以下位置指定应用程序所需的所有模块： --add-modules选项。

  • 从OpenJFX获取JavaFX。

JavaFX脚本应用程序与Java等效脚本应用程序相似，但是Nashorn使您能够简化许多JavaFX构造。通常，JavaFX脚本应用程序仅包含start()功能，相当于start() Java对应的方法。它还可以包含init()和stop()功能。

使用Nashorn简化JavaFX脚本应用程序

Nashorn使您无需声明某些类型，并使您能够更轻松地访问某些方法和属性，从而简化了JavaFX代码。

通过分析JavaFX应用程序示例（HelloWorld.java）和JavaFX脚本应用程序示例 （HelloWorld.js）中的 示例 ，您可以看到Nashorn如何在将JavaFX应用程序编写为脚本时使您简化Java代码：

  • 无需声明变量类型，导入包，使用批注，指定类名并实现其main()方法。

  • 仅必须声明实例化的JavaFX类。

  • JavaBeans不需要get和set前缀，并被视为JavaScript属性，而不是Java方法。请参阅使用JavaBeans 。

  • 实施javafx.event.EventHandler接口不需要您显式指定实现的方法。因为handle()是唯一的方法，Nashorn会自动将提供的功能应用于该方法。请参阅扩展Java类 。

Nashorn可以将JavaFX初级阶段作为全球资产使用$STAGE 。此全局属性使您可以将整个脚本视为一个start()功能（您仍然可以添加init()和stop()功能）。JavaFX脚本应用程序的较简单版本示例（HelloWorldSimple.js）包含JavaFX脚本应用程序的示例（HelloWorld.js）中 JavaFX脚本应用程序的简化版本的源代码。

JavaFX应用程序示例（HelloWorld.java）

下面的例子， HelloWorld.java ，包含一个简单的JavaFX应用程序的源代码，该应用程序显示一个按钮，单击该按钮可打印"Hello World!"到标准输出。 

import javafx.application.Application;
import javafx.event.ActionEvent;
import javafx.event.EventHandler;
import javafx.scene.Scene;
import javafx.scene.control.Button;
import javafx.scene.layout.StackPane;
import javafx.stage.Stage;

public class HelloWorld extends Application {
    public static void main(String[] args) {
        launch(args);
    }
    
    @Override
    public void start(Stage primaryStage) {
        primaryStage.setTitle("Hello World!");
        Button btn = new Button();
        btn.setText("Say 'Hello World'");
        btn.setOnAction(new EventHandler<ActionEvent>() {
 
            @Override
            public void handle(ActionEvent event) {
                System.out.println("Hello World!");
            }
        });
        
        StackPane root = new StackPane();
        root.getChildren().add(btn);
        primaryStage.setScene(new Scene(root, 300, 250));
        primaryStage.show();
    }
}

JavaFX脚本应用程序（HelloWorld.js）的示例

下面的例子， HelloWorld.js ，包含JavaFX脚本应用程序的源代码，该应用程序显示一个按钮，单击该按钮可打印"Hello World!"到标准输出。 

var Button = javafx.scene.control.Button;
var StackPane = javafx.scene.layout.StackPane;
var Scene = javafx.scene.Scene;

function start(primaryStage) {
    primaryStage.title = "Hello World!";
    var button = new Button();
    button.text = "Say 'Hello World'";
    button.onAction = function() print("Hello World!");
    var root = new StackPane();
    root.children.add(button);
    primaryStage.scene = new Scene(root, 300, 250);
    primaryStage.show();
}

JavaFX脚本应用程序的更简单版本的示例（HelloWorldSimple.js）

下面的例子， HelloWorldSimple.js ，使用global属性包含JavaFX脚本应用程序的简化版本的源代码$STAGE 。 

var Button = javafx.scene.control.Button;
var StackPane = javafx.scene.layout.StackPane;
var Scene = javafx.scene.Scene;

$STAGE.title = "Hello World!";
var button = new Button();
button.text = "Say 'Hello World'";
button.onAction = function() print("Hello World!");
var root = new StackPane();
root.children.add(button);
$STAGE.scene = new Scene(root, 300, 250);
$STAGE.show();

load("fx:graphics.js");
