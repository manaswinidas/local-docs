第三部分 Java管理扩展示例

本节中的文件是代码示例，这些示例演示了JMX技术的一些主要功能。您可以使用这些示例来开发更复杂的MBean和功能齐全的JMX代理，以适合您的管理解决方案。

每个示例均包含Java源文件和README文件。README文件解释了示例所涵盖的主题以及有关编译和运行类的说明。有关如何运行示例的更完整说明，请参见Java管理扩展（JMX）技术教程 。

JMX Essentials：介绍JMX API的基本概念，即托管Bean或MBean。

JMX MBean通知：实现MBean通知。

MXBeans：演示MXBean的用法。

MBean描述符：演示MBean描述符的使用。

JMX连接器：提供有关如何连接到MBean以及对其进行远程操作的示例实现。

查找服务： JMX API使用现有的查找技术定义了对查找服务的三个绑定。

  • 服务定位协议（SLP）查找服务
  • Jini查询服务
  • Java命名和目录接口（JNDI）/ LDAP查找服务

安全性： JMX API实现了现有的安全协议以保护您的连接。

  • 简单安全
  • 主题委派的安全性
  • 细粒度的安全
