JMX技术概述

Java管理扩展（JMX）技术最初是通过Java社区流程（JCP）作为Java规范请求（JSR）3，Java管理扩展和JSR 160，JMX远程API开发的。 JMX API是用于管理和监视资源（如应用程序，设备，服务和Java虚拟机）的标准API。

JMX技术的典型用途包括：

  • 监视和更改应用程序配置
  • 累积有关应用程序行为的统计信息并使它们可用
  • 发送状态更改和错误状况的通知。

JMX API包括远程访问，因此远程管理应用程序可以与正在运行的应用程序进行交互以执行这些操作。
