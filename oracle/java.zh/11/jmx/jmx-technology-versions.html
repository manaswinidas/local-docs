7个 JMX技术版本

Java管理扩展技术（JMX）在Java Platform Standard Edition（Java SE）5.0中成为Java平台的标准部分。JMX技术是通过Java社区流程（JCP）作为两个紧密相关的Java规范请求（JSR）开发的。以下各节详细介绍了Java SE 5.0中实现的JSR版本。

JMX规范和代理规范（JSR 3）

Java SE 9平台实现了JMX规范的 1.4版（维护版本，2006年10月）。它包含了下载随附的勘误表中列出的修改。

JMX远程API规范（JSR 160）

Java SE 9平台实现了JMX远程API规范的 1.4版（维护版本，2006年10月）。

除了标准的RMI连接器之外，JSR 160还基于TCP套接字定义了可选的JMX消息协议（JMXMP）连接器。Java SE平台不包括这些可选的连接器。如果需要更高级别的安全性，则可能要使用JMXMP连接器。

如果要使用JMXMP连接器，请从本节开头指定的下载页面下载JSR 160参考实现，然后添加jmxremote_optional.jar文件到您的类路径。您可以在《 JSR 160参考实现》随附的《 JMX远程API教程》中找到有关使用JMXMP连接器的示例。
