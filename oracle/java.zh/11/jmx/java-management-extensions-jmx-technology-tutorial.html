第二部分 Java管理扩展（JMX）技术教程

JMX技术教程概述

本教程提供了有关如何使用Java平台标准版随附的JMX技术的主要功能的示例。

本教程旨在按顺序从头至尾阅读，并按照您的示例进行操作。在本教程的后续部分中可能需要您在本教程开始时执行的操作。因此，从教程中途开始可能会导致您跳过某些示例所需的操作。

在必须执行任务的地方，说明中标有动作编号，子任务中标有小写字母。

使用本教程之前

本教程演示了《 Java管理扩展技术用户指南》中引入的概念和技术。因此，在尝试阅读本教程之前，您应该阅读概述。要充分利用本教程中的信息，您还应该熟悉以下协议和规范：

  • 远程方法调用（RMI）

  • 轻型目录访问协议（LDAP）

  • 服务定位协议（SLP）

  • Jini TM网络技术

  • Java命名和目录接口TM （JNDI）API

  • Java安全套接字扩展（JSSE）

  • Java身份验证和授权服务（JAAS）

  • Java管理扩展规范1.4

本教程的组织方式

本教程提供了以下各章中提供的大类示例。

  • 入门为您提供了一些初始配置信息。
  • JMX API的要点介绍了JMX规范的核心概念。
  • JMX连接器提供了有关如何实现标准和动态MBean类型以及如何在本地和远程对其执行操作的示例。
  • 查找服务演示了可以与JMX技术结合使用的查找服务。
  • 安全性显示了一些安全性配置示例。
