9入门

本章说明了开始使用Java管理扩展（JMX）示例所需要执行的操作。它提供了适用于以下章节中描述的所有示例的说明。

平台资讯

本教程示例中的所有变量分配和命令都是使用UNIX Korn Shell语法定义的。如果在Solaris，Linux或Mac OS X平台上运行的不是Korn外壳程序的外壳程序，则必须使这些命令适应您首选的外壳程序环境。

如果您运行的是Microsoft Windows操作环境，则在大多数情况下，调整命令仅涉及替换正斜杠（ / ），并使用反斜线（ \ ）并替换冒号（ : ）和分号（ ; ）。仅当特定的Microsoft Windows命令与提供的Solaris，Linux或Mac OS X命令有显着差异时，才会给出该命令。
