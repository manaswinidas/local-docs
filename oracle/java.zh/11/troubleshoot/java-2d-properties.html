以下列表说明了如何更改默认值。

  • X11管道是Oracle Solaris和Linux的默认管道。更改此默认值，如下所示：

    • -Dsun.java2d.opengl=true —尝试启用OpenGL管道。

  • DGA的使用受到以下控制：

    • NO_J2D_DGA unset —如果可用，请使用DGA。

    • NO_J2D_DGA set —禁用DGA。

  • MIT共享内存扩展（SHM）的控制如下：

    • 要使用SHM（如果可用），请指定以下任一属性：

      NO_J2D_MITSHM unset

      J2D_USE_MITSHM=true

    • 若要不使用SHM，请指定以下任一属性：

      NO_J2D_MITSHM set

      J2D_USE_MITSHM=false

  • 像素图的一般用法如下控制：

    • -Dsun.java2d.pmoffscreen unset —如果DGA不可用，请使用像素图。

    • -Dsun.java2d.pmoffscreen=true —强制使用像素图。

    • -Dsun.java2d.pmoffscreen=false —禁用像素图。

  • 共享和服务器像素图的使用受到以下控制：

    • J2D_PIXMAPS unset —同时使用两种类型。

    • J2D_PIXMAPS=shared —仅使用共享内存像素图。

    • J2D_PIXMAPS=sserver —仅使用服务器端像素图。

  • 默认视觉效果的选择如下控制：

    • FORCEDEFVIS未设置（默认）—使用最佳视觉效果。

    • FORCEDEFVIS设置为十六进制值—使用ID为十六进制值的视觉对象。

    • FORCEDEFVIS设置为任何其他值-使用默认外观。

Windows上的属性

Windows上有用的属性列表。

下表描述了Windows平台上的一些有用属性。

  • DirectDraw / GDI管道是Windows的默认管道。更改此默认值，如下所示：

    • -Dsun.java2d.noddraw=true —禁用DirectDraw管道。将改为使用GDI。

    • -Dsun.java2d.noddraw=false —启用DirectDraw管道的使用。

    • -Dsun.java2d.d3d=false —禁用Direct3D管道。

    • J2D_D3D=false —禁用Direct3D管道。

    • -Dsun.java2d.d3d=true —启用Direct3D管道的使用。

    • J2D_D3D=true —启用Direct3D管道的使用。

  • 如下控制内置表面打孔机制的使用：

    • -Dsun.java2d.ddforcedram=true —将易失性图像保留在VRAM中。

  • 控制DirectDraw blit操作的使用，如下所示：

    • -Dsun.java2d.ddblit=false —禁止使用DirectDraw blit操作。将使用GDI blit。
