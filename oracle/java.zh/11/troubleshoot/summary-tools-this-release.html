E此版本中的工具摘要

本附录摘要介绍了JDK当前版本中可用的工具以及自先前版本以来的更改。

Oracle Solaris和Linux上的JDK 9中都提供了本文档中描述的所有JDK故障排除工具。

Windows上的JDK 9中还提供了以下JDK故障排除工具：

  • Java任务控制

  • Java飞行记录

  • 如何制作飞行记录

  • 检查飞行记录

  • jcmd

  • J控制台

  • Java虚拟机

  • jdb

  • jinfo

  • jmap

  • jps （当前在Windows 98或Windows ME上不可用）

  • jrunscript

  • jstack

  • jstat （当前在Windows 98或Windows ME上不可用）

  • jstatd （当前在Windows 98或Windows ME上不可用）

  • visualgc （当前在Windows 98或Windows ME上不可用）
