图示说明jconsole-memory-tab.gif

此图显示了JConsole窗口的屏幕截图。在窗口顶部，标题栏包含以下文本：“ Java监视和管理控制台-pid：5500 sun.tools.jconsole。JConsole”。

标题栏下方是一个菜单栏，该菜单栏由三个菜单组成：“连接”，“窗口”和“帮助”。

该窗口分为六个选项卡，位于菜单栏下方：概述，内存，线程，类，VM摘要和MBean。

选择内存选项卡。该选项卡的顶部是三个元素的行：标记为“图表”的下拉列表和选中的“堆内存使用”项；标记为“时间范围”的下拉列表，并且选择了“全部”项；以及标记为“执行GC”的按钮。

在这些元素下方是一个图形，在水平轴上具有四个时间值，范围从15:05到15:08，每个时间间隔为一分钟。垂直轴包含五个内存大小值，范围从10 Mb到50 Mb，每个间隔为10 Mb。在图形线的右边断开的地方，有一个标签，上面写着：“ Used 17,909,604”。

在图表下方，在窗口底部，有一个标签为“详细信息”的组。在该组的左侧，有一个窗格，其中包含五个值的摘要：

Time: 2006-10-02 15:08:26
Used:       18,234 kbytes
Commited:   40,456 kbytes
Max:        65,088 kbytes
GC time: 0.709 seconds on Copy (194 collections)
         1.272 seconds on MarkSweepCompact (12 collections)

在“详细信息”组的右侧，有五个垂直仪表（其中三个位于标为“堆”的框上方，两个位于标为“非堆”的框上方）。量规的左侧是垂直刻度轴，其中五个值的范围从0％到100％，间隔为25％。
