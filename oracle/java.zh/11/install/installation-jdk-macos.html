在macOS上安装JDK的系统要求

以下是在macOS上安装JDK的系统要求：

  • 任何运行macOS的基于Intel的计算机。

  • 管理员权限。

    您不能为单个用户安装Java。在macOS上为所有用户执行JDK安装。在macOS上安装JDK需要管理员特权。

在macOS上确定默认的JDK版本

通过命令行启动Java应用程序时，系统使用默认的JDK。

macOS系统上可以安装多个JDK。

您可以通过输入以下命令确定默认的JDK版本java -version在终端窗口中。如果安装的版本是11 Interim 0，Update 0和Patch 0，则您将看到一个包含以下内容的字符串： 11 。例如：

$  java -version
java version "11"
Java(TM) SE Runtime Environment (build 11)
Java HotSpot(TM) 64-Bit Server VM (build 23.2-b04, mixed mode)

要运行其他版本的Java，请指定完整路径，或使用java_home工具。例如：

$ /usr/libexec/java_home -v 11 --exec javac -version

在macOS上安装JDK

要在macOS上安装JDK：
  1. 下载JDK .dmg文件， jdk-11.interim.update.patch-macosx-x64.dmg 。

    在可以下载文件之前，您必须接受许可协议。

  2. 在浏览器的“ 下载”窗口或文件浏览器中，双击.dmg文件来启动它。
    将出现一个Finder窗口，其中包含一个打开框的图标和该窗口的名称。 .pkg文件。
  3. 双击JDK 11.pkg图标启动安装应用程序。
    安装应用程序将显示“ 简介”窗口。
  4. 点击继续 。
    出现安装类型窗口。
  5. 点击安装 。
    出现一个窗口，显示以下消息：Installer is trying to install new software. Enter your password to allow this.
  6. 输入管理员用户名和密码，然后单击“ 安装软件” 。
    软件已安装，并显示确认窗口。
安装软件后，您可以删除.dmg文件，如果要节省磁盘空间。

在macOS上卸载JDK

要在macOS上卸载JDK：
您必须具有管理员权限。

注意：

不要尝试通过从中删除Java工具来卸载Java。 /usr/bin 。该目录是系统软件的一部分，下一次执行操作系统更新时，Apple将重置所有更改。

  1. 去/Library/Java/JavaVirtualMachines 。
  2. 通过执行以下命令删除名称与以下格式匹配的目录： rm以root用户身份使用命令或使用sudo工具：
    /Library/Java/JavaVirtualMachines/jdk-interim.update.patch.jdk

    例如，要卸载11 Interim 0 Update 0 Patch 0：

    $ rm -rf jdk-11.jdk

macOS平台上的安装常见问题解答

本主题提供以下有关在macOS计算机上安装JDK的常见问题的答案。

1。如何找出系统默认的Java版本？

从命令行运行Java应用程序时，它将使用默认的JDK。如果您不开发Java应用程序，那么您不必为此担心。请参阅在macOS上确定默认的JDK版本 。

2。如何卸载Java？

请参阅在macOS上卸载JDK 。

3。在安装适用于macOS 2012-006的Java之后，我可以继续将Apple的Java 6与适用于Java 11的macOS JDK一起使用吗？

如果要继续使用命令行使用Java 6进行开发，则可以针对自己喜欢的命令环境修改启动脚本。对于bash，请使用以下命令：

$ export JAVA_HOME=`/usr/libexec/java_home -v 11`

一些应用程序使用/usr/bin/java调用Java。安装适用于macOS 2012-006的Java后， /usr/bin/java将找到最新安装的JDK，并将其用于所有与Java相关的命令行工具中/usr/bin 。您可能需要修改这些应用程序以找到Java 6，或者与开发人员联系以获取该应用程序的较新版本。

4。卸载Oracle Java后可以恢复Apple Java吗？

使用以下说明返回到Apple Java：

  1. 通过删除插件文件来卸载Oracle Java。在命令行中，输入：

    $ sudo rm -rf "/Library/Internet Plug-Ins/JavaAppletPlugin.plugin"

  2. 使用以下命令创建符号链接，在单行中输入该命令：

    $ sudo ln -s /System/Library/Java/Support/CoreDeploy.bundle/Contents/JavaAppletPlugin.plugin "/Library/Internet Plug-Ins/JavaAppletPlugin.plugin"

5，应用程序实用程序中的Java首选项应用程序发生了什么？

Java Preferences应用程序是Apple Java安装的一部分，Oracle Java不使用。因此，Apple的不包含Apple Java的macOS版本将不包含Java首选项。
