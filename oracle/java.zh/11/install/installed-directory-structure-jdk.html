7 JDK的已安装目录结构

JDK 11的模块化运行时映像包含以下文件夹：

注意：

在macOS中，JDK安装路径为/Library/Java/JavaVirtualMachines/jdk-11.jdk/Contents/Home 。
/jdk-11

JDK软件安装的根目录。此目录还包含版权，自述文件和发行文件。

/jdk-11.jdk/Contents/Home/

macOS上JDK软件安装的根目录。

/jdk-11/bin

由链接到映像的模块定义的可执行程序和命令行启动器。

/jdk-11/conf

的.properties ， .policy以及打算由开发人员，部署人员和最终用户编辑的其他配置文件。

/jdk-11/lib

运行时系统的私有实现详细信息。这些文件仅供外部使用，不得修改。

的lib macOS上的目录或lib/$ARCH Linux和Solaris上的目录包含运行时系统的动态链接本机库。

/jdk-11/jmods

编译的模块定义。

/jdk-11/legal

每个模块的版权和许可文件。

/jdk-11/lib/src.zip

包含Java平台的源代码的归档文件。

/jdk-11/include

支持使用Java本机接口和Java虚拟机（JVM）调试器接口进行本机代码编程的C语言头文件。

注意：

您可以同时安装两个版本的JDK。JDK的更新版本不会替代以前的版本，但是会创建一个新目录。例如，您可以同时安装JDK 11和JDK 11.0.1。在JDK安装位置， jdk-11和jdk-11.0.1目录被创建。
