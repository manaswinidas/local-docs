1 JDK安装概述

安装类型取决于您的要求和选择安装的平台。

安装要求和相关任务

要开发或运行Java应用程序，请下载并安装JDK。

版本字符串格式

要了解用于区分各种JDK版本的版本字符串方案，请参见Version-String Format 。

安装

JDK可以安装在以下平台上：

  • Oracle Solaris

  • 微软Windows

  • 的Linux

  • 苹果系统

您可以从Java SE Development Kit下载页面下载JDK。

有关受支持的处理器和浏览器，请参见Oracle JDK认证的系统配置 。

JDK文档是单独下载的。请参阅Java SE文档 。

Oracle Solaris

在Oracle Solaris上安装JDK ：说明包括从映像打包系统（IPS）和存档二进制文件安装。

微软Windows

在Microsoft Windows平台上安装JDK ：说明包括在64位Microsoft Windows操作系统上安装JDK的步骤。

的Linux

在Linux平台上安装JDK ：说明包括从存档二进制文件和Red Hat Package Manager（RPM）进行安装。

苹果系统

JDK 11的安装需要macOS 10.11或更高版本。

在macOS上安装JDK ：说明包括在macOS平台上安装JDK的步骤。

目录结构

要了解在JDK安装之后创建的文件和目录，请参阅《 JDK的已安装目录结构》 。
