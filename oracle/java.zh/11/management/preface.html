前言

Java平台标准版11 （Java SE 11 ）提供了实用程序，使您可以监视和管理Java虚拟机（Java VM）以及其中运行的Java应用程序的性能。《 Java SE监视和管理指南》介绍了那些监视和管理实用程序。

听众

本指南面向Java语言的有经验的用户，例如系统管理员和软件开发人员，对于Java平台及其应用程序的性能至关重要。

惯例 含义

黑体字

粗体类型表示与操作或以文本或词汇表定义的术语相关联的图形用户界面元素。

斜体

斜体类型表示要为其提供特定值的书名，重点或占位符变量。

monospace

等宽字体类型表示段落中的命令，URL，示例中的代码，屏幕上显示的文本或您输入的文本。
