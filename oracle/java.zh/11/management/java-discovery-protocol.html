5 Java发现协议（JDP）

Java发现协议（JDP）是一种协议，该协议使技术（尤其是Java Mission Control和Java Flight Recorder）能够跨同一网络子网发现可管理的JVM。

可管理的JVM是运行Java管理扩展（JMX）代理的JVM。JDP是基于多播的，并且像信标一样工作。它广播连接到外部JMX代理所需的JMX服务URL（请参阅类JMXServiceURL ）。这使技术能够检测出故障或不再可用于监视的JVM。

启用和配置JDP

要启用JDP，请在启动Java应用程序时在命令行中指定以下选项：

-Dcom.sun.management.jmxremote.autodiscovery=true

注意：

启用JDP不会影响JMX安全性。要启用和配置JMX安全性，请参阅《 使用JMX技术进行监视和管理》 。
