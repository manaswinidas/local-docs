1 JShell简介

Java Shell工具（JShell）是用于学习Java编程语言和Java代码原型的交互式工具。JShell是一个Read-Evaluate-Print Loop（REPL），它在输入声明，语句和表达式时对它们进行评估，并立即显示结果。该工具从命令行运行。

为什么要使用JShell？

使用JShell，您可以一次输入一个程序元素，立即查看结果，并根据需要进行调整。

Java程序开发通常涉及以下过程：

  • 编写一个完整的程序。

  • 编译并修复所有错误。

  • 运行程序。

  • 找出问题所在。

  • 编辑它。

  • 重复该过程。

JShell可帮助您试用代码并在开发程序时轻松探索选项。您可以测试单个语句，尝试方法的不同变体，并在JShell会话中尝试使用不熟悉的API。JShell不会替代IDE。在开发程序时，请将代码粘贴到JShell中进行尝试，然后将JShell中的工作代码粘贴到程序编辑器或IDE中。

启动和停止JShell

JShell是JDK 9中引入的。要启动JShell，请输入jshell命令行上的命令。

必须在系统上安装JDK 9或更高版本。例如，如果您的路径不包含bin目录java-home/jdk-9/bin ，然后从该目录中启动该工具。

以下示例显示了JShell的命令和响应。您输入的文本以粗体显示：

% jshell
|  Welcome to JShell -- Version 9
|  For an introduction type: /help intro

jshell>

本教程中的示例使用详细模式。在您阅读本教程时，建议采用详细模式，以使您看到的内容与示例匹配。当您更熟悉该工具时，您可能更希望以常规或更简洁的模式运行。

要以详细模式启动JShell，请使用-v选项：

% jshell -v

要退出JShell，请输入/exit ：

jshell> /exit
|  Goodbye
