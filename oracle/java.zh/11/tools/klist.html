列表

您使用klist工具以显示本地凭据缓存和密钥表中的条目。 的ktab该工具使您可以查看本地凭据缓存和密钥表中的条目。

概要

klist [ [-c] [-f] [-e] [-a [-n]] ] [-k [-t] [-K] ] [name] [-help]]

描述

的klist工具会在本地凭据缓存和密钥表中显示条目。使用以下命令修改凭据缓存后kinit工具或使用ktab工具，验证更改的唯一方法是使用klist工具。的klist工具不会更改Kerberos数据库。

指令

-c

指定要列出凭据缓存。

以下是凭据缓存条目的选项：

-f

显示凭证标志。

-e

显示加密类型。

-a

显示地址。

-n

如果-a指定选项，请勿反向解析地址。

-k

指定要列出的密钥选项卡。

列出键表项。以下是密钥表项的选项：

-t

显示密钥标签输入时间戳。

-K

显示密钥表项DES密钥。

-e

显示密钥标签输入密钥类型。

name

指定凭据缓存名称或密钥表名称。基于文件的缓存或密钥表的前缀为FILE: 。如果未指定名称，则klist该工具将默认值用于缓存名称和密钥表。的kinit文档列出了这些默认值。

-help

显示说明。

例子

列出指定的密钥表中的条目，包括密钥表条目时间戳和DES密钥：

klist -k -t -K FILE:\temp\mykrb5cc

列出指定的凭据缓存中的条目，包括凭据标志和地址列表：

klist -c -f FILE:\temp\mykrb5cc
