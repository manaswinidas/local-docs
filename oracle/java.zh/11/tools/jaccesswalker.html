jaccesswalker

您使用jaccesswalker浏览特定Java虚拟机中的组件树，并在树视图中显示层次结构。

描述

您在树中选择一个节点，然后从“ 面板”菜单中选择“ 辅助功能API面板” 。的jaccesswalker工具会在窗口中显示该对象的辅助功能信息。

运行jaccesswalker工具

使用jaccesswalker ，启动jaccesswalker启动Java应用程序后的工具。例如，启动jaccesswalker ，输入以下命令：

%JAVA_HOME%\bin\jaccesswalker.exe

现在，您打开了两个窗口：Java应用程序窗口和jaccesswalker工具。您可以执行两项任务jaccesswalker 。您可以构建Java应用程序的GUI层次结构的树状视图，并且可以查询GUI层次结构中特定元素的Java Accessibility API信息。

建立GUI层次结构

在文件菜单上，选择刷新树菜单。的jaccesswalker该工具将构建属于Java应用程序的顶级窗口的列表。然后，该工具递归地查询那些窗口中的元素，并在系统中运行的所有JVM中的所有Java应用程序中构建所有GUI组件的树。

检查GUI组件

生成GUI树之后，可以通过在树中选择单个GUI组件，然后选择“ 面板”，然后选择“ 显示可访问性信息”来查看有关单个GUI组件的详细可访问性信息 。
