登记处

您使用rmiregistry基于UNIX的系统上的命令，以在当前主机的指定端口上创建并启动远程对象注册表。

概要

rmiregistry options port
options

这代表了rmiregistry命令。查看选项

port

当前主机上要启动远程对象注册表的端口号。

描述

在基于UNIX的系统上， rmiregistry命令在当前主机的指定端口上创建并启动远程对象注册表。如果省略端口，则注册表在端口1099上启动。的rmiregistry该命令不产生任何输出，并且通常在后台运行，例如：

rmiregistry &

远程对象注册表是引导命名服务，同一主机上的RMI服务器使用它来将远程对象绑定到名称。然后，本地和远程主机上的客户端可以查找远程对象并进行远程方法调用。

注册表通常用于定位应用程序需要在其上调用方法的第一个远程对象。然后，该对象为查找其他对象提供了特定于应用程序的支持。

的方法java.rmi.registry.LocateRegistry类用于获取在本地主机或本地主机和端口上运行的注册表。

基于URL的方法java.rmi.Naming该类在注册表上运行，可用于：

  • 将指定名称绑定到远程对象

  • 返回注册表中绑定的名称的数组

  • 返回与指定名称关联的远程对象的引用（存根）

  • 将指定名称重新绑定到新的远程对象

  • 销毁与远程对象关联的指定名称的绑定
