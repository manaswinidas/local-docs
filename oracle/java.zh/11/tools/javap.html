Java程序

您使用javap命令反汇编一个或多个类文件。

概要

javap [options] classes...
options

指定命令行选项。请参阅javap的选项 。

classes

指定一个或多个用空格分隔的类，以对其进行注释处理。您可以指定一个类，该类可以通过其文件名，URL或完全限定的类名在类路径中找到。

例子：

path/to/MyClass.class
jar:file:///path/to/MyJar.jar!/mypkg/MyClass.class
 java.lang.Object

描述

的javap命令反汇编一个或多个类文件。输出取决于所使用的选项。如果不使用任何选项， javap命令显示受保护的字段和公共字段，以及传递给它的类的方法。

的javap该命令不支持多版本JAR。使用命令的类路径形式可以查看所有JAR文件（无论是否为multirelease）中的基本条目。使用URL形式，可以使用参数的URL形式来指定要反汇编的类的特定版本。

的javap命令将其输出打印到stdout 。

注意：

在支持的工具中--样式选项，GNU样式选项可以使用等号（=）代替空格，以将选项名称与其值分开。

javap的选项

-help ， --help ， 要么-?

打印帮助信息javap命令。

-version

打印发行信息。

-verbose要么-v

打印有关所选类的其他信息。

-l

打印行和局部变量表。

-public

仅显示公共班级和成员。

-protected

仅显示受保护的和公开的班级和成员。

-package

显示软件包/受保护的/公共的类和成员（默认）。

-private要么-p

显示所有班级和成员。

-c

为该类中的每个方法打印反汇编的代码，例如构成Java字节码的指令。

-s

打印内部类型签名。

-sysinfo

显示正在处理的类的系统信息（路径，大小，日期，MD5哈希）。

-constants

演出static final常数。

--module module要么-m module

指定包含要反汇编的类的模块。

--module-path path

指定在哪里找到应用程序模块。

--system jdk

指定在何处查找系统模块。

--class-path path -classpath path ， 要么-cp path

指定路径javap该命令用于查找用户类文件。它会覆盖默认值或CLASSPATH设置时的环境变量。

-bootclasspath path

覆盖引导类文件的位置。

-Joption

将指定的选项传递给JVM。例如：

javap -J-version

javap -J-Djava.security.manager -J-Djava.security.policy=MyPolicy MyClassName

请参阅Java选项概述 。

javap示例

编译以下HelloWorldFrame类：

import java.awt.Graphics;
 
import javax.swing.JFrame;
import javax.swing.JPanel;
 
public class HelloWorldFrame extends JFrame {
 
   String message = "Hello World!";
 
   public HelloWorldFrame(){
        setContentPane(new JPanel(){
            @Override
            protected void paintComponent(Graphics g) {
                g.drawString(message ,15, 30);
            }
        });
        setSize(100,100);
    }
    public static void main(String[] args) {
        HelloWorldFrame frame = new HelloWorldFrame();
        frame.setVisible(true);
 
    }
 
}
