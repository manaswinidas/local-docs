2个用于创建和构建应用程序的主要工具

您可以使用基础JDK工具和命令来创建和构建应用程序。

以下各节描述可用于创建和构建应用程序的工具和命令：

  • javac ： 您可以使用javac工具及其选项，以读取Java类和接口定义并将其编译为字节码和类文件。

  • javap ： 您使用javap命令反汇编一个或多个类文件。

  • javadoc ： 您使用javadoc工具及其选项，可从Java源文件生成API文档的HTML页面。

  • java ： 您可以使用java命令来启动Java应用程序。

  • jar ： 您可以使用jar命令来为类和资源创建档案，并从档案中操纵或还原单个类或资源。

  • jlink ： 您可以使用jlink工具，用于将一组模块及其依赖项组装和优化为自定义运行时映像。

  • jmod ： 您使用jmod创建JMOD文件并列出现有JMOD文件内容的工具。

  • jdeps ： 您使用jdeps命令以启动Java类依赖关系分析器。

  • jdeprscan ： 您使用jdeprscan作为静态分析工具的工具，它会扫描jar文件（或类文件的某些其他聚合）以查找已弃用的API元素的使用。
