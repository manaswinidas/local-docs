7监控工具和命令

您可以使用Java虚拟机（JVM）监视工具和命令来监视和管理Java应用程序和JVM。

以下各节描述了用于监视和管理Java应用程序和JVM的JDK工具和命令：

  • jconsole ： 您使用jconsole命令以启动图形控制台来监视和管理Java应用程序。

注意：

标识为“ 实验”的工具不受支持，将来的JDK版本中可能不可用。

  • JPS ： 实验 可以使用jps命令以列出目标系统上已检测的JVM。

  • jstat ： 实验性 使用jstat命令以监视JVM统计信息。此命令是实验性的，不受支持。

  • jstatd ： 实验性 的jstatd命令以监视已检测Java HotSpot VM的创建和终止。此命令是实验性的，不受支持。
