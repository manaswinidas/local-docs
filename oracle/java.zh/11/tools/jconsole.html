控制台

您使用jconsole命令以启动图形控制台来监视和管理Java应用程序。

概要

jconsole [-interval=n] [-notile] [-plugin path] [-version] [connection ... ] [-J input arguments]
jconsole -help
-interval

将更新间隔设置为n秒（默认值为4秒）。

-notile

不为两个或多个连接平铺窗口。

-pluginpath path

指定路径jconsole用于查找插件。外挂程式path应该包含一个名为的提供程序配置文件META-INF/services/com.sun.tools.jconsole.JConsolePlugin每个插件包含一行。该行指定实现该类的类的全限定类名。 com.sun.tools.jconsole.JConsolePlugin类。

-version

打印程序版本。

connection = pid | host:port | jmxURL

  • 的pid value是目标进程的进程ID。JVM必须使用与运行JVM的用户ID相同的用户ID运行。 jconsole命令。

  • 的host:port值是运行JVM的主机系统的名称，以及系统属性指定的端口号com.sun.management.jmxremote.port JVM启动时。

  • 的jmxUrl value是要连接的JMX代理的地址，如JMXServiceURL中所述。

-J input arguments

通行证input arguments到其上的JVM jconsole命令运行。

-help要么--help

显示命令的帮助消息。

描述

的jconsole命令启动图形控制台工具，该工具可让您监视和管理本地或远程计算机上的Java应用程序和虚拟机。
