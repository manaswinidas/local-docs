8 Java可访问性实用程序和命令

Java Access Bridge 2.0.2包含Java可访问性实用程序，用于检查有关Java虚拟机（JVM）中的对象和特定Java虚拟机中的组件树的可访问信息。

以下主题描述了Java可访问性实用程序及其命令：

  • jaccessinspector ： 您使用jaccessinspector Java Accessibility Utilities API的可访问性评估工具，用于检查有关Java虚拟机中对象的可访问信息。

  • jaccesswalker ： 您使用jaccesswalker浏览特定Java虚拟机中的组件树，并在树视图中显示层次结构。
