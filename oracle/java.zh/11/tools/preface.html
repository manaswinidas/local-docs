前言

Java平台标准版（Java SE）命令参考描述了Java SE命令的有效选项和参数。在许多情况下，都包含示例以显示正确用法。

听众

本文档适用于希望使用JDK中提供的工具和命令的Java SE开发人员。

约定

本文档中使用以下文本约定：

惯例 含义

黑体字

粗体类型表示与操作或以文本或词汇表定义的术语相关联的图形用户界面元素。

斜体

斜体类型表示要为其提供特定值的书名，重点或占位符变量。

monospace

等宽字体类型表示段落中的命令，URL，示例中的代码，屏幕上显示的文本或您输入的文本。
