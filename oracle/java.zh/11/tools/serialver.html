串行器

您使用serialver命令返回serialVersionUID一个或多个类的形式适合复制到一个不断发展的类中。

概要

serialver [ options ] [ classnames ]
options

这代表了命令行选项serialver命令。请参阅serialver的选项 。

classnames

的课程serialVersionUID将被退回。

描述

的serialver命令返回serialVersionUID一个或多个类的形式适合复制到一个不断发展的类中。在不带参数的情况下调用serialver命令显示用法行。

serialver的选项

-classpath path-files

设置应用程序类和资源的搜索路径。用冒号（:)分隔类和资源。

-Joption

通过指定option到Java虚拟机， option是Java应用程序启动器参考页上描述的选项之一。例如， -J-Xms48m将启动内存设置为48 MB。

笔记

的serialver该命令会在其虚拟机中加载并初始化指定的类，默认情况下，它不会设置安全管理器。如果serialver命令将与不受信任的类一起运行，那么可以使用以下选项设置安全管理器：

-J-Djava.security.manager

必要时，可以使用以下选项指定安全策略：

-J-Djava.security.policy=policy file
