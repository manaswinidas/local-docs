Apache Geronimo Service and Support

Below is a list of companies already providing services and support for Apache Geronimo:

Chariot Solutions

Chariot Solutions is a Java/J2EE architecture and development consulting firm. With deep expertise in enterprise architecture, systems integration, application security, performance tuning, technology mentoring, and open source, Chariot can help make the most of any Geronimo deployment. For more, see how Chariot is involved with Geronimo and visit our recent presentations.

Covalent Technologies

Covalent Technologies offers full commercial development and production-level technical support and professional services for Apache Geronimo. Founded in 1998, Covalent Technologies is the leading provider of products and services for the Apache Tomcat Application Server, Apache Geronimo Application Server, Apache Axis Web Services Framework, and Apache HTTP, the world's leading Web server. Covalent is one of the few sources for full commercial support for Apache, Tomcat, Geronimo and Axis on a global basis.

IBM

As the market leader in application middleware, IBM is an active contributor to the Apache Geronimo project. IBM provides expert technical support through the IBM Support for Apache Geronimo web site and offers developer articles and tutorials through IBM developerWorks. IBM also provides a lightweight J2EE application server based on Apache Geronimo, visit the WebSphere Application Server Community Edition web site for additional information.

Savoir Technologies, Inc

Savoir Technologies specializes in JavaEE and SOA "best in breed" software development and open source solutions. Savoir provides support for Apache Geronimo, ServiceMix, and many other open source initiatives . Savoir offers best practice architecture, design, and training for the Apache Geronimo and ServiceMix platforms.
