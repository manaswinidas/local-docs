November 8, 2007 - - Webinar: An Alternative to Traditional JEE Application Servers

Thursday, November 8, 2007 11:00am PDT (2:00pm EDT) Jason Warner and Erik Craig, Software Developers at IBM and contributors to Apache Geronimo along with Filip Hanik, member of the Covalent Technical Team, PMC member for Apache Tomcat, and contributor for Apache Geronimo present An Alternative to Traditional JEE Application Servers.

The application server market continues to grow and mature. As the needs of organizations change, they are looking to alternative technologies such as open source to address these constantly changing business needs. This webinar will take a look at the application server market today, why there is a need for alternatives, and why you should consider open source. Apache Geronimo will be introduced as one such open source alternative, including how it compares to the popular JBoss, and how you can easily migrate from JBoss to Geronimo.

12/06/06 - - Webcast: Deploying and Configuring a Sample Application in Geronimo
Tim McConnell- Software Engineer for IBM Presents on Apache Geronimo.

This presentation will provide a walkthrough of a sample application, Apache Roller, and demonstration of configuration and deployment to Apache Geronimo. A brief introduction and description of Roller will be given along with a list of prerequisites and versions of what is being used. The steps required for configuring Geronimo for Roller deployment will be shown including Geronimo console steps, Command line steps, and various XML configuration files. We'll conclude by showing a demo of Roller running under Geronimo.

10/25/06 – Webcast: Tomcat to Geronimo

Jeff Genender - Apache Geronimo Committer and PMC member

Have you thought about moving your web applications to an application server, but were afraid to try? Geronimo may be the application server for you! This presentation describes migrating from Tomcat to Geronimo. It covers the differences in the configuration files and the steps required to move your Tomcat based web applications to the Geronimo stack. A walk through of the web console and setting up database pools, security realms, and configuring connectors will show how easy it is to migrate your Tomcat applications, and make the leap into Geronimo.
