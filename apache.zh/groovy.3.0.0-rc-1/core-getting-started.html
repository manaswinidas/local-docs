1。下载

在此下载区域中，您将能够下载发行版（二进制和源代码），Windows安装程序以及Groovy的文档。

要在Mac OSX，Linux或Cygwin上快速轻松地启动，可以使用SDKMAN！ （软件开发工具包管理器）以下载和配置您选择的任何Groovy版本。基本说明可在下面找到。

1.1。稳定

  • 下载zip ： 二进制发行版 | 来源发布

  • 下载文档 ： JavaDoc和压缩的在线文档

  • 二进制/源代码/文档组合包 ： 分发包

您可以在发行说明或变更日志中了解有关此版本的更多信息。

如果您打算使用invokedynamic支持，请阅读这些说明 。

1.2。快照

对于那些想要测试Groovy的最新版本并且生活在最前沿的人，可以使用我们的快照构建 。一旦在我们的持续集成服务器上构建成功，就会将快照部署到Artifactory的OSS快照存储库中。

1.3。先决条件

Groovy 3.0需要Java 6+以及对Java 8的全面支持。使用Java 9快照时，在某些方面当前存在一些已知问题。的groovy-nio模块需要Java 7+。使用Groovy的invokeDynamic功能需要Java 7+，但我们建议使用Java 8。

Groovy CI服务器对于确认不同Groovy版本支持的Java版本也很有用。测试套件（接近10000个测试）针对每个流支持的所有主要Java版本在当前支持的Groovy流中运行。

3。SDKMAN！（软件开发套件管理器）

该工具使在任何Bash平台（Mac OSX，Linux，Cygwin，Solaris或FreeBSD）上安装Groovy非常容易。

只需打开一个新终端并输入：

$ curl -s get.sdkman.io | bash

按照屏幕上的说明完成安装。

打开一个新终端或键入命令：

$ source "$HOME/.sdkman/bin/sdkman-init.sh"

然后安装最新的稳定Groovy：

$ sdk install groovy

安装完成后，将其设置为默认版本，请使用以下命令进行测试：

$ groovy -version

这里的所有都是它的！

5，安装二进制

这些说明描述了如何安装Groovy的二进制发行版。

  • 首先， 下载 Groovy的二进制发行版，并将其解压缩到本地文件系统上的某个文件中。

  • 设置你的GROOVY_HOME环境变量到您解压缩发行版的目录。

  • 加GROOVY_HOME/bin给你PATH环境变量。

  • 设置你的JAVA_HOME环境变量以指向您的JDK。在OS X上，这是/Library/Java/Home ，在其他Unix上通常/usr/java等等。如果您已经安装了Ant或Maven之类的工具，则可能已经完成了此步骤。

现在，您应该已经正确安装了Groovy。您可以通过在命令外壳程序中键入以下命令来测试它：

groovysh

哪个应该创建一个交互式Groovy Shell，您可以在其中键入Groovy语句。或运行Swing交互式控制台类型：

groovyConsole

要运行特定的Groovy脚本类型：

groovy SomeScript
