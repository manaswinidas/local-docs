WSDL转CORBA

名称

wsdl2corba --

概要

wsdltoidl -corba -idl -i <port-type-name> -b <binding-name> -d <output-directory> -w <wsdloutput-file> 
-o <idl-output-file> -props <corba-typemap-namespace> -wrapped -a <corba-address> -f <corba-address-file> -h -v 
-verbose -quiet <wsdlurl>

描述

选件

选项

解释

-科尔巴

生成CORBA绑定。

-idl

从wsdl生成idl。

-一世

指定要使用的端口类型。

-b

指定要使用的绑定。

-d

生成的wsdl / idl所在的目录

-w

wsdl输出文件名。

-o

idl输出文件名。

-道具

corba类型映射名称空间。

-包裹

使用包装器类型生成corba绑定操作，而不是将其展开为单独的参数。

-一种

指定用于corba：address位置属性的值。

-F

使用文件的内容作为corba：address location属性的值。

？，-h，-帮助

显示选项的详细信息。

-v

显示工具的版本号。

-冗长

在代码生成过程中显示注释。

-安静

在代码生成过程中禁止注释。
