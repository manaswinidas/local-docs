工具类

  • Maven cxf-codegen-plugin（WSDL到Java）
  • Maven Java2WSDL插件（仅CXF2.0.x。在2.1中删除，并替换为Java2WS）
  • Maven Java2WS插件
  • 在Maven中使用CXF

  • WSDL到Java
