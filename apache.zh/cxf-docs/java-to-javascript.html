Java到Javascript

名称

java2js --

概要

java2js -databinding <jaxb or aegis> -jsutils -o <output-file> -d <resource-directory> -beans <pathname of 
the bean definition file.>* -cp <class-path> -soap12 -h -v -verbose -quiet <classname>

描述

...

选件

选项

解释

-数据绑定

指定数据绑定（宙斯盾或jaxb）。对于jaxws前端，默认值为jaxb；对于简单前端，默认值为aegis。

-jsutils

在输出的开头编写JavaScript通用实用程序。

-o

指定输出wsdl文件名

-d

输出wsdl和其他资源文件所在的目录

-豆*

指定定义其他Spring Bean的文件的路径名，以自定义数据绑定配置。

-cp 类路径

指定目录和zip / jar文件的SEI和类型类搜索路径

-soap12

指定生成SOAP 1.2绑定

？，-h，-帮助

显示选项的详细信息。

-v

显示工具的版本号。

-冗长

在代码生成过程中显示注释。

-安静

在代码生成过程中禁止注释。
