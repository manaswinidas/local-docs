CXF还支持使用JAX-WS Provider / Dispatch和XML绑定来构建REST样式的Web服务。可以从samples \ restful中找到一个样本。REST服务器提供以下服务：

URL http：// localhost：9000 / customerservice / customers上提供了RESTful客户服务。用户访问此URI来查询或更新客户信息。

对URL http：// localhost：9000 / customerservice / customers的HTTP GET请求返回客户超链接的列表。这允许客户端浏览应用程序状态。返回的XML文档：

<Customers>
  <Customer href="http://localhost/customerservice/customer?id=1234">
      <id>1234</id>
  </Customer>
  <Customer href="http://localhost/customerservice/customer?id=1235"> 
      <id>1235</id>
  </Customer>
  <Customer href="http://localhost/customerservice/customer?id=1236"> 
      <id>1236</id>
  </Customer>
</Customers>

对URL http：// localhost：9000 / customerservice / customers？id = 1234的HTTP GET请求返回一个id为1234的客户实例。返回的XML文档：

<Customer>
  <id>1234</id>
  <name>John</name>
  <phoneNumber>123456</phoneNumber>
</Customer>

对URL http：// localhost：9000 / customerservice / customers的HTTP POST请求，其中包含以下数据：

<Customer>
  <id>1234</id>
  <name>John</name>
  <phoneNumber>234567</phoneNumber>
</Customer>

使用提供的数据更新客户1234。

客户端代码演示了如何使用JAX-WS Dispatch发送带有XML数据的HTTP POST，以及如何使用URL.openStream（）发送HTTP GET。该服务器代码演示了如何通过JAX-WS Provider接口构建RESTful端点。
