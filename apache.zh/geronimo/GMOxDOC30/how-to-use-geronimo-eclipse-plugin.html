如何使用Geronimo Eclipse插件

提供了有关如何使用Apache Geronimo Eclipse插件的视频桌面演示。

在这里，我们将说明GEP的各种有用功能。这将包括添加服务器运行时，添加Java运行时，启动，停止，部署，重新部署，取消部署和其他功能。

在Eclipse中添加目标服务器和Java运行时

  1. 启动Eclipse。默认情况下，将启动JEE透视图。选择服务器选项卡。右键单击并选择New-> Server 。





  2. 在下一个屏幕上，选择Apache-> Apache Geronimo V2.2 Server 。如果您在列表中没有看到Geronimo，请遵循如何安装Geronimo Eclipse Plugin v2.2中提到的步骤。





  3. 在下一个屏幕上，选择Installed JRE，以安装新的Java Runtime Environment。





  4. 这将启动已安装的JRE的窗口，从中可以看到所有已安装的JRE。当前，您只能看到一个已安装的JRE。选择添加 。





  5. 在“ 添加JRE”窗口中，选择“ 标准VM” 。选择下一步 。





  6. 在下一个窗口中，浏览到Installed JRE的根目录。





  7. 完成后，您可以看到JRE主目录和JRE名称字段已填充。选择完成 。





  8. 选中复选框以选择新添加的JRE。选择确定 。





  9. 在“新服务器”窗口上，选择下拉框，然后选择新添加的JRE。





    但是，对于当前的示例，我们将继续使用Java 6。
  10. 接下来，浏览至Geronimo Server安装的根文件夹。如果您没有安装服务器，请选择如图所示的链接以下载geornimo。





  11. 指定所有工件后，选择“下一步”。





  12. 保留接下来2个窗口的默认值，然后选择“下一步”。选择完成。这将在“ 服务器”选项卡下创建一个Geronimo服务器。





    这样就完成了添加服务器的过程

启动和停止服务器

  1. 右键单击Geronimo服务器实例，然后选择Start 。





  2. 服务器将启动，您可以看到状态已更改为“已启动”。





  3. 停止服务器右键单击服务器实例，然后选择“ 停止” 。





    部署运行取消部署并重新部署应用程序

为了说明这些功能，我们将添加一个简单的Web项目，然后与它一起使用以部署/运行/取消部署/重新部署应用程序。

  1. 右键单击Project Explorer下的，然后选择New-> Dynamic Web Project 。





  2. 将项目命名为Hello World 。选择下一步 。





  3. 将默认值保留在下一个窗口中。选择下一步 。





  4. 在Geronimo Deployment Plan窗口上，将Artifact Id命名为Hello。选择完成 。这将添加一个Web项目。





  5. 展开该项目，然后右键单击Web Content 。选择“ 新建”->“ JSP” 。将jsp命名为index.jsp并选择Next 。





  6. 在下一个窗口中，选择完成。





  7. 在index.jsp中，将标题添加为Hello World ，在正文中，添加这是我在G / GEP上的第一个应用程序 。保存更改。





  8. 现在，您已经准备好在Geronimo上运行您的第一个应用程序。右键单击您的项目，然后选择Run-> Run on Server 。





  9. 在下一个屏幕上，选择Apache Geronimo并选择Next 。





  10. 您可以将Hello World视为已配置的项目之一。选择完成 。这将在Geronimo服务器上部署您的应用程序。





  11. 启动Web浏览器，如图所示。





  12. 运行网址http：// localhost：8080 / Hello World。这将运行您的应用程序。



  13. 接下来修改index.jsp并追加Hello World！ 在身体里。






  14. 保存更改后，Geronimo服务器将自动发布Web应用程序。刷新浏览器以验证您所做的更改是否得到体现。





  15. 从服务器取消部署应用程序。右键单击Geronimo服务器实例，然后选择添加和删除项目 。





  16. 在下一个屏幕上，选择Hello World项目，然后选择完成 。这将取消部署 Geronimo的项目。





启动Geronimo控制台和支持

您也可以从Eclipse启动Geronimo Console。

  1. 如图所示，选择您的首选项以启动geronimo控制台。因此，将在Eclipse内部浏览器或外部浏览器中启动geronimo管理控制台。





  2. 右键单击geronimo服务器实例，然后选择启动Geronimo控制台以启动geronimo管理控制台。



  3. 同样，选择Geronimo支持。这将启动Apache Geronimo网站。
