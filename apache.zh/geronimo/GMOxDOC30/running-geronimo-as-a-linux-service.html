将Geronimo作为Linux服务运行
编辑页面 浏览空间 添加页面 新增新闻

本文将向您展示如何配置Apache Geronimo服务器以作为Linux服务运行。

软件先决条件

要将Apache Geronimo作为Linux服务运行，您需要在Linux上安装Geronimo服务器。请遵循《 快速入门》中有关安装Apache Geronimo的说明。在本文的其余部分中，安装目录将称为 。

安装Geronimo服务

确保您以root用户身份登录系统。然后我们需要一个Linux Shell脚本register-service运行Geronimo服务器作为服务。该文件默认位于/bin 。

现在，您可以通过从以下命令运行以下命令，从命令行将Geronimo作为服务运行： /bin ：

./register-service add <service_name>

哪里是您为服务指定的名称。您可以为服务器创建多个服务名称。

将Geronimo服务器作为服务安装后，该服务将在下次启动操作系统时自动启动。

管理已安装的Geronimo服务

启动，停止或重新启动服务

您可以通过运行以下命令来启动服务：

<service_name> start

同样，停止服务b运行此命令：

<service_name> stop

重新启动服务b运行以下命令：

<service_name> restart

哪里是服务的名称。

检查服务状态

要查找Geronimo服务的状态，请运行以下命令：

<service_name> status

哪里是服务的名称。

查找服务的映射目录

如果在服务器上安装了多个服务，则可能要查找每个服务的映射目录。运行以下命令以查找映射目录：

./register-service list <service_name>

哪里是服务的名称。

删除Geronimo服务

要删除Apache Geronimo服务，请运行以下命令：

./register-service delete <service_name>

哪里是您要删除的服务的名称。
