配置Geronimo Eclipse插件以将POJO项目发布为共享库
编辑页面 浏览空间 添加页面 添加新闻

使用Web应用程序说明了如何配置GEP以将POJO项目发布为共享库，并添加一个POJO项目作为依赖项。为了创建一个新的Web项目并将其对POJO项目的依赖关系作为shareld库发布，请执行以下步骤：

  1. 按照开发Hello World Web应用程序中的说明创建一个HelloWorld Dynamic Web项目。
  2. 创建一个POJO项目，如下所示：
    • 单击文件 -> 新建 -> 项目 。
    • 选择Java Project ，然后单击Next 。
    • 输入CurrencyConverterPojo作为项目名称，然后单击完成 。（如果要求切换到Java透视图，则可以选择No ）。
    • 在Project Explorer视图中，右键单击CurrencyConverterPojo ，选择New- > Class 。
    • 输入myPackage作为包裹名称， CurrencyConverter作为类名，然后单击完成 。
    • 替换内容CurrencyConverjer.java包含CurrencyConverter.java中的内容。
    • 按Ctrl + S保存更改。
  3. 如下所示，将POJO项目添加到Web应用的Java构建路径中：
    • 在Project Explorer视图中，右键单击HelloWorld ，然后单击Properties 。
    • 单击“ Java构建路径” ，然后切换到“ 项目”选项卡。
    • 点击添加 。选择CurrencyConverterPojo作为要添加的项目，然后单击“ 确定” 。再次单击确定 。
