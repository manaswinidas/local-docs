创建和使用自定义服务器程序集

在本节中，我们讨论如何使用Geronimo Eclipse插件来创建和使用定制服务器程序集。在开始之前，您必须安装Eclipse和Geronimo Eclipse插件 ，并在Eclipse中定义Geronimo服务器和服务器运行时 。

创建自定义服务器程序集

为了创建定制服务器程序集，您必须已经安装并启动了Geronimo Server。

  1. 双击Geronimo服务器将在主窗口中打开服务器编辑器。
  2. 选择“ 插件”页面并单击“ 创建自定义程序集”将弹出“ 新服务器自定义程序集”对话框，如下所示。
  3. 选择要从服务器提取的相关插件或插件组。可在“ 插件组”页面上找到用于服务器的项目。
  4. 输入相关信息，然后选择您希望在新服务器中拥有的所有项目。按Ctrl键可以在列表中选择多个项目。注意，必须选择Geronimo Assembly :: Boilerplate或Geronimo插件组:: Framework才能创建工作服务器。



    组装好的服务器将放置在您指定的目录下。一种.tar球或.zip文件也将放置在/var/temp正在运行的服务器的目录，以便于分发。

使用服务器程序集

现在我们已经创建了一个定制的服务器程序集，是时候使用它了。停止并卸载原始服务器（通过选择Windows-> Preferences-> Server并删除服务器运行时环境，这很容易做到。有关详细信息，请参见定义Geronimo服务器运行时和服务器页面。使用与安装原始服务器相同的方法，唯一的区别是。这次，“ Application Server安装目录”的位置应该是新创建的服务器被提取到的位置。



现在，代替开始时使用的一整套Geronimo服务器，您应该拥有一个有效且自定义的Geronimo Server进行开发。
