本文演示了如何配置JMS资源组以及如何使用Geronimo管理控制台将其卸载。

创建一个新的JMS资源组

创建JMS资源组意味着在Geronimo上部署模块连接器，该模块连接器至少包括2个资源适配器，一个连接工厂和一个目标。

您可以使用以下步骤来创建JMS资源组：

  1. 从JMS资源portlet中选择For ActiveMQ设置向导。在设置页面上，输入JMS资源组的属性信息，然后单击Next 。

  2. 在下一页上，可以分别单击添加连接工厂或添加目标按钮来选择添加连接工厂或队列目标。

  3. 如果选择添加Connection Factory ，则会看到以下向导页面。从下拉菜单中选择JMS Factory的类型，然后单击Next 。

  4. 在下一页上，提供连接工厂的属性，然后单击“ 下一步” 。在此示例中，连接工厂的名称为ActiveMQConnectionFactory 。

  5. 现在，您可以在此页面上选择添加连接工厂，添加队列目标，显示此操作的计划或部署新的连接工厂。根据要添加的资源适配器的类型，单击“ 添加连接工厂”或“ 添加目标”按钮，以添加所需的资源适配器。单击立即部署以部署新的连接工厂。

  6. 您的新连接工厂应该正在运行，并准备创建新连接。

从控制台卸载JMS资源

  1. 在左侧导航面板上，选择J2EE连接器 。
  2. 单击要卸载的JMS资源后面的“卸载”，例如， console.jms/jmsResources/1.0/car 。
