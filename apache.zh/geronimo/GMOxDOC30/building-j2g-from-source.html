从源头建造

如果您只想使用J2G迁移应用程序，则本文可能不适合您，因为本文旨在作为从源代码编译J2G的说明。有关下载J2G二进制副本的信息，请访问Geronimo开发工具页面。

从最新资源下载并构建J2G允许对工具集进行修改和添加，前提条件和说明如下。

先决条件

  • Apache Maven（2.0.6+）
  • 颠覆（1.2+）
  • JDK 5.0+（J2SE 1.5.0+）

获取源代码

安装必备组件后，就该从存储库中签出代码了。为此，请从命令行使用“ svn co”命令。当前，j2g的存储库为https://svn.apache.org/repos/asf/geronimo/devtools/j2g

svn co http://svn.apache.org/repos/asf/geronimo/devtools/j2g/trunk j2g

建造

从根J2G目录：

mvn install assembly:assembly

禁用测试

要在构建时禁用单元测试：

mvn install -Dtest=false assembly:assembly

部署中

构建该工具后，可以使用以下步骤进行部署：

  1. 解压在/ target中创建的可部署文件
  2. 在命令行上，导航到可部署目录中的bin目录
  3. 为您的操作系统运行适当的j2g-configure脚本
