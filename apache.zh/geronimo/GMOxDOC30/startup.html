启动

启动脚本在后台启动Apache Geronimo服务器。该命令具有以下语法：

startup [options]

可用的选项有：

- 安静
禁止正常的启动进度栏。通常在将控制台输出重定向到文件或从IDE或其他工具启动服务器时使用。

- 长
将启动进度以适合于将控制台输出重定向到文件或从IDE或其他工具启动服务器的格式写入控制台（如果不指定，则不使用换行来更新默认情况下使用的进度信息--quiet或--long）。

-v-详细
将控制台日志级别设置为INFO ，从而导致控制台输出比通常显示的更多。

-vv-非常详细
将控制台日志级别设置为DEBUG ，从而获得更多的控制台输出。

-override [ configId ]
覆盖中的配置/var/config.xml这样就只会启动命令行中列出的配置。注意，许多J2EE
功能取决于要启动的某些配置，因此您应非常小心忽略的内容。-override之后的所有参数均假定为配置名称。

启动命令也可以通过使用Java的罐子命令启动：

java -Djava.endorsed.dirs=lib/endorsed -javaagent:bin/jpa.jar -jar bin/server.jar [options]
