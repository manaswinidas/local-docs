How to get daily Geronimo Eclipse Plugin builds
Edit Page   Browse Space   Add Page   Add News

Daily builds are available and can be downloaded from Geronimo Eclipse Plugin nightly build site http://people.apache.org/dist/geronimo/eclipse/unstable/

  • Find updatesite and deployable zip files in their release number directory, eg, 2.1.3.
  • The updates directory is an Eclipse update site, with the full contents of the production GEP update site http://www.apache.org/dist/geronimo/eclipse/updates/, as well as those of releases currently under development.
    • To install the nightly build via the Eclipse update manager from this image, define a remote update site in Eclipse with this URL: *http://people.apache.org/dist/geronimo/eclipse/unstable/updates/.
    • For details, see Installing GEP using Eclipse update manager, using the above URL for the Eclipse update manager site.

The available builds should also be used with all official releases of Apache Geronimo.

In case you find any issues see How to open a bug for Geronimo Eclipse Plugin.
