本教程将带您完成在Apache Geronimo中针对已经部署在服务器上的Web服务开发，部署和测试RESTful Web Service客户端所需的步骤。

我们将创建一个基于Web的客户端，该客户端可以通过GET和POST方法访问RESTful Web服务。您可以轻松创建此客户端，而无需使用服务器环境以外的任何外部工具。

要运行本教程，至少需要您安装以下必备软件。

  • Sun JDK 5.0+（J2SE 1.5）
  • Apache Geronimo 2.x
  • 适用于Java EE开发人员的Eclipse IDE-Europa版本
  • Geronimo Eclipse插件2.x

“ 开发环境”部分中提供了有关安装Eclipse的详细信息。

部署和测试Web客户端

部署

  • 右键单击服务器视图中存在的Apache Geronimo Server Runtime，然后选择“ 添加和删除项目”

  • 将jaxws-rest-converterclient添加到已配置的项目列表中，然后单击“ 完成”。



  • 等待一段时间，直到服务器状态更改为“已同步”

测试中

  • 右键单击存在于我们项目的WebContent目录下的index.jsp ，然后选择“运行方式”->“在服务器上运行”

  • 在弹出窗口中，选中复选框“ 在运行项目时始终使用此服务器” ，然后单击“ 完成”。

  • 现在，Eclipse将尝试在Web浏览器中打开jsp，该浏览器向您显示一个表格，以美元为单位输入金额。

  • 输入任何金额，然后按Submit ，jsp将显示Web服务返回的结果。
