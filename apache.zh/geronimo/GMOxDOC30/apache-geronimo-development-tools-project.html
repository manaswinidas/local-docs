Geronimo Eclipse插件（GEP）

Geronimo Eclipse插件（GEP）在Eclipse ，其Web工具平台（WTP）项目和Geronimo之间提供了集成，从而允许用户利用WTP功能来为Apache Geronimo应用程序服务器开发，部署，调试和运行应用程序。

GEP3.0-M1尚未发布，完成发布工作后将更新此部分。

Geronimo Eclipse插件的当前版本是v2.2，于2010年2月2日发布。有关详细信息，请参见：

  • 公告
  • 2.2发行说明

先决条件

  • Java J2SE 1.6.0或更高版本
  • Java EE开发人员的Eclipse IDE（（Helios（Eclipse 3.6）或更高版本）
    • Eclipse Helios下载
  • Apache Geronimo 3.0-M1服务器
    • Apache Geronimo下载

问题与疑问

有关GEP的问题可以在Geronimo 邮件列表中提出 。
Eclipse Platform或WTP问题应在Eclipse Bugzilla下打开。
与Eclipse，WTP或任何Eclipse Project有关的其他问题应在Eclipse Newsgroups上提出。

每晚构建

每晚构建都放在Eclipse更新站点http://people.apache.org/dist/geronimo/eclipse/unstable/updates上 。

该站点的结构和内容与生产Eclipse更新站点http://apache.org/dist/geronimo/eclipse/updates/相同 ，并添加了当前正在开发的GEP版本的最新夜间版本（trunk）。

与生产站点一样，可以在http://people.apache.org/dist/geronimo/eclipse/unstable/中找到updatesite和可部署的zip文件。

JBoss到Geronimo（J2G）迁移工具

JBoss到Geronimo的迁移工具（J2G）是一个Eclipse插件，旨在帮助将为JBoss应用程序服务器或Java 2 Enterprise Edition（J2EE）编写的应用程序源迁移到Apache Geronimo平台。可以在此处找到更多信息。
