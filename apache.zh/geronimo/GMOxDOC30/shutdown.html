关闭脚本停止Apache Geronimo服务器

。该命令具有以下语法：

关机[选项]

可用的选项有：

- 用户
指定有权停止服务器的用户名。默认情况下，您通常使用system作为用户名。

- 密码
指定刚输入的用户名的密码。默认情况下，您通常将使用管理员作为密码。

- 港口
指定要连接到服务器的RMI命名端口（例如JMX连接端口）。默认情况下，使用端口1099 。

- 主办
指定要尝试停止的服务器的主机名。默认情况下使用本地主机 。此参数仅在Geronimo 2.1.3和更高版本中可用。

- 安全
使用安全通道与JMX服务器通信，有关详细信息，请参见#Security 。此参数仅在Geronimo 2.1.3和更高版本中可用。

如果未指定任何参数，则此命令将提示您输入用户名和密码，并将连接到本地主机上的端口1099。

还可以使用java -jar命令启动shutdown命令：

java -Djava.endorsed.dirs = lib / endorsed -jar bin / shutdown.jar [选项]

如果您具有通过deploy命令的login选项保存的凭据，则无需指定或重新输入用户名和密码。
