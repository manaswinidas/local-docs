总览

Geronimo管理控制台是一个基于Web的界面，它提供了一种方便，用户友好的方式来管理Geronimo服务器的许多方面。启动Apache Geronimo服务器后（请参阅启动和停止服务器 ），可以通过将浏览器导航到以下URL来访问管理控制台：

http：// localhost：8080 / console

通过将system指定为用户名并以manager作为密码登录。登录后，您将被重定向到控制台的欢迎页面。控制台有两个主要区域，左侧是导航面板，右侧是主要内容区域。

左侧的导航面板使您可以轻松访问各个任务。它始终存在，并且可以轻松地从一个任务过渡到另一个任务。右侧的主要内容区域是实际工作所在的位置。当您在左侧的导航面板中选择其他任务时，此区域将更改。每个视图都包含一个或多个portlet，这些portlet通常在标题中包含一个帮助链接。请参见下图作为示例。

Web服务器Web服务器

此选项提供两个portlet，即Web服务器管理器和网络侦听器 。

  • Web服务器管理器
    启用此portlet后，它将收集服务器性能统计信息。被测量的值是：
    • 要求总数
    • 总计1xx，2xx，3xx，4xx，5xx响应
    • 活动请求计数（当前，低，高）
    • 要求期限
  • 网络监听器
    此portlet显示Jetty和Tomcat Web容器的所有连接器侦听器（HTTP，HTTPS和AJP）和状态。您可以从此portlet添加新的侦听器，并更改现有连接器的状态（停止，重新启动，编辑，删除）。

数据库管理员

此选项提供两个portlet， DB VIewer和Run SQL 。

  • 数据库查看器
    此portlet显示所有可用的数据库及其表，并显示表的内容。要查看特定数据库中的表，请单击数据库链接。这将显示数据库中的所有表。要查看表的内容，请单击“ 查看目录”链接。在DB Viewer portlet的底部，有一些链接可以带您回到View Databases或View Tables 。
  • 运行SQL
    通过此portlet，您可以运行SQL命令来处理表中的数据，创建新数据库或删除数据库。您可以在“ SQL命令”文本框中输入SQL命令，选择将要使用的数据库，然后单击“ 运行SQL”以执行命令。

Web App战争Web App战争

此portlet显示已安装的Web应用程序及其状态。从此portlet，您可以停止，启动和卸载可用的Web应用程序。此portlet提供了以下两个选项来帮助管理Web应用程序：

  • 专家模式 ，启用组件的所有操作。可以将这种模式理解为替代功能，该功能授予您执行在组件上执行选项（停止，重新启动和卸载）的权限，否则这些组件将变灰并可能导致应用程序服务器发生故障。您可以通过选择以下专家模式复选框来使用此模式。
  • 显示父项和子项选项 ，该选项显示已安装的Web应用程序的父项和子项。父组件是指已安装的Web应用程序所依赖的组件，而子组件是依赖于已安装的Web应用程序的组件。您可以通过选择以下复选框来使用此选项。

服务器日志服务器日志

此选项提供以下portlet： 日志管理器 ， 服务器日志查看器和Web访问日志查看器 。

  • 日志管理器
    该portlet允许用户选择用于记录的配置文件和/或更改日志级别和刷新间隔。缺省的“配置文件”是标准的log4j文件，它定义了Geronimo Server将记录Geronimo输出的位置。配置文件还定义了日志级别，可以更好地控制要记录的内容。
  • 服务器日志查看器
    此portlet显示Geronimo服务器日志，对于调试Geronimo Console和已部署的应用程序中的问题很有帮助。
  • Web访问日志查看器
    此portlet显示并过滤Web容器日志文件。该日志文件包含HTTP访问以及来自Geronimo上运行的JSP和servlet应用程序的消息。默认情况下，此portlet显示当前日期的日志文件的内容。您可以应用不同的过滤条件以仅查看您感兴趣的数据。

  • Derby日志查看器

JMX查看器

此portlet提供了类似树的表示形式，供您浏览不同类型的MBean。每个MBean将被表示为一个树形节点，该树形节点显示其对象名称表示。所有MBean都会为您提供按其域分组的MBean列表。

LDAP查看器

通过此portlet，您可以连接到任何LDAP服务器，并浏览其内容，但不能对其进行编辑。该Portlet是预定义的，可以访问Geronimo中嵌入的LDAP，在连接到服务器之前，请确保该服务正在运行。

ClassLoader查看器

该视图可用于服务器中涉及的所有类加载器及其加载的类。

JNDI查看器

该视图可用于查看各种模块的JNDI上下文。

依赖查看器
