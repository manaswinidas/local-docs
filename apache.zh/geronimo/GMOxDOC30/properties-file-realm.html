属性文件安全领域

此领域类型使您可以配置Web应用程序以对其进行身份验证。

输入安全领域的名称，然后从“ 领域类型：”菜单中选择“ 属性文件领域 ”。单击下一步，然后按照向导进行操作。

通过管理控制台创建新的属性文件领域时，提供的键值位于配置选项字段中，该字段是用户和组属性文件的位置和文件名，例如：

usersURI = var / security / vhost1_users .properties
groupsURI = var / security / vhost1_groups .properties

缺省情况下，服务器用来验证管理用户以进行控制台和部署者访问的安全领域是geronimo-admin属性文件领域。
