安装OSGi应用程序开发工具
编辑页面 浏览空间 添加页面 添加新闻

用于Eclipse的OSGi应用程序开发工具可帮助您开发和构建动态的模块化OSGi应用程序。您可以：

  • 创建和编辑OSGi捆绑软件，组合捆绑软件和应用程序
  • 导入和导出OSGi捆绑软件，复合捆绑软件和应用程序
  • 将现有的Java EE资产转换为OSGi捆绑包
  • 创建和编辑OSGi蓝图配置文件
  • 调试和诊断您的捆绑软件和Aries应用程序

使用OSGi应用程序开发工具之前，您必须已安装GEP。有关如何操作，请参阅安装Geronimo Eclipse插件 。

安装OSGi应用程序开发工具

请按照以下说明为Eclipse安装OSGi工具：

  1. 从安装目录启动Eclipse。
  2. 在主菜单中，单击下载>下载OSGi应用程序开发工具 。
  3. 请按照新打开的网页上的说明进行操作，或参考以下步骤作为快速入门。
    1. 单击“ 帮助”菜单下的“ 安装新软件 ”。
    2. 单击添加以在后续窗口中添加新的存储库位置。
    3. 类型http://public.dhe.ibm.com/ibmdl/export/pub/software/rational/OSGiAppTools在位置字段中，然后单击确定 。
    4. 在可用软件列表中选择OSGi Application Development ，然后单击Next 。
    5. 阅读许可协议后，单击我接受许可协议的条款。
    6. 单击“ 确定”获取任何警告消息，然后选择“ 立即重新启动”以激活OSGi工具。

卸载OSGi应用程序开发工具

在安装更新版本的OSGi应用程序开发工具之前，您必须卸载以前安装的插件：

  1. 启动Eclipse。
  2. 在主菜单中，单击帮助>安装新软件 。
  3. 在“安装”对话框中，单击“ 已经安装了什么？”。 链接。
  4. 在“ Eclipse安装详细信息”窗口中的“已安装的软件”选项卡上，按住Ctrl键，然后单击以下功能：
    • OSGi应用程序开发工具
    • OSGi应用程序开发工具UI
    • OSGi上下文相关帮助
    • OSGi帮助文档
    • OSGi帮助文档
  5. 点击卸载 ，然后点击
  6. 单击完成以从Eclipse中除去功能部件。
