在Geronimo中，提供了一组预定义的图形和视图用于服务器监视，并且添加了一个新帐户监视器 ，该帐户监视器仅具有查看为服务器监视定义的图形和视图的权限。

使用新添加的帐户监视服务器

在使用新帐户监视服务器之前，必须使用默认帐户和密码检查先决条件 ：系统/管理器。

先决条件要求预定义图先决条件

要开始监视服务器，必须首先检查要监视的服务器在服务器列表中。通过使用默认帐户和密码：system / manager登录到管理控制台后，单击“ 服务器”类别下的“ 监视”可以查看该列表。如果服务器不在服务器列表中，则需要添加服务器 。一旦服务器显示在服务器列表中，您就可以查看信息，例如IP地址或主机名，状态（在线/禁用），快照线程（运行/停止）和快照持续时间。

在使用新添加的帐户监视服务器之前，请确保要监视的服务器的状态为联机且快照线程正在运行。如果服务器的状态为禁用，请单击与服务器关联的启用链接以启用服务器。如果快照线程已停止，请单击与服务器关联的“ 启用查询”链接以启用快照查询。启用快照查询后，监视代理将立即开始以默认的5分钟间隔拍摄统计信息快照，可通过单击与服务器关联的“ 编辑”链接来更改该默认间隔。

如何使用新帐户

请按照以下步骤使用新帐户监视器监视服务器。该帐户的默认密码是password 。

  1. 使用用户名Monitor和密码password登录到管理控制台。页面左侧的导航区域仅包含服务器类别，在该类别下有四个功能：
    • 信息显示服务器已激活多长时间，并显示有关启动服务器时Java环境的信息。
      注意 ：IE无法在管理控制台中显示JVM内存使用情况图，因为不再支持Adobe SVG插件。
    • Java系统信息显示Java环境中的当前系统属性。
    • 线程池显示公共线程池的状态。
    • 监视可帮助您查看许多服务器统计信息并监视Web服务器性能。
  2. 单击监视以查看服务器列表，并使用已定义的图形和视图监视服务器性能。

注意 ：此帐户仅有权查看已定义的图形/视图。使用用户名系统和密码管理器登录到管理控制台以执行任何配置，包括启用/禁用快照收集，添加/编辑视图或图形。有关如何监视服务器性能的更多信息，请参阅监视Geronimo Server上的组件 。

图形预定义图

下表列出了为服务器监视预定义的一组图形及其定义：

  • 当前JVM堆大小-JVM的当前堆大小

  • 活动请求计数 -连接器的活动连接数
  • 忙线程最大值 -用于连接器忙的最大线程数
  • 忙线程电流 -为连接器当前忙碌线程数
  • 最小繁忙线程 —连接器的最小繁忙线程数
  • 接收的字节数—连接器接收的字节数
  • 发送的字节数-连接器发送的字节数
  • 错误计数 -连接器检测到的错误数
  • 打开的连接电流 -为连接器当前打开的连接数
  • 打开的连接最大值 -用于连接器打开的连接的最大数目
  • 打开的连接闽 -用于连接器打开连接的最小数量
  • Request Time CurrentTime-连接器的当前请求处理时间
  • Request Time MaxTime-连接器的最大请求处理时间
  • Request Time MinTime-连接器的最小请求处理时间
  • Request Time TotalTime-连接器的总请求处理时间

  • SSL活动请求计数 -SSL连接器的活动连接数
  • 当前SSL繁忙线程-SSL连接器的当前繁忙线程数
  • SSL繁忙线程数最大值 -SSL连接器的最大繁忙线程数
  • SSL繁忙线程数最小值 -SSL连接器的最小繁忙线程数
  • 收到的SSL字节 -通过SSL连接接收的字节数
  • 发送的SSL字节数-SSL连接器发送的字节数
  • SSL错误计数 -SSL连接器检测到的错误数
  • SSL当前打开连接数-SSL连接器当前打开的连接数
  • SSL打开连接数上限-SSL连接器的最大打开连接数
  • SSL打开连接数最小值 -SSL连接器的最小打开连接数
  • SSL请求时间CurrentTime -SSL连接器的当前请求处理时间
  • SSL Request Time MaxTime -SSL连接器的最大请求处理时间
  • SSL Request Time MinTime -SSL连接器的最小请求处理时间
  • SSL Request Time TotalTime -SSL连接器的总请求处理时间
