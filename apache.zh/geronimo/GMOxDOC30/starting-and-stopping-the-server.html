通过控制台启动和停止Geronimo

有一些命令行替代方法可以启动服务器。从命令行窗口或终端将目录更改为/ bin并运行startup要么geronimo run要么geronimo start命令。有关这些命令的所有选项，请参阅“ 工具和命令”部分的“ 启动和Geronimo” 。

要停止服务器，您必须打开一个新的命令行窗口或终端，将目录更改为/ bin并运行shutdown命令。系统将提示您输入用户名和密码。或者，您可以在Geronimo运行的终端上按Ctrl + C以终止该进程。有关这些命令的可用选项，请参阅“ 工具和命令”部分的“ 关机”和“ Geronimo” 。

如果服务器已经在运行，则可以选择使用Geronimo管理控制台简介来远程关闭服务器。在服务器启动并运行的情况下，打开Web浏览器并指向以下URL以访问控制台：

http：// localhost：8080 / console

登录到Geronimo管理控制台，然后在左侧的“控制台导航”菜单上单击“ 服务器” >“ 关闭 ”。

当单击服务器管理器Portlet中的“ 关闭”按钮时，系统将提示您再次确认服务器关闭。单击“ 确定”确认并关闭服务器。管理控制台中的连接将丢失，您需要从终端或命令行窗口重新启动服务器。
