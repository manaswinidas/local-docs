本教程将带您完成在Apache Geronimo中开发，部署和测试EJB无状态会话Bean Web服务所需的步骤。完成本教程之后，您应该能够了解如何使用Eclipse开发环境在Apache Geronimo中开发简单的，与JAX-WS兼容的EJB Web服务。

强烈建议您在进入本教程之前，先阅读“ 开发JAX-WS POJO Web服务”教程。我们将使用教程中的相同示例来开发EJB Web服务。

客户发展

本教程不包括客户机开发，因为为EJB Stateless Session Bean Web服务和POJO Web服务创建客户机没有区别。

您可以参考“ 开发JAX-WS POJO Web服务”教程，以获取有关如何为Web Services开发客户端的更多知识。

要运行本教程，至少需要您安装以下必备软件。

  1. Sun JDK 6.0+（J2SE 1.6）
  2. 面向特定平台的Java EE开发人员的Eclipse IDE
  3. Apache Geronimo Eclipse插件2.1.x
  4. Apache Geronimo服务器2.1.x

    在本教程中使用了Geronimo 2.1.x版，Java 1.5运行时和Eclipse Ganymede，但是可以使用其他版本（例如Geronimo 2.2版，Java 1.6，Eclipse Europa）

开发环境部分中提供了有关安装Eclipse的详细信息。本教程将带您完成以下步骤：

创建Web服务实现代码

为了部署Converter EJB Service，我们将创建一个远程接口和一个实现该接口的无状态会话bean。所需步骤如下：

  1. 右键单击ejbModule并选择New- > Package




  2. 将包命名为org.apache.geronimo.samples.jaxws并单击“ 完成”。




  3. 右键单击新软件包，然后选择“ 新建->接口”




  4. 将接口命名为Converter并使用org.apache.geronimo.samples.jaxws包，然后单击“ 完成”。




  5. 将以下代码添加到Converter类： solid Converter.java package org.apache.geronimo.samples.jaxws;导入java.math。BigDecimal;导入javax.ejb。远程;导入javax.jws。网络服务; @Remote @WebService（name =“ ConverterPortType”，targetNamespace =“ http://jaxws.samples.geronimo.apache.org”）公共接口转换器{public BigDecimal dollarToRupees（BigDecimal dollars）; public BigDecimal rupeesToEuro（BigDecimal rupees）; }

  6. 右键单击新包，然后选择新建->类




  7. 将类命名为ConverterBean并使用org.apache.geronimo.samples.jaxws包，然后单击“ 完成”。




  8. 将以下代码添加到ConverterBean类： 固态 ConverterBean.java 包org.apache.geronimo.samples.jaxws;导入java.math。BigDecimal;导入javax.ejb。*;导入javax.jws。网络服务; @Stateless @WebService（serviceName =“ Converter”，portName =“ ConverterPort”，endpointInterface =“ org.apache.geronimo.samples.jaxws。Converter“，targetNamespace =” http://jaxws.samples.geronimo.apache.org“）公共类ConverterBean实现Converter {private BigDecimal rupeeRate = new BigDecimal（” 40.58“）; private BigDecimal euroRate = new BigDecimal（” 0.018368“） ; public BigDecimal dollarToRupees（BigDecimal dollars）{BigDecimal result = dollars.multiply（rupeeRate）; return result.setScale（2，BigDecimal。围捕）; }公开BigDecimal rupeesToEuro（BigDecimal rupees）{BigDecimal结果= rupees.multiply（euroRate）;返回result.setScale（2，BigDecimal。围捕）; }}

部署和测试Web服务

现在，我们将研究在没有任何客户端的情况下部署和测试Web服务所涉及的步骤。

部署

  1. 右键单击服务器视图中存在的Apache Geronimo Server Runtime，然后选择“ 添加或删除项目”

  2. 在弹出对话框中，选择jaxws-converterejb项目，然后单击“ 添加”。




  3. 确保jaxws-converterejb在已配置的项目列表中，然后单击“ 完成”。




  4. 等待一段时间，直到服务器状态更改为已同步。 部署时发生错误

    如果您在部署时看到任何错误，例如无法读取WSDL文件null ，则只需忽略它们。



测试中

  1. 将应用程序部署到服务器上后，启动浏览器并转到以下URL：

    http：// localhost：8080 / Converter / ConverterPortType
