Apache Geronimo v3.0
新功能和增强
编辑页面 浏览空间 添加页面 新增新闻

此版本中进行了许多改进，增强和错误修复。后面部分列出的项目是最重要的。

功能增强和增加

Java EE 6完整的Profile支持

Geronimo V3.0提供对Java EE 6完整概要文件规范的全面支持。

OSGi支持

Geronimo V3.0为OSGi软件包的部署和管理提供支持。

白羊座编程模型支持

Geronimo V3.0提供对Aries编程模型的支持，例如EBA（企业捆绑包存档）和CBA（综合捆绑包存档）。

Karaf Shell命令扩展

Geronimo V3.0提供了用于服务器管理的扩展gogo命令。

可用性改进

导航项目快速搜索框

搜索框用于更轻松，快速地查找菜单项。

新的OSGi捆绑包管理Portlet

新的OSGi portlet用于为服务器安装，搜索和管理OSGi捆绑软件。您可以查看绑定到捆绑软件的软件包或服务。有关更多详细信息，请参见管理OSGi捆绑软件 。

新的EBA portlet

新的EBA portlet用于管理EBA应用程序的方式与处理传统Java EE应用程序的方式相同。

开发工具改进

支持最新的Eclipse版本

GEP可以安装在Eclipse Helios（也称为Eclipse 3.6）上。

OSGi工具安装

OSGi开发的蓝图编辑器
