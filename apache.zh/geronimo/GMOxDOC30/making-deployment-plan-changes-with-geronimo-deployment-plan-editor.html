使用Geronimo部署计划编辑器更改部署计划
编辑页面 浏览空间 添加页面 添加新闻

本文介绍了如何使用Geronimo部署计划编辑器来配置Geronimo部署计划。

要在Geronimo Deployment Plan Editor中修改部署计划，至少需要安装以下必备软件。

  • Sun JDK 6.0+（J2SE 1.6）
  • Eclipse 3.4.1（Eclipse Ganymede SR1发行版）或Eclipse 3.5（Eclipse Galileo）
  • Apache Geronimo v2.2
  • Geronimo Eclipse插件v2.2

在本教程的其余部分中，我们将考虑使用Currency Converter示例。

  1. 在WebEJB项目中，选择WebContent- > WEB- INF- > geronimo-web.xml ，右键单击并选择Open With- > Geronimo Deployment Plan Editor 。



  2. 选择“ 命名”选项卡，您可以配置Web应用程序的所有引用。



  3. 如有必要，可以在安全选项卡中添加安全选项。
  4. 在“ 部署”选项卡中，您可以编辑Web应用程序的所有服务器依赖项或类加载器配置。



  5. 完成后，打开“ 源”选项卡，您可以看到部署计划的源代码。
  6. 在CurrencyConvertEJB项目中，可以选择ejbModule- > META- INF- > openejb-jar.xml ，右键单击并选择Open With- > Geronimo Deployment Plan Editor 。



  7. 同样，您可以更改openejb-jar.xml在编辑器中。完成配置后，打开“ 源”选项卡，您可以看到部署计划的源代码。
