欢迎来到Apache Geronimo开发工具子项目

该子项目的目的是为Geronimo提供一套丰富的开发工具，重点是应用程序开发，迁移和IDE集成。当前，这些工具是：

Geronimo Eclipse插件（GEP）

Geronimo Eclipse插件（GEP）在Eclipse ，其Web工具平台（WTP）项目和Geronimo之间提供了集成，从而允许用户利用WTP功能来为Apache Geronimo应用程序服务器开发，部署，调试和运行应用程序。

Geronimo Eclipse插件的当前版本为v3.0.1，于2013年6月3日发布。有关详细信息，请参见：

  • 公告
  • 3.0.1发行说明

先决条件

  • Java J2SE 1.6.0或更高版本
  • 适用于Java EE开发人员的Eclipse IDE（Indigo（Eclipse 3.7.x）或Juno（Eclipse 4.2.x））
    • Eclipse Indigo下载
    • Eclipse Juno下载
  • Apache Geronimo 1.1.x，2.0.x，2.1.x，2.2.1和3.0.x服务器
    • Apache Geronimo下载

每晚构建

每晚构建都放在Eclipse更新站点http://people.apache.org/builds/geronimo/eclipse/nightly/中 。

该站点的结构和内容与生产Eclipse更新站点http://apache.org/dist/geronimo/eclipse/updates/相同 ，并添加了当前正在开发的GEP版本的最新夜间版本（trunk）。

与生产站点一样，可以在http://people.apache.org/builds/geronimo/eclipse/nightly/中找到updatesite和可部署的zip文件。

JBoss到Geronimo的迁移工具（J2G）

JBoss到Geronimo的迁移工具（J2G）是一个Eclipse插件，旨在帮助将为JBoss应用程序服务器或Java 2 Enterprise Edition（J2EE）编写的应用程序源迁移到Apache Geronimo平台。可以在此处找到更多信息。

先决条件

  1. Sun JDK 5.0+（J2SE 1.5）
  2. 带有JDT Core插件的Eclipse SDK 3.3
  3. Apache Geronimo 1.1.1及更高版本
  4. 为JBoss 3.2和更高版本或J2EE 1.2和更高版本编写的应用程序
