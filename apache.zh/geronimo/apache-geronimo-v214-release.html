Apache Geronimo v2.1.4-2009年3月31日发布





Apache Geronimo v2.1.4服务器已通过SUN的JEE5认证测试套件。提供Windows和Unix（包括Linux，Mac OS X）操作系统的下载。每个Geronimo服务器软件包都包括两个可能的Web容器之一，即Tomcat或Jetty。请选择下载名称中带有首选Web容器的下载。Geronimo完全支持Tomcat和Jetty。有关每个背景的更多信息，请参见Tomcat主页和Jetty主页 。

小小G

Apache Geronimo的此发行版包括Web容器和Geronimo其他模块的子集。这不是兼容Java EE 5的服务器，并且对于希望使用特定功能实现较小占用空间的用户有吸引力。Little-G完全支持Tomcat 6和Jetty 6。
