2006-12-22
Apache Geronimo 2.0-M1已发布，可供用户下载。

这个版本的Geronimo代表了我们向Java Enterprise Edition 5.0 Server运行时迈进的时间点。它包括最新的Jetty和Tomcat Servlet引擎，因此用户可以尝试Servlet 2.5，JSP 2.1以及JSF 1.1和JSTL 1.2。还包括Apache的Java持久性OpenJPA以及最新的Javamail 1.4和Java激活框架。这是一个里程碑版本，因此不建议用于高负载部署。Apache Geronimo团队非常希望获得用户反馈，因此我们将继续努力实现“五驱动”，从而满足您的需求。

2006-12-22
Apache Geronimo 1.2-beta可供用户下载和评估。

Apache Geronimo 1.2包含几个新功能。它们是全局JNDI支持，作为持久性选项的OpenJPA，EJB的改进性能以及对CMP实体bean的新锁定支持。此版本尚未通过TCK。还包含了Apache Yoko，我们正在Java 5虚拟机上对该发行版进行认证。请参阅下载页面以获取下载信息。

2006-11-13
Geronimo Eclipse插件v1.2.0已发布，可供用户下载。

Geronimo Eclipse插件v1.2.0主要是一个错误修复版本，提高了稳定性并增加了对Geronimo 1.1.1版本的支持。有关下载信息，请参见“ 开发工具”部分。

2006-10-12
Geronimo v1.1.1通过Azul Compute Appliances认证

Apache Geronimo项目很高兴宣布在Azul Compute Appliance上运行的Apache Geronimo v1.1.1的J2EE 1.4认证。测试是在1920B型Azul Compute Appliance模型（8芯片/ 192核，64GB内存）和Azul VM版本2.3.1.2（HotSpotJava 1.4.2_12），64位服务器模式下进行的。Apache Geronimo项目希望感谢Azul Systems在用于该认证工作的Azul Compute Appliance上提供了机器时间。有关Azul计算设备的更多信息，请访问http://www.azulsystems.com 。

加入Apache Geronimo开发人员社区的成员，以获取技术概述并解释Apache Geronimo v1.1.1（来自Apache Software Foundation的功能齐全，经过全面认证的J2EE服务器）中的功能。您将学习如何下载，安装和运行Geronimo。您将获得管理控制台的演练，该控制台提供了易于使用的用户界面，并了解了新的插件架构，该架构提供了简单的用户体验。

Geronimo v1.1.1已发布，可供用户下载

Apache Geronimo v1.1.1主要是一个错误修复版本，因此它解决了v1.1版本中报告的大多数错误。除了这些修复程序，此版本还提供了可用性方面的改进，例如：
