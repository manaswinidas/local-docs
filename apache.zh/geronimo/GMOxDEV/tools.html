总览

该页面列出了Apache Geronimo项目使用的各种工具。并非所有开发人员都使用每种工具。几乎可以肯定缺少一些工具。如果您是提交者，则可以直接编辑此页面；如果您不是提交者，则请发送邮件至dev@geronimo.apache.org或user@geronimo.apache.org（如果要添加geronimo使用的工具）。

Apache工具

Geronimo由Maven构建。

开源工具

http://www.eclipse.org/

Geronimo开发人员使用了各种IDE。其中之一是Eclipse IDE。

商业工具

http://www.yourkit.com/

YourKit，LLC为YourKit Java Profiler提供免费许可证，以开放源代码项目。YourKit Java Profiler主要用于调查Apache Geronimo中的性能问题。确定性能瓶颈特别有用。

http://msdn.microsoft.com/

微软恳请有需要的Apache提交者免费提供MSDN许可证。
