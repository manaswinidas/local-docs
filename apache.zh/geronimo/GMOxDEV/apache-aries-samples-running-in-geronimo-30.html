在Geronimo 3.0中运行的Apache Aries示例
编辑页面 浏览空间 添加页面 添加新闻


可以在Apache Geronimo 3.0-SNAPSHOT中安装和运行多个Apache Aries示例EBA。这些是启用OSGi的Web应用程序和其他捆绑包，它们以OSGi指定的方式利用诸如Blueprint，JNDI，JDBC，JPA和JTA之类的功能。样本打包在EBA中以进行部署。当前的示例是Aries Blog和AriesTrader。

您可以在此处找到有关样本的更多详细信息：

  • 白羊座博客样本
  • AriesTrader示例

获取Geronimo 3.0-M1服务器

您可以从以下位置下载“ Geronimo未经认证的Web容器”（Tomcat或Jetty）的图像：
Geronimo 3.0-M1下载页面

您必须使用Web容器映像，而不是服务器的“ Little-G”（最小）版本。Little-G版本不包含用于部署Apache Aries示例的必要组件。

请遵循Apache Geronimo文档解压缩并启动服务器映像（提示：解压缩或解压缩服务器映像，并使用“ ./bin/geronimo.sh run”之类的内容来启动服务器）。

获取白羊座样本

有多种选择可获取Apache Aries EBA示例。

  1. 您可以选择获取源代码并自己构建示例。如果您将样本用作学习工具来构建自己的应用程序，这将很有用。
  2. 您也可以选择仅从最新的Apache Aries示例发行版下载EBA，而无需构建Apache Aries。

无论哪种情况，都可以在以下位置找到必要的下载文件（源或EBA工件）：
最新的Apache Aries版本

构建Apache Aries示例

如果您选择不构建样本，则可以跳过以部署Aries样本数据源
下载并解压缩最新的Aries示例的Apache Aries版本的源zip并构建项目。

cd samples-0.1-incubating/
mvn install

此过程将从最新版本及其依赖项中提取二进制文件。

您真正感兴趣的是Blog和AriesTrader的示例EBA，可以在以下位置找到：

  • 博客jdbc样本-样本0.1-孵化/博客/博客-jdbc-eba / target / org.apache.aries.samples.blog.jdbc.eba-0.1-incubating.eba
  • Blog JPA样本-样本0.1-incubating / blog / blog-jpa-eba / target / org.apache.aries.samples.blog.jpa.eba-0.1-incubating.eba
  • AriesTrader jdbc样本-samples-0.1-incubating / ariestrader / assembly / ariestrader-jdbc-eba / target / org.apache.aries.samples.ariestrader.jdbc-0.1-incubating.eba
  • AriesTrader所有样本-samples-0.1-incubating / ariestrader / assemblies / ariestrader-all-eba / target / org.apache.aries.samples.ariestrader.all-0.1-incubating.eba

部署白羊座样本数据源

两个样本中的每个样本都需要一个数据源。但是，在部署数据源之前，您需要获取tranql-connector-derby-embed-xa rar。不幸的是，它没有包含在Geronimo 3.0-M1服务器映像中（但是将包含在以后的映像中）。

  • 对于3.0-M1版本，您可以从任何ibiblio镜像中获取必要的rar： 用于derby的Tranql RAR
  • 您还必须下载用于数据源定义的XML-位于以下位置： aries-datasource.xml 。

在本地文件系统中拥有两个必需文件后，使用以下命令部署数据源：

./bin/deploy -u system -p manager deploy tranql-connector-derby-embed-xa-1.6.rar aries-datasource.xml

部署示例EBA

使用Geronimo Web控制台或以下命令部署Blog或AriesTrader EBA（注意：您一次只能部署每个样本的一个版本-例如，仅jdbc Blog或jpa Blog样本，而不能同时部署）：

Blog jdbc
./bin/deploy -u system -p manager deploy org.apache.aries.samples.blog.jdbc.eba-0.1-incubating.eba
Blog jpa
./bin/deploy -u system -p manager deploy org.apache.aries.samples.blog.jpa.eba-0.1-incubating.eba
AriesTrader jdbc
./bin/deploy -u system -p manager deploy org.apache.aries.samples.ariestrader.jdbc-0.1-incubating.eba
AriesTrader all
./bin/deploy -u system -p manager deploy org.apache.aries.samples.ariestrader.all-0.1-incubating.eba

运行样本

现在，您应该能够按以下方式访问样本：

  • 博客-http：// localhost：8080 / blog
  • AriesTrader- http：// localhost：8080 / ariestrader

有关如何与样本和样本结构进行交互的详细信息，请参考以下链接：

  • 白羊座博客样本 -特别参考“关于博客样本”部分。
  • AriesTrader示例 -特别参考“访问和使用AriesTrader示例”以及随后的“关于AriesTrader示例”部分。
