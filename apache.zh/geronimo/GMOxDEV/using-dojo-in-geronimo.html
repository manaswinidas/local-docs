什么是道场？

Dojo是一种流行的开源AJAX库，根据BSD和Academic Free许可提供。它提供了许多有用的浏览器小部件，例如树，表等。Dojo当前被引入Daytrader应用程序和Geronimo管理控制台中，以增强用户界面。有关Dojo的更多信息，请参见http://dojotoolkit.org/ 。

Dojo通常在Web应用程序中如何使用？

AJAX开发人员通常通过在其Web应用程序中复制其静态文件（javascript，css，gifs等）并从其servlet和JSP引用这些文件的方式，将Dojo库合并到其应用程序中。这种方法的缺点是，每个应用程序都有一个单独的AJAX库副本，从而增加了服务器的总体占用空间，并阻止了浏览器使用其缓存中库文件的一个副本。另一个缺点是AJAX库不能独立于包含它的Web应用程序进行升级或管理。例如，跨集群部署的Web应用程序可能需要从中央位置提供静态Dojo文件。将Dojo文件托管在单独的Webapp中可以帮助解决这些问题。

如何在Geronimo中使用Dojo？

Geronimo的Dojo插件为Web应用程序提供了Dojo文件的共享副本。安装该插件后，Dojo文件将在上下文“ / dojo”中可用，用于从该服务器中部署的任何Web应用程序中引用该库的HTML如下所示：

Geronimo管理控制台中的调试视图使用Dojo插件。因此，如果已安装并激活了调试视图，则Dojo文件应该已经在服务器上的“ / dojo”上下文中可用。否则，您可以使用deploy.sh命令行工具或管理控制台中的Plugins portlet手动安装Dojo插件。

我怎么知道Geronimo中有哪个版本的Dojo？

要查看服务器上当前可用的Dojo版本，请在http：// localhost：8080 / dojo / dojo / dojo.js中查找“ dojo.version”。
