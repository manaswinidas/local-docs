描述

服务器计划和通用元素

http://geronimo.apache.org/xml/ns/deployment-1.2

用于在独立计划中的Geronimo中部署新服务，并且还包含许多其他计划使用的通用元素。

Geronimo插件描述符

http://geronimo.apache.org/xml/ns/plugins-1.3

Geronimo插件上的元数据或可用Geronimo插件的列表。

安全映射

http://geronimo.apache.org/xml/ns/security-2.0

其他计划使用的通用安全元素。

安全领域

http://geronimo.apache.org/xml/ns/loginconfig-2.0

用于配置安全领域和登录模块GBean的缩写语法。您可以手动配置多个GBean或为该领域声明一个GBean，使用它来配置所有登录模块。

命名

http://geronimo.apache.org/xml/ns/naming-1.2

引用其他组件（EJB，数据库池，JMS资源，J2EE连接器，Web服务等）的公共元素。

主键生成器

http://openejb.apache.org/xml/ns/pkgen-2.1

为CMP实体bean配置主键生成器的缩写语法。避免手动配置和连接PK生成器GBean。

CORBA CSS配置

http://openejb.apache.org/xml/ns/corba-css-config-2.1

用于为通过CORBA访问远程EJB的客户端配置安全性的缩写语法。

CORBA TSS配置

http://openejb.apache.org/xml/ns/corba-tss-config-2.1

为通过CORBA公开的EJB配置安全性的缩写语法。

config.xml

http://geronimo.apache.org/xml/ns/attributes-1.2

的格式var/config/config.xml文件。

Tomcat Web App配置

http://geronimo.apache.org/xml/ns/web/tomcat/config-1.0

如果您使用通用（ geronimo-web-2.0.xsd ）网络应用程序配置中，您可以在container-config元素中使用这些元素来配置特定于Tomcat的行为。

Jetty Web App配置

http://geronimo.apache.org/xml/ns/web/jetty/config-1.0.1

如果您使用通用（ geronimo-web-2.0.xsd ）网络应用程序配置，您可以在container-config元素中使用这些元素来配置Jetty特定的行为。
