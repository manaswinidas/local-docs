介绍

Server元素代表整个Catalina Servlet容器。因此，它必须是conf/server.xml配置文件。它的属性代表了整个servlet容器的特征。

属性

共同属性

Server的所有实现均支持以下属性：

属性描述
className

使用的Java类名称。此类必须实现org.apache.catalina.Server接口。如果未指定类名，则将使用标准实现。

address

该服务器等待关闭命令的TCP / IP地址。如果未指定地址， localhost用来。

port

该服务器等待关闭命令的TCP / IP端口号。调成-1禁用关闭端口。

注意：当使用Apache Commons Daemon启动Tomcat（在Windows上作为服务运行，或者在un * xes上使用jsvc运行）时，禁用关闭端口非常有效。但是，当使用标准shell脚本运行Tomcat时，不能使用它，因为它将阻止shutdown.bat | .sh和catalina.bat | .sh正常停止运行。

portOffset

适用于的偏移量port以及任何嵌套连接器的端口。它必须是一个非负整数。如果未指定，则默认值为0用来。

shutdown

为了关闭Tomcat，必须通过与指定端口号的TCP / IP连接接收的命令字符串。

utilityThreads

此服务将用于各种实用程序任务（包括重复执行的任务）的线程数。特殊值0将导致值Runtime.getRuntime().availableProcessors()正在使用。负值将导致Runtime.getRuntime().availableProcessors() + value除非小于1，否则将使用1个线程。预设值为1。

标准实施

Server的标准实现是org.apache.catalina.core。标准服务器 。它支持以下其他属性（除了上面列出的常见属性之外）：

属性描述
utilityThreadsAsDaemon

为实用程序线程设置守护程序标志值。默认值为false 。

periodicEventDelay

此值表示在此服务器上配置的生命周期侦听器的定期生命周期事件调用之间的秒数延迟。该值以秒为单位，并且负数或零值将禁用调用。如果未指定，则此属性的默认值为10秒。

嵌套组件

以下组件可能嵌套在Server元素内：

  • 服务 -一个或多个服务元素。
  • GlobalNamingResources-为服务器配置JNDI全局资源。

特殊功能

没有与服务器关联的特殊功能。
