支持的操作概述

本文档定义了可以对Tomcat管理应用程序支持的受管对象执行的受支持操作 。并非所有操作都必须通过实现的每个管理应用程序可用。但是，如果有给定的操作可用，则应与此处的描述保持一致。

以下受管理对象描述了受支持的操作：

  • 存取记录器
  • 连接器
  • 语境
  • 默认上下文
  • 发动机
  • 环境进入
  • 主办
  • JDBC资源
  • 装载机
  • 经理
  • 领域
  • 请求过滤器
  • 服务器
  • 服务

存取记录器

从特定的访问记录器的角度来看，应该可以执行以下管理操作：

  • 导航到拥有的引擎 ， 主机或上下文 。
  • 编辑此对象的可配置属性。

连接器

从特定连接器的角度来看，应该可以执行以下管理操作：

  • 导航到所属服务 。
  • 编辑此对象的可配置属性。

语境

从特定上下文的角度来看，应该可以执行以下管理操作：

  • 导航到所属的主机 。
  • 编辑此对象的可配置属性。
  • 创建并配置与此对象关联的新的Access Logger 。
  • 编辑关联的访问记录器的可配置属性。
  • 删除关联的访问记录器 。
  • 创建和配置与此对象关联的新环境条目 。
  • 选择并编辑关联的环境条目的可配置属性。
  • 删除关联的环境条目 。
  • 创建和配置与此对象关联的新JDBC资源 。
  • 选择并编辑关联的JDBC资源的可配置属性。
  • 删除关联的JDBC资源 。
  • 创建并配置与此对象关联的新Loader 。
  • 编辑关联的Loader的可配置属性。
  • 卸下相关的装载机 。
  • 创建并配置与此对象关联的新Manager 。
  • 编辑关联的Manager的可配置属性。
  • 删除关联的Manager 。
  • 创建和配置与此对象关联的新领域 。
  • 编辑关联的Realm的可配置属性。
  • 删除关联的领域 。
  • 创建和配置与此对象关联的新请求过滤器 。
  • 选择并编辑关联的请求过滤器的可配置属性
  • 删除关联的请求过滤器 。

默认上下文

从特定的默认上下文的角度来看，应该可以执行以下管理操作：

  • 导航到拥有的Engine或Host 。
  • 编辑此对象的可配置属性。
  • 创建和配置与此对象关联的新环境条目 。
  • 选择并编辑关联的环境条目的可配置属性。
  • 删除关联的环境条目 。
  • 创建和配置与此对象关联的新JDBC资源 。
  • 选择并编辑关联的JDBC资源的可配置属性。
  • 删除关联的JDBC资源 。

发动机

从特定引擎的角度来看，应该可以执行以下管理操作：

  • 导航到所属服务 。
  • 编辑此对象的可配置属性。
  • 创建并配置与此对象关联的新的Access Logger 。
  • 编辑关联的访问记录器的可配置属性。
  • 删除关联的访问记录器 。
  • 创建和配置与此对象关联的新的默认上下文 。
  • 编辑关联的默认上下文的可配置属性。
  • 删除关联的默认上下文 。
  • 创建并配置与此对象关联的新主机 。
  • 选择并编辑关联主机的可配置属性。
  • 删除关联的主机 。
  • 创建和配置与此对象关联的新领域 。
  • 编辑关联的Realm的可配置属性。
  • 删除关联的领域 。
  • 创建和配置与此对象关联的新请求过滤器 。
  • 选择并编辑关联的请求过滤器的可配置属性
  • 删除关联的请求过滤器 。

环境进入

从特定的环境条目的角度来看，应该可以执行以下管理操作：

  • 导航到所属上下文或默认上下文 。
  • 编辑此对象的可配置属性。

主办

从特定主机的角度来看，应该可以执行以下管理操作：

  • 导航到所属引擎 。
  • 编辑此对象的可配置属性。
  • 创建并配置与此对象关联的新的Access Logger 。
  • 编辑关联的访问记录器的可配置属性。
  • 删除关联的访问记录器 。
  • 创建并配置与此对象关联的新上下文 。
  • 选择并编辑关联Context的可配置属性。
  • 删除关联的Context 。
  • 创建和配置与此对象关联的新的默认上下文 。
  • 编辑关联的默认上下文的可配置属性。
  • 删除关联的默认上下文 。
  • 创建和配置与此对象关联的新领域 。
  • 编辑关联的Realm的可配置属性。
  • 删除关联的领域 。
  • 创建和配置与此对象关联的新请求过滤器 。
  • 选择并编辑关联的请求过滤器的可配置属性
  • 删除关联的请求过滤器 。

JDBC资源

从特定的JDBC资源的角度来看，应该可以执行以下管理操作：

  • 导航到所属上下文或默认上下文 。
  • 编辑此对象的可配置属性。

装载机

从特定加载程序的角度来看，应该可以执行以下管理操作：

  • 导航到所属上下文 。
  • 编辑此对象的可配置属性。

经理

从特定经理的角度来看，应该可以执行以下管理操作：

  • 导航到所属上下文 。
  • 编辑此对象的可配置属性。

领域

从特定领域的角度来看，应该可以执行以下管理操作：

  • 导航到拥有的引擎 ， 主机或上下文 。
  • 编辑此对象的可配置属性。

请求过滤器

从特定的请求过滤器的角度来看，应该可以执行以下管理操作：

  • 导航到拥有的引擎 ， 主机或上下文 。
  • 编辑此对象的可配置属性。

服务器

从整个服务器的角度来看，应该可以执行以下管理操作：

  • 编辑此对象的可配置属性。
  • 创建并配置与此对象关联的新服务 。
  • 选择并编辑关联服务的可配置属性。

服务

从特定服务的角度来看，应该可以执行以下管理操作：

  • 导航到所属服务器 。
  • 编辑此对象的可配置属性。
  • 创建并配置与此对象关联的新连接器 。
  • 选择并编辑关联连接器的可配置属性。
  • 删除关联的连接器 。
  • 创建并配置与此对象关联的新引擎 。
  • 编辑关联的Engine的可配置属性。
  • 删除关联的引擎 。
