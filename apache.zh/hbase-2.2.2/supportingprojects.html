配套项目

此页面是与HBase相关的项目的列表。要添加您的项目，请归档文档JIRA或通过电子邮件将相关信息发送给hbase-dev 。如果您发现过时的信息，请使用相同的方式进行报告。

这些项目是用户提交的，HBase团队对其准确性不承担任何责任。

向HBase添加新功能的项目

mis弥斯
Themis基于Google的Percolator在HBase上提供跨行/跨表交易。
特非拉
Cask Tephra在Apache HBase之上提供全局一致的事务。
海印莎
Haeinsa是用于HBase的线性可扩展多行，多表事务库。
HBase TA聚合器
一个HBase协处理器，用于基于时间序列的聚合。
Apache Trafodion
Apache Trafodion是一种Webscale-SQL-on-Hadoop解决方案，支持Hadoop上的事务或操作工作负载。
阿帕奇凤凰
Apache Phoenix是基于HBase的关系数据库层，作为客户端嵌入式JDBC驱动程序提供，其目标是针对HBase数据的低延迟查询。
色相HBase浏览器
随Hue发行的HBase简易而强大的WebUI。
HBase SEP
HBase副作用处理器，一个基于HBase复制的异步可靠地监听HBase突变事件的系统。
Lily HBase索引器
通过侦听复制流（使用HBase SEP）将HBase内容索引到Solr。
症结
-HBase报告和分析，支持简单和组合键，获取和范围扫描，基于列的过滤，图表。
奥米德
-在HBase之上提供快照隔离的无锁事务支持。
帕里
HBase的ORM
HBase的作家
Heritrix2处理器，用于将爬网写入HBase。
皮吉项目
Pigi项目是一个类似于ORM的框架。它包括一个可配置的索引系统和一个简单的对象到HBase的映射框架（如果需要，还可以为HBase进行索引）。设计供Web应用程序使用。
HBase节俭
hbase-thrift为HBase生成并安装Perl和Python Thrift绑定。
欧姆
OHM是用于HBase的弱关系ORM，它提供对象映射和列索引。它具有自己的编译器，能够生成多种语言的接口代码。当前是C＃（通过Thrift API），并且正在开发中的Java支持。该编译器很容易扩展，以添加对其他语言的支持。
数据存储
旨在使用HBase而不是bigtable在Java中实现Google应用引擎数据存储区的实现。
数据核
DataNucleus是Java JDO / JPA / REST实现。它支持HBase和许多其他数据存储。
昆德拉
Kundera是用于HBase，Cassandra和MongoDB的基于JPA 2.0的对象-数据存储映射库。
佐姆格
Zohmg是使用HBase作为其后备存储的时间序列数据存储。
Grails支持
Grails HBase插件。
大唱片
是Ruby on Rails的基于active_record的对象映射层。
hbase-星际之门
HBase Stargate的Ruby客户端。
聚会。比尼奥
聚会。Beeno是基于注释的简单HBase Java“ beans”映射框架。它包括一个基本的高级查询API，该API生成适当的服务器端过滤器。
春季Hadoop
-Spring Hadoop项目为编写Apache Hadoop应用程序提供支持，这些应用程序受益于Spring，Spring Batch和Spring Integration的功能。
Spring Framework HBase模板
Spring Framework HBase模板提供的HBase数据访问模板与Spring中为JDBC，Hibernate，iBatis等提供的模板类似。如果您觉得有用，请投票将其包含在Spring Framework中。
Clojure-HBase
一个方便从Clojure访问HBase的库。
HBase索引库
一个用于构建和查询基于HBase表的索引的库。
HBasene
Lucene + HBase-使用HBase作为Lucene所需的TF-IDF表示的后备存储。此外，还包含一个用于从HBase模式构造Lucene索引的库。
JMX工具包
HBase量身定制的JMX工具包，可使用Cacti进行监视并使用Nagios或类似工具进行检查。
信息库
IHBASE通过索引区域来提供更快的扫描，每个区域都有其自己的索引。索引列是用户定义的，索引可以在单个查询中相交或连接。
HBASE EC2脚本
这一系列的bash脚本使您能够在Amazon的Elastic Compute Cloud（EC2）服务上运行HBase群集，并结合最佳实践。
星际之门
Stargate提供增强的RESTful接口。
HBase的trx
HBase-trx提供了HBase的事务（JTA）和索引扩展。
HBase Thrift Python客户端Debian软件包
用于HBase Thrift Python客户端的Debian软件包（请参阅自述文件以获取sources.list设置）
卡普雷
capjure是HBase的持久性助手。它是用Clojure语言编写的，并且支持本机哈希映射的持久化。
HBaseHUT
（HBase的高更新吞吐量）它着重于记录更新期间的写入性能（避免在每个更新记录上执行Get操作）。
HBaseWD
即使它们的键是连续的，HBase Writes Distributor也会在整个群集上分布记录，同时仍然允许对其进行快速范围扫描
HBase UI工具和实用程序
HBase UI工具和实用程序是HBase UI客户端和简单的util模块。它可以像jdo（不是持久性api）一样更轻松地处理hbase
