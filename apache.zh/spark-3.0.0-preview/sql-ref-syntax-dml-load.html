加载数据

描述

LOAD DATA语句将数据从用户指定的目录或文件加载到表中。如果指定了目录，则将加载该目录中的所有文件。如果指定了文件，则仅加载单个文件。另外LOAD DATA语句采用可选的分区规范。指定分区时，数据文件（当输入源是目录时）或单个文件（当输入源是文件时）都将加载到目标表的分区中。

句法

参量

path
文件系统的路径。它可以是绝对路径，也可以是相对路径。
table_name
现有表的名称。
PARTITION ( partition_col_name = partition_col_val [ , ... ] )
指定一个或多个分区列和值对。
LOCAL
如果指定，它将导致INPATH将针对本地文件系统而不是默认文件系统（通常是分布式存储）进行解析。
OVERWRITE
默认情况下，新数据将附加到表中。如果OVERWRITE使用时，该表将被新数据覆盖。
