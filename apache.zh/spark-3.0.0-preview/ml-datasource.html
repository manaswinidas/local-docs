数据源

在本节中，我们介绍如何在ML中使用数据源加载数据。除了诸如Parquet，CSV，JSON和JDBC之类的常规数据源外，我们还为ML提供了一些特定的数据源。

目录

  • 图像数据源
  • LIBSVM数据源

图像数据源

该图像数据源用于从目录加载图像文件，它可以通过以下方式将压缩图像（jpeg，png等）加载为原始图像表示形式： ImageIO在Java库中。加载的DataFrame有一个StructType列：“图像”，包含存储为图像架构的图像数据。的架构image列是：

  • 起源： StringType （代表图像的文件路径）
  • 高度： IntegerType （图像的高度）
  • 宽度： IntegerType （图像的宽度）
  • nChannels： IntegerType （图像通道数）
  • 模式： IntegerType （OpenCV兼容类型）
  • 数据： BinaryType （以与OpenCV兼容的顺序显示图像字节：大多数情况下按行BGR）

LIBSVM数据源

这个LIBSVM数据源用于从目录加载“ libsvm”类型的文件。加载的DataFrame有两列：包含以double形式存储的标签的标签和包含以Vectors存储的特征向量的特征。列的模式为：

  • 标签： DoubleType （代表实例标签）
  • 特征： VectorUDT （代表特征向量）
