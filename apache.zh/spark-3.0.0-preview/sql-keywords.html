Spark SQL关键字

什么时候spark.sql.ansi.enabled是的，Spark SQL有两种关键字：

  • 保留关键字：保留的关键字，不能用作表，视图，列，函数，别名等的标识符。
  • 非保留关键字：仅在特定上下文中具有特殊含义并且可以在其他上下文中用作标识符的关键字。例如， SELECT 1 WEEK是一个间隔文字，但WEEK可以在其他地方用作标识符。

什么时候spark.sql.ansi.enabled如果为假，Spark SQL有两种关键字：

  • 非保留关键字：与以下定义相同spark.sql.ansi.enabled=true 。
  • 严格非保留关键字：非保留关键字的严格版本，不能用作表别名。

默认spark.sql.ansi.enabled是错误的。
