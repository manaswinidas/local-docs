更改表

描述

ALTER TABLE语句更改表的架构或属性。

改名

ALTER TABLE RENAME语句更改数据库中现有表的表名。

句法

参量

old_table_name
现有表的名称。
db_name
现有数据库的名称。
new_table_name
用来重命名表的新名称。
partition_spec
要重命名的分区。

加栏

ALTER TABLE ADD COLUMNS语句将提到的列添加到现有表中。

句法

参量

table_name
现有表的名称。
COLUMNS (col_spec)
指定要添加的要重命名的列。

设置和取消设置

设置表属性

ALTER TABLE SET命令用于设置表属性。如果已经设置了特定的属性，则此属性会用新属性覆盖旧值。

ALTER TABLE UNSET用于删除表属性。

句法

设置序列

ALTER TABLE SET命令用于在Hive表中设置SERDE或SERDE属性。如果已经设置了特定属性，
这会用新值覆盖旧值。

句法

设置位置和设置文件格式

ALTER TABLE SET该命令还可以用于更改现有表的文件位置和文件格式。

句法

参量

table_name
现有表的名称。
PARTITION (part_spec)
指定必须在其上设置属性的分区。
SERDEPROPERTIES (key1=val1, key2=val2, ...)
指定要设置的SERDE属性。
