描述

的DESCRIBE QUERY语句用于返回查询输出的元数据。速记DESC可以代替DESCRIBE描述查询输出。

句法

参量

QUERY
该子句是可选的，可以省略。
input_statement
指定结果集产生语句，并且可以是以下之一：
  • 一种SELECT声明
  • 一种CTE(Common table expression)声明
  • 一个INLINE TABLE声明
  • 一种TABLE声明
  • 一种FROM声明
请参阅select语句以获取查询参数的详细语法。
