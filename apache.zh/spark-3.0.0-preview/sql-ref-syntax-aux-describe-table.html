描述表

描述

DESCRIBE TABLE语句返回表的基本元数据信息。元数据信息包括列名称，列类型和列注释。可选地，可以指定分区规范或列名称，以分别返回与分区或列有关的元数据。

句法

参量

format
指定描述输出的可选格式。如果指定了“ EXTENDED”，则返回其他元数据信息（例如父数据库，所有者和访问时间）。
table_identifier
指定一个表名，该表名可以可选地用数据库名限定。

句法： [database_name.]table_name
partition_spec
一个可选参数，用于指定分区的键和值对的逗号分隔列表。指定后，将返回其他分区元数据。

句法： PARTITION (partition_col_name = partition_col_val [ , ... ])
col_name
可选参数，用于指定需要描述的列名称。提供的列名称可以是可选的限定。参数`partition_spec`和`col_name`是互斥的，不能一起指定。当前不允许指定嵌套列。

句法： [database_name.][table_name.]column_name
