分析表

描述

的ANALYZE TABLE语句收集有关查询优化器将使用的表的统计信息，以查找更好的查询执行计划。

句法

参量

table_name
现有表的名称。
PARTITION ( partition_col_name [ = partition_col_val ] [ , ... ] )
指定一个或多个分区列和值对。分区值是可选的。
[ NOSCAN | FOR COLUMNS col [ , ... ] | FOR ALL COLUMNS ]
  • 如果未指定分析选项， ANALYZE TABLE收集表的行数和大小（以字节为单位）。
  • 诺斯坎
    仅收集表的大小（以字节为单位）（不需要扫描整个表）。
  • 对于列col [，...] |对于所有列
    收集指定的每个列（或每个列）的列统计信息以及表统计信息。
