显示功能

描述

应用可选的正则表达式模式后，返回函数列表。给定Spark支持的功能数量非常大，可以将此语句与describe函数结合使用，以快速找到功能并了解其用法。的LIKE子句是可选的，仅在与其他系统兼容时才受支持。

句法

参量

function_kind
指定要搜索的函数的名称空间。有效的名称空间是：
  • USER-在用户定义的功能中查找功能。
  • SYSTEM-在系统定义的功能中查找功能。
  • ALL-在用户定义的功能和系统定义的功能之间查找功能。
function_name
指定系统中现有功能的名称。函数名称可以可选地用数据库名称限定。如果`function_name`具有数据库资格，则从用户指定的数据库解析该函数，否则从当前数据库解析该函数。

句法： [database_name.]function_name
regex_pattern
指定用于限制语句结果的正则表达式模式。
  • 只允许使用*和`|作为通配符模式。
  • 除`*`和`|`外，其余模式遵循regex语义。
  • 在处理之前，将在输入模式中修剪前导和尾随的空白。
