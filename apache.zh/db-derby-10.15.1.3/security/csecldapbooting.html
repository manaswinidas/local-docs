引导LDAP服务器

首先，启动OpenDS QuickSetup JNLP（Java Web Start）安装程序，然后按照安装步骤设置目录服务器。

您可以通过单击“使用QuickSetup安装”链接从https://opends.java.net/获得安装程序。

在此安装过程中，您将指定一个密码，我们将其称为YOUR_SELECTED_PASSWORD 。

接下来，将一些凭据加载到目录服务器中。下载此凭证示例文件： http : //today.java.net/today/2007/03/22/secArticle.LDIF 。现在，使用加载到目录服务器import-ldif中的工具bin OpenDS安装目录。（在导入凭据时，请确保OpenDS不在运行；否则，您将收到一条错误消息，指示导入实用程序无法获取存储锁定。）

import-ldif --backendID userRoot --ldifFile secArticle.LDIF

现在，通过运行以下命令启动OpenDS服务器： start-ds中的脚本bin OpenDS安装目录。
