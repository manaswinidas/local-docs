其他服务器命令

其他服务器命令（ shutdown ， ping ， sysinfo ， runtimeinfo ， logconnections ， maxthreads ， timeslice ， trace和tracedirectory ）作为客户端实施，并且就SSL而言，它们的行为与客户端完全相同。

有关更多信息，请参见使用SSL / TLS运行客户端 。

SSL模式是使用属性设置的derby.drda.sslMode或服务器命令选项-ssl 。

例子

以下命令将关闭启用SSL的服务器：

java -jar derbyrun.jar server shutdown -ssl basic

同样，如果您有peerAuthentication在两侧，使用以下命令：

java -Djavax.net.ssl.keyStore=clientKeyStore.key \
-Djavax.net.ssl.keyStorePassword=qwerty \
-Djavax.net.ssl.trustStore=clientTrustStore.key \   
-Djavax.net.ssl.trustStorePassword=qwerty \
-jar derbyrun.jar server shutdown -ssl peerAuthentication
