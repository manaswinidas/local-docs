编程应用程序以进行Derby用户身份验证

要将用户身份验证编程到Derby应用程序中，请使用DriverManager.getConnection调用以指定用户名和密码。

应用程序可以通过以下方式提供用户名和密码。

  • 另外，作为以下方法签名的参数： getConnection(String url, String user, String password) 
    Connection conn = DriverManager.getConnection(
    "jdbc:derby:myDB", "mary", "little7xylamb");
  • 作为数据库连接URL的属性：
    Connection conn = DriverManager.getConnection(
    "jdbc:derby:myDB;user=mary;password=little7xylamb");
  • 通过设置user和password物业Properties对象以及其他连接URL属性：
    Properties p = new Properties();
p.put("user", "mary");
p.put("password", "little7xylamb");
Connection conn = DriverManager.getConnection(
    "jdbc:derby:myDB", p);
注意：密码未加密。在服务器框架的环境中使用Derby时，该框架应负责在网络上加密密码。如果您的框架未加密密码，则强烈建议您使用SSL / TLS保护网络连接（请参阅配置SSL / TLS ）。

有关Derby系统中用户名处理的信息，请参阅用户和授权标识符 。
