使用NetworkServerControl API

您需要创建一个实例NetworkServerControl类，如果您使用的是API。

该类有四个构造函数。

注意：在启用来自其他系统的连接之前，请确保您正在安全管理器下运行。
  • NetworkServerControl()

    此构造函数创建一个实例，该实例侦听默认端口（1527）或由端口设置的端口。 derby.drda.portNumber属性。它还将侦听由derby.drda.host属性或未设置该属性的回送地址。这是默认的构造函数。它不允许远程连接。相当于打电话NetworkServerControl(InetAddress.getByName("localhost"),1527)如果未设置任何属性。

  • NetworkServerControl(InetAddress address, int portNumber)
    此构造函数创建一个实例，该实例侦听指定的portNumber在指定的地址上。的InetAddress将传递给ServerSocket 。NULL是无效的地址值。下面的示例说明如何允许网络服务器接受来自其他主机的连接：
    // accepts connections from other hosts on an IPv4 system
NetworkServerControl serverControl = 
  new NetworkServerControl(InetAddress.getByName("0.0.0.0"),1527);
    // accepts connections from other hosts on an IPv6 system
NetworkServerControl serverControl = 
  new NetworkServerControl(InetAddress.getByName("::"),1527);
  • NetworkServerControl(String userName, String password)

    如果网络服务器应使用用户身份验证运行，则某些操作如NetworkServerControl.shutdown()要求您提供用户凭据。此构造函数使用用户凭据创建一个实例，然后将其用于需要它们的操作。在所有其他方面，此构造函数的行为类似于NetworkServerControl() 。

  • NetworkServerControl(InetAddress address, int portNumber, String userName, String password)

    此构造函数使用用户凭据创建一个实例，然后将其用于需要它们的操作。在所有其他方面，此构造函数的行为类似于NetworkServerControl(InetAddress address, int portNumber) 。
