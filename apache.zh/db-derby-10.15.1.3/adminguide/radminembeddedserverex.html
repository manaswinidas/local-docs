嵌入式服务器示例

您可以通过在Derby启动时自动在另一个线程中启动网络服务器。 derby.drda.startNetworkServer属性，也可以使用程序启动它。

有关此属性的详细信息，请参阅设置网络服务器属性 。

下面的示例演示如何使用程序启动网络服务器：

import org.apache.derby.drda.NetworkServerControl;
import java.net.InetAddress;
NetworkServerControl server = new NetworkServerControl
	(InetAddress.getByName("localhost"),1527);
server.start(null);

启动网络服务器的程序可以通过使用嵌入式驱动程序或网络客户端驱动程序来访问数据库。服务器框架引导本地JDBC驱动程序的尝试将被忽略，因为它已经在应用程序的JVM中被引导。服务器框架仅访问已引导的Derby实例。应用程序和服务器框架之间没有冲突。

然后，远程客户端可以通过Derby客户端驱动程序进行连接：

String nsURL="jdbc:derby://localhost:1527/sample";  
java.util.Properties props = new java.util.Properties();
props.setProperty("user","usr");
props.setProperty("password","pwd");

Connection conn = DriverManager.getConnection(nsURL, props);

/*interact with Derby*/
Statement s = conn.createStatement();

ResultSet rs = s.executeQuery(
"SELECT * FROM HotelBookings");
