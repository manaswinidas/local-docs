设置端口号

默认情况下， Derby Network Server侦听TCP / IP端口号1527。如果要使用其他端口号，则可以在启动网络服务器时在命令行上指定它。

例如：

java org.apache.derby.drda.NetworkServerControl start -p 1088
但是，最好使用以下任何一种方法来指定端口号：
  • 改变startNetworkServer.bat要么startNetworkServer.ksh脚本
  • 使用derby.drda.portNumber财产derby.properties
有关更多信息，请参见启动网络服务器 。
