本指南的组织方式

本指南描述了如何运行和管理Derby网络服务器。

第一部分： Derby服务器指南

  • 多用户环境中的Derby

    描述了将Derby嵌入服务器框架中的不同选项，并说明了“网络服务器”选项。

  • 将网络服务器与预先存在的Derby应用程序一起使用

    描述如何更改现有的Derby应用程序以与网络服务器一起使用。

  • 管理Derby网络服务器

    描述如何使用外壳程序脚本，命令行和网络服务器API来管理网络服务器。

  • 使用Java管理扩展（JMX）技术

    描述如何使用Derby MBean和其他JMX功能。

  • 使用Servlet接口远程管理Derby Network Server

    描述如何使用servlet接口来管理网络服务器。（servlet接口仅应用于测试目的，而不能用于生产中。）

  • Derby Network Server高级主题

    为Derby Network Server用户介绍高级主题。

  • Derby Network Server示例程序

    为网络服务器用户描述几个Derby网络服务器示例程序。

  • 维护数据库完整性

  • 在单独的设备上登录

  • 获取锁定信息

  • 回收未使用的空间
