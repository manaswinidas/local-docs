使用API关机

您可以使用NetworkServerControl API从Java应用程序中关闭网络服务器。

您用来关闭网络服务器的方法的名称是shutdown() 。

例如，以下命令使用默认端口号（1527）关闭当前计算机上运行的网络服务器：

NetworkServerControl server = new NetworkServerControl();
server.shutdown();

要关闭以用户身份验证运行的服务器，您需要使用通过用户凭据创建的NetworkServerControl实例：

NetworkServerControl server = new NetworkServerControl(username, password);
server.shutdown();
