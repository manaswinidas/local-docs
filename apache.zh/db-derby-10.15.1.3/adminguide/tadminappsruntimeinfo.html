获取网络服务器运行时信息

使用runtimeinfo命令或getRuntimeInfo获取有关网络服务器的内存使用情况和当前会话信息（包括用户，数据库和准备好的语句信息）的方法。

  • 跑步runtimeinfo从命令行： 
    java org.apache.derby.drda.NetworkServerControl runtimeinfo 
   [-h <hostname>][<-p portnumber>]
  • 的getRuntimeInfo方法返回与相同的信息runtimeinfo命令。的签名getRuntimeInfo方法是 
    String getRuntimeInfo()

    例如：

    NetworkServerControl serverControl = new NetworkServerControl();
String myinfo = serverControl.getRuntimeInfo();
