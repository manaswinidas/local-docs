验证启动

要验证Derby Network Server当前正在运行，请使用ping命令。

您可以使用ping通过以下方式命令：

  • 您可以使用脚本NetworkServerControl.bat对于Windows系统或NetworkServerControl对于带有ping命令。例如： 
    NetworkServerControl ping [-h <hostname>;] [-p <portnumber>]
  • 您可以使用NetworkServerControl ping命令： 
    java org.apache.derby.drda.NetworkServerControl
   ping [-h <hostname>] [-p <portnumber>]
  • 您可以使用NetworkServerControl API来从Java应用程序中验证启动： 
    ping();
以下示例使用一种方法来验证启动。它将尝试验证指定的秒数：
private static boolean isServerStarted(NetworkServerControl server, int ntries) {
    for (int i = 1; i <= ntries; i ++) {
        try {
            Thread.sleep(500);
            server.ping();
            return true;
        } catch (Exception e) {
            if (i == ntries) {
                return false;
            }
        }
    }
    return false;
}
