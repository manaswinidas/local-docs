将Derby ij工具与网络服务器一起使用

要使用ij工具与网络客户端驱动程序，请按照下列步骤。

  1. 开始ij通过以下方式之一。有关详细信息，请参见《 Derby工具和实用程序指南 》中的“启动ij”。
    1. 使用脚本。 跑过ij.bat Windows系统和ij UNIX系统上的脚本。这些脚本位于$DERBY_HOME/bin目录。
    2. 跑过ij使用工具$DERBY_HOME/lib/derbyrun.jar文件。 
      java -jar derbyrun.jar ij
    3. 跑过ij通过指定类名称的工具。 
      java org.apache.derby.tools.ij
  2. 通过指定URL进行连接： 
    ij> CONNECT 'jdbc:derby://localhost:1527/sample'
	USER 'judy' PASSWORD 'no12see';
    有关其他URL示例，请参见网络客户端驱动程序示例。
