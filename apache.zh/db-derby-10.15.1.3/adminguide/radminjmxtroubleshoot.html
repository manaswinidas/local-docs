对JMX连接问题进行故障排除

如果您在使用JMX远程连接到MBean服务器时遇到问题，获取一些跟踪信息可能会有所帮助。

有关远程连接到MBean服务器的详细信息，请参见使用JConsole访问Derby MBean和使用自定义Java代码访问Derby MBean 。

Oracle JDK中的JMX实现使用java.util.logging记录JMX跟踪的API。例如，为了跟踪SSL连接问题，请设置系统属性java.util.logging.config.file如下图所示：

java -Djava.util.logging.config.file=logging.properties MyJmxClient

使用JConsole时，如果在启动JConsole时指定以下选项，则将显示一个单独的日志记录窗口（请参阅使用JConsole访问Derby MBeans ），只要logging.properties找到文件：

-J-Djava.util.logging.config.file=logging.properties

的logging.properties文件应指定日志处理程序和日志记录级别，如以下示例所示：

handlers = java.util.logging.ConsoleHandler
.level = INFO

java.util.logging.ConsoleHandler.level=FINEST
java.util.logging.ConsoleHandler.formatter=java.util.logging.SimpleFormatter

 // Level FINEST is suitable for diagnosing SSL-related JMX remote
 //  connection issues.
javax.management.level=FINEST
javax.management.remote.level=FINEST

博客条目https://blogs.oracle.com/jmxetc/entry/troubleshooting_connection_problems_in_jconsole提供了其他提示。
