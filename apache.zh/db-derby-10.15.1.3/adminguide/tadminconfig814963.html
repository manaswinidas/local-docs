从Java应用程序启动网络服务器

使用后请务必正确关闭网络服务器，因为这样做可能会导致不可预知的副作用，例如服务器上的端口被阻塞。

有两种方法可以从Java应用程序启动网络服务器。

  • 您可以将以下行添加到derby.properties文件： 
    derby.drda.startNetworkServer=true
    这将在默认端口1527上启动服务器，监听localhost （所有接口）。

    要在其中指定其他端口或特定接口derby.properties文件，分别包含以下几行：

    derby.drda.portNumber=1110
derby.drda.host=myhost
    您还可以指定startNetworkServer和portNumber使用Java命令设置属性：
    java -Dderby.drda.startNetworkServer=true \
-Dderby.drda.portNumber=1110 \
-Dderby.drda.host=myhost yourApp
  • 您可以使用NetworkServerControl API从Java应用程序中的单独线程启动网络服务器： 
    NetworkServerControl server = new NetworkServerControl();
server.start (null);
