运行SimpleNetworkServerSample程序

要运行Derby Network Server示例程序，请按照下列步骤操作。

  1. 打开命令提示符并将目录更改为%DERBY_HOME%\demo\nserverdemo目录，在哪里%DERBY_HOME%是您安装Derby的目录。
  2. 设置类路径以包括当前目录（“。”）和以下jar文件：
    derbynet.jar
    网络服务器jar文件。它必须在您的CLASSPATH中，因为在此程序中启动了网络服务器。
    derby.jar
    数据库引擎jar文件。
    derbtools.jar
    数据源。
    derbyshared.jar
    通用实用程序方法。
  3. 通过运行以下Java命令来测试CLASSPATH设置： 
    java org.apache.derby.tools.sysinfo
    此命令显示类路径中的Derby jar文件。
  4. 正确设置环境后，运行SimpleNetworkServerSample来自同一目录的程序： 
    java SimpleNetworkServerSample
    如果程序成功运行，您将收到与以下示例所示类似的输出：
    Starting Network Server
Testing if Network Server is up and running!
Derby Network Server now running
Got an embedded connection.
Testing embedded connection by executing a sample query
number of rows in sys.systables = 16
While my app is busy with embedded work, ij might connect like this:

        $ java -Dij.user=me -Dij.password=pw -Dij.protocol=
			jdbc:derby:\\localhost:1527\ org.apache.derby.tools.ij
        ij> connect 'NSSimpleDB';

Clients can continue to connect:
Press [Enter] to stop Server

此时请勿按Enter。在运行服务器时，保持服务器运行SimpleNetworkClientSample程序。

运行SimpleNetworkServerSample程序还会创建以下新目录和文件：
NSSimpleDB
该目录构成了NSSimpleDB数据库。
derby.log
该日志文件包含Derby进度和错误消息。
