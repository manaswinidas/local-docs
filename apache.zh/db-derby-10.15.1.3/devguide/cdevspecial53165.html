触发动作概述

触发动作是一个简单的SQL语句。

例如：

CREATE TRIGGER . . .
DELETE FROM FlightAvailability 
  WHERE flight_id IN (SELECT flight_id FROM FlightAvailability 
    WHERE YEAR(flight_date) < 2005);

触发动作不仅包括triggeredSQLStatement在《 Derby参考手册 》的“ CREATE TRIGGER语句”主题的“ triggeredSQLStatement”部分中进行了介绍， [ FOR EACH { ROW | STATEMENT } ]条款和WHEN条款。

触发动作有局限性。例如，它不能包含动态参数或更改定义触发器的表。这些限制在Derby参考手册的“ CREATE TRIGGER语句”主题的“ triggeredSQLStatement”部分中进行了详细说明 。
