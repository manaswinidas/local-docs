限制/偏移量子句的JDBC转义语法

LIMIT转义子句可以在查询中出现OFFSET / FETCH FIRST子句的位置出现。

有关更多信息，请参见结果偏移量和获取第一条款 。

句法

{ LIMIT rowCount [ OFFSET startRow ] }

rowCount是一个非负整数，用于指定要返回的行数。如果rowCount为0，则返回从startRow开始的所有行。

startRow是一个非负数，它指定返回结果之前要跳过的行数。

相当于

OFFSET startRow FETCH NEXT rowCount ROWS ONLY

例子

-- return the first two rows of sorted table t
SELECT * FROM t
ORDER BY a
{ LIMIT 2 }
-- return two rows of sorted table t, starting with the eleventh row
SELECT * FROM t
ORDER BY a
{ LIMIT 2 OFFSET 10 }
