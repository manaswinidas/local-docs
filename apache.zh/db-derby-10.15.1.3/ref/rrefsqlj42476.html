USER功能

在外部存储例程中使用时，USER，CURRENT_USER和SESSION_USER函数均返回创建SQL会话的用户的授权标识符。

有关这些功能的详细信息，请参见CURRENT_USER函数和SESSION_USER函数。

在存储的例程中使用时，SESSION_USER也始终返回此值。

但是，如果在用EXTERNAL SECURITY DEFINER创建的存储例程中使用，则USER和CURRENT_USER返回拥有该例程架构的用户的授权标识符。尽管数据库所有者也可以是创建者，但这通常是创建用户。

有关定义者和调用者权利的信息，请参见CREATE PROCEDURE语句或CREATE FUNCTION语句 。

句法

USER

例

VALUES USER
