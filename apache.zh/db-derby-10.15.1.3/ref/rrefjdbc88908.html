fn关键字的JDBC转义语法

您可以使用JDBC转义语法指定函数fn关键词。

句法

{ fn functionCall }

其中functionCall是下面列出的标量函数之一的名称。这些功能具有以下类型：

  • 数值函数
  • 字符串函数
  • 日期和时间功能
  • 系统功能

时间戳
使用TIMESTAMPADD用于将时间间隔的值添加到时间戳的函数。该函数根据间隔类型将整数应用于指定的时间戳，并返回总和作为新的时间戳。您可以使用负整数从时间戳中减去。

TIMESTAMPADD是JDBC转义函数，并且只能通过使用JDBC转义函数语法进行访问。

TIMESTAMPADD ( interval, integerExpression, timestampExpression )

去表演TIMESTAMPADD在日期和时间上，有必要将日期和时间转换为时间戳。通过在时间字段中输入00：00：00.0，将日期转换为时间戳。通过将当前日期放入日期字段，可以将时间转换为时间戳。

不要在WHERE子句的时间戳算术函数中放入datetime列，因为优化器不会在该列上使用任何索引。

时差
使用TIMESTAMPDIFF函数以指定的间隔查找两个时间戳值之间的差。例如，该函数可以返回两个指定时间戳之间的分钟数。

的TIMESTAMPDIFF是JDBC转义函数，并且只能通过使用JDBC转义函数语法进行访问。

TIMESTAMPDIFF ( interval, timestampExpression1, timestampExpression2 )

去表演TIMESTAMPDIFF在日期和时间上，有必要将日期和时间转换为时间戳。通过在时间字段中输入00：00：00.0，将日期转换为时间戳。通过将当前日期放入日期字段，可以将时间转换为时间戳。

不要在WHERE子句的时间戳算术函数中放入datetime列，因为优化器不会在该列上使用任何索引。

TIMESTAMPADD和TIMESTAMPDIFF的有效间隔

的TIMESTAMPADD和TIMESTAMPDIFF函数用于执行带时间戳的算术运算。这两个函数使用以下有效间隔进行算术运算：
  • SQL_TSI_DAY
  • SQL_TSI_FRAC_SECOND
  • SQL_TSI_HOUR
  • SQL_TSI_MINUTE
  • SQL_TSI_MONTH
  • SQL_TSI_QUARTER
  • SQL_TSI_SECOND
  • SQL_TSI_WEEK
  • SQL_TSI_YEAR

TIMESTAMPADD和TIMESTAMPDIFF转义函数的示例

要比当前时间戳晚一个月返回时间戳值，请使用以下语法：

{fn TIMESTAMPADD( SQL_TSI_MONTH, 1, CURRENT_TIMESTAMP)}

要返回从现在到指定时间（2008年1月1日）之间的星期数，请使用以下语法：

{fn TIMESTAMPDIFF(SQL_TSI_WEEK, CURRENT_TIMESTAMP, 
  timestamp('2008-01-01-12.00.00.000000'))}
