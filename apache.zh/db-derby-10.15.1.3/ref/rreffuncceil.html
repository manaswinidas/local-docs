CEIL或CEILING功能

CEIL或CEILING函数将指定的数字四舍五入，并返回大于或等于指定数字的最小数字。

指定的数字必须是DOUBLE PRECISION数字。
  • 如果指定的数字为NULL，则返回值为NULL。
  • 如果指定数字等于数学整数，则返回值与指定数字相同。
  • 如果指定的数字为零（0），则返回值为零。
  • 如果指定的数字小于零但大于-1.0，则返回值为零。

返回的值是大于（或等于指定数字）的最小（最接近负无穷大）双浮点值。返回的值等于一个数学整数。返回值的数据类型是DOUBLE PRECISION号。

句法

CEIL ( number )
CEILING ( number )
