traceFileAppend = true属性

traceFileAppend = true属性指定Derby Network Client应该将JDBC跟踪信息附加到跟踪文件。

可以通过traceFile = path属性指定文件。如果您未指定跟踪文件，但指定了traceDirectory = path属性，则跟踪信息将附加到默认文件中。如果未指定traceFileAppend = true ，则将覆盖跟踪文件的任何先前版本。

有关跟踪的更多信息，请参阅《 Derby服务器和管理指南 》中的“网络客户端跟踪”。有关与跟踪相关的其他属性，请参见traceDirectory = path和traceLevel = value 。

与其他属性结合

必须与traceFile = path属性或traceDirectory = path属性一起指定此属性。您也可以将此属性与其他属性组合。

例子

-- enable tracing on an existing database, appending to the
--   specified file
jdbc:derby://localhost:1527/mydb;traceFile=trace.out;
    traceFileAppend=true
-- enable tracing on an existing database, appending to the default file 
--   within the specified directory, relative to the Derby home directory
jdbc:derby://localhost:1527/mydb;traceDirectory=mytracedir;
    traceFileAppend=true
