LCASE或LOWER功能

LCASE或LOWER函数将字符表达式作为参数，并返回一个字符串，在该字符串中所有字母字符均已转换为小写。

句法

LCASE ( characterExpression )
LOWER ( characterExpression )

characterExpression是CHAR，VARCHAR或LONG VARCHAR数据类型，或者是任何隐式转换为字符串（位表达式除外）的内置类型。

如果参数类型为CHAR或LONG VARCHAR，则返回类型为CHAR或LONG VARCHAR。否则，返回类型为VARCHAR。

返回值的长度和最大长度与参数的长度和最大长度相同。

如果characterExpression计算为null，则此函数返回null。

例子

-- returns 'asd1#w'
VALUES LOWER('aSD1#w')

SELECT LOWER(flight_id) FROM Flights
