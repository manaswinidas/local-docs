UCASE或UPPER功能

UCASE或UPPER函数将字符表达式作为参数，并返回一个字符串，在该字符串中所有字母字符均已转换为大写。

句法

UCASE ( characterExpression )
UPPER ( characterExpression )
如果参数类型为CHAR，则返回类型为CHAR。否则，返回类型为VARCHAR。
注意： UPPER和LOWER遵循数据库的语言环境。有关指定语言环境的更多信息，请参见region = ll_CC属性 。

返回值的长度和最大长度与参数的长度和最大长度相同。

例

返回字符串aSD1#w大写使用以下子句：
VALUES UPPER('aSD1#w')
返回的值是ASD1#W 。
