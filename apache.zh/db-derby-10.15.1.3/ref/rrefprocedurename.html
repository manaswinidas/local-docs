procedureName

procedureName表示Java存储过程。要创建过程，请使用CREATE PROCEDURE语句 。

句法

[ schemaName. ] SQLIdentifier

您可以使用schemaName限定过程名称。如果指定了合格的过程名称，则架构名称不能以SYS开头。

例

-- SALES.TOTAL_REVENUE is a procedureName that includes a schemaName
CREATE PROCEDURE SALES.TOTAL_REVENUE(IN S_MONTH INTEGER,
                                     IN S_YEAR INTEGER, 
                                     OUT TOTAL DECIMAL(10,2))
PARAMETER STYLE JAVA READS SQL DATA LANGUAGE JAVA EXTERNAL NAME 
'com.example.sales.calculateRevenueByMonth'
