定位功能

LOCATE函数用于在另一个字符串中搜索一个字符串。如果找到了所需的字符串，则LOCATE返回找到它的索引。如果找不到所需的字符串，则LOCATE返回0。

句法

LOCATE ( characterExpression, characterExpression [ , startPosition ] )
LOCATE函数有两个必需的参数，第三个可选参数。
  • 第一characterExpression指定要搜索的字符串。
  • 第二个characterExpression指定要在其中搜索的字符串。
  • 第三个参数是startPosition ，它指定第二个参数中开始搜索的位置。如果未提供第三个参数，则LOCATE函数在第二个参数的开头开始搜索。

LOCATE的返回类型是整数。LOCATE函数返回一个整数，该整数指示第一个参数首次位于的第二个参数中的索引位置。索引位置从1开始。如果在第二个参数中找不到第一个参数，则LOCATE返回0。如果第一个参数是一个空字符串（ '' ），即使第二个参数也是一个空字符串，LOCATE也会返回第三个参数的值（如果未提供，则返回1）。如果为任一characterExpression参数传递了NULL值，则返回NULL。

例子

-- returns 2, since 'love' is found at index position 2:
VALUES LOCATE('love', 'clover')
-- returns 0, since 'stove' is not found in 'clover':
VALUES LOCATE('stove', 'clover')
-- returns 5 (note the start position is 4):
VALUES LOCATE('iss', 'Mississippi', 4)
-- returns 1, because the empty string is a special case:
VALUES LOCATE('', 'ABC')
-- returns 0, because 'AAA' is not found in '':
VALUES LOCATE('AAA', '')
-- returns 3
VALUES LOCATE('', '', 3)
