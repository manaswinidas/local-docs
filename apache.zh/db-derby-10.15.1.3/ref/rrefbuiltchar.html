CHAR功能

CHAR函数返回固定长度的字符串表示形式。

表示为：
  • 字符串（如果第一个参数是任何类型的字符串）。
  • 如果第一个参数是日期，时间或时间戳，则为datetime值。
  • 如果第一个参数是十进制数字，则为十进制数字。
  • 如果第一个参数是DOUBLE或REAL，则为双精度浮点数。
  • 如果第一个参数是SMALLINT，INTEGER或BIGINT，则为整数。
第一个参数必须是内置数据类型。CHAR函数的结果是固定长度的字符串。如果第一个参数可以为null，则结果可以为null。如果第一个参数为null，则结果为null值。第一个参数不能是XML值。要将XML值转换为指定长度的CHAR，必须使用SQL / XML序列化运算符XMLSERIALIZE。

字符到字符语法

CHAR ( characterExpression [ , integer ] )
characterExpression
该表达式返回一个值为CHAR，VARCHAR，LONG VARCHAR或CLOB数据类型的值。
整数
生成的固定长度字符串的length属性。该值必须在0到254之间。

如果characterExpression的长度小于结果的长度属性，则在结果中填充空格，直到结果的长度为止。如果characterExpression的长度大于结果的长度属性，则将执行截断。除非截断的字符全部为空白并且characterExpression不是长字符串（LONG VARCHAR或CLOB），否则将返回警告。

整数到字符语法

CHAR ( integerExpression )
integerExpression
该表达式返回的值是整数数据类型（SMALLINT，INTEGER或BIGINT）。

结果是以SQL整数常量的形式表示参数的字符串表示形式。结果由n个字符组成，它们是有效数字，如果参数为负，则表示参数的值并带有一个前减号。结果左对齐。

  • 如果第一个参数是SMALLINT：结果的长度为6。如果结果中的字符数少于6个，则结果将在右边填充空白，长度为6。
  • 如果第一个参数是INTEGER：结果的长度为11。如果结果中的字符数少于11，则结果将在右边填充空白，长度为11。
  • 如果第一个参数是BIGINT：结果的长度为20。如果结果中的字符数少于20，则结果将在右边填充空白，长度为20。

日期时间到字符语法

CHAR ( datetimeExpression )
datetimeExpression
该表达式是以下三种数据类型之一：
  • 日期：结果是日期的字符表示。结果的长度为10。
  • TIME：结果是时间的字符表示。结果的长度为8。
  • TIMESTAMP：结果是时间戳的字符串表示形式。结果的长度为26。

小数到字符

CHAR ( decimalExpression )
小数表达式
该表达式返回一个十进制数据类型的值。

浮点到字符语法

CHAR ( floatingPointExpression )
floatPointExpression
该表达式返回一个浮点数据类型（DOUBLE或REAL）的值。

例

使用CHAR函数以固定长度的字符串返回EDLEVEL（定义为smallint）的值：
SELECT CHAR(EDLEVEL) FROM EMPLOYEE
将返回EDLEVEL 18作为CHAR（6）值'18'（18后跟四个空格）。
