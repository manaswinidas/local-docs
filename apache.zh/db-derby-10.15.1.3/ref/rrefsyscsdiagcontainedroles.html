SYSCS_DIAG.CONTAINED_ROLES诊断表功能

SYSCS_DIAG.CONTAINED_ROLES诊断表函数返回指定角色中包含的所有角色。

传递给此表函数的参数应该是角色的名称（在引号中以字符串形式指定），或者是特殊的关键字CURRENT_ROLE，它指示当前有效的角色。有关角色包含的定义，请参见GRANT语句中的 “角色语法”。

例如：

SELECT * FROM TABLE (SYSCS_DIAG.CONTAINED_ROLES('READER')) AS T1
SELECT * FROM TABLE (SYSCS_DIAG.CONTAINED_ROLES(CURRENT_ROLE)) AS T2

无论数据库是否启用了身份验证和SQL授权，所有用户都可以访问此诊断表功能。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。

返回的表具有下表中显示的列。

表格1。SYSCS_DIAG.CONTAINED_ROLES表函数返回的列
栏名 类型 长度 可空 内容
角色 VARCHAR 128 假 角色的标识符。
