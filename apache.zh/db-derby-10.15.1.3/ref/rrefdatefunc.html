DATE功能

DATE函数从值返回日期。

参数必须是日期，时间戳记，小于或等于2,932,897的正数，日期或时间戳记的有效字符串表示形式或长度不是7的字符串，不是CLOB，LONG VARCHAR或XML值。如果参数是长度为7的字符串，则它必须以yyyynnn的形式表示有效日期，其中yyyy是表示年份的数字，而nnn是介于001和366之间的数字，表示该年的某一天。该函数的结果是一个日期。如果参数可以为null，则结果可以为null；否则，结果为null。如果参数为null，则结果为null值。

其他规则取决于指定的参数的数据类型：
  • 如果参数是日期，时间戳记或日期或时间戳记的有效字符串表示形式：结果是值的日期部分。
  • 如果参数是数字：结果是1970年1月1日之后n -1天的日期，其中n是数字的整数部分。
  • 如果参数是长度为7的字符串：结果是该字符串表示的日期。

句法

DATE ( expression )

例子

本示例的内部表示为'1988-12-25'。

VALUES DATE('1988-12-25')

本示例的内部表示为“ 1972-02-28”。

VALUES DATE(789)
