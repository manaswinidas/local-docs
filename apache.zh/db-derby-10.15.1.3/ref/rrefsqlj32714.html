VARCHAR FOR BIT DATA数据类型

VARCHAR FOR BIT DATA类型允许您存储小于或等于指定长度的二进制字符串。对于字符串不合适的非结构化数据（例如图像）很有用。

句法

{ VARCHAR | CHAR VARYING | CHARACTER VARYING } (length) FOR BIT DATA

length是一个无符号整数文字，以字节为单位指定长度。

与CHAR FOR BIT DATA类型的情况不同，VARCHAR FOR BIT DATA类型没有默认长度 。长度值的最大大小为32672字节。

JDBC元数据类型（java.sql。类型）

VARBINARY

VARCHAR FOR BIT DATA存储可变长度的字节字符串。与CHAR FOR BIT DATA值不同，VARCHAR FOR BIT DATA值不会填充到目标长度。

对VARCHAR FOR BIT DATA和CHAR FOR BIT DATA值（例如，级联）进行运算会产生VARCHAR FOR BIT DATA值。

字节文字的类型始终是VARCHAR FOR BIT DATA，而不是CHAR FOR BIT DATA。
