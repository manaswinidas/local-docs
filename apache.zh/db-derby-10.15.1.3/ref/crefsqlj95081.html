创建语句

使用CREATE语句创建函数，索引，过程，角色，模式，同义词，表，触发器和视图。

  • CREATE DERBY AGGREGATE语句
    CREATE DERBY AGGREGATE语句创建用户定义的聚合（UDA）。UDA是自定义聚合运算符。
  • CREATE FUNCTION语句
    CREATE FUNCTION语句创建一个Java函数，然后可以在表达式中使用它。
  • CREATE INDEX语句
    CREATE INDEX语句在表上创建索引。索引可以在表中的一列或多列上。
  • CREATE PROCEDURE语句
    CREATE PROCEDURE语句创建一个Java存储过程，然后可以使用CALL PROCEDURE语句调用它。
  • CREATE ROLE语句
    CREATE ROLE语句创建一个SQL角色。当数据库有许多用户时，角色对于管理特权很有用。
  • CREATE SCHEMA语句
    CREATE SCHEMA语句创建一个模式，该模式是在单个集合中对对象进行逻辑分组并为对象提供唯一名称空间的一种方式。
  • CREATE SEQUENCE语句
    CREATE SEQUENCE语句创建一个序列生成器，该生成器是一种用于一次生成一个精确数值的机制。
  • 创建SYNONYM语句
    CREATE SYNONYM语句为同一模式或其他模式中存在的表或视图提供了备用名称。
  • CREATE TABLE语句
    CREATE TABLE语句创建一个表。表包含列和约束，以及数据必须遵循的规则。
  • CREATE TRIGGER语句
    CREATE TRIGGER语句创建一个触发器，该触发器定义了在指定表上发生数据库事件时执行的一组操作。
  • CREATE TYPE语句
    CREATE TYPE语句创建用户定义的类型（UDT）。UDT是可序列化的Java类，其实例存储在列中。
  • CREATE VIEW语句
    CREATE VIEW语句创建一个视图，该视图是由查询形成的虚拟表。
