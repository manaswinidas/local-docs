重命名COLUMN语句

RENAME COLUMN语句重命名任何模式（模式SYS除外）中现有表中的现有列。

要重命名列，您必须是数据库所有者或表所有者。

其他类型的表更改也是可能的；有关更多信息，请参见ALTER TABLE语句 。

句法

RENAME COLUMN tableName.simpleColumnName TO simpleColumnName

例子

要将表employee中的manager列重命名为supervisor ，请使用以下语法：
RENAME COLUMN EMPLOYEE.MANAGER TO SUPERVISOR
您可以结合使用ALTER TABLE和RENAME COLUMN来修改列的数据类型。要将表t的列c1更改为新数据类型NEWTYPE ：

    ALTER TABLE t ADD COLUMN c1_newtype NEWTYPE
    UPDATE t SET c1_newtype = c1
    ALTER TABLE t DROP COLUMN c1
    RENAME COLUMN t.c1_newtype TO c1

使用说明

限制：如果生成的列的视图，触发器，检查约束或generationClause引用了该列，则对其重命名的尝试将生成错误。
限制：如果有任何打开的游标引用正在更改的列，则不允许RENAME COLUMN语句。
注意：如果在列上定义了索引，则仍可以重命名该列。索引会自动更新以通过其新名称引用该列
