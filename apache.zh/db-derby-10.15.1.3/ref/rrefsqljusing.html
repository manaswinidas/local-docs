使用子句

USING子句指定在连接两个表时要测试哪些列的相等性。

在具有显式join子句的JOIN操作中 ，可以使用它代替ON子句。

句法

USING ( simpleColumnName [ , simpleColumnName ]* )

USING子句中列出的列必须同时存在于要连接的两个表中。USING子句将转换为ON子句，该子句检查两个表中命名列之间的相等性。

指定USING子句后，查询的选择列表中的星号（*）将扩展到以下列列表（按此顺序）：

  • USING子句中的所有列
  • USING子句中未指定的第一个（左）表的所有列
  • USING子句中未指定的第二个（右）表的所有列

由表名限定的星号（例如COUNTRIES。*）将扩展到该表的未在USING子句中列出的每一列。

如果在未使用表名限定的情况下引用了USING子句中的列，则如果联接是INNER JOIN或LEFT OUTER JOIN，则该列引用指向第一个（左）表中的列。如果它是RIGHT OUTER JOIN，则对USING子句中一列的无条件引用指向第二个（右）表中的列。

例子

以下查询在COUNTRIES.COUNTRY等于CITIES.COUNTRY的条件下执行COUNTRIES表和CITIES表之间的内部联接：

SELECT * FROM COUNTRIES JOIN CITIES
     USING (COUNTRY)

下一个查询与上面的查询类似，但是它具有附加的联接条件，即COUNTRIES.COUNTRY_ISO_CODE等于CITIES.COUNTRY_ISO_CODE：

SELECT * FROM COUNTRIES JOIN CITIES
    USING (COUNTRY, COUNTRY_ISO_CODE)
