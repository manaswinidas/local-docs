TIMESTAMP数据类型

TIMESTAMP用于存储组合的DATE和TIME值。它允许的小数秒值最多为九位数。

请参见DATE数据类型和TIME数据类型 。

句法

TIMESTAMP

对应的编译时Java类型

java.sql。时间戳记

JDBC元数据类型（java.sql。类型）

时间戳

日期，时间和时间戳不能在表达式中相互混合。

Derby支持以下格式的TIMESTAMP：
yyyy-mm-dd hh:mm:ss[.nnnnnnnnn]
yyyy-mm-dd-hh.mm.ss[.nnnnnnnnn]
上面两种格式中的第一种是java.sql。时间戳格式。

年份必须始终有四个数字。月，日和小时可以有一个或两个数字。分钟和秒必须有两位数字。纳秒（如果存在）的位数可能在1到9之间。年，月和日组成部分必须为正整数。

Derby还使用数据库服务器的语言环境接受特定于语言环境的日期时间格式的字符串。如果有歧义，则以上内置格式优先。

例子

VALUES '1960-01-01 23:03:20'
VALUES TIMESTAMP('1962-09-23 03:23:34.234')
VALUES TIMESTAMP('1960-01-01 23:03:20')
VALUES TIMESTAMP('1962-09-23-03:23:34.987654321')
