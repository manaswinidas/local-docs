TRIM功能

TRIM函数采用一个字符表达式，并返回该表达式并删除前导和/或尾随填充字符。

可选参数指示是否应删除前导，尾随或前导和尾随的填充字符，并指定要删除的填充字符。

句法

TRIM( [ trimOperands ] trimSource )

trimSource是characterExpression 。

trimOperands

{ trimType [ trimCharacter ] FROM | trimCharacter FROM }

trimCharacter是characterExpression 。

trimType

{ LEADING | TRAILING | BOTH }
如果未指定trimType ，则默认为BOTH。如果未指定trimCharacter ，它将默认为空格字符（''）。否则， trimCharacter表达式必须为以下值之一：
  • 长度恰好为一个的字符串
  • 空值
如果trimCharacter或trimSource的值为NULL，则TRIM函数的结果为NULL。否则，TRIM函数的结果定义如下：
  • 如果trimType为LEADING，则结果将是trimSource值， 并删除所有出现的trimCharacter 。
  • 如果trimType为TRAILING，则结果将是trimSource值， 并且删除所有出现的trimCharacter 。
  • 如果trimType是BOTH，其结果将是除去与trimCharacter的所有开头和结尾出现的trimSource值。

如果trimSource的数据类型为CHAR或VARCHAR，则TRIM函数的返回类型为VARCHAR。否则，TRIM函数的返回类型将为CLOB。

例子

-- returns 'derby' (no spaces)
VALUES TRIM('  derby ')
-- returns 'derby' (no spaces)
VALUES TRIM(BOTH ' ' FROM '  derby ')
-- returns 'derby ' (with a space at the end)
VALUES TRIM(LEADING ' ' FROM '  derby ')
-- returns '  derby' (with two spaces at the beginning)
VALUES TRIM(TRAILING ' ' FROM '  derby ')
-- returns NULL
VALUES TRIM(cast (null as char(1)) FROM '  derby ')
-- returns NULL
VALUES TRIM(' ' FROM cast(null as varchar(30)))
-- returns ' derb' (with a space at the beginning)
VALUES TRIM('y' FROM ' derby')
-- results in an error because trimCharacter can only be 1 character
VALUES TRIM('by' FROM ' derby')
