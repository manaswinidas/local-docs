CHAR数据类型

CHAR提供固定长度的字符串存储。

句法

CHAR[ACTER] [(length)]

length是一个无符号整数文字，以字节为单位指定长度。CHAR的默认长度为1，最大长度为254。

对应的编译时Java类型

java.lang。串

JDBC元数据类型（java.sql。类型）

焦炭

Derby插入空格以填充比预期长度短的字符串值。Derby将从字符串值中截断比预期长度更长的空格。除空格以外的其他字符都会引发异常。将比较布尔运算符应用于CHAR时，较短的字符串将用空格填充到较长的字符串的长度。

当在表达式中混合使用CHAR和VARCHAR时，较短的值将用空格填充到较长的值的长度上。

字符串常量的类型为CHAR。

例子

-- within a string constant use two single quotation marks
-- to represent a single quotation mark or apostrophe 
VALUES 'hello this is Joe''s string'
-- create a table with a CHAR field
CREATE TABLE STATUS (
    STATUSCODE CHAR(2) NOT NULL
        CONSTRAINT PK_STATUS PRIMARY KEY,
    STATUSDESC VARCHAR(40) NOT NULL
);
