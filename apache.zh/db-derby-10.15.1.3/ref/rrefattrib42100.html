bootPassword =键属性

bootPassword = key属性指定启动密码（加密密钥）。

该属性指定用于以下目的的键：
  • 加密新数据库
  • 配置现有的未加密数据库进行加密
  • 引导现有的加密数据库
指定至少八个字符的字母数字字符串。

有关数据加密的信息，请参阅《 Derby安全指南 》中的“配置数据库加密”。

与其他属性结合

创建新数据库时，必须将bootPassword = key属性与create = true和dataEncryption = true属性结合使用。

当配置现有的未加密数据库进行加密时，必须将bootPassword = key属性与dataEncryption = true属性结合使用。对于同时启用身份验证和SQL授权的现有未加密数据库，只有数据库所有者才能执行加密。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。

bootPassword = key属性可以选择与encryptionProvider = providerName ， encryptionAlgorithm = algorithm和/或encryptionKeyLength = length属性结合使用。

例子

-- create a new, encrypted database
jdbc:derby:newDB;create=true;dataEncryption=true;
    bootPassword=cseveryPlace
-- configure an existing unencrypted database for encryption
jdbc:derby:salesdb;dataEncryption=true;bootPassword=cseveryPlace
-- boot an existing encrypted database
jdbc:derby:encryptedDB;bootPassword=cseveryPlace
-- configure an existing encrypted database for encryption, 
-- specifying the encryption algorithm
jdbc:derby:encryptedDB;dataEncryption=true;bootPassword=cseveryPlace;
    encryptionAlgorithm=DESede/CBC/NoPadding
