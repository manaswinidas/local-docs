traceFile =路径属性

traceFile = path属性指定Derby Network Client将JDBC跟踪信息发送到的文件。

如果未指定绝对路径名，则将文件放置在derby.system.home目录中（有关详细信息，请参阅《 Derby开发人员指南 》中的“定义系统目录”）。

如果同时指定traceFile = path和traceFileAppend = true ，则输出将附加到指定的文件（如果存在）。如果指定traceFile = path但未指定traceFileAppend = true ，则该文件的文件的任何先前版本都会被覆盖。

有关跟踪的更多信息，请参阅《 Derby服务器和管理指南 》中的“网络客户端跟踪”。有关与跟踪相关的其他属性，请参见traceDirectory = path和traceLevel = value 。

与其他属性结合

您可以将此属性与其他属性组合。

例

-- enable tracing on a new database
jdbc:derby://localhost:1527/mydb;create=true;traceFile=trace.out
