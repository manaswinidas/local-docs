SUM功能

SUM是一个聚合函数，用于评估一组行中表达式的总和。

有关这些功能的更多信息，请参见聚合（设置功能） 。

仅对计算结果为数字数据类型的表达式允许SUM。

句法

SUM ( [ DISTINCT | ALL ] expression )

DISTINCT和ALL限定词消除或保留重复项。如果未指定ALL或DISTINCT，则假定ALL。例如，如果列包含值1、1、1、1和2，则SUM（col）返回的值大于SUM（DISTINCT col）。

每个selectExpression仅允许一个DISTINCT聚合表达式。例如，不允许以下查询：
SELECT AVG (DISTINCT flying_time), SUM (DISTINCT miles)
FROM Flights

该表达式可以包含多个列引用或表达式，但不能包含另一个聚合或子查询。它必须求值为内置数字数据类型。如果表达式的计算结果为NULL，则聚合将跳过该值。

结果数据类型与其在其上运行的表达式相同（可能会溢出）。

例子

-- find all economy seats available:
SELECT SUM (economy_seats) FROM Airlines;

-- use SUM on multiple column references
-- (find the total number of all seats purchased):
SELECT SUM (economy_seats_taken + business_seats_taken + firstclass_seats_taken)
as seats_taken FROM FLIGHTAVAILABILITY;
