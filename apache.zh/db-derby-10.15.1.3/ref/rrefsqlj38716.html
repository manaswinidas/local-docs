COUNT功能

COUNT是一个聚合函数，用于计算在表达式中访问的行数。所有类型的表达式都可以使用COUNT。

有关这些功能的更多信息，请参见聚合（设置功能） 。

句法

COUNT ( [ DISTINCT | ALL ] expression )

DISTINCT限定词可消除重复项。ALL限定词保留重复项。如果未指定ALL或DISTINCT，则假定ALL。例如，如果列包含值1、1、1、1和2，则COUNT（col）返回的值大于COUNT（DISTINCT col）。

每个selectExpression仅允许一个DISTINCT聚合表达式。例如，不允许以下查询：

-- query not allowed
SELECT COUNT (DISTINCT flying_time), SUM (DISTINCT miles)
FROM Flights

一个表达式可以包含多个列引用或表达式，但不能包含另一个聚合或子查询。如果表达式的计算结果为NULL，则不会为该值处理聚合。

结果数据类型COUNT为INTEGER 。

例

-- Count the number of countries in each region,
-- show only regions that have at least 2
SELECT COUNT (country), region
FROM Countries
GROUP BY region
HAVING COUNT (country) > 1
