DROP语句

使用DROP语句删除函数，索引，过程，角色，模式，同义词，表，触发器和视图。

  • DROP DERBY AGGREGATE语句
    DROP DERBY AGGREGATE语句删除指定的用户定义的聚合（UDA）。
  • DROP FUNCTION语句
    DROP FUNCTION语句删除指定的Java函数。
  • DROP INDEX语句
    DROP INDEX语句删除指定的索引。
  • DROP PROCEDURE语句
    DROP PROCEDURE语句删除指定的Java存储过程。
  • DROP ROLE语句
    DROP ROLE语句删除指定的SQL角色。
  • DROP SCHEMA声明
    DROP SCHEMA语句删除指定的架构。
  • DROP SEQUENCE语句
    DROP SEQUENCE语句删除指定的序列生成器。
  • DROP SYNONYM语句
    DROP SYNONYM语句从表或视图中删除指定的同义词。
  • DROP TABLE语句
    DROP TABLE语句删除指定的表。
  • DROP TRIGGER语句
    DROP TRIGGER语句删除指定的触发器。
  • DROP TYPE语句
    DROP TYPE语句删除指定的用户定义类型（UDT）。
  • DROP VIEW语句
    DROP VIEW语句删除指定的视图。
