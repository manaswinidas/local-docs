EXTERNAL NAME子句

EXTERNAL NAME子句指定要在CREATE FUNCTION或CREATE PROCEDURE语句中调用的Java方法，并在CREATE AGGREGATE或CREATE TYPE语句中指定Java类。

有关更多信息，请参见CREATE FUNCTION语句 ， CREATE PROCEDURE语句 ， CREATE DERBY AGGREGATE语句和CREATE TYPE语句 。

句法

EXTERNAL NAME singleQuotedString

singleQuotedString不能有任何多余的空格。

在CREATE FUNCTION或CREATE PROCEDURE语句中指定的方法名称通常采用以下形式：

'class_name.method_name[(parameterTypes)]'

当从SQL声明确定的Java签名不明确时，需要可选的parameterTypes规范。

如果该类是静态嵌套类，或者该方法在静态嵌套类中，请在外部类和静态类之间使用美元符号（$）作为分隔符。例如，假设您具有以下类和方法定义：

public class TestFuncs {
    public static final class MyMath {
        public static double pow( double base, double power ) {
            return Math.pow( base, power );
        }
    }
}

如果使用CREATE FUNCTION将此pow方法绑定到用户定义的函数，则外部名称应为TestFuncs $ MyMath.pow ，而不是TestFuncs。MyMath.pow 。

例子

-- Specify the Mode class as an external name
CREATE DERBY AGGREGATE mode FOR INT
EXTERNAL NAME 'com.example.myapp.aggs.Mode';
-- Specify the pow method in the static class MyMath
CREATE FUNCTION MYPOWER ( X DOUBLE, Y DOUBLE )
RETURNS DOUBLE
PARAMETER STYLE JAVA
NO SQL LANGUAGE JAVA
EXTERNAL NAME 'TestFuncs$MyMath.pow'
-- create a function to round a Double to a specified number of decimal places
CREATE FUNCTION DoubleFormat (value FLOAT, places INTEGER)
RETURNS FLOAT
PARAMETER STYLE JAVA
LANGUAGE JAVA
EXTERNAL NAME
'utils.Utils.Double(java.lang.Float,java.lang.Integer)'
