SYSCS_UTIL.SYSCS_UNFREEZE_DATABASE系统过程

的SYSCS_UTIL.SYSCS_UNFREEZE_DATABASE系统过程在备份后解冻数据库。

有关使用此过程的更多信息，请参见《 Derby服务器和管理指南 》中的“在冻结和解冻系统过程中使用操作系统命令来执行联机备份”。

句法

SYSCS_UTIL.SYSCS_UNFREEZE_DATABASE()

此过程不返回结果集。

执行权限

如果同时启用了身份验证和SQL授权，则默认情况下，只有数据库所有者对此过程具有执行特权。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。数据库所有者可以将访问权限授予其他用户。

例

String backupdirectory = "c:/mybackups/" + JCalendar.getToday();
CallableStatement cs = conn.prepareCall
("CALL SYSCS_UTIL.SYSCS_FREEZE_DATABASE()");
cs.execute();
cs.close();
// user supplied code to take full backup of "backupdirectory"
// now unfreeze the database once backup has completed:
CallableStatement cs = conn.prepareCall
("CALL SYSCS_UTIL.SYSCS_UNFREEZE_DATABASE()");
cs.execute();
cs.close();
