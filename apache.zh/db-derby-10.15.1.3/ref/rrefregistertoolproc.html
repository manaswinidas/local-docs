SYSCS_UTIL.SYSCS_REGISTER_TOOL系统过程

的SYSCS_UTIL.SYSCS_REGISTER_TOOL系统过程加载和卸载可选工具包。

句法

SYSCS_UTIL.SYSCS_REGISTER_TOOL(IN TOOLNAME VARCHAR(128), 
                               IN REGISTER BOOLEAN, 
                               IN OPTIONALARGS VARCHAR(128) ... )

此过程不返回结果集。

工具名
可选工具的名称。必须是以下区分大小写的名称之一： 'databaseMetaData' ， 'foreignViews' ， 'luceneSupport' ， 'simpleJson' ， 要么'rawDBReader' 。
寄存器
值为true指示Derby加载工具。值false指示Derby卸载工具。
选件
特定于每个工具的可选的区分大小写的字符串参数。

执行权限

如果同时启用了身份验证和SQL授权，则默认情况下，只有数据库所有者对此过程具有执行特权。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。数据库所有者可以将访问权限授予其他用户。

用法

支持以下可选工具：

  • databaseMetaData

    这个可选工具创建函数和表函数以将方法包装在java.sql中。DatabaseMetaData ，允许您在查询中使用DatabaseMetaData方法。例如，您可以加入和过滤DatabaseMetaData方法返回的ResultSet 。该工具不需要任何可选参数。要创建元数据函数和表函数，请执行以下操作：

    call syscs_util.syscs_register_tool( 'databaseMetaData', true )

    要删除函数和表函数，请执行以下操作：

    call syscs_util.syscs_register_tool( 'databaseMetaData', false )
  • foreignViews

    此可选工具为外部数据库中的所有用户表创建架构，表功能和便利视图。表功能和视图对于将外部数据批量导入Derby很有用。此工具有两个附加参数：

    CONNECTION_URL
    这是一个连接URL字符串，适用于通过调用DriverManager.getConnection（）创建与外部数据库的连接。
    SCHEMA_PREFIX
    这是一个可选字符串，以该工具创建的所有架构名称为前缀。该参数可以省略。如果省略它，则该工具将创建与外部数据库中的架构具有相同名称的架构。

    要在外部数据上创建视图，请执行以下操作：

    call syscs_util.syscs_register_tool( 'foreignViews', true,
    'foreignDatabaseURL', 'XYZ_' )

    要删除外部数据上的视图，请执行以下操作：

    call syscs_util.syscs_register_tool( 'foreignViews', false,
    'foreignDatabaseURL', 'XYZ_' )

有关如何使用这些工具的更多信息，请参见《 Derby工具和实用程序指南》 。在运行可选工具之前，请确保您的类路径包含Derby jar文件，包括derbytools.jar 。
