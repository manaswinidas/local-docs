SYSCS_UTIL.SYSCS_SET_XPLAIN_MODE系统过程

当捕获运行时统计信息时，您可以使用以下命令控制处理模式SYSCS_UTIL.SYSCS_SET_XPLAIN_MODE程序。

XPLAIN模式设置为1时，将编译和优化语句，但不执行；当XPLAIN模式设置为0（默认值）时，语句将被编译，优化和正常执行。

XPLAIN模式仅在启用XPLAIN样式时才重要。见SYSCS_UTIL.SYSCS_SET_XPLAIN_SCHEMA有关更多信息的系统过程。

有关其他信息，请参见Tuning Derby中的 “使用RunTimeStatistics”。

句法

SYSCS_UTIL.SYSCS_SET_XPLAIN_MODE(IN SMALLINT NOEXECUTE)

执行权限

如果同时启用了身份验证和SQL授权，则默认情况下，只有数据库所有者对此过程具有执行特权。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。数据库所有者可以将访问权限授予其他用户。

例

要让Derby在不执行语句的情况下解释它：

        call syscs_util.syscs_set_runtimestatistics(1);
        call syscs_util.syscs_set_xplain_schema('STATS');
        call syscs_util.syscs_set_xplain_mode(1);

        select country from countries;

        call syscs_util.syscs_set_runtimestatistics(0);
        call syscs_util.syscs_set_xplain_schema('');
        call syscs_util.syscs_set_xplain_mode(0);
