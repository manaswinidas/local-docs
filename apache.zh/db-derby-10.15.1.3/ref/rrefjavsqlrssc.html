结果集和流列

如果基础对象本身是OutputStream类，则getBinaryStream直接返回该对象。

要使用流式列从ResultSet中获取字段，可以使用getXXXStream方法（如果该类型支持）。有关支持各种流的类型的列表，请参见Streamable JDBC数据类型 。（另请参见java.sql的映射。类型转换为SQL类型 。）

您可以从支持的数据类型列之一中将数据作为流检索，无论是否将其存储为流。

以下代码片段显示了用户如何检索LONG VARCHAR列作为流：
    // retrieve data as a stream
    ResultSet rs = s.executeQuery("SELECT b FROM atable");
    while (rs.next()) {
        // use a java.io.Reader to get the data
        java.io.Reader ip = rs.getCharacterStream(1);

        // process the stream--this is just a generic way to
        // print the data
        char[] buff = new char[128];
        int size;
        while ((size = ip.read(buff)) != -1) {
            String chunk = new String(buff, 0, size);
            System.out.print(chunk);
        }
    }
    rs.close();
    s.close();
    conn.commit();
