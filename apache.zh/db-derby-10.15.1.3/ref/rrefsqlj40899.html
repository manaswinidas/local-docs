串联运算符

串联运算符， || ，将其右操作数连接到其左操作数的末尾。它对字符或位表达式进行操作。

因为所有内置数据类型都隐式转换为字符串，所以此函数可以对所有内置数据类型起作用。

句法

{
    characterExpression || characterExpression |
    bitExpression || bitExpression
}

对于字符串，如果左右操作数均为CHAR类型，则结果类型为CHAR。否则为VARCHAR。 CHAR和VARCHAR的常规空白填充/整理规则适用于此运算符的结果。

结果字符串的长度是两个操作数的长度之和。

对于位字符串，如果左右操作数均为CHAR FOR BIT DATA类型，则结果类型为CHAR FOR BIT DATA；否则，结果类型为CHAR FOR BIT DATA。否则为VARCHAR FOR BIT DATA。

例子

-- returns 'supercalifragilisticexbealidocious(sp?)'
VALUES 'supercalifragilistic' || 'exbealidocious' || '(sp?)'
-- returns NULL
VALUES CAST (null AS VARCHAR(7))|| 'AString'
-- returns '130asdf'
VALUES '130' || 'asdf'
