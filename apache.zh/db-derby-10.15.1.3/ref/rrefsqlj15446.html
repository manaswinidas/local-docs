CREATE VIEW语句

CREATE VIEW语句创建一个视图，该视图是由查询形成的虚拟表。

视图是可以使用的字典对象，直到将其删除。意见不可更新。

如果指定了合格的视图名称，则架构名称不能以SYS开头。

视图以视图所有者的特权进行操作。有关详细信息，请参见《 Derby安全指南 》中的“配置细粒度的用户授权”和“视图，触发器，约束和生成的列的特权”。

视图所有者自动在视图上获得SELECT特权。不能从视图所有者撤消SELECT特权。数据库所有者将自动在视图上获得SELECT特权，并能够将此特权授予其他用户。不能从数据库所有者撤消SELECT特权。

如果视图所有者还拥有基础对象，则视图所有者只能将SELECT特权授予其他用户。

如果视图引用的基础对象不属于视图所有者，则必须授予视图所有者适当的特权。例如，如果授权ID user2尝试创建一个名为user2.v2引用表user1.t1和功能user1.f_abs() ， 然后user2必须对表具有SELECT特权user1.t1以及功能的EXECUTE特权user1.f_abs() 。

授予SELECT特权的特权无法撤消。如果撤销了该视图引用的基础对象之一的必需特权，则该视图将被删除。

句法

CREATE VIEW viewName
    [ ( simpleColumnName [ , simpleColumnName ]* ) ]
AS query [ ORDER BY clause ]
         [ result offset clause ]
         [ fetch first clause ]

视图定义可以包含可选的视图列列表，以显式命名视图中的列。如果没有列列表，则视图从基础查询继承列名称。视图中的所有列都必须唯一命名。

例子

CREATE VIEW SAMP.V1 (COL_SUM, COL_DIFF)
	AS SELECT COMM + BONUS, COMM - BONUS
	FROM SAMP.EMPLOYEE;

CREATE VIEW SAMP.VEMP_RES (RESUME)
	AS VALUES 'Delores M. Quintana', 'Heather A. Nicholls', 'Bruce Adamson';

CREATE VIEW SAMP.PROJ_COMBO 
	(PROJNO, PRENDATE, PRSTAFF, MAJPROJ) 
	AS SELECT PROJNO, PRENDATE, PRSTAFF, MAJPROJ
	FROM SAMP.PROJECT UNION ALL 
SELECT PROJNO, EMSTDATE, EMPTIME, EMPNO 
	FROM SAMP.EMP_ACT 
	WHERE EMPNO IS NOT NULL;

语句依赖系统

视图定义取决于视图定义中引用的表和视图。包含视图引用的DML（数据操作语言）语句取决于这些视图以及视图所依赖的视图定义中的对象。引用该视图的语句取决于该视图使用的索引。根据查询的优化方式，视图使用的索引可以在语句之间更改。例如，给定：
CREATE TABLE T1 (C1 DOUBLE PRECISION);

CREATE FUNCTION SIN (DATA DOUBLE) 
	RETURNS DOUBLE EXTERNAL NAME 'java.lang.Math.sin'
	LANGUAGE JAVA PARAMETER STYLE JAVA;

CREATE VIEW V1 (C1) AS SELECT SIN(C1) FROM T1;
下面的SELECT：
SELECT * FROM V1

取决于视图V1 ，表T1和外部标量函数SIN。
