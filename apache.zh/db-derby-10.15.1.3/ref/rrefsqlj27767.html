表达式中的数字类型提升

在仅使用整数类型的表达式中， Derby将结果的类型提升为至少INTEGER。在将整数与非整数类型混合的表达式中， Derby会将表达式的结果提升为表达式中的最高类型。

下表显示了表达式中数据类型的提升。

表格1。表达式中的类型提升
表达中出现的最大类型 结果表达类型
双精度 双精度
真实 双精度
十进制 十进制
比金特 比金特
整数 整数
SMALLINT 整数
例如：
-- returns a double precision
VALUES 1 + 1.0e0
-- returns a decimal
VALUES 1 + 1.0
-- returns an integer
VALUES CAST (1 AS INT) + CAST (1 AS INT)
