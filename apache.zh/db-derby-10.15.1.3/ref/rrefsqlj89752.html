REFERENCING
{
    { OLD | NEW } [ ROW ] [ AS ] correlationName 
        [ { OLD | NEW } [ ROW ] [ AS ] correlationName ]
  | 
    { OLD_TABLE | NEW_TABLE } [ AS ] identifier 
        [ { OLD_TABLE | NEW_TABLE } [ AS ] identifier ] 
}
注意： OLD_TABLE | NEW_TABLE不赞成使用该语法，因为它不兼容SQL，并且旨在实现向后兼容性和DB2兼容性。
