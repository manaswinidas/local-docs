STDDEV_POP函数

STDDEV_POP是一个聚合函数，用于评估总体上表达式的标准偏差。

有关这些功能的更多信息，请参见聚合（设置功能） 。

STDDEV_POP仅在计算结果为数字数据类型的表达式上允许。

句法

STDDEV_POP ( expression )

该表达式可以包含多个列引用或表达式，但不能包含另一个聚合或子查询。它必须求值为内置数字数据类型。如果表达式的计算结果为NULL，则聚合将跳过该值。

结果数据类型为DOUBLE（可能会溢出）。

式

标准偏差的计算如下：


sqrt( var_pop() )

例

-- find the standard deviation in flight time per aircraft
SELECT AIRCRAFT, STDDEV_POP( flying_time )
FROM flights GROUP BY aircraft;
