newBootPassword = newPassword属性

newBootPassword = newPassword属性为加密的数据库指定新的启动密码。

引擎在内部生成新的加密密钥，并使用新的启动密码保护该密钥。新生成的加密密钥会加密数据库，包括现有数据。有关此属性的更多信息，请参阅《 Derby安全指南 》中的“使用新的引导密码加密数据库” 。

与其他属性结合

newBootPassword = newPassword属性必须与bootPassword = key属性结合使用。

使用newBootPassword = newPassword属性时，不能更改加密提供程序或加密算法。

对于同时启用身份验证和SQL授权的现有加密数据库，只有数据库所有者才能执行重新加密。有关更多信息，请参见《 Derby安全指南 》中的“配置用户身份验证”和“配置用户授权”。

更改启动密码后，请确保检查SQLWarning 。仅当没有SQLWarning或SQLException时 ，更改成功。

例

-- specify a new boot password for a database
jdbc:derby:salesdb;bootPassword=abc1234xyz;newBootPassword=new1234xyz
