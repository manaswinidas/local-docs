只读命令

的Readonly命令将当前连接设置为“只读”连接，就好像当前用户被定义为readOnlyAccess用户。

有关数据库授权的更多信息，请参阅《 Derby安全指南 》中的“配置用户授权”。

句法

READONLY { ON | OFF }

例

ij> connect 'jdbc:derby:menuDB';
ij> readonly on;
ij> SELECT * FROM menu;
COURSE    |ITEM                |PRICE
-----------------------------------------------
entree    |lamb chop           |14
dessert   |creme brulee        |6
appetizer |baby greens         |7
entree    |lamb chop           |14
entree    |lamb chop           |14
dessert   |creme brulee        |6
6 rows selected
ij> UPDATE menu set price = 3;
ERROR 25502: An SQL data change is not permitted for a read-only
connection, user or database.
