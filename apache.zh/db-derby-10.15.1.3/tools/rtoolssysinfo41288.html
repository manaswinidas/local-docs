系统信息

使用sysinfo该实用程序可显示有关Java环境和Derby的信息（包括版本信息）。

使用sysinfo ，请执行以下任一操作：

  • 如果您是Java编程语言的新手，请按照Derby入门中“设置环境”中的说明进行操作。 DERBY_HOME和JAVA_HOME环境变量并添加DERBY_HOME/bin到你的路。然后使用以下命令：
    sysinfo
  • 如果您是Java的普通用户，但不熟悉Derby ，请设置DERBY_HOME环境变量，然后使用java命令来调用derbyrun.jar文件：
    (UNIX) java [options] -jar $DERBY_HOME/lib/derbyrun.jar sysinfo

    (Windows) java [options] -jar %DERBY_HOME%\lib\derbyrun.jar sysinfo
  • 如果您既熟悉Java编程语言又熟悉Derby ，那么您已经设置好了DERBY_HOME 。设置类路径以包含Derby jar文件。然后使用java命令来调用sysinfo直接上课。 
    java org.apache.derby.tools.sysinfo
