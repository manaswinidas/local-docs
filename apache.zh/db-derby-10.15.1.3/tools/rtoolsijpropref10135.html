ij。连接。 connectionName属性

的ij.connection.connectionName属性在以下情况下创建到给定数据库连接URL的命名连接ij启动。

该属性等效于Connect AS Identifier命令。如果数据库连接URL的格式为ij.protocol已指定。每个会话可以多次指定此属性，从而创建多个连接。什么时候ij启动时，它将显示以这种方式创建的所有连接的名称。如果有多个连接，它还会显示当前连接的名称。 ij提示。

句法

ij.connection.connectionName=databaseConnectionURL

在命令行上指定时，不应将databaseConnectionURL用单引号引起来。但是，如果数据库路径包含特殊字符（例如空格），则必须将其用双引号引起来。

例

本示例连接到现有数据库sample并创建然后连接到数据库anotherDB 。

java -Dij.connection.sample1=jdbc:derby:sample \
-Dij.connection.anotherConn=jdbc:derby:anotherDB;create=true \
org.apache.derby.tools.ij
ij version 10.15
ANOTHERCONN* -  jdbc:derby:anotherDB;create=true
SAMPLE1 -        jdbc:derby:sample
* = current connection
ij(ANOTHERCONN)>
