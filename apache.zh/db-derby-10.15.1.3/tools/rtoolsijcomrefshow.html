显示命令

的Show命令显示有关活动连接和数据库对象的信息。

句法

SHOW
{
   CONNECTIONS |
   FUNCTIONS [ IN schemaName ] |
   INDEXES [ IN schemaName | FROM tableName ] |
   PROCEDURES [ IN schemaName ] |
   ROLES |
   ENABLED_ROLES |
   SETTABLE_ROLES |
   SCHEMAS |
   SYNONYMS [ IN schemaName ] |
   TABLES [ IN schemaName ] |
   VIEWS [ IN schemaName ] |

}

显示连接

如果没有连接，则SHOW CONNECTIONS命令返回“无可用连接”。

否则，该命令将显示连接名称和用于连接到它们的URL的列表。当前活动的连接（如果有的话）标有星号（ * ）后的名字。

显示连接示例

ij> connect 'sample' as sample1;
ij> connect 'newDB;create=true' as newDB;
ij(NEWDB)> show connections;
SAMPLE1 -        jdbc:derby:sample
NEWDB* -        jdbc:derby:newDB;create=true
* = current connection
ij(NEWDB)>

显示功能

的SHOW FUNCTIONS命令显示数据库中的所有功能。默认情况下，系统功能和用户定义的功能都出现在输出中。

如果IN schemaName指定后，仅显示指定模式中的功能。

显示功能示例

如果您创建了TO_DEGREES Derby参考手册中“ CREATE FUNCTION语句”中描述的函数，CREATE FUNCTION语句的输出和SHOW FUNCTIONS命令如下所示：

ij> connect 'jdbc:derby:firstdb';
ij> CREATE FUNCTION TO_DEGREES ( RADIANS DOUBLE )
> RETURNS DOUBLE
> PARAMETER STYLE JAVA
> NO SQL LANGUAGE JAVA
> EXTERNAL NAME 'java.lang.Math.toDegrees';
0 rows inserted/updated/deleted
ij> show functions in app;
FUNCTION_SCHEM|FUNCTION_NAME               |REMARKS
-------------------------------------------------------------------------
APP           |TO_DEGREES                  |java.lang.Math.toDegrees           

1 row selected

显示角色，显示ENABLED_ROLES和显示SETTABLE_ROLES

的SHOW ROLES命令显示创建的所有角色的名称，无论是否可为当前会话设置。

的SHOW ENABLED_ROLES命令显示其特权可用于当前会话的所有角色的名称。也就是说，它显示当前角色以及当前角色中包含的任何角色。（有关角色包含的定义，请参阅《 Derby安全指南 》中的“使用SQL角色”。）

的SHOW SETTABLE_ROLES命令显示当前会话可以设置的所有角色，即已授予当前用户或PUBLIC的所有角色。

这些命令显示的角色按升序排序。

SHOW ROLES，SHOW ENABLED_ROLES和SHOW SETTABLE_ROLES示例

在以下示例中，CASUALUSER和POWERUSER都包含ANYUSER，但是ANYUSER不能直接设置。

ij> show roles;
ROLEID                        
------------------------------
ANYUSER                             
CASUALUSER                             
POWERUSER                             

3 rows selected
ij> show enabled_roles;
ROLEID                        
------------------------------
ANYUSER                             
CASUALUSER                             

2 rows selected
ij> show settable_roles;
ROLEID                        
------------------------------
CASUALUSER                             
POWERUSER

2 rows selected

显示同步

的SHOW SYNONYMS命令显示数据库中所有使用CREATE SYNONYMS语句创建的同义词。

如果IN schemaName指定时，仅显示指定架构中的同义词。

显示同义词示例

ij> show synonyms;
TABLE_SCHEM         |TABLE_NAME                    |REMARKS
------------------------------------------------------------------------
APP                 |MYAIRLINES                    |
