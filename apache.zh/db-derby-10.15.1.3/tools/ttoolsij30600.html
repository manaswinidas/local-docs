使用属性启动ij

你可以设定ij可以通过以下任何一种方式来设置属性。

  • 通过使用-D命令行上的选项
  • 通过使用-p propertyfile命令行上的选项
记得： ij属性名称区分大小写，而命令则区分大小写。

例子

以下示例说明了如何使用ij属性。

开始ij通过使用名为ij.properties ，请使用以下命令（并添加文件路径）：

java -jar derbyrun.jar -p ij.properties

开始ij带着ij.maximumDisplayWidth的1000：

java -Dij.maximumDisplayWidth=1000 -jar derbyrun.jar

开始ij带着ij.protocol的jdbc:derby:和ij.database的sample ，请使用以下命令：

java -Dij.protocol=jdbc:derby: -Dij.database=sample derbyrun.jar

开始ij有两个命名的连接，使用ij.connection.connectionName属性，请使用如下命令：

java -Dij.connection.sample=jdbc:derby:sample \
-Dij.connection.History=jdbc:derby:History \
-Dderby.system.home=c:\derby\demo\databases \
-jar c:\derby\lib\derbyrun.jar

要查看连接名称和用于连接它们的URL的列表，请使用以下命令。（如果当前有活动的连接，它将显示一个星号（ * ）。

ij version 10.15
ij(HISTORY)> show connections;
HISTORY* -    jdbc:derby:History
SAMPLE -     jdbc:derby:sample
* = current connection
ij(HISTORY)>
