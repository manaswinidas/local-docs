运行ij脚本

您可以在以下位置运行脚本ij以下列任何一种方式。

  • 将输入文件命名为命令行参数。例如：
    java org.apache.derby.tools.ij myscript.sql
  • 重定向标准输入以来自文件。例如：
    java org.apache.derby.tools.ij < myscript.sql
  • 使用Run来自的命令ij命令行。例如：
    ij> run 'myscript.sql';
注意：如果您将输入文件命名为命令行参数，或者使用Run命令， ij回显文件的输入。如果您将标准输入重定向为来自文件， ij不回显命令。

您可以通过以下任意一种方式保存输出：

  • 通过将输出重定向到文件：
    java org.apache.derby.tools.ij myscript.sql > myoutput.txt
  • 通过设置ij.outfile属性：
    java -Dij.outfile=myoutput.txt org.apache.derby.tools.ij 
    myscript.sql

ij进入时退出Exit命令，或者，如果执行脚本，则到达命令文件的末尾。当您使用Exit命令， ij通过发出以下命令自动关闭嵌入式Derby系统connect jdbc:derby:;shutdown=true请求。如果它在服务器框架中运行，它不会关闭Derby 。
