ij.dataSource属性

的ij.dataSource属性指定用于访问数据库的数据源。

指定数据源时， ij不使用DriverManager机制建立连接。

句法

当您设置ij.dataSource属性， ij将自动尝试连接到数据库。使用以下命令建立到特定数据库的连接ij.dataSource ，将ij.dataSource.databaseName属性。如果您未设置此属性， ij将以错误开头。如果要创建数据库，请指定ij.dataSource.createDatabase财产以及ij.dataSource.databaseName 。不指定ij.protocol设定时ij.dataSource ，因为那样会激活DriverManager机制。

ij.dataSource=datasourceClassName
ij.dataSource.databaseName=databaseName
[ij.dataSource.createDatabase=create]

如果不指定ij.dataSource.databaseName并显示一条错误消息，表明未找到数据库，您仍然可以使用以下命令连接到数据库ij的Connect命令。您不应该指定协议（例如， jdbc:derby: ） 在里面connect使用时的命令ij.dataSource 。

例

在以下示例中， ij连接到名为sample使用EmbeddedDataSource 。的sample如果数据库尚不存在，则会创建该数据库。 

java -Dij.dataSource=org.apache.derby.jdbc.EmbeddedDataSource \
-Dij.dataSource.databaseName=sample \
-Dij.dataSource.createDatabase=create org.apache.derby.tools.ij
ij version 10.15 
CONNECTION0*
* = current connection
ij>
