SQL语法

SQL语法以修改后的BNF表示法表示。

下表列出了BNF的元符号。

表格1。BNF元符号
符号 含义
| 要么。选择其中一项。
[ ] 包含可选项目。
* 标记可重复0次或多次的项目。在某些SQL语句中具有特殊含义。
{ } 将所需项目分组，以便可以用符号标记| 。
( ) . , 语法中的其他标点符号。
以下示例显示了如何显示SQL语法：
CREATE [ UNIQUE ] INDEX indexName
  ON tableName ( simpleColumnName [ , simpleColumnName ]* )
