Derby随附的脚本

Derby包括一些脚本，这些脚本可以启动Derby工具，将适当的库添加到CLASSPATH中，以及启动和停止网络服务器。这些脚本仅在您下载Derby的bin发行版时可用。

Derby脚本位于安装Derby的/ bin目录中。要运行脚本，您需要设置几个环境变量，以使脚本正确运行。请参阅设置环境变量以设置环境变量。

您必须在适当的命令行环境（例如Windows命令提示符或UNIX Shell）中运行Derby脚本。

Derby随附的脚本的完整列表为：
bin / dblook
运行dblook工具。
bin / ij
开始ij工具。
bin / NetworkServerControl
运行NetworkServerControl。
bin / setEmbeddedCP
将用于嵌入式环境的所有Derby库放在CLASSPATH中。
bin / setNetworkClientCP
将连接到Derby Network Server所需的库放入CLASSPATH。
bin / setNetworkServerCP
将启动Derby Network Server所需的库放入CLASSPATH。
bin / startNetworkServer
在本地计算机上启动网络服务器。
bin / stopNetworkServer
停止本地计算机上的网络服务器。
bin / sysinfo
运行sysinfo工具。
