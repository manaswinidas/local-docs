derbyrun.jar文件的语法

的derbyrun.jar文件是一个特殊的JAR文件，可简化您调用Derby工具和网络服务器的方式。

随着derbyrun.jar文件，则可以使用缩写名称运行Derby工具和实用程序，而无需设置Java CLASSPATH环境变量。的derbyrun.jar文件必须与其他Derby JAR文件位于同一文件夹中。

句法

下表显示了使用的语法derbyrun.jar用于每个Derby工具。

表格1。使用语法derbyrun.jar
操作系统 命令
UNIX（Korn Shell）

java -jar $DERBY_HOME/lib/derbyrun.jar ij [-p propertiesfile] [sql_script]
java -jar $DERBY_HOME/lib/derbyrun.jar sysinfo [-cp ...] [-cp help]
java -jar $DERBY_HOME/lib/derbyrun.jar dblook [arg]* （或没有使用参数）
java -jar $DERBY_HOME/lib/derbyrun.jar server [arg]* （或没有使用参数）
视窗

java -jar %DERBY_HOME%\lib\derbyrun.jar ij [-p propertiesfile] [sql_script]
java -jar %DERBY_HOME%\lib\derbyrun.jar sysinfo [-cp ...] [-cp help]
java -jar %DERBY_HOME%\lib\derbyrun.jar dblook [arg]* （或没有使用参数）
java -jar %DERBY_HOME%\lib\derbyrun.jar server [arg]* （或没有使用参数）

要查看此语法提示，请运行命令java -jar derbyrun.jar没有参数。

附加信息

请参阅手动设置CLASSPATH / MODULEPATH环境变量 ，以获取有关设置类路径和运行ij工具。

替代使用derbyrun.jar jar文件-jar参数，您可以改用derbyrun.jar jar文件-cp参数或CLASSPATH环境变量中。

使用此技术时，请添加derbyrun.jar将JAR文件添加到CLASSPATH的作用是将所有Derby类包含在类路径中。有关此用途的更多信息derbyrun.jar ，请参阅《 Derby服务器和管理指南》 。
