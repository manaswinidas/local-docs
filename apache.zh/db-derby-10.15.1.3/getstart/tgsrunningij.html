运行ij

德比 ij该工具是JDBC工具，可用于对Derby数据库运行脚本或交互式查询。

  • 选择将用于运行ij脚本。下表列出了这些方法。
    表格1。运行方法ij效用
    方法 何时使用 命令
    跑ij作为独立命令。 如果您是Java编程语言的新手，还是Derby的新手，请使用此方法。 在运行操作系统之前，请按照设置环境变量中的步骤进行操作。 ij工具使用此方法。运行ij在命令行中使用以下脚本：
    ij

    您必须添加DERBY_HOME/bin目录到PATH环境变量，然后才能运行ij工具。

    的ij脚本设置相应的环境变量（包括CLASSPATH），并启动ij工具。
    跑ij使用位于以下目录中的jar文件ij居住。 如果您不熟悉Derby ，但熟悉Java编程语言，请使用此方法。 您必须先设置DERBY_HOME环境变量，然后才能运行ij工具使用此方法。

    在UNIX上，命令是：

    java -jar $DERBY_HOME/lib/derbyrun.jar ij

    在Windows上，命令为：

    java -jar %DERBY_HOME%\lib\derbyrun.jar ij
    跑ij使用java命令。 如果您既熟悉Java编程语言，又熟悉Derby ，请使用此方法，并且已经添加了java可执行文件添加到命令执行路径。 您必须设置CLASSPATH。使用手动设置CLASSPATH / MODULEPATH环境变量中指定的步骤。然后在java命令。例如：
    java org.apache.derby.tools.ij
    跑ij使用java带有模块路径的命令。 如果您既熟悉Java编程语言又熟悉Derby ，请使用此方法，并且已经添加了java可执行文件添加到命令执行路径，并希望通过模块路径运行Derby。 您必须设置您的MODULEPATH。使用手动设置CLASSPATH / MODULEPATH环境变量中指定的步骤。然后在工具栏中指定工具模块和ij类名称java命令。

    在UNIX上，命令是：

    java -p $MODULEPATH \
  -m org.apache.derby.tools/org.apache.derby.tools.ij

    在Windows上，命令为：

    java -p %MODULEPAT% ^
  -m org.apache.derby.tools/org.apache.derby.tools.ij
  • 当您准备离开时ij工具，输入： 
    ij> exit;
