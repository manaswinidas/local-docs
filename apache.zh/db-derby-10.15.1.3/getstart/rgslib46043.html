德比提供的图书馆

本节显示了Derby使用的不同库（模块）及其用途。

下表显示了Derby共享库。所有Derby配置都需要此模块。

表格1。引擎库
图书馆名称 使用
derbyshared.jar 无论您运行的是嵌入式引擎，网络服务器，远程客户端还是数据库工具，所有配置均需要此模块。

下表显示了Derby引擎库。

表2。引擎库
图书馆名称 使用
derby.jar 对于嵌入式数据库。

对于嵌入式环境，您始终需要此库。对于客户端/服务器环境，您仅在服务器上需要此库。

对于嵌入式环境，您需要在类路径/模块路径中使用另一个模块才能使用这些工具。对于客户端/服务器环境，您仅需要客户端上的该库。下表显示了工具模块。

表3。工具库
图书馆名称 使用
derbytools.jar 运行所有Derby工具（例如ij ， dblook ，以及导入/导出）。如果您正在运行网络服务器，则也需要此模块。
derbyrun.jar 可执行的jar文件，可用于启动Derby工具。

下表显示了Derby Network Server库。您需要此模块以及共享，引擎和工具模块。

表4。网络服务器库
图书馆名称 使用
derbynet.jar 启动Derby Network Server所需。

下表显示了Derby网络客户端库。

表5。网络客户端库
图书馆名称 使用
derbyclient.jar 使用Derby网络客户端驱动程序是必需的。

下表显示了Derby语言环境库。

表6。语言环境库
图书馆名称 使用
derbyLocale_cs.jar 提供捷克语言环境的翻译后的消息所必需。
derbyLocale_de_DE.jar 提供德语语言环境的翻译后消息所必需。
derbyLocale_es.jar 提供西班牙语语言环境的翻译后消息所必需。
derbyLocale_fr.jar 提供法语区域设置的翻译消息所必需。
derbyLocale_hu.jar 提供匈牙利语言环境的翻译后消息所必需。
derbyLocale_it.jar 必须提供意大利语区域的翻译后消息。
derbyLocale_ja_JP.jar 提供日语语言环境的翻译消息所必需。
derbyLocale_ko_KR.jar 提供韩文语言环境的翻译后的消息所必需。
derbyLocale_pl.jar 提供波兰语言环境的翻译后的消息所必需。
derbyLocale_pt_BR.jar 提供巴西葡萄牙语语言环境的翻译后消息所必需。
derbyLocale_ru.jar 需要提供俄语语言环境的翻译消息。
derbyLocale_zh_CN.jar 必须提供简体中文语言环境的翻译后的消息。
derbyLocale_zh_TW.jar 需要提供繁体中文语言环境的翻译后消息。

概括而言，下表显示了每个Derby配置所需的库（模块）。

表7。所需的库
组态 必修模块
Remote client derbyclient.jar，derbyshared.jar
Embedded engine derby.jar，derbyshared.jar
Network server derbynet.jar，derbytools.jar，derby.jar，derbyshared.jar
Embedded tools derbytools.jar，derby.jar，derbyshared.jar
Convenient startup derbyrun.jar，derbynet.jar，derbytools.jar，derby.jar，derbyshared.jar
