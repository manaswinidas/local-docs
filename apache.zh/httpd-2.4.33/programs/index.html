服务器和支持程序

此页面记录了Apache HTTP Server随附的所有可执行程序。

指数

httpd
Apache超文本传输协议服务器
apachectl
Apache HTTP服务器控制界面
ab
Apache HTTP服务器基准测试工具
apxs
APache eXtenSion工具
configure
配置源树
dbmmanage
创建和更新DBM格式的用户身份验证文件以进行基本身份验证
fcgistarter
启动FastCGI程序
htcacheclean
清理磁盘缓存
htdigest
创建和更新用户身份验证文件以进行摘要身份验证
htdbm
操作DBM密码数据库。
htpasswd
创建和更新用户身份验证文件以进行基本身份验证
httxt2dbm
创建用于RewriteMap的dbm文件
logresolve
在Apache日志文件中解析IP地址的主机名
log_server_status
定期记录服务器状态
rotatelogs
旋转Apache日志而不必终止服务器
split-logfile
将多虚拟机日志文件拆分为每个主机的日志文件
suexec
切换用户执行
