httxt2dbm-生成用于RewriteMap的dbm文件

httxt2dbm用于从文本输入生成dbm文件，以用于RewriteMap与dbm地图类型。

如果输出文件已经存在，则不会被截断。新密钥将被添加，现有密钥将被更新。

  • 概要
  • 选件
  • 例子

也可以看看

  • httpd

概要

httxt2dbm [ -v ] [ -f DBM_TYPE ] -i SOURCE_TXT -o OUTPUT_DBM

选件

-v
更详细的输出
-f DBM_TYPE
指定用于输出的DBM类型。如果未指定，将使用APR默认值。可用的类型有： GDBM对于GDBM文件， SDBM对于SDBM文件， DB对于berkeley DB文件， NDBM对于NDBM文件， default默认的DBM类型。
-i SOURCE_TXT
要从中创建dbm的输入文件。该文件的格式应为每行一条记录，格式为： key value 。请参阅有关的文档RewriteMap有关此文件格式和含义的更多详细信息。
-o OUTPUT_DBM
输出dbm文件的名称。

例子

httxt2dbm -i rewritemap.txt -o rewritemap.dbm
httxt2dbm -f SDBM -i rewritemap.txt -o rewritemap.dbm
