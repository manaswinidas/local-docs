fcgistarter-启动FastCGI程序

  • 注意
  • 概要
  • 选件

也可以看看

  • mod_proxy_fcgi

注意

当前仅适用于Unix系统。

概要

fcgistarter -c command -p port [ -i interface ] -N num

选件

-c command
FastCGI程序
-p port
程序将监听的端口
-i interface
程序将监听的接口
-N num
程序的实例数
