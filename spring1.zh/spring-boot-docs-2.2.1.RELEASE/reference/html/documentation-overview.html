2。获得帮助

如果您在使用Spring Boot时遇到问题，我们将为您提供帮助。

  • 尝试使用方法文档 。他们为最常见的问题提供解决方案。

  • 了解Spring基础知识。Spring Boot建立在许多其他Spring项目上。在spring.io网站上查看大量参考文档。如果您是从Spring开始的，请尝试其中的指南之一 。

  • 问一个问题。我们会监视stackoverflow.com中是否有标有的问题spring-boot 。

  • 在github.com/spring-projects/spring-boot/issues上报告Spring Boot的错误 。

所有的Spring Boot都是开源的，包括文档。如果您发现文档有问题或想要改进它们，请参与其中 。
