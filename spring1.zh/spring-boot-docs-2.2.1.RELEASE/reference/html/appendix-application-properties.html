您可以在内部指定各种属性application.properties文件，在您的内部application.yml文件，或作为命令行开关。本附录提供了常见的Spring Boot属性列表以及对使用它们的基础类的引用。

Spring Boot提供了各种具有高级值格式的转换机制，请务必查看属性转换部分 。
属性贡献可能来自类路径上的其他jar文件，因此您不应将其视为详尽的列表。另外，您可以定义自己的属性。

spring.cache.jcache.provider

CachingProvider实现的完全限定名称，用于检索符合JSR-107的缓存管理器。仅当类路径上有多个JSR-107实现可用时才需要。
