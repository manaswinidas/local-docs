如果出现以下情况，Spring Boot Actuator将显示与构建相关的信息： META-INF/build-info.properties文件存在。的build-info目标使用项目坐标和构建时间生成此类文件。它还允许您添加任意数量的其他属性：

<project>
  ...
  <build>
    ...
    <plugins>
      ...
      <plugin>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-maven-plugin</artifactId>
        <version>2.2.1.RELEASE</version>
        <executions>
          <execution>
            <goals>
              <goal>build-info</goal>
            </goals>
            <configuration>
              <additionalProperties>
                <encoding.source>UTF-8</encoding.source>
                <encoding.reporting>UTF-8</encoding.reporting>
                <java.source>${maven.compiler.source}</java.source>
                <java.target>${maven.compiler.target}</java.target>
              </additionalProperties>
            </configuration>
          </execution>
        </executions>
        ...
      </plugin>
      ...
    </plugins>
    ...
  </build>
  ...
</project>

此配置将生成一个build-info.properties在预期的位置，还有四个附加键。注意maven.compiler.source和maven.compiler.target预计将是项目中可用的常规属性。它们将按照您的期望进行插值。
