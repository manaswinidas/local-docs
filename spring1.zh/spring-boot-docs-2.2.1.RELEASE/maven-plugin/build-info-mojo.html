全名

org.springframework.boot：spring-boot-maven-plugin：2.2.1。发布：build-info

说明 ：

根据当前MavenProject的内容生成一个build-info.properties文件。

属性 ：

  • 需要执行一个Maven项目。
  • 由于版本：1.4.0。
  • 默认情况下绑定到生命周期阶段 ：generate-resources。

可选参数

名称 类型 以来 描述
地图 1.4.0 要存储在build-info.properties中的其他属性。每个条目都以build为前缀。在生成的build-info.properties中。
文件 1.4.0 生成的build-info.properties的位置。
默认值为 ：$ {project.build.outputDirectory} /META-INF/build-info.properties。
串 2.2.0 用于build.time属性的值采用适合Instant.parse（）的形式。默认为session.request.startTime。要完全禁用build.time属性，请使用“ off”。
