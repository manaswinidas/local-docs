总览

该项目使用Git来管理其源代码。有关Git使用的说明，请参见http://git-scm.com/documentation 。

Web浏览器访问

以下是到源存储库的可浏览版本的链接：

https://github.com/spring-projects/spring-boot

匿名访问

可以使用以下命令从Git匿名检出源（请参见http://git-scm.com/docs/git-clone ）：

$ git clone git://github.com/spring-projects/spring-boot.git

开发人员访问

只有项目开发人员才能通过此方法访问Git树（请参阅http://git-scm.com/docs/git-clone ）。

$ git clone ssh://git@github.com/spring-projects/spring-boot.git

从防火墙后面进行访问

有关在防火墙后面进行访问的更多信息，请参考所用SCM的文档。
