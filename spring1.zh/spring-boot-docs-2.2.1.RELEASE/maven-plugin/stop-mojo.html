春季引导：停止

全名

org.springframework.boot：spring-boot-maven-plugin：2.2.1。发布：停止

说明 ：

停止已由“启动”目标启动的spring应用程序。通常在测试套件完成后调用。

属性 ：

  • 需要执行一个Maven项目。
  • 由于版本：1.3.0。
  • 默认情况下绑定到生命周期阶段 ：集成测试后。

可选参数

名称 类型 以来 描述
布尔型 1.3.0 指示停止过程是否已分叉的标志。默认情况下，该值是从MavenProject继承的。如果已设置，则它必须与StartMojo启动过程所使用的值匹配。
用户属性是 ：spring-boot.stop.fork。
串 1.3.0 自动部署的MBean的JMX名称，用于管理应用程序的生命周期。
整型 1.3.0 如果应用程序已经分叉，则用于查找平台MBeanServer的端口。
布尔值 1.3.2 跳过执行。
默认值为 ：false。
用户属性是 ：spring-boot.stop.skip。

参数明细

指示停止过程是否已分叉的标志。默认情况下，该值是从MavenProject继承的。如果已设置，则它必须与StartMojo启动过程所使用的值匹配。
  • 类型 ：java.lang。布尔型
  • 由于 ：1.3.0
  • 必填 ：否
  • 用户属性 ：spring-boot.stop.fork

自动部署的MBean的JMX名称，用于管理应用程序的生命周期。
  • 类型 ：java.lang。串
  • 由于 ：1.3.0
  • 必填 ：否

如果应用程序已经分叉，则用于查找平台MBeanServer的端口。
  • 类型 ：int
  • 由于 ：1.3.0
  • 必填 ：否

跳过执行。
  • 类型 ：布尔值
  • 由于 ：1.3.2
  • 必填 ：否
  • 用户属性 ：spring-boot.stop.skip
  • 默认值 ：false
