春季启动：重新包装

全名

org.springframework.boot：spring-boot-maven-plugin：2.2.1。发行：重新包装

说明 ：

重新打包现有的JAR和WAR归档文件，以便可以使用java -jar从命令行执行它们。使用layout = NONE也可以简单地用于打包具有嵌套依赖关系的JAR（并且没有主类，因此不是可执行文件）。

属性 ：

  • 需要执行一个Maven项目。
  • 需要范围内工件的依赖项解析：编译+运行时。
  • 由于版本：1.0.0。
  • 默认情况下绑定到生命周期阶段 ：包。
