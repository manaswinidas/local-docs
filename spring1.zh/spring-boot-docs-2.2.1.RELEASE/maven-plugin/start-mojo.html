Spring Boot：开始

全名

org.springframework.boot：spring-boot-maven-plugin：2.2.1。发布：开始

说明 ：

启动spring应用程序。与运行目标相反，这不会阻止并允许其他目标在应用程序上运行。此目标通常用于集成测试方案中，在该方案中，应用程序在测试套件之前启动，然后在测试套件之后停止。

属性 ：

  • 需要执行一个Maven项目。
  • 需要范围：测试中的工件的依赖项解析。
  • 由于版本：1.3.0。
  • 默认情况下绑定到生命周期阶段 ：pre-integration-test。
