插件文档

此插件可用的目标：

目标 描述
春季启动：建设信息 产生一个build-info.properties基于当前文件的内容MavenProject 。
春季引导：帮助 在spring-boot-maven-plugin上显示帮助信息。
呼叫mvn spring-boot:help -Ddetail=true -Dgoal=显示参数详细信息。
春季启动：重新包装 重新打包现有的JAR和WAR归档文件，以便可以使用java -jar从命令行执行它们。用layout=NONE也可以简单地用于打包具有嵌套依赖关系的JAR（并且没有主类，因此不能执行）。
春季启动：运行 运行可执行的存档应用程序。
春季启动：开始 启动spring应用程序。与...相反run目标，这不会阻止并允许其他目标在应用程序上运行。此目标通常用于集成测试方案中，在该方案中，应用程序在测试套件之前启动，然后在测试套件之后停止。
春季引导：停止 停止已由“启动”目标启动的spring应用程序。通常在测试套件完成后调用。

系统要求

下面指定了运行此Maven插件的最低要求：

马文 2.0
JDK 1.8
记忆 没有最低要求。
磁盘空间 没有最低要求。

用法

您应该在项目的插件配置中指定版本：

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>org.springframework.boot</groupId>
          <artifactId>spring-boot-maven-plugin</artifactId>
          <version>2.2.1.RELEASE</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-maven-plugin</artifactId>
        <version>2.2.1.RELEASE</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
</project>

有关更多信息，请参见“配置插件指南”。
