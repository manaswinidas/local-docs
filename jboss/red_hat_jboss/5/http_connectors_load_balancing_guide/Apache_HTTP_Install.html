⁠Chapter 2. Download and install

Apache HTTP Server is included in the JBoss Enterprise Web Server binary you download from https://access.redhat.com.
Apache mod_jk is included in the native installation binaries for JBoss Enterprise Application Platform and JBoss Enterprise Web Server.
Follow the procedures in the JBoss Enterprise Application Platform or JBoss Enterprise Web Server Installation Guide to download and install the correct platform and native binaries.
