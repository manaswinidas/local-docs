8.7小结

  • 标准awk提供一维关联数组（由字符串值索引的数组）。所有数组都是关联的；数字索引会自动转换为字符串。
  • 数组元素被引用为array[indx] 。如果先前不存在引用元素，则会创建该元素。
  • 查看数组是否具有具有给定索引的元素的正确方法是使用in运算子： indx in array '。
  • 使用 ' for (indx in array) …以扫描数组的所有单个元素。在循环的主体中indx依次采用每个元素的索引值。
  • ' for (indx in array) '循环遍历数组在POSIX中未定义awk并因实现而异。 gawk可让您通过将特殊的预定义值分配给PROCINFO["sorted_in"] 。
  • 使用 ' delete array[indx] '删除单个元素。要删除数组中的所有元素，请使用' delete array '。后者的功能已成为多年的通用扩展，现已成为标准功能，但可能并非所有商用版本都支持awk 。
  • 标准awk通过用逗号分隔下标值来模拟多维数组。这些值被连接成单个字符串，并由的值分隔SUBSEP 。这样的下标是通过这种方式创建的事实并未保留；因此，改变SUBSEP可能会有意想不到的后果。您可以使用 ' (sub1, sub2, …) in array '以查看是否存在这样的多维下标array 。
  • gawk提供数组的真实数组。您为此类数组中的每个维度使用单独的一组方括号： data[row][col] ， 例如。因此，数组元素可以是标量值（数字或字符串）或其他数组。
  • 使用isarray()内置函数确定数组元素本身是否为子数组。
