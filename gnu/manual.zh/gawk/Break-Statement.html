7.4.6 break声明

的break语句从最里面跳出来for ， while ， 要么do封闭它的循环。下面的示例查找任何整数的最小除数，并标识素数：

# find smallest divisor of num
{
    num = $1
    for (divisor = 2; divisor * divisor <= num; divisor++) {
        if (num % divisor == 0)
            break
    }

    if (num % divisor == 0)
        printf "Smallest divisor of %d is %d\n", num, divisor
    else
        printf "%d is prime\n", num
}

当第一个的余数为零时if声明， awk立刻突破包含for环。这意味着awk立即进入循环后的语句并继续处理。（这与exit语句，它将停止整个awk程序。见节的exit声明 。）

以下程序说明了condition的for要么while语句可以替换为break里面if ：

# find smallest divisor of num
{
    num = $1
    for (divisor = 2; ; divisor++) {
        if (num % divisor == 0) {
            printf "Smallest divisor of %d is %d\n", num, divisor
            break
        }
        if (divisor * divisor > num) {
            printf "%d is prime\n", num
            break
        }
    }
}
