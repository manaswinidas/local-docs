11.3.5生成单词使用计数

当处理大量文本时，知道不同单词出现的频率可能很有趣。例如，作者可能会过度使用某些单词，在这种情况下，他或她可能希望找到同义词来代替出现频率很高的单词。本小节开发了一个程序，用于对单词进行计数并以有用的格式显示频率信息。

乍看起来，像这样的程序似乎可以完成这项工作：

# wordfreq-first-try.awk --- print list of word frequencies

{
    for (i = 1; i <= NF; i++)
        freq[$i]++
}


END {
    for (word in freq)
        printf "%s\t%d\n", word, freq[word]
}

该程序依赖awk的默认字段拆分机制，用于将每一行分解为“单词”，并使用名为freq ，由每个单词索引，以计算单词出现的次数。在里面END规则，它将打印计数。

该程序有一些问题，使其无法在实际文本文件中使用：

  • 的awk语言认为大写和小写字符是不同的。因此，“酒保”和“酒保”不会被视为同一词。这是不希望的，因为如果单词在普通文本中以句子开头时会被大写，并且频率分析仪不应对大写敏感。
  • 使用awk约定字段仅由空格分隔。输入中的其他字符（换行符除外）对awk 。这意味着标点符号算作单词的一部分。
  • 输出不会以任何有用的顺序出现。您更可能对哪个单词出现频率最高或对每个单词出现频率的字母顺序表感兴趣。

第一个问题可以通过使用解决tolower()删除区分大小写。第二个问题可以通过使用解决gsub()删除标点符号。最后，我们通过使用系统解决了第三个问题sort实用程序来处理输出awk脚本。这是程序的新版本：

# wordfreq.awk --- print list of word frequencies

{
    $0 = tolower($0)    # remove case distinctions
    # remove punctuation
    gsub(/[^[:alnum:]_[:blank:]]/, "", $0)
    for (i = 1; i <= NF; i++)
        freq[$i]++
}

END {
    for (word in freq)
        printf "%s\t%d\n", word, freq[word]
}

正则表达式/[^[:alnum:]_[:blank:]]/可能已经写了/[[:punct:]]/ ，但下划线也将被删除，我们希望保留它们。

假设我们已经将该程序保存在一个名为wordfreq.awk ，并且数据在file1 ，以下管道：

awk -f wordfreq.awk file1 | sort -k 2nr

产生一张出现在其中的单词表file1按频率递减的顺序。

的awk程序会适当地按摩数据并产生一个字频表，该表未排序。的awk脚本的输出然后按sort实用程序并打印在屏幕上。

给的选项sort指定使用每条输入行的第二个字段的排序（跳过一个字段），则排序键应视为数字量（否则' 15 '会先于' 5 '），并且排序应以降序（反向）进行。

的sort甚至可以通过更改END采取行动：

END {
    sort = "sort -k 2nr"
    for (word in freq)
        printf "%s\t%d\n", word, freq[word] | sort
    close(sort)
}

这种排序方式必须用于在命令行（或批处理文件）级别没有真正管道的系统上。有关如何使用操作系统的更多信息，请参见常规操作系统文档。 sort程序。
