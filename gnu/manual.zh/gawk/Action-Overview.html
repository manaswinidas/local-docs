7.3行动

一个awk程序或脚本由散布的一系列规则和功能定义组成。（功能将在后面介绍。请参阅用户定义的功能部分 。）规则包含一个模式和一个操作，可以省略其中的一个（但不能同时删除两者）。行动的目的是告诉awk找到匹配的模式后该怎么做。因此，总的来说， awk程序通常如下所示：

[pattern]  { action }
 pattern  [{ action }]
…
function name(args) { … }
…

一个动作由一个或多个组成awk 语句 ，括在括号（” {…} '）。每个语句指定要做的一件事。语句用换行符或分号分隔。即使动作仅包含一条语句，或者根本不包含任何语句，也必须使用括号括起来。但是，如果您完全省略该动作，则也要省略花括号。省略的动作等同于' { print $0 } '：

/foo/  { }     match foo, do nothing — empty action
/foo/          match foo, print the record — omitted action

以下类型的语句受支持： awk ：

表达方式

调用函数或为变量分配值（请参见“ 表达式”部分）。执行这种语句只是计算表达式的值。当表达式有副作用时，这很有用（请参阅“ 赋值表达式”部分）。

控制声明

指定控制流awk程式。的awk语言为您提供类似C的构造（ if ， for ， while和do ）以及一些特殊的内容（请参见“操作中的控制语句”部分）。

复合陈述

用括号将一个或多个语句括起来。使用复合语句以便将多个语句放到一个主体中。 if ， while ， do ， 要么for声明。

输入语句

使用getline命令（请参阅使用以下命令进行显式输入getline ）。也提供awk是next声明（见部分的next声明 ）和nextfile声明（见部分的nextfile声明 ）。

输出语句

如print和printf 。请参阅“ 打印输出”部分。

删除声明

用于删除数组元素。见节的delete声明 。
