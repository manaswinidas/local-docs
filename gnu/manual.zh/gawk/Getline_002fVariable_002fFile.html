4.10.4使用getline从文件变成变量

使用 ' getline var < file '从文件中读取输入file ，并将其放在变量中var 。像以前一样file是一个字符串值表达式，它指定要从中读取文件。

在这个版本中getline ，则所有预定义变量均不会更改，并且记录也不会拆分为字段。唯一更改的变量是var 。 24例如，以下程序将所有输入文件复制到输出中，但记录为“ @include filename ' 。这样的记录被文件内容替换filename ：

{
     if (NF == 2 && $1 == "@include") {
          while ((getline line < $2) > 0)
               print line
          close($2)
     } else
          print
}

注意这里额外的输入文件的名称是如何没有内置到程序中的；它直接从数据中获取，特别是从第二个字段中获取@include线。

的close()调用函数以确保两个相同@include行出现在输入中，整个指定文件被包含两次。请参阅“ 关闭输入和输出重定向”部分 。

该程序的一个不足是它不处理嵌套@include声明（即， @include包含文件中的语句），就像真正的宏预处理程序那样。有关处理嵌套的程序，请参见使用库函数的简便方法 @include陈述。
