5.3输出分离器

如前所述， print语句包含用逗号分隔的项目列表。在输出中，项目通常由单个空格分隔。但是，不必如此。单个空格只是默认设置。通过设置预定义变量，可以将任何字符串用作输出字段分隔符 OFS 。此变量的初始值为字符串" " （即单个空格）。

整个输出print语句称为输出记录 。每print语句输出一个输出记录，然后输出一个字符串，称为输出记录分隔符 （或ORS ）。的初始值ORS是字符串"\n" （即换行符）。因此，每个print语句通常单独一行。

为了更改输出字段和记录的分隔方式，请为变量分配新值OFS和ORS 。通常这样做的地方是在BEGIN规则（见节的BEGIN和END特殊模式 ），以便在处理任何输入之前发生。也可以通过在命令行上，在输入文件名之前的分配或使用-v命令行选项（请参阅“ 命令行选项”部分 ）。下面的示例打印每个输入记录的第一和第二字段，用分号分隔，并在每个换行符后添加一个空行：

$ awk 'BEGIN { OFS = ";"; ORS = "\n\n" }
>            { print $1, $2 }' mail-list
-| Amelia;555-5553
-|
-| Anthony;555-3412
-|
-| Becky;555-7685
-|
-| Bill;555-1675
-|
-| Broderick;555-0542
-|
-| Camilla;555-2912
-|
-| Fabius;555-1234
-|
-| Julie;555-6699
-|
-| Martin;555-6480
-|
-| Samuel;555-3430
-|
-| Jean-Paul;555-2127
-|

如果值ORS不包含换行符，该程序的输出在一行上一起运行。
