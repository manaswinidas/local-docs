14.3.3查看和更改数据

用于查看和更改内部变量的命令gawk是：

display [ var | $n ]

添加变量var （或字段$n ）到显示列表。每次程序停止时都会显示变量或字段的值。添加到列表中的每个变量均由唯一编号标识：

gawk> display x
-| 10: x = 1

这将显示分配的项目编号，变量名称及其当前值。如果显示变量引用了一个函数参数，则在执行到达上下文中（其中不存在任何给定名称的变量）后，立即将其从列表中静默删除。没有争执， display在列表上显示项目的当前值。

eval "awk statements"

评估awk statements在运行程序的上下文中。你可以做任何事情awk程序将执行以下操作：将值分配给变量，调用函数等。

注意：您不能使用eval执行包含以下任何一项的语句： exit ， getline ， next ， nextfile ， 要么return 。

evalparam ，…
awk statements
end

这种形式的eval相似，但是它允许您定义存在于awk statements ，而不使用程序定义的变量或函数参数。

printvar1 [ ,var2 …]
pvar1 [ ,var2 …]

打印一个的值gawk变量或字段。字段必须由常量引用：

gawk> print $3

这会打印输入记录中的第三个字段（如果指定的字段不存在，则会打印“ Null field '）。变量可以是数组元素，下标是常量字符串值。要打印数组的内容，请在数组名前添加' @ '符号：

gawk> print @a

这将为数组中的所有元素打印索引和相应的值a 。

printfformat [ ,arg …]

打印格式化的文本。的format可能包含转义序列，例如“ \n '（请参阅转义序列部分）。除非指定了换行符，否则不打印换行符。

set var=value

将常量（数字或字符串）值分配给awk变量或字段。字符串值必须用双引号（ " … " ）。

您还可以设置特殊awk变量，例如FS ， NF ， NR ， 等等。

watchvar | $n [ "expression" ]
wvar | $n [ "expression" ]

添加变量var （或字段$n ）到监视列表。然后，只要变量或字段的值发生更改，调试器就会停止。每个监视项目都分配有一个编号，该编号可用于使用以下命令将其从监视列表中删除unwatch命令。

使用观察点，您还可以提供条件。这是一awk达到观察点时调试器将评估的表达式（用双引号引起来）。如果条件为真，则调试器将停止执行并提示输入命令。除此以外， gawk继续执行程序。

undisplay [ n ]

删除商品编号n （或所有项目，如果没有参数，则为自动显示列表中的内容）。

unwatch [ n ]

删除商品编号n （或所有项目，如果没有参数）。
