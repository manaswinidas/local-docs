4.5.6现场拆分摘要

重要的是要记住，当您将字符串常量指定为FS ，它经历正常awk字符串处理。例如，使用Unix awk和gawk ， 分配 ' FS = "\.." '分配字符串".."至FS （反斜杠被去除）。这将创建一个正则表达式，表示“字段由出现的任何两个字符分隔开。”相反，如果您希望字段之间用英文句号和任何单个字符分隔，请使用' FS = "\\.." '。

以下列表根据的值总结了如何拆分字段FS （' == '表示“等于”）：

FS == " "

字段之间用空格隔开。前导和尾随空格将被忽略。这是默认值。

FS == any other single character

字段由每次出现的字符分隔。多个连续出现界定空白字段，前导和尾随出现也是如此。该字符甚至可以是一个正则表达式元字符。它不需要逃脱。

FS == regexp

字段由出现的匹配字符分隔regexp 。的领先和落后比赛regexp分隔空字段。

FS == ""

记录中的每个字符都变成一个单独的字段。（这是一个常见的扩展； POSIX标准未指定。）

FS和IGNORECASE

的IGNORECASE变量（请参阅控制的内置变量部分awk ） 仅在FS是一个正则表达式。何时没有作用FS是单个字符，即使该字符是字母也是如此。因此，在以下代码中：

FS = "c"
IGNORECASE = 1
$0 = "aCa"
print $1

输出为“ aCa '。如果您确实想在忽略大小写的同时拆分字母字符中的字段，请使用适合您的正则表达式（例如，“ FS = "[c]" '）。在这种情况下， IGNORECASE将生效。
