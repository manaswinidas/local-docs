10.3.3检查可读数据文件

通常，如果你给awk不可读取的数据文件，它会因致命错误而停止。有时候，您可能只想忽略此类文件并继续运行。 71您可以通过在您的计算机之前添加以下程序来执行此操作awk程序：

# readable.awk --- library file to skip over unreadable files

BEGIN {
    for (i = 1; i < ARGC; i++) {
        if (ARGV[i] ~ /^[a-zA-Z_][a-zA-Z0-9_]*=.*/ \
            || ARGV[i] == "-" || ARGV[i] == "/dev/stdin")
            continue    # assignment or standard input
        else if ((getline junk < ARGV[i]) < 0) # unreadable
            delete ARGV[i]
        else
            close(ARGV[i])
    }
}

这行得通，因为getline不会致命的。从中删除元素ARGV与delete跳过文件（因为它不再在列表中）。另请参阅使用ARGC和ARGV 。

因为awk变量名称仅允许使用英文字母，正则表达式检查故意不使用字符类，例如' [:alpha:] '和' [:alnum:] '（请参阅使用括号表达式部分）。
