10.2.1将字符串转换为数字

的strtonum()函数（请参阅“ 字符串操作函数”一节）是一个gawk延期。以下功能为其他版本的awk ：

# mystrtonum --- convert string to number

function mystrtonum(str,        ret, n, i, k, c)
{
    if (str ~ /^0[0-7]*$/) {
        # octal
        n = length(str)
        ret = 0
        for (i = 1; i <= n; i++) {
            c = substr(str, i, 1)
            # index() returns 0 if c not in string,
            # includes c == "0"
            k = index("1234567", c)

            ret = ret * 8 + k
        }
    } else if (str ~ /^0[xX][[:xdigit:]]+$/) {
        # hexadecimal
        str = substr(str, 3)    # lop off leading 0x
        n = length(str)
        ret = 0
        for (i = 1; i <= n; i++) {
            c = substr(str, i, 1)
            c = tolower(c)
            # index() returns 0 if c not in string,
            # includes c == "0"
            k = index("123456789abcdef", c)

            ret = ret * 16 + k
        }
    } else if (str ~ \
  /^[-+]?([0-9]+([.][0-9]*([Ee][0-9]+)?)?|([.][0-9]+([Ee][-+]?[0-9]+)?))$/) {
        # decimal number, possibly floating point
        ret = str + 0
    } else
        ret = "NOT-A-NUMBER"

    return ret
}

该函数首先查找C样式的八进制数字（以8为底）。如果输入字符串与描述八进制数字的正则表达式匹配，则mystrtonum()遍历字符串中的每个字符。它设置k到索引"1234567"当前八进制数字。返回值可以是与数字相同的数字，如果字符不存在，则返回零，对于' 0 '。这是安全的，因为在if确保仅转换八进制值。

类似的逻辑适用于检查和转换以'开头的十六进制值的代码。 0x ' 要么 ' 0X '。指某东西的用途tolower()简化了为每个十六进制数字查找正确数值的计算。

最后，如果该字符串与正则十进制整数或浮点数相匹配的（相当复杂的）正则表达式，则计算“ ret = str + 0 ' 让我们awk将值转换为数字。
