2.1调用awk

有两种运行方式awk -使用显式程序或一个或多个程序文件。这是它们的模板；这些模板中[…]内的项目是可选的：

awk [options] -f progfile [--] file …
awk [options] [--] 'program' file …

除了传统的单字母POSIX样式选项之外， gawk还支持GNU长选项。

可以调用awk使用空程序：

awk '' datafile1 datafile2

但是，这样做没有任何意义。 awk给定一个空程序时，它会静默退出。（直流）如果--lint已在命令行中指定， gawk发出警告，该程序为空。
