17.4.4构造函数

该API提供了许多用于创建字符串和数字值的构造函数，以及许多便利宏。本小节以扩展代码使用它们的方式将它们全部展示为函数原型：

static inline awk_value_t *
make_const_string(const char *string, size_t length, awk_value_t *result);

此函数在awk_value_t指向的变量result 。期望string成为C字符串常量（或其他字符串数据），并自动创建数据副本以存储在result 。它返回result 。

static inline awk_value_t *
make_malloced_string(const char *string, size_t length, awk_value_t *result);

此函数在awk_value_t指向的变量result 。期望string成为一个 ' char *的值指向先前从中获取的数据gawk_malloc() ， gawk_calloc() ， 要么gawk_realloc() 。这里的想法是将数据直接传递到gawk ，对此承担责任。它返回result 。

static inline awk_value_t *
make_null_string(awk_value_t *result);

这个专门的函数会在其中创建一个空字符串（“未定义”值） awk_value_t指向的变量result 。它返回result 。

static inline awk_value_t *
make_number(double num, awk_value_t *result);

此函数只是在awk_value_t指向的变量result 。

static inline awk_value_t *
make_number_mpz(void *mpz, awk_value_t *result);

此函数在以下位置创建GMP编号值result 。的mpz必须来自get_mpz_ptr() （因此属于真正的基础类型mpz_ptr ）。 gawk拥有此内存的所有权。

static inline awk_value_t *
make_number_mpfr(void *mpfr, awk_value_t *result);

此函数在以下位置创建MPFR编号值result 。的mpfr必须来自get_mpfr_ptr() 。（因此属于真正的基础类型mpfr_ptr ） gawk拥有此内存的所有权。

static inline awk_value_t *
make_const_user_input(const char *string, size_t length, awk_value_t *result);

此功能与make_const_string() ，但字符串被标记为用户输入，如果字符串的内容为数字，则应将其视为strnum值。

static inline awk_value_t *
make_malloced_user_input(const char *string, size_t length, awk_value_t *result);

此功能与make_malloced_string() ，但字符串被标记为用户输入，如果字符串的内容为数字，则应将其视为strnum值。

static inline awk_value_t *
make_const_regex(const char *string, size_t length, awk_value_t *result);

此函数通过分配字符串的副本来创建强类型的regexp值。 string是长度的正则表达式len 。

static inline awk_value_t *
make_malloced_regex(const char *string, size_t length, awk_value_t *result);

此函数创建一个强类型的regexp值。 string是长度的正则表达式len 。期望string成为一个 ' char *的值指向先前从中获取的数据gawk_malloc() ， gawk_calloc() ， 要么gawk_realloc() 。
