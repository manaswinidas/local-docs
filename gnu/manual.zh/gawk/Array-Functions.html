17.4.11.2数组函数

以下函数与单个数组元素有关：

awk_bool_t get_element_count(awk_array_t a_cookie, size_t *count);

对于由表示的数组a_cookie ， 放入*count它包含的元素数量。子数组算作单个元素。如果有错误，则返回false。

awk_bool_t get_array_element(awk_array_t a_cookie,
                             const awk_value_t *const index,
                             awk_valtype_t wanted,
                             awk_value_t *result);

对于由表示的数组a_cookie ，返回*result索引为的元素的值index 。 wanted指定要检索的值的类型。如果返回false wanted与实际类型不匹配，或者index不在数组中（请参阅表17.2 ）。

的价值index可以是数字，在这种情况下gawk将其转换为字符串。可以使用非整数值，但要求您了解如何将这些值转换为字符串（请参见“字符串和数字的转换”一节 ）。因此，使用整数值是最安全的。

与所有传入的字符串一样gawk从扩展名开始，字符串值index必须来自gawk_malloc() ， gawk_calloc() ， 要么gawk_realloc()和gawk释放存储。

awk_bool_t set_array_element(awk_array_t a_cookie,
                             const awk_value_t *const index,
                             const awk_value_t *const value);

在由表示的数组中a_cookie ，创建或修改其索引为的元素index 。的ARGV和ENVIRON数组可能不会更改，尽管PROCINFO数组即可。

awk_bool_t set_array_element_by_elem(awk_array_t a_cookie,
                                     awk_element_t element);

喜欢set_array_element() ，但是index和value从element 。这是一个便利宏。

awk_bool_t del_array_element(awk_array_t a_cookie,
                             const awk_value_t* const index);

从由表示的数组中删除具有给定索引的元素a_cookie 。如果删除了元素，则返回true；如果数组中不存在该元素，则返回false。

以下功能与整个数组有关：

awk_array_t create_array(void);

创建一个可以添加元素的新数组。有关如何创建新数组并向其中添加元素的讨论，请参见如何创建和填充数组 。

awk_bool_t clear_array(awk_array_t a_cookie);

清除由表示的数组a_cookie 。如果存在某种问题，则返回false，否则返回true。该数组仍然是一个数组，但是在调用此函数之后，它没有任何元素。这等效于使用delete声明（见部分的delete声明 ）。

awk_bool_t flatten_array_typed(awk_array_t a_cookie,
                               awk_flat_array_t **data,
                               awk_valtype_t index_type,
                               awk_valtype_t value_type);

对于由表示的数组a_cookie ，创建一个awk_flat_array_t结构并使用所需类型的索引和值填写。设置地址传递为的指针data指向这个结构。成功返回true，否则返回false。请参阅“使用数组的所有元素”一节 ，以获取有关如何展平数组并对其进行处理的讨论。

awk_bool_t flatten_array(awk_array_t a_cookie, awk_flat_array_t **data);

对于由表示的数组a_cookie ，创建一个awk_flat_array_t结构并填写AWK_STRING索引和AWK_UNDEFINED价值观。被此取代flatten_array_typed() 。它作为宏提供，为了方便起见以及与API的先前版本的源代码兼容性而保留。

awk_bool_t release_flattened_array(awk_array_t a_cookie,
                                   awk_flat_array_t *data);

完成展平阵列后，请使用此功能释放存储。您必须同时传递原始数组cookie和创建的地址awk_flat_array_t结构体。该函数在成功时返回true，否则返回false。
