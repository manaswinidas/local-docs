6.3.1对与错awk

许多编程语言对于“ true”和“ false”的概念都有特殊的表示。此类语言通常使用特殊常量true和false ，或者它们的大写等效项。然而， awk是不同的。它从C借用了一个非常简单的对与错概念。在awk ，则任何非零数字值或任何非空字符串值都为true。其他任何值（零或空字符串， "" ）是错误的。以下程序打印“ A strange truth value三遍：

BEGIN {
   if (3.1415927)
       print "A strange truth value"
   if ("Four Score And Seven Years Ago")
       print "A strange truth value"
   if (j = 57)
       print "A strange truth value"
}

“非零或非空”规则产生了令人惊讶的结果：字符串常量"0"实际上是正确的，因为它不是null。（直流）
