B.3.2.3安装gawk在VMS上

使用gawk ，您只需要一个“外国”命令，这是一个DCL其符号以美元符号开头的符号。例如：

$ GAWK :== $disk1:[gnubin]gawk

替换实际位置gawk.exe为$disk1:[gnubin] '。该符号应放在login.com任何想要运行的用户gawk ，以便在用户每次登录时进行定义。或者，可以将符号放置在系统范围内sylogin.com程序，允许所有用户运行gawk 。

如果你的gawk是由PCSI套件安装到GNV$GNU:目录树，该程序将被称为GNV$GNU:[bin]gnv$gawk.exe帮助文件将是GNV$GNU:[vms_help]gawk.hlp 。

PCSI套件还安装了一个GNV$GNU:[vms_bin]gawk_verb.cld可用于添加的文件gawk和awk作为DCL命令。

对于当前流程，您可以使用：

$ set command gnv$gnu:[vms_bin]gawk_verb.cld

或者系统管理员可以使用GNV$GNU:[vms_bin]gawk_verb.cld添加gawk和awk到系统范围的“ DCLTABLES '。

DCL语法记录在gawk.hlp文件。

（可选） gawk.hlp条目可以加载到VMS帮助库中：

$ LIBRARY/HELP sys$help:helplib [.vms]gawk.hlp

（您可能要替换特定于站点的帮助库，而不是标准的VMS库' HELPLIB '。）加载帮助文本后，命令：

$ HELP GAWK

提供有关gawk实施与awk编程语言。

逻辑名' AWK_LIBRARY '可以为awk程序文件。为了-f选项，如果指定的文件名中没有设备或目录路径信息， gawk首先在当前目录中查找，然后在“ AWK_LIBRARY '如果找不到该文件。在两个目录中搜索后，如果仍然找不到该文件， gawk附加后缀' .awk '作为文件名，然后重试文件搜索。如果“ AWK_LIBRARY '没有定义，默认值为' SYS$LIBRARY: '用于它。
