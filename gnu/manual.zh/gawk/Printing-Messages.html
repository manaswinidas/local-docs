17.4.6打印消息

您可以在扩展程序中打印各种警告消息，如此处所述。请注意，对于这些功能，您必须传入从gawk加载扩展程序时： 105

void fatal(awk_ext_id_t id, const char *format, ...);

打印一条消息，然后导致gawk立即退出。

void nonfatal(awk_ext_id_t id, const char *format, ...);

打印非致命错误消息。

void warning(awk_ext_id_t id, const char *format, ...);

打印警告消息。

void lintwarn(awk_ext_id_t id, const char *format, ...);

打印“棉绒警告”。通常，这与打印警告消息相同，但是如果gawk被用' --lint=fatal '，然后棉绒警告变成致命错误消息。

所有这些功能都像C printf()功能家族，其中format参数是混合了文字字符和格式代码的字符串。

脚注

（105）

因为API仅使用ISO C 90功能，所以它无法利用ISO C 99可变参数宏功能来隐藏该参数。更可惜。
