11.3.6从未排序的文本中删除重复项

的uniq程序（请参阅“ 打印文本的非重复行”一节）从排序的数据中删除重复的行。

但是，假设您需要从数据文件中删除重复的行，但是您想要保留行所在的顺序。外壳历史记录文件就是一个很好的例子。历史记录文件保留了您输入的所有命令的副本，并且连续多次重复执行命令并不罕见。有时，您可能希望通过删除重复的条目来压缩历史记录。然而，期望维持原始命令的顺序。

这个简单的程序可以完成工作。它使用两个数组。的data数组由每一行的文本索引。对于每一行， data[$0]增加。如果以前没有看到过特定行，则data[$0]是零。在这种情况下，该行的文本存储在lines[count] 。每个元素lines是唯一的命令，并且的索引lines指示遇到这些行的顺序。的END规则只是按顺序打印行：

# histsort.awk --- compact a shell history file
# Thanks to Byron Rakitzis for the general idea


{
    if (data[$0]++ == 0)
        lines[++count] = $0
}



END {
    for (i = 1; i <= count; i++)
        print lines[i]
}

该程序还为生成其他有用信息提供了基础。例如，使用以下print中的陈述END规则指示使用特定命令的频率：

print data[lines[i]], lines[i]

这行得通，因为data[$0]每次看到一条线都会增加。
