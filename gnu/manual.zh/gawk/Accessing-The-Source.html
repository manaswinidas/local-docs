C.2.1访问gawk Git存储库

如gawk是免费软件，源代码始终可用。 的gawk发行版描述了如何获取和构建正式的发行版本gawk 。

但是，如果要修改gawk并贡献您的更改，您可能希望使用开发版本。为此，您需要访问gawk源代码存储库。该代码使用Git分布式版本控制系统进行维护。如果您的系统没有它，则需要安装它。完成后，请使用以下命令：

git clone git://git.savannah.gnu.org/gawk.git

这克隆了gawk资料库。如果您位于不允许您使用Git本机协议的防火墙后面，则仍可以使用以下方法访问存储库：

git clone https://git.savannah.gnu.org/r/gawk.git

进行更改后，您可以使用“ git diff '产生补丁，并将其发送到gawk维护者有关如何执行的信息，请参阅报告问题和错误 。

从前，有一个Git–CVS网关供无法安装Git的人使用。但是，此网关不再起作用，因此，使用像Bazaar这样的更现代的版本控制系统可能会更好，该系统具有用于处理Git存储库的Git插件。
