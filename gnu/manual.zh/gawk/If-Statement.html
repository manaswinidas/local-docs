7.4.1 if -- else声明

的if -- else声明是awk的决策声明。看起来像这样：

if (condition) then-body [else else-body]

的condition是一个控制语句其余部分做什么的表达式。如果condition是真的， then-body被执行;除此以外， else-body被执行。的else该语句的一部分是可选的。如果该条件的值为零或空字符串，则认为该条件为假；否则，条件为真。请参阅以下内容：

if (x % 2 == 0)
    print "x is even"
else
    print "x is odd"

在此示例中，如果表达式“ x % 2 == 0 '是正确的（即，如果x被两个均分），然后第一个print语句执行；否则，第二print语句被执行。如果else关键字与then-body和then-body不是复合语句（即，不要用大括号括起来），则必须用分号分隔then-body来自else 。为了说明这一点，可以将前面的示例重写为：

if (x % 2 == 0) print "x is even"; else
        print "x is odd"

如果“ ; '被排除在外， awk无法解释该语句，它产生语法错误。实际不要以这种方式编写程序，因为人类读者可能看不到else如果这不是第一件事。
