17.7.5字符和数值： ord()和chr()

的ordchr扩展添加了两个函数，名为ord()和chr() ， 如下：

@load "ordchr"

这是您加载扩展程序的方式。

number = ord(string)

返回第一个字符的数值string 。

char = chr(number)

返回一个字符串，其第一个字符为number 。

这些功能受同名的Pascal语言功能的启发。这是一个例子：

@load "ordchr"
…
printf("The numeric value of 'A' is %d\n", ord("A"))
printf("The string value of 65 is %s\n", chr(65))
