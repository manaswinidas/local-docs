10.2.3舍入数字

方式printf和sprintf() （请参见使用部分printf进行舍入打印的语句 ）通常取决于系统的C语言。 sprintf()子程序。在许多机器上sprintf()四舍五入是无偏见的 ，这意味着它并不总是将尾随的.5向上舍入，这与幼稚的期望相反。在无偏舍入中，.5舍入为偶数而不是总是向上舍入，因此1.5舍入为2，但4.5舍入为4。这意味着，如果您使用的是四舍五入的格式（例如， "%.0f" ），则应检查系统的功能。以下函数进行传统的舍入；如果您的awk的printf进行无偏舍入：

# round.awk --- do normal rounding

function round(x,   ival, aval, fraction)
{
   ival = int(x)    # integer part, int() truncates

   # see if fractional part
   if (ival == x)   # no fraction
      return ival   # ensure no decimals

   if (x < 0) {
      aval = -x     # absolute value
      ival = int(aval)
      fraction = aval - ival
      if (fraction >= .5)
         return int(x) - 1   # -2.5 --> -3
      else
         return int(x)       # -2.3 --> -2
   } else {
      fraction = x - ival
      if (fraction >= .5)
         return ival + 1
      else
         return ival
   }
}
