11.3.1在文档中查找重复的单词

撰写大量散文时的一个常见错误是偶然地重复单词。通常，您会在文本中看到类似“程序将执行以下操作……”的内容。当文本处于联机状态时，重复的单词通常出现在一行的结尾和另一行的开头，这使它们很难被发现。

这个程序dupword.awk ，一次扫描文件一行，并查找同一单词的相邻出现位置。它还将最后一个单词保存在一行中（在变量中prev ）以与下一行的第一个单词进行比较。

前两个语句确保该行全部为小写，例如，“ The”和“ the”彼此比较相等。下一条语句将非字母数字和非空格字符替换为空格，因此标点符号也不会影响比较。字符将被替换为空格，以便格式控件不会创建无用的字词（例如，Texinfo' @code{NF} '变成' codeNF '（如果只删除了标点符号）。然后将记录重新拆分为字段，仅产生行上的实际单词，并确保没有空字段。

如果删除所有标点符号后没有剩余字段，则跳过当前记录。否则，程序将循环遍历每个单词，并将其与上一个单词进行比较：

# dupword.awk --- find duplicate words in text
{
    $0 = tolower($0)
    gsub(/[^[:alnum:][:blank:]]/, " ");
    $0 = $0         # re-split
    if (NF == 0)
        next
    if ($1 == prev)
        printf("%s:%d: duplicate %s\n",
            FILENAME, FNR, $1)
    for (i = 2; i <= NF; i++)
        if ($i == $(i-1))
            printf("%s:%d: duplicate %s\n",
                FILENAME, FNR, $i)
    prev = $NF
}
