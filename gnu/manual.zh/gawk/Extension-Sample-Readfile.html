17.7.10读取整个文件

的readfile扩展添加了一个名为readfile() ，以及输入解析器：

@load "readfile"

这是您加载扩展程序的方式。

result = readfile("/some/path")

参数是要读取的文件的名称。返回值是一个字符串，其中包含请求的文件的全部内容。发生错误时，该函数返回空字符串并进行设置ERRNO 。

BEGIN { PROCINFO["readfile"] = 1 }

此外，该扩展添加了一个输入解析器，如果PROCINFO["readfile"]存在。激活后，每个输入文件将全部返回为$0 。 RT设置为空字符串。

这是一个例子：

@load "readfile"
…
contents = readfile("/path/to/file");
if (contents == "" && ERRNO != "") {
    print("problem reading file", ERRNO) > "/dev/stderr"
    ...
}
