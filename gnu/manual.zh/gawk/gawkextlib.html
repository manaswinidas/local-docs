17.8 gawkextlib项目

的gawkextlib项目提供了一些gawk扩展名，包括用于处理XML文件的扩展名。这是原始的演变xgawk （XML gawk ）项目。

有许多扩展。一些更有趣的是：

  • abort延期。它可以让您立即退出awk程序而不运行END规则。
  • json延期。这会将多维数组序列化为JSON字符串，并且可以将JSON字符串反序列化为gawk数组。此扩展很有趣，因为它是用C ++而不是C编写的。
  • MPFR库扩展。这提供了对许多MPFR功能的访问，这些功能gawk的本机MPFR支持不支持。
  • 选择扩展名。它提供了基于select()系统调用。
  • XML解析器扩展，使用Expat XML解析库

您可以查看代码gawkextlib项目使用Git分布式源代码控制系统。命令如下：

git clone git://git.code.sf.net/p/gawkextlib/code gawkextlib-code

您需要安装RapidJson JSON解析器库才能构建和使用json延期。

您将需要安装Expat XML解析器库才能构建和使用XML扩展名。

另外，必须安装GNU Autotools（ Autoconf ， Automake ， Libtool和GNU gettext ）。

构建和测试的简单方法gawkextlib如下。一，搭建安装gawk ：

cd .../path/to/gawk/code
./configure --prefix=/tmp/newgawk     Install in /tmp/newgawk for now
make && make check                    Build and check that all is OK
make install                          Install gawk

接下来，转到https://sourceforge.net/projects/gawkextlib/files下载gawkextlib以及您要构建的所有扩展。的README该站点上的文件说明了如何构建代码。如果您已安装gawk在非标准位置，您需要指定./configure --with-gawk=/path/to/gawk找到它。您可能需要使用sudo安装两者的实用程序gawk和gawkextlib ，具体取决于您的系统如何工作。

如果您编写了一个扩展，希望与其他人共享gawk用户，请考虑通过gawkextlib项目。有关更多信息，请参见项目的网站。
