9.1.7获取类型信息

gawk提供了两个函数，可让您区分变量的类型。这对于编写遍历数组数组的每个元素（请参见Arrays of Arrays ）以及在其他上下文中的代码是必需的。

isarray(x)

如果返回真值x是一个数组。否则，返回false。

typeof(x)

根据以下类型返回以下字符串之一x ：

"array"

x是一个数组。

"regexp"

x是强类型的正则表达式（请参阅强类型的正则表达式常量 ）。

"number"

x是一个数字。

"string"

x是一个字符串。

"strnum"

x是一个从用户输入开始的数字，例如字段或呼叫结果split() 。（即， x具有strnum属性；请参阅“ 字符串类型与数字类型”部分 。）

"unassigned"

x是尚未分配值的标量变量。例如：

BEGIN {
    # creates a[1] but it has no assigned value
    a[1]
    print typeof(a[1])  # unassigned
}
"untyped"

x尚未使用过；它可以成为标量或数组。甚至可以想象，同一程序的运行之间甚至可能有所不同！例如：

BEGIN {
    print "initially, typeof(v) = ", typeof(v)

    if ("FOO" in ENVIRON)
        make_scalar(v)
    else
        make_array(v)

    print "typeof(v) =", typeof(v)
}

function make_scalar(p,    l) { l = p }

function make_array(p) { p[1] = 1 }

isarray()用于两种情况。第一个是遍历多维数组时：您可以测试元素本身是否为数组。第二个位于用户定义函数（尚未讨论；请参阅用户定义函数 ）的主体内部，以测试参数是否为数组。

注意：使用isarray()在全局级别上测试变量没有任何意义。因为您是编写程序的人，所以应该知道变量是否为数组。实际上，由于gawk如果您传递以前未使用过的变量名，则可以使用isarray() ， gawk最终将其变成标量。

的typeof()功能一般；它使您可以确定变量或函数参数是标量，数组还是强类型的正则表达式。
