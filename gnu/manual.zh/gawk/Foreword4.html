第四版前言

有些事情不会改变。十三年前，我写道：“如果您使用AWK或想学习如何使用，请读这本书。”那时是正确的，今天仍然是正确的。

学习使用编程语言不仅仅是掌握语法。人们需要了解如何使用语言的功能来解决实际的编程问题。本书的重点是许多示例，展示了如何使用AWK。

有些事情确实会改变。我们的计算机速度更快，并且具有更多的内存。因此，高级语言的速度和存储效率不那么重要。由于性能原因，在AWK中进行原型制作然后出于性能原因在C中进行重写的情况较少，因为原型通常足够快。

当然，有些计算操作最好用C或C ++完成。用gawk 4.1及更高版本，您不必在用AWK或C / C ++编写程序之间进行选择。您可以使用AWK编写大多数程序，并且可以使用C / C ++编写需要C / C ++功能的方面，然后在gawk模块将C / C ++模块作为动态插件加载。 编写扩展gawk ，其中包含所有详细信息，并且如预期的那样，提供了许多示例来帮助您学习来龙去脉。

我喜欢用AWK编程，并且很有趣（重新）读了这本书。我想你也会。 

Michael Brennan
Author of mawk
October 2014
