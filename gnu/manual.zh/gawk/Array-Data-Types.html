17.4.11.1数组数据类型

与数组关联的数据类型如下：

typedef void *awk_array_t;

如果请求数组变量的值，则返回awk_array_t值。该值对于扩展名是不透明的110 。它唯一地标识数组，但是只能通过将其传递到API函数或从API函数接收它来使用。这与“ FILE * '值与库例程。

typedef struct awk_element {
    /* convenience linked list pointer, not used by gawk */
    struct awk_element *next;
    enum {
        AWK_ELEMENT_DEFAULT = 0,  /* set by gawk */
        AWK_ELEMENT_DELETE = 1    /* set by extension */
    } flags;
    awk_value_t index;
    awk_value_t value;
} awk_element_t;

的awk_element_t是“展平”的数组元素。 awk在内部产生这些数组awk_flat_array_t （请参阅下一项）。单个元素可以标记为删除。为此，必须使用单独的API分别一次添加一个新元素。字段如下：

struct awk_element *next;

该指针是为了方便扩展作者。它允许扩展创建新元素的链接列表，然后可以在遍历列表的循环中将其添加到数组中。

enum { … } flags;

一组标志值，用于在扩展名和扩展名之间传递信息gawk 。当前只有一个： AWK_ELEMENT_DELETE 。设置它会导致gawk在释放展平数组时从原始数组中删除元素。
