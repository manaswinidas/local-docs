4.10.3使用getline从文件

使用 ' getline < file '从中读取下一条记录file 。这里， file是指定文件名的字符串值表达式。 ' < file之所以称为“ 重定向 ”，是因为它会引导输入来自其他位置。例如，以下程序从文件读取其输入记录secondary.input当它在当前输入文件中遇到第一个等于10的字段时：

{
    if ($1 == 10) {
         getline < "secondary.input"
         print
    } else
         print
}

由于未使用主输入流，因此NR和FNR没有改变。但是，它读取的记录将以常规方式分为多个字段，因此$0且其他字段已更改，从而产生新值NF 。 RT也设置。

根据POSIX，“ getline < expression如果是if昧的expression包含除“ $ ';例如， ' getline < dir "/" file '是不明确的，因为串联运算符（尚未讨论；请参见字符串串联一节）未加括号。您应该将其写为“ getline < (dir "/" file) '如果您想让程序可移植到所有人awk实现。
