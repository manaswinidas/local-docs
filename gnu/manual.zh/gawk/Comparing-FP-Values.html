16.4.1.2谨慎比较值

由于基础表示可能与精确值略有出入，因此比较浮点值以查看它们是否完全相等通常是一个坏主意。这是一个示例，它无法正常运行：

$ gawk 'BEGIN { print (0.1 + 12.2 == 12.3) }'
-| 0

比较浮点值时，通常的智慧是看它们是否在彼此的小范围内（称为delta或lerance ）。您必须决定对您而言重要的增量有多小。执行此操作的代码如下所示：

delta = 0.00001                 # for example
difference = abs(a - b)         # subtract the two values
if (difference < delta)
    # all ok
else
    # not ok

（我们假设您有一个简单的绝对值函数，名为abs()在程序的其他位置定义。）如果您编写了一个将值与增量进行比较的函数，则应确保使用' difference < abs(delta) '，以防有人输入负增量值。
