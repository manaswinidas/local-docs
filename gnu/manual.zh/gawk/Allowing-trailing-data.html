4.6.3捕获可选的尾随数据

有时，固定宽度数据后可能会跟随没有固定长度的其他数据。此类数据可能存在也可能不存在，但如果存在，应该可以从awk程序。

从4.2版开始，为了提供一种方式来说“记录中定义字段之后的其他内容”， gawk允许您添加最终的“ *字符的价值FIELDWIDTHS 。只能有一个这样的字符，并且必须是最后一个非空格字符FIELDWIDTHS 。例如：

$ cat fw.awk                         Show the program
-| BEGIN { FIELDWIDTHS = "2 2 *" }
-| { print NF, $1, $2, $3 }
$ cat fw.in                          Show sample input
-| 1234abcdefghi
$ gawk -f fw.awk fw.in               Run the program
-| 3 12 34 abcdefghi
