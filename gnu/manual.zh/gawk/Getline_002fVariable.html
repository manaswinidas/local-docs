4.10.2使用getline变成变量

您可以使用 ' getline var '从中读取下一条记录awk的输入变量var 。没有其他处理。例如，假设下一行是注释或特殊字符串，并且您希望在不触发任何规则的情况下阅读它。这种形式的getline允许您读取该行并将其存储在变量中，从而使主读取行和检查每个规则循环awk从来没有看到它。以下示例每两行输入交换一次：

{
     if ((getline tmp) > 0) {
          print tmp
          print $0
     } else
          print $0
}

它包含以下列表：

wan
tew
free
phore

并产生以下结果：

tew
wan
phore
free

的getline以这种方式使用的命令仅设置变量NR ， FNR和RT （而且当然， var ）。记录未拆分为字段，因此字段的值（包括$0 ）和NF不要换。
