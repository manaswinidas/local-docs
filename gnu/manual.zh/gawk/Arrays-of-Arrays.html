8.6数组数组

gawk超越标准awk的多维数组访问，并提供数组的真实数组。子数组的元素由其自己的索引括在方括号中来引用，就像主数组的元素一样。例如，以下代码在索引处创建一个包含两个元素的子数组1主数组的a ：

a[1][1] = 1
a[1][2] = 2

这将模拟真实的二维数组。每个子数组元素可以包含另一个子数组作为值，而该子数组又可以保存其他数组。这样，您可以创建三个或更多个维度的数组。索引可以是任何awk表达式，包括用逗号分隔的标量（即，正则awk模拟多维下标）。所以以下内容在gawk ：

a[1][3][1, "name"] = "barney"

每个子数组和主数组的长度可以不同。实际上，数组或其子数组的元素不必都具有相同的类型。这意味着主数组及其任何子数组可以是非矩形的，也可以是锯齿状的。您可以为索引分配标量值4主数组的a ， 即使a[1]它本身是一个数组，而不是一个标量：

a[4] = "An element in a jagged array"

当将维度 ， 行和列应用于此类数组时，这些术语毫无意义，但是此后我们将使用“维”来表示引用现有元素所需的最大索引数。不能通过分配其他类型的值来更改已分配的任何元素的类型。您必须先删除当前元素，这样才能使gawk忘记该索引处的元素：

delete a[4]
a[4][5][6][7] = "An element in a four-dimensional array"

这将从索引中删除标量值4然后插入一个包含标量的三级嵌套子数组。您还可以删除整个子数组或子数组的子数组：

delete a[4][5]
a[4][5] = "An element in subarray a[4]"

但是请记住，您不能删除主数组a然后将其用作标量。

带有数组参数的内置函数也可以与子数组一起使用。例如，以下代码片段使用length() （请参阅“ 字符串操作函数”一节），以确定主数组中的元素数a及其子数组：

print length(a), length(a[1]), length(a[1][3])

这将为我们的主数组生成以下输出a ：

2, 3, 1

' subscript in array '表达式（请参阅“ 引用数组元素”一节 ）对于两个常规awk样式的数组和数组的数组。例如，测试“ 1 in a '，' 3 in a[1] '和' (1, "name") in a[1][3] '我们的数组的所有值都为一个（true） a 。

' for (item in array)如果语句的结构是矩形的，则可以嵌套'语句（请参见扫描数组的所有部分）以扫描数组的所有元素。为了打印二维数组数组的内容（标量值）（即每个第一级元素本身就是一个数组，不一定具有相同的长度），可以使用以下代码：

for (i in array)
    for (j in array[i])
        print array[i][j]

的isarray()函数（请参阅“ 获取类型信息”部分）可让您测试数组元素本身是否为数组：

for (i in array) {
    if (isarray(array[i])) {
        for (j in array[i]) {
            print array[i][j]
        }
    }
    else
        print array[i]
}

如果预先知道数组的锯齿状数组的结构，则通常可以使用控制语句设计解决方法。例如，以下代码输出我们主数组的元素a ：

for (i in a) {
    for (j in a[i]) {
        if (j == 3) {
            for (k in a[i][j])
                print a[i][j][k]

        } else
            print a[i][j]
    }
}

有关“ 遍历 ”任意尺寸数组的用户定义函数，请参见遍历数组数组一节。

回想一下，对未初始化的数组元素的引用产生的值为"" ，空字符串。当您打算将子数组用作函数的参数时，这具有一个重要的含义，如以下示例所示：

$ gawk 'BEGIN { split("a b c d", b[1]); print b[1][1] }'
error→ gawk: cmd. line:1: fatal: split: second argument is not an array

解决此问题的方法是先发力b[1]通过创建任意索引成为数组：

$ gawk 'BEGIN { b[1][1] = ""; split("a b c d", b[1]); print b[1][1] }'
-| a
