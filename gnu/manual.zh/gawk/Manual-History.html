GNU项目和本书

自由软件基金会（FSF）是一个非营利组织，致力于生产和分发可自由分发的软件。它由Emacs原始编辑的作者Richard M. Stallman创建。GNU Emacs是当今使用最广泛的Emacs版本。

GNU 6项目是自由软件基金会的不懈努力，旨在创建一个完整的，可自由分发的，符合POSIX的计算环境。FSF使用GNU通用公共许可证（GPL）来确保其软件的源代码始终可供最终用户使用。GPL的副本包含在此网页中，以供您参考（请参阅GNU通用公共许可证 ）。GPL适用于以下语言的C语言源代码gawk 。要在线了解有关FSF和GNU项目的更多信息，请参见GNU项目的主页 。该网页也可以从GNU网站上阅读。

一个外壳程序，一个编辑器（Emacs），高度可移植的优化C，C ++和Objective-C编译器，一个符号调试器以及数十种大型和小型实用程序（例如gawk ），都已完成，可以免费获得。GNU操作系统内核（HURD）已发布，但仍处于开发的早期阶段。

在更全面地开发GNU操作系统之前，您应该考虑使用GNU / Linux，这是一种可自由分发的，类似于Unix的操作系统，适用于Intel，Power Architecture，Sun SPARC，IBM S / 390和其他系统。 7许多GNU / Linux发行版都可以从Internet下载。

您正在阅读的网页实际上是免费的-至少其中的信息对任何人都是免费的。Web页面的机器可读源代码随附gawk 。（花点时间检查GNU Free Documentation License中的Free Documentation License 。）

该网页本身已经经历了多个以前的版本。保罗•鲁宾（Paul Rubin）撰写了“ GAWK手册”的初稿;它长约40页。Diane Close和Richard Stallman对其进行了改进，生成了大约90页的版本，几乎没有描述原始的“旧”版本awk 。

我从1988年秋天开始使用该版本。随着工作的进行，FSF发布了几个初步版本（编号为0。 x ）。1996年，发布了1.0版gawk 3.0.0。FSF发布了前两个版本，标题为《 GNU Awk用户指南》 。

此版本保持先前版本的基本结构。对于FSF 4.0版，对内容进行了彻底的检查和更新。所有参考gawk 4.0之前的版本已删除。该版本值得注意的是增加了Debugging awk程序 。

对于FSF版本5.0，内容已重新组织成几个部分，主要的新增加内容是算术运算和具有任意精度的任意精度算术gawk和编写扩展gawk 。
