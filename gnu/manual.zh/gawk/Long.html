1.1.3运行长程序

有时awk程序很长。在这些情况下，将程序放在单独的文件中更为方便。为了告诉awk将该文件用于其程序，请键入：

awk -f source-file input-file1 input-file2 …

的-f指示awk实用程序以获取awk文件中的程序source-file （请参阅“ 命令行选项”部分 ）。任何文件名都可以用于source-file 。例如，您可以将程序放在：

BEGIN { print "Don't Panic!" }

进入文件advice 。然后这个命令：

awk -f advice

与此功能相同：

awk 'BEGIN { print "Don\47t Panic!" }'

前面已经对此进行了解释（请参阅“ 运行”部分awk没有输入文件 ）。请注意，您通常不需要在指定文件名时用单引号引起来。 -f ，因为大多数文件名不包含Shell的任何特殊字符。请注意advice ， awk程序周围没有单引号。引号仅适用于awk命令行。（此外，将程序放置在文件中还可以使我们在程序文本中使用文字单引号，而不是魔术' \47 '。）

如果您想清楚地识别awk这样的程序文件，您可以添加扩展名.awk文件名。这不会影响awk程序，但它确实使“内务管理”更加容易。
