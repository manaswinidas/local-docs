17.4.5.2注册退出回调函数

退出回调函数是gawk退出前调用。如果您具有应在扩展中执行的常规“清理”任务（例如关闭数据库连接或其他资源释放），则此类功能很有用。您可以使用gawk使用以下功能：

void awk_atexit(void (*funcp)(void *data, int exit_status),
                void *arg0);

参数为：

funcp

指向之前要调用的函数的指针gawk退出。的data参数将是的原始值arg0 。的exit_status参数是退出状态值gawk打算传递给exit()系统调用。

arg0

指向私有数据的指针gawk保存以传递给指向的函数funcp 。

退出回调函数按照后进先出（LIFO）的顺序调用，即以与注册时相反的顺序进行调用gawk 。
