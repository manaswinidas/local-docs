8.1.2引用数组元素

使用数组的主要方法是引用其元素之一。数组引用是如下表达式：

array[index-expression]

这里， array是数组的名称。表达方式index-expression是数组所需元素的索引。

数组引用的值是该数组元素的当前值。例如， foo[4.3]是一个引用数组元素的表达式foo在索引“ 4.3 '。

对没有记录值的数组元素的引用产生的值为"" ，空字符串。这包括没有被分配任何值，该值已经被删除的元素以及元素（参见部分的delete声明 ）。

注意：对不存在的元素的引用会自动使用空字符串作为其值来创建该数组元素。（在某些情况下，这很不幸，因为它可能会浪费内部的内存awk ）

新手awk程序员经常通过检查值是否为空来犯错检查元素是否存在的错误：

# Check if "foo" exists in a:         Incorrect!
if (a["foo"] != "") …

这是不正确的，原因有两个。首先，它创建 a["foo"]如果以前不存在！其次，将数组元素设置为等于空字符串是有效的（如果有些不正常）。

要确定某个元素在数组中是否存在某个索引，请使用以下表达式：

indx in array

此表达式测试特定索引是否indx存在，没有创建该元素（如果不存在）的副作用。如果以下条件，则表达式的值为一（true） array[indx]存在，如果不存在则为零（假）。（我们用indx在这里，因为index '是内置函数的名称。）例如，此语句测试数组是否frequencies包含索引“ 2 '：

if (2 in frequencies)
    print "Subscript 2 is present."

请注意，这不是测试数组是否frequencies包含一个值为 2的元素。除了扫描所有元素，没有其他方法可以做到。另外，这不会创建frequencies[2] ，而以下（错误的）替代方法可以：

if (frequencies[2] != "")
    print "Subscript 2 is present."
