10.2.6将数组合并为字符串

在进行字符串处理时，通常可以将数组中的所有字符串连接成一个长字符串。以下功能， join() ，完成此任务。稍后将在一些应用程序中使用它（请参见“ 实用”部分awk程序 ）。

好的功能设计很重要；此功能需要通用，但也应具有合理的默认行为。它使用数组以及要合并的数组中元素的开始和结束索引进行调用。假设数组索引是数字，这是一个合理的假设，因为数组可能是使用split() （请参阅“ 字符串操作函数”部分 ）：

# join.awk --- join an array into a string

function join(array, start, end, sep,    result, i)
{
    if (sep == "")
       sep = " "
    else if (sep == SUBSEP) # magic value
       sep = ""
    result = array[start]
    for (i = start + 1; i <= end; i++)
        result = result sep array[i]
    return result
}

可选的附加参数是将字符串重新连接在一起时要使用的分隔符。如果呼叫者提供非空值， join()使用它；如果未提供，则为空值。在这种情况下， join()使用单个空格作为字符串的默认分隔符。如果值等于SUBSEP ， 然后join()连接字符串，字符串之间没有分隔符。 SUBSEP用作“魔术”值，以指示组件字符串之间不应有分隔符。 70
