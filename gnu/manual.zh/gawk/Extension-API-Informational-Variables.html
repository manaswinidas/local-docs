17.4.13.3信息变量

该API提供对几个变量的访问权限，这些变量描述了在以下情况下是否启用了相应的命令行选项： gawk被调用。变量是：

do_debug

如果该变量为true gawk被调用--debug选项。

do_lint

如果该变量为true gawk被调用--lint选项。

do_mpfr

如果该变量为true gawk被调用--bignum选项。

do_profile

如果该变量为true gawk被调用--profile选项。

do_sandbox

如果该变量为true gawk被调用--sandbox选项。

do_traditional

如果该变量为true gawk被调用--traditional选项。

的价值do_lint可以改变，如果awk代码修改了LINT预定义变量（请参阅预定义变量部分）。其他不应在执行期间更改。
