6.4函数调用

函数是特定计算的名称。这使您可以在程序中的任何位置按名称要求。例如功能sqrt()计算数字的平方根。

内置了一组固定的功能，这意味着它们在每个awk程序。的sqrt()功能就是其中之一。有关内置功能及其说明的列表，请参见内置功能一节。此外，您可以定义要在程序中使用的功能。有关如何执行此操作的说明，请参见“ 用户定义的功能”部分。最后， gawk使您可以用C或C ++编写可以从程序中调用的函数（请参见编写扩展gawk ）。

使用函数的方法是使用函数调用表达式，该表达式由函数名称组成，后跟括号中的参数列表。参数是为函数的计算提供原材料的表达式。当有多个参数时，它们之间用逗号分隔。如果没有参数，则只需输入' ()在函数名称之后。以下示例显示带有和不带有参数的函数调用：

sqrt(x^2 + y^2)        one argument
atan2(y, x)            two arguments
rand()                 no arguments

小心：请勿在函数名称和左括号之间留任何空格！用户定义的函数名称看起来就像变量的名称一样-一个空格会使表达式看起来像是带有括号内的表达式的变量的串联。对于内置函数，圆括号前的空格是无害的，但是最好不要养成使用空格的习惯，以避免用户定义的函数出错。

每个函数都需要特定数量的参数。例如， sqrt()函数必须使用单个参数调用，该参数的个数取平方根：

sqrt(argument)

一些内置函数具有一个或多个可选参数。如果未提供这些参数，则函数使用合理的默认值。有关完整的详细信息，请参见内置功能一节。如果在调用用户定义函数时省略了参数，则这些参数将被视为局部变量。如果在需要字符串值的地方引用，则此类局部变量的作用类似于空字符串；在需要数字值的地方引用，此类局部变量的作用类似于零（请参阅用户定义的函数一节）。

作为一项高级功能， gawk提供间接函数调用，这是一种选择在运行时调用函数的方法，而不是在将源代码写入程序时使用的方法。我们将对该功能的讨论推迟到稍后；请参见间接函数调用 。

像其他所有表达式一样，函数调用具有一个值，通常称为返回值 ，该值由函数根据您为其提供的参数进行计算。在此示例中，返回值' sqrt(argument) '是...的平方根argument 。以下程序读取数字，每行一个数字，并打印每个数字的平方根：

$ awk '{ print "The square root of", $1, "is", sqrt($1) }'
1
-| The square root of 1 is 1
3
-| The square root of 3 is 1.73205
5
-| The square root of 5 is 2.23607
Ctrl-d

函数也可能有副作用，例如为某些变量分配值或执行I / O。该程序显示了match()函数（请参阅“ 字符串操作函数”一节）更改变量RSTART和RLENGTH ：

{
    if (match($1, $2))
        print RSTART, RLENGTH
    else
        print "no match"
}

这是一个示例运行：

$ awk -f matchit.awk
aaccdd  c+
-| 3 2
foo     bar
-| no match
abcdefg e
-| 5 1
