B.3.2.1编译gawk在VMS上

编译gawk在VMS下，有一个DCL发出所有必要命令的命令过程CC和LINK命令。还有一个Makefile与MMS和MMK实用程序。在源目录中，使用以下任一方法：

$ @[.vms]vmsbuild.com

要么：

$ MMS/DESCRIPTION=[.vms]descrip.mms gawk

要么：

$ MMK/DESCRIPTION=[.vms]descrip.mms gawk

MMK是开源的，免费的，近乎克隆的MMS并可以更好地处理大小写文件名的ODS-5卷。 MMK可从https://github.com/endlesssoftware/mmk获得 。

启用ODS-5卷并启用扩展解析后，可能需要精确确定目标参数的大小写。

gawk已使用Compaq C V6.4在VAX / VMS 7.3和Alpha / VMS 7.3-1以及Alpha / VMS 7.3，Alpha / VMS 7.3-2和IA64 / VMS 8.3下进行了测试。最新的版本在Alpha VMS 8.3上使用HP C V7.3，而Alpha和IA64 VMS 8.4都使用HP C 7.3。 116

有关建筑物的信息，请参见“ VMS GNV项目”部分。 gawk作为与GNV产品兼容的PCSI套件。
