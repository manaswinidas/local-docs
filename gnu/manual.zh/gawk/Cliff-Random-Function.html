10.2.4悬崖随机数生成器

Cliff随机数发生器是一个非常简单的随机数发生器，“通过不显示任何结构，通过了噪声领域的随机性测试”。它易于编程，少于10行awk码：

# cliff_rand.awk --- generate Cliff random numbers

BEGIN { _cliff_seed = 0.1 }

function cliff_rand()
{
    _cliff_seed = (100 * log(_cliff_seed)) % 1
    if (_cliff_seed < 0)
        _cliff_seed = - _cliff_seed
    return _cliff_seed
}

此算法要求初始“种子”为0.1。每个新值都将当前种子用作计算的输入。如果内置rand()函数（请参见“ 数字函数”一节）不够随机，您可以尝试使用此函数。
