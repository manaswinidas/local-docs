4.5.3使每个字符分开

有时您可能想单独检查记录的每个字符。这可以在gawk通过简单地分配空字符串（ "" ） 至FS 。 （ce）在这种情况下，记录中的每个单个字符将成为一个单独的字段。例如：

$ echo a b | gawk 'BEGIN { FS = "" }
>                  {
>                      for (i = 1; i <= NF; i = i + 1)
>                          print "Field", i, "is", $i
>                  }'
-| Field 1 is a
-| Field 2 is
-| Field 3 is b

传统上， FS等于""未定义。在这种情况下，大多数版本的Unix awk只需将整个记录视为只有一个字段即可。（直流）在兼容模式下（请参阅“ 命令行选项”一节），如果FS是空字符串，然后gawk也表现出这种方式。
