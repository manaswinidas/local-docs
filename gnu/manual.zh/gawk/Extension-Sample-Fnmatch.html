17.7.2与fnmatch()

该扩展提供了C库的接口fnmatch()功能。用法是：

@load "fnmatch"

这是您加载扩展程序的方式。

result = fnmatch(pattern, string, flags)

成功返回值为零， FNM_NOMATCH如果字符串与模式不匹配，或者如果发生错误，则返回另一个非零值。

除了fnmatch()功能， fnmatch扩展名添加一个常数（ FNM_NOMATCH ），以及一个名为的标志值数组FNM 。

的论点fnmatch()是：

pattern

要匹配的文件名通配符

string

文件名字符串

flag

要么为零，要么为一个或多个标志的按位或FNM数组

标志如下：

数组元素对应的标志定义为fnmatch()
FNM["CASEFOLD"]FNM_CASEFOLD
FNM["FILE_NAME"]FNM_FILE_NAME
FNM["LEADING_DIR"]FNM_LEADING_DIR
FNM["NOESCAPE"]FNM_NOESCAPE
FNM["PATHNAME"]FNM_PATHNAME
FNM["PERIOD"]FNM_PERIOD

这是一个例子：

@load "fnmatch"
…
flags = or(FNM["PERIOD"], FNM["NOESCAPE"])
if (fnmatch("*.a", "foo.c", flags) == FNM_NOMATCH)
    print "no match"
