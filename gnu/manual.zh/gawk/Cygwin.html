B.3.1.4使用gawk在Cygwin环境中

gawk如果使用的是Cygwin环境，则可以在MS-Windows下“直接使用”构建和使用。该环境使用Bash，GCC，GNU Make和其他GNU程序，可以对GNU / Linux进行出色的仿真。Cygwin的编译和安装与Unix系统相同：

tar -xvpzf gawk-5.0.0.tar.gz
cd gawk-5.0.0
./configure
make && make check

与同一系统上的GNU / Linux相比，“ configure '踩Cygwin所需的时间要长得多。但是，它确实完成了，然后“ make照常进行。

Cygwin的最新版本以二进制模式打开所有文件。这意味着您应该使用“ RS = "\r?\n"为了能够处理带有回车符和换行符的标准MS-Windows文本文件。

Cygwin环境同时支持“ |&运营商和TCP / IP网络（请参阅使用gawk用于网络编程 ）。
