11.2.3打印出用户信息

的id实用程序列出了用户的真实和有效用户ID号，真实和有效的组ID号以及用户的组集（如果有）。 id仅在有效用户ID和组ID与实际用户ID和组ID不同时才打印它们。如果可能的话， id还提供相应的用户名和组名。输出可能如下所示：

$ id
-| uid=1000(arnold) gid=1000(arnold) groups=1000(arnold),4(adm),7(lp),27(sudo)

此信息是提供的内容的一部分gawk的PROCINFO数组（请参见预定义变量部分）。然而id实用程序提供的输出比单个数字更可口。

这是一个简单的版本id写在awk 。它使用A库的用户数据库库功能（请参阅“ 读取用户数据库”一节）和组数据库库功能（请参阅“ 读取组数据库”一节） 。 awk功能 。

该程序非常简单。所有工作都在BEGIN规则。用户和组ID号从以下位置获得PROCINFO 。该代码是重复的。用户数据库中真实用户ID号的条目在“ : '。名称是第一个字段。类似的代码用于有效的用户ID号和组号：

# id.awk --- implement id in awk
#
# Requires user and group library functions
# output is:
# uid=12(foo) euid=34(bar) gid=3(baz) \
#             egid=5(blat) groups=9(nine),2(two),1(one)


BEGIN {
    uid = PROCINFO["uid"]
    euid = PROCINFO["euid"]
    gid = PROCINFO["gid"]
    egid = PROCINFO["egid"]


    printf("uid=%d", uid)
    pw = getpwuid(uid)
    pr_first_field(pw)


    if (euid != uid) {
        printf(" euid=%d", euid)
        pw = getpwuid(euid)

        pr_first_field(pw)
    }


    printf(" gid=%d", gid)
    pw = getgrgid(gid)
    pr_first_field(pw)

    if (egid != gid) {
        printf(" egid=%d", egid)
        pw = getgrgid(egid)
        pr_first_field(pw)
    }

    for (i = 1; ("group" i) in PROCINFO; i++) {
        if (i == 1)
            printf(" groups=")
        group = PROCINFO["group" i]
        printf("%d", group)
        pw = getgrgid(group)
        pr_first_field(pw)
        if (("group" (i+1)) in PROCINFO)
            printf(",")
    }

    print ""
}

function pr_first_field(str,  a)
{
    if (str != "") {
        split(str, a, ":")
        printf("(%s)", a[1])
    }
}

测试中for循环值得注意。在任何补充组PROCINFO数组具有索引"group1"通过"groupN"对于一些N （即补充组的总数）。但是，我们事先不知道有多少这样的小组。

该循环从一个开始，将值与"group" ，然后使用in查看该值是否在数组中（请参阅“ 引用数组元素”一节 ）。最终， i超过数组中的最后一个组，循环退出。

如果没有补充组，则循环也是正确的。则条件在第一次测试时为假，并且循环体从不执行。

的pr_first_field()函数简单地将一些重复使用的代码隔离开，从而使整个程序更短，更简洁。特别是，将空字符串的检查移到该函数中可以节省几行代码。
