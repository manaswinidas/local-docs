4.5.5使整行成为单个字段

有时，将整个输入行视为一个字段很有用。通过设置，可以轻松便捷地完成此操作FS至"\n" （换行符）： 21 

awk -F'\n' 'program' files …

当您这样做时$1是相同的$0 。

改变中FS不影响字段

根据POSIX标准， awk假定其行为就像在读取记录时将每个记录拆分为多个字段一样。特别是，这意味着如果您更改FS读取记录后，字段的值（即字段的拆分方式）应反映旧的值FS ，而不是新的。

但是，许多较旧的实现awk不要这样工作。相反，它们将字段拆分推迟到实际引用某个字段为止。使用的当前值拆分字段FS ！（直流）此行为可能难以诊断。下面的示例说明了两种方法之间的区别：

sed 1q /etc/passwd | awk '{ FS = ":" ; print $1 }'

通常会打印：

root

在不正确的实现上awk ，而gawk打印文件的第一行，例如：

root:x:0:0:Root:/:

（ sed 22命令仅打印第一行/etc/passwd ）
