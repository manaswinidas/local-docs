13.3国际化awk程式

gawk为国际化提供以下变量：

TEXTDOMAIN

此变量指示应用程序的文本域。为了与GNU兼容gettext ，默认值为"messages" 。

_"your message here"

标有前导下划线的字符串常量是运行时转换的候选对象。没有前导下划线的字符串常量不会被翻译。

gawk为国际化提供以下功能：

dcgettext(string [, domain [, category]])

返回翻译string在文本域中domain对于语言环境类别category 。的默认值domain是的当前值TEXTDOMAIN 。的默认值category是"LC_MESSAGES" 。

如果您提供category ，它必须是等于上一节中描述的已知语言环境类别之一的字符串。您还必须提供文本域。使用TEXTDOMAIN如果要使用当前域。

注意：参数的顺序awk版本dcgettext()功能故意不同于C版本的顺序。的awk选择版本的顺序是为了简单并允许合理awk样式的默认参数。

dcngettext(string1, string2, number [, domain [, category]])

返回用于的复数形式number的翻译string1和string2在文本域中domain对于语言环境类别category 。 string1是消息的英文单数变体，并且string2是同一封邮件的英文复数形式。的默认值domain是的当前值TEXTDOMAIN 。的默认值category是"LC_MESSAGES" 。

关于参数顺序的注释与dcgettext()功能适用。

bindtextdomain(directory [, domain ])

更改目录gettext寻找.gmo文件，以防它们无法或不能放置在标准位置（例如，在测试过程中）。返回其中的目录domain被“绑定”。

默认值domain是...的价值TEXTDOMAIN 。如果directory是空字符串（ "" ）， 然后bindtextdomain()返回给定的当前绑定domain 。

在您的房间中使用这些设施awk程序，请按照下列步骤操作：

  1. 设置变量TEXTDOMAIN到程序的文本域。最好在BEGIN规则（见节的BEGIN和END特殊模式 ），也可以通过-v命令行选项（请参阅“ 命令行选项”部分 ）：
    BEGIN {
    TEXTDOMAIN = "guide"
    …
}
  2. 将所有可翻译的字符串标记为下划线（' _ '）字符。它必须与字符串的开头引号相邻。例如：
    print _"hello, world"
x = _"you goofed"
printf(_"Number of users is %d\n", nusers)
  3. 如果您要动态创建字符串，则仍然可以使用dcgettext()内置功能： 90 
    if (groggy)
    message = dcgettext("%d customers disturbing me\n", "adminprog")
else
    message = dcgettext("enjoying %d customers\n", "adminprog")
printf(message, ncustomers)

    在这里， dcgettext()提供不同的文本域（ "adminprog" ）在其中查找消息，但它使用默认值"LC_MESSAGES"类别。

    前面的示例仅在以下情况下有效ncustomers大于一。这个例子最好用dcngettext() ：

    if (groggy)
    message = dcngettext("%d customer disturbing me\n",
                         "%d customers disturbing me\n",
                         ncustomers, "adminprog")
else
    message = dcngettext("enjoying %d customer\n",
                         "enjoying %d customers\n",
                         ncustomers, "adminprog")
printf(message, ncustomers)
  4. 在开发过程中，您可能需要将.gmo文件放在专用目录中进行测试。这是通过bindtextdomain()内置功能：
    BEGIN {
   TEXTDOMAIN = "guide"   # our text domain
   if (Testing) {
       # where to find our files
       bindtextdomain("testdir")
       # joe is in charge of adminprog
       bindtextdomain("../joe/testdir", "adminprog")
   }
   …
}

有关示例程序的示例程序，请参见“简单国际化示例”部分， 其中显示了从中创建和使用翻译的步骤。 awk 。
