6.2.1算术运算符

的awk语言在评估表达式时使用通用算术运算符。所有这些算术运算符都遵循正常的优先级规则，并且可以按您期望的那样工作。

下面的示例使用一个名为grades ，其中包含学生姓名列表以及每个学生三个测试分数（这是一门小班）：

Pat   100 97 58
Sandy  84 72 93
Chris  72 92 89

该程序将文件grades并打印分数的平均值：

$ awk '{ sum = $2 + $3 + $4 ; avg = sum / 3
>        print $1, avg }' grades
-| Pat 85
-| Sandy 83
-| Chris 84.3333

下表提供了算术运算符awk ，从最高优先级到最低优先级：

x ^ y
x ** y

求幂; x提出来y功率。 ' 2 ^ 3 '的值为八；字符序列' ** '等价于' ^ '。 （ce）

- x

否定。

+ x

一元加；表达式将转换为数字。

x * y

乘法。

x / y

师;因为所有数字都在awk是浮点数，结果未四舍五入为整数-' 3 / 4 '的值为0.75。（这是一个常见的错误，尤其是对于C程序员而言，忘记了所有数字awk是浮点数，看似常量的除法会产生一个实数，而不是整数。）

x % y

余;在此列表之后的正文中提供了进一步的讨论。

x + y

加成。

x - y

减法。

一元加号和减号具有相同的优先级，乘法运算符都具有相同的优先级，加法和减法具有相同的优先级。

计算“ x % y '，商将零取整为整数并乘以y 。从中减去该结果x ;该操作有时称为“ trunc-mod”。以下关系始终成立：

b * int(a / b) + (a % b) == a

此余数定义的一个可能的不良影响是“ x % y如果为负x是负面的。从而：

-17 % 8 = -1

其他awk在实施中，其余的签名可能取决于机器。

注意： POSIX标准仅指定使用' ^以求幂。为了获得最大的便携性，请勿使用“ ** '运算符。
