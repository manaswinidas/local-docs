1.3一些简单的例子

下面的命令运行一个简单的awk搜索输入文件的程序mail-list对于字符串“ li '（一组字符通常称为字符串 ；术语字符串是基于英语中的类似用法，例如“一串珍珠”或“火车上的一列汽车”）：

awk '/li/ { print $0 }' mail-list

当包含“ li找到了，之所以打印它们是因为print $0 ' 表示打印当前行。（只是print本身就意味着同一件事，因此我们可以写成它。）

您会注意到斜杠（' / '）包围字符串' li ' 在里面awk程序。斜线表示“ li '是要搜索的模式。这种类型的模式称为正则表达式 ，稍后将对其进行详细介绍（请参见正则表达式一节）。该模式允许匹配单词的各个部分。左右有单引号awk程序，以便外壳程序不会将其中的任何内容解释为特殊的外壳程序字符。

这是程序打印的内容：

$ awk '/li/ { print $0 }' mail-list
-| Amelia       555-5553     amelia.zodiacusque@gmail.com    F
-| Broderick    555-0542     broderick.aliquotiens@yahoo.com R
-| Julie        555-6699     julie.perscrutabor@skeeve.com   F
-| Samuel       555-3430     samuel.lanceolis@shu.edu        A

在一个awk规则，模式或操作都可以省略，但不能两者都省略。如果省略该模式，则对每条输入线执行该动作。如果省略该操作，则默认操作是打印所有与该模式匹配的行。

因此，我们可以省略该操作（ print上例中的语句和花括号），结果将是相同的： awk打印所有与模式' li '。相比之下，省略了print语句但保留大括号将执行一个不执行任何操作的空操作（即，不打印任何行）。

许多实用awk程序只有一两行。以下是一些有用的简短程序，可以帮助您入门。其中一些程序包含尚未涵盖的构造。（该程序的描述将使您对正在发生的事情有一个很好的了解，但是您需要阅读Web页面的其余部分才能成为awk专家！）大多数示例使用一个名为data 。这只是一个占位符；如果您自己使用这些程序，请用您自己的文件名替换data 。供将来参考，请注意，通常有多种方法可以awk 。在某个时候，您可能想回顾一下这些示例，看看是否可以提出不同的方法来完成此处显示的相同操作：

  • 打印每行超过80个字符的行：
    awk 'length($0) > 80' data

    唯一的规则将关系表达式作为其模式，并且没有任何操作，因此它使用默认操作来打印记录。

  • 打印最长输入行的长度：
    awk '{ if (length($0) > max) max = length($0) }
     END { print max }' data

    与相关的代码END读取所有输入后执行；是硬币的另一面BEGIN 。

  • 打印中的最长行的长度data ：
    expand data | awk '{ if (x < length($0)) x = length($0) }
                   END { print "maximum line length is " x }'

    此示例与上一个示例稍有不同：输入由输入处理expand实用程序将TAB更改为空格，因此比较的宽度实际上是右边缘的列，而不是每行输入字符的数量。

  • 打印至少包含一个字段的每一行：
    awk 'NF > 0' data

    这是从文件中删除空白行的简便方法（或者，类似于旧文件创建一个新文件，但空白行已从中删除）。

  • 打印0到100之间的七个随机数，包括：
    awk 'BEGIN { for (i = 1; i <= 7; i++)
                 print int(101 * rand()) }'
  • 打印以下对象使用的总字节数files ：
    ls -l files | awk '{ x += $5 }
                   END { print "total bytes: " x }'
  • 打印用户使用的千字节总数files ：
    ls -l files | awk '{ x += $5 }
   END { print "total K-bytes:", x / 1024 }'
  • 打印所有用户的登录名的排序列表：
    awk -F: '{ print $1 }' /etc/passwd | sort
  • 计算文件中的行数：
    awk 'END { print NR }' data
  • 打印数据文件中的偶数行：
    awk 'NR % 2 == 0' data

    如果您使用表达式“ NR % 2 == 1 '，程序将打印奇数行。
