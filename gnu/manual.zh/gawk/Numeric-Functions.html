9.1.2数值函数

下表描述了所有与数字一起使用的内置函数。可选参数括在方括号（[]）中：

atan2(y, x)

返回的反正切y / x以弧度为单位。您可以使用 ' pi = atan2(0, -1) '以获取pi的值。

cos(x)

返回的余弦x ，带有x以弧度为单位。

exp(x)

返回的指数x （ e ^ x ）或报告错误，如果x超出范围。取值范围x取决于您的计算机的浮点表示形式。

int(x)

返回最接近的整数到x ，位于x和零，并被截断为零。例如， int(3)是3 int(3.9)是3 int(-3.9)是-3，并且int(-3)也是-3

log(x)

返回的自然对数x如果x是肯定的；否则，返回NaN （“非数字”）在IEEE 754系统上。另外， gawk在以下情况时显示警告消息x是负面的。

rand()

返回一个随机数。的值rand()在零和一之间均匀分布。该值可以为零，但不能为一。 43

通常，需要使用随机整数代替。以下是用户定义的函数，可用于获取小于的随机非负整数n ：

function randint(n)
{
    return int(n * rand())
}

乘法产生一个大于或等于零且小于n 。使用int() ，则此结果将成为介于0和之间的整数n -1（含）。

以下示例使用类似的函数来生成介于和之间的随机整数n 。该程序为每个输入记录打印一个新的随机数：

# Function to roll a simulated die.
function roll(n) { return 1 + int(rand() * n) }

# Roll 3 six-sided dice and
# print total number of points.
{
    printf("%d points\n", roll(6) + roll(6) + roll(6))
}

注意：在大多数情况下awk实现，包括gawk ， rand()每次您运行时，都会从相同的起始编号或seed开始生成编号awk 。 44因此，程序每次运行都会产生相同的结果。这些数字在一个内是随机的awk运行，但可以预测运行。这对于调试很方便，但是如果希望程序在每次使用时都做不同的事情，则必须将种子更改为每次运行时都不同的值。为此，请使用srand() 。

sin(x)

返回的正弦x ，带有x以弧度为单位。

sqrt(x)

返回的正平方根x 。 gawk打印警告消息，如果x是负面的。从而， sqrt(4)是2。

srand( [ x ])

将生成随机数的起点或种子设置为值x 。

每个种子值导致特定的随机数序列。 45因此，如果第二次将种子设置为相同的值，则会再次产生相同的随机数序列。

注意：不同awk实现内部使用不同的随机数生成器。不要期望一样awk由不同版本的程序执行时产生相同系列的随机数的程序awk 。

如果论点x被省略，如' srand() '，则将当前日期和时间用于种子。这是获得真正不可预测的随机数的方法。

脚注

（43）

C版rand()在许多Unix系统上，众所周知会产生相当差的随机数序列。但是，没有任何要求awk实现使用C rand()实施awk版本rand() 。实际上，多年来， gawk使用了BSD random()功能，比rand() ，以产生随机数。从4.1.4版开始，由Nelson HF Beebe提供， gawk使用贝叶斯-达勒姆（Bayes-Durham）混洗缓冲算法，该算法大大延长了随机数生成器的周期，并消除了原始生成器中可能存在的短距离和长距离相关性。

计算机生成的随机数实际上并不是真正的随机数。它们在技术上称为伪随机 。这意味着尽管序列中的数字看起来是随机的，但实际上您可以一次又一次地生成相同的随机数序列。
