7.4.5 switch声明

本节介绍了gawk特定的功能。如果gawk处于兼容模式（请参阅“ 命令行选项”一节），它不可用。

的switch语句允许对表达式的求值和基于语句的语句的执行case比赛。按照定义的顺序检查Case语句是否匹配。如果不合适case被发现， default如果提供，则执行该节。

每case包含单个常量，可以是数字，字符串或正则表达式。的switch计算表达式，然后每个case的常量与结果依次进行比较。常量的类型确定比较：数字或字符串进行常规比较。regexp常量会将正则表达式与原始表达式的字符串值进行匹配。的一般形式switch语句如下所示：

switch (expression) {
case value or regular expression:
    case-body
default:
    default-body
}

控制流switch语句的工作方式与C中相同。与指定案例匹配后，case语句主体将执行直到break ， continue ， next ， nextfile ， 要么exit遇到或结束switch声明本身。例如：

while ((c = getopt(ARGC, ARGV, "aksx")) != -1) {
    switch (c) {
    case "a":
        # report size of all files
        all_files = TRUE;
        break
    case "k":
        BLOCK_SIZE = 1024       # 1K block size
        break
    case "s":
        # do sums only
        sum_only = TRUE
        break
    case "x":
        # don't cross filesystems
        fts_flags = or(fts_flags, FTS_XDEV)
        break
    case "?":
    default:
        usage()
        break
    }
}

请注意，如果此处指定的所有语句均未停止匹配的执行case语句，执行进入下一个case直到执行停止。在此示例中， case对于"?"落入default情况，即调用一个名为usage() 。（ getopt() 处理命令行选项中描述了此处调用的函数。）
