3.3.1中的正则表达式运算符awk

前面在转义序列中描述的转义序列在正则表达式内有效。它们由“ \ '和被识别并转换为相应的实字符，这是处理正则表达式的第一步。

这是元字符列表。不是转义序列且未在此处列出的所有字符都代表自己：

\

这样可以抑制匹配时字符的特殊含义。例如， ' \$ “匹配字符” $ '。

^

这匹配字符串的开头。' ^@chapter ' 火柴 ' @chapter例如，在字符串的开头，可用于标识Texinfo源文件中的章节开头。' ^ '称为锚 ，因为它锚定模式以仅在字符串的开头进行匹配。

重要的是要意识到“ ^ '与行的开头不符（'之后的点\n （换行符）嵌入在字符串中。在以下示例中，条件不成立：

if ("line1\nLINE 2" ~ /^L/) …
$

这类似于“ ^ '，但仅在字符串末尾匹配。例如， ' p$ '匹配以'结尾的记录p '。' $ '是锚点，与行尾不符（' \n （换行符）嵌入在字符串中。以下示例中的条件不正确：

if ("line1\nLINE 2" ~ /1$/) …
. （期）

这匹配任何单个字符， 包括换行符。例如， ' .P '匹配任何单个字符，后跟一个' P在字符串中。使用串联，我们可以制作一个正则表达式，例如' U.A '，与以'开头的任何三个字符的序列匹配U '并以'结尾A '。

在严格的POSIX模式下（请参见“ 命令行选项 ”部分），“ . '与NUL字符不匹配， NUL字符是所有位等于零的字符。否则， NUL只是另一个字符。其他版本awk可能无法匹配NUL字符。

[ …]

这称为方括号表达式 。 15它匹配方括号中包含的任何一个字符。例如， ' [MVX] '匹配任何一个字符' M '，' V '， 要么 ' X在字符串中。使用括号表达式中将对括号表达式的方括号内的内容进行全面讨论。

[^ …]

这是一个补充的括号表达式 。'之后的第一个字符[ ' 必须是' ^ '。它的任何字符匹配除了方括号。例如， ' [^awk] '匹配非'的任何字符a '，' w '， 要么 ' k '。

|

这是交替运算符 ，用于指定替代项。' | '在所有正则表达式运算符中具有最低的优先级。例如， ' ^P|[aeiouy] '匹配任何一个与' ^P ' 要么 ' [aeiouy] '。这意味着它匹配以'开头的任何字符串P ”（或在其中的任何位置）包含小写的英语元音。

交替适用于两侧最大的正则表达式。

( …)

像算术一样，圆括号用于将正则表达式分组。它们可用于连接包含交替运算符' | '。例如， ' @(samp|code)\{[^}]+\} '都匹配' @code{foo} '和' @samp{bar} '。（这些是Texinfo格式化控制序列。' +在此列表中进一步说明。）

*

该符号表示，前面的正则表达式应根据需要重复多次以找到匹配项。例如， ' ph* '应用' * '前面的符号' h '并寻找一个' p '后跟任意数量的' h的。这也只匹配“ p ' 如果不 ' h的存在。

要了解如何“ *的作品。首先， * '仅适用于单个前面的正则表达式组成部分（例如， ph* '，它仅适用于' h '）。导致“ * '要应用于较大的子表达式，请使用括号：' (ph)* ' 火柴 ' ph '，' phph '，' phphph '， 等等。

其次， * '找到尽可能多的重复。如果要匹配的文本是“ phhhhhhhhhhhhhhooey '，' ph* '匹配所有' h的。

+

该符号类似于“ * '，但前面的表达式必须至少匹配一次。这意味着 ' wh+y “会匹配” why '和' whhy '， 但不是 ' wy '，而' wh*y '会匹配所有三个。

?

该符号类似于“ * '，除了前面的表达式可以一次匹配或根本不匹配。例如， ' fe?d ' 火柴 ' fed '和' fd '， 但没有别的。

{n}
{n,}
{n,m}

花括号内的一个或两个数字表示间隔表达式 。如果花括号中有一个数字，则重复前面的正则表达式n次。如果两个数字之间用逗号分隔，则重复前面的正则表达式n至m次。如果后跟一个逗号，则前一个正则表达式至少重复一次n时间：

wh{3}y

火柴 ' whhhy '， 但不是 ' why ' 要么 ' whhhhy '。

wh{3,5}y

火柴 ' whhhy '，' whhhhy '， 要么 ' whhhhhy ' 只要。

wh{2,}y

火柴 ' whhy '，' whhhy '， 等等。

在正则表达式中，“ * '，' + '和' ? '运算符，以及大括号' { '和' } '，具有最高优先级，然后是串联，最后是' | '。与算术一样，括号可以更改运算符的分组方式。

在POSIX中awk和gawk ， * '，' + '和' ?当正则表达式中没有任何运算符时，运算符将代表自己。例如， /+/匹配文字加号。但是，许多其他版本的awk将此用法视为语法错误。
