17.7.8双向I / O示例

的revtwoway扩展程序添加了一个简单的双向处理器，该处理器可以反转发送给它的每行中的字符，以便由awk程序。它的主要目的是显示如何编写双向处理器，尽管它可能也很有趣。以下示例显示了如何使用它：

@load "revtwoway"

BEGIN {
    cmd = "/magic/mirror"
    print "don't panic" |& cmd
    cmd |& getline result
    print result
    close(cmd)
}

该程序的输出也为：' cinap t'nod '。
