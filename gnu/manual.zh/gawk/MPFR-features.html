16.3中的任意精度算术功能gawk

默认， gawk使用由其运行的系统硬件提供的双精度浮点值。但是，如果将其编译为这样做，并且-M提供了命令行选项， gawk使用GNU MPFR和GNU MP （GMP）库对数字进行任意精度的算术运算。您可以看到MPFR支持是否可用，如下所示：

$ gawk --version
-| GNU Awk 4.1.2, API: 1.1 (GNU MPFR 3.1.0-p3, GNU MP 5.0.2)
-| Copyright (C) 1989, 1991-2015 Free Software Foundation.
…

（您可能会看到与此处显示的版本号不同的版本号。没关系;重要的是要确保输出中列出了GNU MPFR和GNU MP。）

此外， PROCINFO数组以提供有关MPFR和GMP库的信息（请参阅传达信息的内置变量部分）。

MPFR库提供了对精度和舍入模式的精确控制，并给出了正确舍入，可再现，与平台无关的结果。随着-M命令行选项，所有浮点算术运算符和数值函数都可以将结果产生到MPFR支持的任何所需的精度级别。

两个预定义的变量PREC和ROUNDMODE ，提供对工作精度和舍入模式的控制。精度和舍入模式是针对每个后续操作全局设置的。有关更多信息，请参见设置精度和设置舍入模式部分 。
