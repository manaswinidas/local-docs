B.6小结

  • 的gawk可以从GNU Project的主要分发站点进行分发， ftp.gnu.org 。规范的构建方法是：
    wget https://ftp.gnu.org/gnu/gawk/gawk-5.0.0.tar.gz
tar -xvpzf gawk-5.0.0.tar.gz
cd gawk-5.0.0
./configure && make && make check

    注意：由于“ https:// 'URL，您可能需要提供--no-check-certificate选择wget下载文件。

  • gawk也可以在非POSIX系统上构建。当前支持的系统是使用MSYS，DJGPP，MinGW和Cygwin的MS-Windows，以及Vax / VMS和OpenVMS。附录中包含每个系统的说明。
  • 错误报告应通过电子邮件发送至bug-gawk@gnu.org 。错误报告应为英文，并应包含gawk ，如何编译以及演示该问题的简短程序和数据文件。
  • 还有许多其他免费提供的awk实现。许多都符合POSIX；其他人则不是如此。
