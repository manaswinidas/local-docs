5.5.4使用示例printf

以下简单示例显示了如何使用printf制作对齐表：

awk '{ printf "%-10s %s\n", $1, $2 }' mail-list

此命令显示人员的姓名（ $1 ）在文件中mail-list作为由10个左对齐的字符组成的字符串。它还会打印电话号码（ $2 ）接下来就行了。这将生成一个对齐的两列名称和电话号码表，如下所示：

$ awk '{ printf "%-10s %s\n", $1, $2 }' mail-list
-| Amelia     555-5553
-| Anthony    555-3412
-| Becky      555-7685
-| Bill       555-1675
-| Broderick  555-0542
-| Camilla    555-2912
-| Fabius     555-1234
-| Julie      555-6699
-| Martin     555-6480
-| Samuel     555-3430
-| Jean-Paul  555-2127

在这种情况下，电话号码必须打印为字符串，因为电话号码之间用短划线隔开。将电话号码打印为数字只会产生前三位数字： 555 '。这本来就很令人困惑。

不必为电话号码指定宽度，因为电话号码排在最后。他们不需要后面有空格。

通过将标题添加到列的顶部，可以使表格看起来更好。这是通过使用BEGIN规则（见节的BEGIN和END特殊模式 ），这样标题就只能在开始时打印一次awk程序：

awk 'BEGIN { print "Name      Number"
             print "----      ------" }
           { printf "%-10s %s\n", $1, $2 }' mail-list

前面的示例混合print和printf同一程序中的语句。仅使用printf语句可以产生相同的结果：

awk 'BEGIN { printf "%-10s %s\n", "Name", "Number"
             printf "%-10s %s\n", "----", "------" }
           { printf "%-10s %s\n", $1, $2 }' mail-list

使用与列元素相同的格式规范打印每个列标题，以确保标题与列一样对齐。

通过将同一格式规范存储在变量中，可以强调使用相同格式规范三次的事实，如下所示：

awk 'BEGIN { format = "%-10s %s\n"
             printf format, "Name", "Number"
             printf format, "----", "------" }
           { printf format, $1, $2 }' mail-list
