7.4.3 do -- while声明

的do循环是while循环语句。的do循环执行body一次，然后重复body只要condition是真的。看起来像这样：

do
  body
while (condition)

即使condition一开始是假的， body至少执行一次（只有一次，除非执行body使condition真正）。与此相对应while声明：

while (condition)
    body

该语句不执行body即使一次condition首先是错误的。以下是一个例子do声明：

{
    i = 1
    do {
        print $0
        i++
    } while (i <= 10)
}

该程序将每个输入记录打印10次。但是，这不是一个非常现实的示例，因为在这种情况下， while也会做得很好。这种情况反映了实际经验；只是偶尔有一个真正的用途do声明。
