10.7遍历数组

数组描述了数组 gawk提供数组的数组。特别地，数组的任何元素可以是标量或另一个数组。的isarray()函数（请参阅“ 获取类型信息”一节）可让您将数组与标量区分开。以下功能， walk_array()递归地遍历数组，打印元素索引和值。您可以使用数组和代表数组名称的字符串来调用它：

function walk_array(arr, name,      i)
{
    for (i in arr) {
        if (isarray(arr[i]))
            walk_array(arr[i], (name "[" i "]"))
        else
            printf("%s[%s] = %s\n", name, i, arr[i])
    }
}

它通过遍历数组的每个元素来工作。如果任何给定元素本身是数组，则该函数以递归方式调用自身，并传递子数组和代表当前索引的新字符串。否则，该函数仅打印元素的名称，索引和值。这是一个演示的主要程序：

BEGIN {
    a[1] = 1
    a[2][1] = 21
    a[2][2] = 22
    a[3] = 3
    a[4][1][1] = 411
    a[4][2] = 42

    walk_array(a, "a")
}

运行时，程序将产生以下输出：

$ gawk -f walk_array.awk
-| a[1] = 1
-| a[2][1] = 21
-| a[2][2] = 22
-| a[3] = 3
-| a[4][1][1] = 411
-| a[4][2] = 42

刚刚提供的函数仅打印每个标量数组元素的名称和值。但是，通过传递在遍历数组时要调用的函数的名称，可以很容易地将其概括。修改后的函数如下所示：

function process_array(arr, name, process, do_arrays,   i, new_name)
{
    for (i in arr) {
        new_name = (name "[" i "]")
        if (isarray(arr[i])) {
            if (do_arrays)
                @process(new_name, arr[i])
            process_array(arr[i], new_name, process, do_arrays)
        } else
            @process(new_name, arr[i])
    }
}

参数如下：

arr

数组。

name

数组的名称（字符串）。

process

要调用的函数的名称。

do_arrays

如果为真，则该函数可以处理属于子数组的元素。

如果要处理子数组，请在进一步处理之前完成。

使用以下脚手架运行时，该函数产生的结果与早期版本的相同。 walk_array() ：

BEGIN {
    a[1] = 1
    a[2][1] = 21
    a[2][2] = 22
    a[3] = 3
    a[4][1][1] = 411
    a[4][2] = 42

    process_array(a, "a", "do_print", 0)
}

function do_print(name, element)
{
    printf "%s = %s\n", name, element
}
