7.4.2 while声明

在编程中， 循环是程序的一部分，可以连续执行两次或多次。的while语句是中最简单的循环语句awk 。只要条件为真，它就会重复执行一条语句。例如：

while (condition)
  body

body是一个称为循环主体的语句，并且condition是控制循环保持运行时间的表达式。第一件事while声明确实是测试condition 。如果condition是真的，它执行语句body 。后body已经执行了condition经过再次测试，如果仍然正确， body再次执行。重复此过程，直到condition不再是真的。如果condition最初为false，循环的主体从不执行， awk继续循环后的语句。本示例打印每条记录的前三个字段，每行一个：

awk '
{
    i = 1
    while (i <= 3) {
        print $i
        i++
    }
}' inventory-shipped

该循环的主体是用大括号括起来的复合语句，其中包含两个语句。循环以以下方式工作：首先， i设置为一个。然后， while语句测试是否i小于或等于三。这是真的i等于一，所以i字段被打印。然后， i++ '增加值i并且循环重复。循环在以下时间终止i达到四。

在条件和身体之间不需要换行符；但是，除非主体是复合语句，否则使用一个可使程序更清晰。也不需要在复合语句开始的大括号后面的换行符，但是如果没有它，则程序将更难以阅读。
