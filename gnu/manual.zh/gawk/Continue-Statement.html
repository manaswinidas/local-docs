7.4.7 continue声明

相似break ， continue语句仅在内部使用for ， while和do循环。它跳过了循环主体的其余部分，导致围绕循环的下一个循环立即开始。与之对比break ，它们完全跳出了循环。

的continue中的陈述for循环指令awk跳过循环的其余部分，并使用for声明。下面的程序说明了这一事实：

BEGIN {
     for (x = 0; x <= 20; x++) {
         if (x == 5)
             continue
         printf "%d ", x
     }
     print ""
}

该程序将打印从0到20的所有数字，但5除外。 printf被跳过。因为增量' x++ '不会被跳过， x不会停留在5。对比for从前面的示例循环如下while环：

BEGIN {
     x = 0
     while (x <= 20) {
         if (x == 5)
             continue
         printf "%d ", x
         x++
     }
     print ""
}

这个程序永远循环一次x达到5，因为增量（' x++ '）永远不会到达。

的continue该陈述对于switch语句，在循环体之外使用时也没有任何意义。的历史版本awk治疗一个continue循环外的声明与他们对待break循环外的语句：好像是一个next声明（见部分的next声明 ）。（直流）BWK的最新版本awk不再以这种方式工作，也不再gawk 。
