32.3支持哪个版本的POSIX

巨集： long int _POSIX_VERSION

该常量表示实现所遵循的POSIX.1标准的版本。对于符合1995 POSIX.1标准的实现，该值为整数199506L 。

_POSIX_VERSION始终被定义（在unistd.h ）在任何POSIX系统中。

使用注意：不要尝试通过包含以下内容来测试系统是否支持POSIX： unistd.h然后检查是否_POSIX_VERSION被定义为。在非POSIX系统上，这可能会失败，因为没有unistd.h 。我们不知道您可以在编译时可靠地测试目标系统是否支持POSIX的任何方式unistd.h存在。

巨集： long int _POSIX2_C_VERSION

该常数表示库和系统内核支持的POSIX.2标准的版本。我们不知道这对POSIX.2标准的第一个版本会有什么价值，因为该价值基于正式采用该标准的年份和月份。

该符号的值与系统上安装的实用程序无关。

使用注意：您可以使用此宏来判断POSIX.1系统库是否也支持POSIX.2。任何POSIX.1系统都包含unistd.h ，因此请包含该文件，然后进行测试defined (_POSIX2_C_VERSION) 。
