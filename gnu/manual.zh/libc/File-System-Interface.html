14文件系统界面

本章介绍了GNU C库用于处理文件的功能。与输入和输出功能（请参阅Streams上的I / O ；请参阅低级I / O ）不同，这些功能与文件本身而不是内容的操作有关。

本章介绍的功能包括检查或修改目录的功能，重命名和删除文件的功能以及检查和设置文件属性（例如访问权限和修改时间）的功能。

• 工作目录  这用于解析相对文件名。
• 访问目录  找出目录包含的文件。
• 使用目录树  将操作应用于所有文件或目录层次结构的可选子集。
• 硬链接  将备用名称添加到文件。
• 符号链接  “指向”文件名的文件。
• 删除文件  如何删除文件，这意味着什么。
• 重命名文件  更改文件名。
• 创建目录  仅用于创建目录的系统调用。
• 文件属性  单个文件的属性。
• 制作特殊文件  如何创建特殊文件。
• 临时文件  命名和创建临时文件。
