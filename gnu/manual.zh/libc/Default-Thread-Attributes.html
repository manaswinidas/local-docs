35.2.2.1为线程属性设置进程范围的默认值

GNU C库提供非标准的API函数，以设置和获取在进程中创建线程时使用的默认属性。

函数： int pthread_getattr_default_np （pthread_attr_t * attr ）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

获取默认属性值并设置attr匹配。成功时此函数返回0 ，失败时返回非零错误代码。

功能： int pthread_setattr_default_np （pthread_attr_t * attr ）

初步： MT安全| AS不安全堆锁| AC不安全锁记忆|请参阅POSIX安全概念 。

设置默认属性值以匹配其中的值attr 。该函数在成功时返回0 ，在失败时返回非零错误代码。为此功能定义了以下错误代码：

EINVAL

中的至少一个值attr对属性无效，或者在属性中设置了堆栈地址。

ENOMEM

系统没有足够的内存。

函数： int sem_clockwait （sem_t * sem ，clockid_t clockid ，

const struct timespec * abstime ）表现像sem_timedwait除了时间abstime相对于指定的时钟进行测量clockid而不是CLOCK_REALTIME 。目前， clockid必须是CLOCK_MONOTONIC要么CLOCK_REALTIME 。

函数： int pthread_cond_clockwait （pthread_cond_t * cond ，pthread_mutex_t * mutex ，

clockid_t clockid ，const struct timespec * abstime ）初步：| MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

表现像pthread_cond_timedwait除了时间abstime相对于指定的时钟进行测量clockid而不是在以下情况下指定或默认的时钟pthread_cond_init被称为。目前， clockid必须是CLOCK_MONOTONIC要么CLOCK_REALTIME 。

功能： int pthread_rwlock_clockrdlock （pthread_rwlock_t * rwlock ，

clockid_t clockid ，const struct timespec * abstime ）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

表现像pthread_rwlock_timedrdlock除了时间abstime相对于指定的时钟进行测量clockid而不是CLOCK_REALTIME 。目前， clockid必须是CLOCK_MONOTONIC要么CLOCK_REALTIME ， 除此以外EINVAL返回。

功能： int pthread_rwlock_clockwrlock （pthread_rwlock_t * rwlock ，

clockid_t clockid ，const struct timespec * abstime ）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

表现像pthread_rwlock_timedwrlock除了时间abstime相对于指定的时钟进行测量clockid而不是CLOCK_REALTIME 。目前， clockid必须是CLOCK_MONOTONIC要么CLOCK_REALTIME ， 除此以外EINVAL返回。
