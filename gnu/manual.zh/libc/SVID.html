1.2.4 SVID（系统V接口说明）

System V接口说明 （SVID）是描述AT＆T Unix System V操作系统的文档。在某种程度上，它是POSIX标准的超集（请参阅POSIX ）。

GNU C库定义了SVID所需的大多数工具，而ISO C 或POSIX标准也不需要这些工具，以与System V Unix和其他包含这些工具的Unix系统（例如SunOS）兼容。但是，不包括SVID所需的许多较晦涩和不太通用的工具。（实际上，Unix System V本身并不能全部提供。）

System V支持的功能包括进程间通信和共享内存的方法， hsearch和drand48功能族fmtmsg以及一些数学函数。
