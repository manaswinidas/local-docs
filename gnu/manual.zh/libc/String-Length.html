5.3字符串长度

您可以使用来获取字符串的长度strlen功能。该函数在头文件中声明string.h 。

功能： size_t strlen （const char * s ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的strlen函数返回字符串的长度s以字节为单位。（换句话说，它返回数组中终止空字节的偏移量。）

例如，

strlen ("hello, world")
    ⇒ 12

当应用于数组时， strlen函数返回存储在那里的字符串的长度，而不是分配的大小。您可以使用来获取包含字符串的数组的分配大小sizeof操作员：

char string[32] = "hello, world";
sizeof (string)
    ⇒ 32
strlen (string)
    ⇒ 12

但是请注意，除非string是数组本身，而不是指向它的指针。例如：

char string[32] = "hello, world";
char *ptr = string;
sizeof (string)
    ⇒ 32
sizeof (ptr)
    ⇒ 4  /* (on a machine with 4 byte pointers) */

当您使用带有字符串参数的函数时，这是一个容易犯的错误。这些参数始终是指针，而不是数组。

还必须注意，对于多字节编码的字符串，返回值不必与字符串中的字符数相对应。要获得此值，可以将字符串转换为宽字符，然后wcslen可以使用或可以使用以下代码：

/* The input is in string.
   The length is expected in n.  */
{
  mbstate_t t;
  char *scopy = string;
  /* In initial state.  */
  memset (&t, '\0', sizeof (t));
  /* Determine number of characters.  */
  n = mbsrtowcs (NULL, &scopy, strlen (scopy), &t);
}

如果经常需要字符数（而不是字节），这样做很麻烦。最好使用宽字符。

等价的宽字符在wchar.h 。

功能： size_t wcslen （const wchar_t * ws ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的wcslen功能是相当于的宽字符strlen 。返回值是由指向的宽字符串中的宽字符数ws （这也是的终止空宽字符的偏移量ws ）。

由于没有多个宽字符序列组成一个宽字符，因此返回值不仅是数组中的偏移量，而且还是宽字符的数量。

在ISO C90 修订1 引入了此功能。

功能： size_t strnlen （const char * s ，size_t maxlen ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果数组s大小maxlen包含一个空字节， strnlen函数返回字符串的长度s以字节为单位。否则返回maxlen 。因此，此功能等效于(strlen (s) < maxlen ? strlen (s) : maxlen)但它效率更高，即使s只要不以null终止， maxlen不超过s的数组。 

char string[32] = "hello, world";
strnlen (string, 32)
    ⇒ 12
strnlen (string, 5)
    ⇒ 5

此函数是GNU扩展，在中声明string.h 。

功能： size_t wcsnlen （const wchar_t * ws ，size_t maxlen ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

wcsnlen等于strnlen 。的maxlen参数指定最大宽度字符。

此函数是GNU扩展，在中声明wchar.h 。
