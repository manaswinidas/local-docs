26.7流程完成状态

如果子进程的退出状态值（请参阅程序终止 ）为零，则由waitpid要么wait也为零。您可以使用以下宏测试返回状态值中编码的其他类型的信息。这些宏在头文件中定义sys/wait.h 。

宏： int WIFEXITED （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果子进程正常终止，则此宏返回一个非零值exit要么_exit 。

巨集： int WEXITSTATUS （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果WIFEXITED是真的status ，此宏返回子进程退出状态值的低8位。请参阅退出状态 。

宏： int WIFSIGNALED （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果子进程由于接收到未处理的信号而终止，则此宏返回非零值。请参阅信号处理 。

宏： int WTERMSIG （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果WIFSIGNALED是真的status ，此宏返回终止子进程的信号的信号号。

巨集： int WCOREDUMP （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果子进程终止并产生了核心转储，则此宏返回一个非零值。

宏： int WIFSTOPPED （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果子进程已停止，则此宏返回一个非零值。

宏： int WSTOPSIG （int status ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果WIFSTOPPED是真的status ，此宏返回导致子进程停止的信号的信号号。
