A.2.2.1可变参数的语法

接受可变数量参数的函数必须使用声明为原型的声明。您可以像往常一样编写固定参数，然后在“ … '表示可能有其他参数。ISO C 的语法在“ … '。例如，

int
func (const char *a, int b, …)
{
  …
}

定义一个功能func返回一个int并接受两个必需的参数const char *和int 。这些后面跟着任意数量的匿名参数。

可移植性说明：对于某些C编译器，必须声明最后一个必需参数register在函数定义中。此外，此参数的类型必须是自我提升的 ：也就是说，默认提升不能更改其类型。这排除了数组和函数类型，以及float ， char （无论是否签名）和short int （无论是否签名）。这实际上是ISO C 要求。
