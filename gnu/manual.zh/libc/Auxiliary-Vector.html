25.5辅助向量

执行程序时，它将从操作系统接收有关其运行环境的信息。此信息的形式是键-值对表，其中键来自“ AT_ '中的值elf.h 。某些数据是由内核提供的，供libc使用，并且可以通过普通接口获得，例如sysconf 。但是，在逐平台的基础上，可能存在其他任何方式都无法获得的信息。

25.5.1的定义getauxval

函数： unsigned long int getauxval （无符号long int type ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此功能用于查询辅助向量中的条目。的type参数应为“ AT_ '中定义的符号elf.h 。如果找到匹配的条目，则返回该值；否则，返回0。如果找不到该条目，则返回零，然后errno被设定为ENOENT 。

对于某些平台，关键AT_HWCAP是查询运行时可用的任何指令集扩展的最简单方法。在这种情况下，（必要）将有特定于平台的“ HWCAP_屏蔽在一起的值描述了在其上执行程序的CPU的功能。
