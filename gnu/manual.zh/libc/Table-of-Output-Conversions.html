12.12.3输出转换表

下表总结了所有不同的转换的作用：

' %d '，' %i '

打印一个整数作为带符号的十进制数字。有关详细信息，请参见整数转换 。 ' %d '和' %i '是输出的同义词，但与scanf输入（请参阅输入转换表 ）。

' %o '

打印一个整数作为无符号的八进制数。有关详细信息，请参见整数转换 。

' %u '

将整数打印为无符号十进制数字。有关详细信息，请参见整数转换 。

' %x '，' %X '

将整数打印为无符号十六进制数。 ' %x '使用小写字母和' %X '使用大写。有关详细信息，请参见整数转换 。

' %f '

以常规（定点）表示法打印浮点数。有关详细信息，请参见浮点转换 。

' %e '，' %E '

以指数表示法打印浮点数。 ' %e '使用小写字母和' %E '使用大写。有关详细信息，请参见浮点转换 。

' %g '，' %G '

以常规或指数表示法打印浮点数，以更适合其大小的方式为准。 ' %g '使用小写字母和' %G '使用大写。有关详细信息，请参见浮点转换 。

' %a '，' %A '

以十六进制小数形式打印浮点数，以2为底的指数以十进制数字表示。 ' %a '使用小写字母和' %A '使用大写。有关详细信息，请参见浮点转换 。

' %c '

打印单个字符。请参阅其他输出转换 。

' %C '

这是“ %lc支持与Unix标准兼容。

' %s '

打印一个字符串。请参阅其他输出转换 。

' %S '

这是“ %ls支持与Unix标准兼容。

' %p '

打印指针的值。请参阅其他输出转换 。

' %n '

获取到目前为止打印的字符数。请参阅其他输出转换 。请注意，此转换规范永远不会产生任何输出。

' %m '

打印对应于值的字符串errno 。（这是一个GNU扩展。）请参阅其他输出转换 。

' %% '

打印文字' %字符。请参阅其他输出转换 。

如果转换规范的语法无效，则会发生无法预料的事情，因此请不要这样做。如果没有足够的函数参数为模板字符串中的所有转换规范提供值，或者参数的类型不正确，则结果将不可预测。如果您提供的参数多于转换规范，则多余的参数值将被忽略；这有时很有用。
