21.3.1 CPU时间查询

要获取进程的CPU时间，您可以使用clock功能。该功能在头文件中声明time.h 。

在典型用法中，您称clock在您要计时的时间间隔的开始和结束处进行功能，减去值，然后除以CLOCKS_PER_SEC （每秒的时钟滴答数）来获取处理器时间，如下所示：

#include <time.h>

clock_t start, end;
double cpu_time_used;

start = clock();
… /* Do the work. */
end = clock();
cpu_time_used = ((double) (end - start)) / CLOCKS_PER_SEC;

不要将单个CPU时间用作时间量；它不是那样的。如上所示进行减法或直接查询处理器时间。请参阅处理器时间 。

不同的计算机和操作系统在跟踪CPU时间方面有很大的不同。内部处理器时钟的分辨率通常在百分之一秒和百万分之一秒之间。

巨集： int CLOCKS_PER_SEC

此宏的值是每秒时钟滴答数，由clock功能。POSIX要求该值应为一百万，与实际分辨率无关。

资料类型： clock_t

这是返回值的类型clock功能。类型值clock_t是时钟滴答声的数量。

函数： clock_t 时钟 （无效）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此函数返回调用进程的当前CPU时间。如果CPU时间不可用或无法表示， clock返回值(clock_t)(-1) 。
