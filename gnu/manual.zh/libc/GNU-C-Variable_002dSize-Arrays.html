3.2.7.4 GNU C可变大小数组

在GNU C中，您可以替换大多数alloca具有可变大小的数组。这是怎么open2然后会看起来：

int open2 (char *str1, char *str2, int flags, int mode)
{
  char name[strlen (str1) + strlen (str2) + 1];
  stpcpy (stpcpy (name, str1), str2);
  return open (name, flags, mode);
}

但alloca由于以下几个原因，它并不总是等效于可变大小的数组：

  • 大小可变的数组的空间在数组名称范围的末尾释放。分配的空间alloca一直保留到函数结束。
  • 可以使用alloca在循环中，在每次迭代中分配一个额外的块。对于可变大小的数组，这是不可能的。

注意：如果您混合使用alloca和一个函数中的可变大小数组，退出其中声明了可变大小数组的范围将释放分配有alloca在执行该范围期间。
