32.2总体系统选项

POSIX定义了并非所有POSIX系统都支持的某些特定于系统的选项。由于这些选项是在内核中提供的，而不是在库中提供的，因此仅使用GNU C库并不能保证支持这些功能。这取决于您使用的系统。

您可以使用本节中的宏以及功能来测试给定选项的可用性sysconf 。仅当包含以下内容时才定义宏unistd.h 。

对于以下宏，如果宏定义在unistd.h ，则支持该选项。否则，该选项可能不受支持；使用sysconf找出答案。参见Sysconf 。

巨集： int _POSIX_JOB_CONTROL

如果定义了此符号，则表明系统支持作业控制。否则，实现的行为就好像会话中的所有进程都属于一个进程组。请参阅作业控制 。符合POSIX 2001或更高版本的系统将始终定义此符号。

巨集： int _POSIX_SAVED_IDS

如果定义了此符号，则表示系统在执行设置了set-user-ID或set-group-ID位的可执行文件之前，会记住进程的有效用户名和组ID，并明确更改有效用户或允许将组ID恢复为这些值。如果未定义此选项，则如果非特权进程将其有效用户或组ID更改为该进程的真实用户或组ID，则无法再次将其更改。请参阅启用/禁用Setuid 。

对于以下宏，如果宏定义在unistd.h ，则其值指示是否支持该选项。值-1表示“否”，其他任何值表示“是”。如果未定义宏，则可能支持或可能不支持该选项。使用sysconf找出答案。参见Sysconf 。

巨集： int _POSIX2_C_DEV

如果定义了此符号，则表明系统具有POSIX.2 C编译器命令， c89 。GNU C库始终将其定义为1 ，假设您没有C编译器就不会安装它。

巨集： int _POSIX2_FORT_DEV

如果定义了此符号，则表明系统具有POSIX.2 Fortran编译器命令， fort77 。GNU C库从来没有定义这个，因为我们不知道系统有什么。

巨集： int _POSIX2_FORT_RUN

如果定义了此符号，则表明系统具有POSIX.2。 asa解释Fortran托架控制的命令。GNU C库从来没有定义这个，因为我们不知道系统有什么。

巨集： int _POSIX2_LOCALEDEF

如果定义了此符号，则表明系统具有POSIX.2。 localedef命令。GNU C库从来没有定义这个，因为我们不知道系统有什么。

巨集： int _POSIX2_SW_DEV

如果定义了此符号，则表明系统具有POSIX.2命令ar ， make和strip 。GNU C库始终将其定义为1 ，前提是您必须ar和make安装库，这不太可能strip当那些存在时将不存在。
