30.13.1描述用户的数据结构

在头文件中声明用于访问系统用户数据库的功能和数据结构pwd.h 。

数据类型： struct passwd

的passwd数据结构用于保存有关系统用户数据库中条目的信息。它至少具有以下成员：

char *pw_name

用户的登录名。

char *pw_passwd

从历史上看，此字段将保存用户密码的单向哈希。如今，几乎总是单字符' x '，表示哈希在影子数据库中。

uid_t pw_uid

用户标识号。

gid_t pw_gid

用户的默认组ID号。

char *pw_gecos

一个字符串，通常包含用户的真实姓名，并可能包含其他信息，例如电话号码。

char *pw_dir

用户的主目录或初始工作目录。这可能是空指针，在这种情况下，解释取决于系统。

char *pw_shell

用户的默认外壳程序或用户登录时运行的初始程序。这可能是空指针，指示应使用系统默认值。
