35.1 ISO C线程

本节描述了GNU C库ISO C线程的实现。为了更深入地了解此API，强烈建议您阅读ISO / IEC 9899：2011第7.26节，其中最初指定了ISO C线程。所有类型和函数原型都在头文件中声明threads.h 。

• ISO C线程返回值  代表函数返回值的符号常量。
• ISO C线程管理  支持基本线程。
• 拨打一次  单次调用函数和宏。
• ISO C互斥体  互斥的低级机制。
• ISO C条件变量  用于线程同步的高级对象。
• ISO C线程本地存储  支持线程本地存储的功能。
