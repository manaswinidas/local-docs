4.1字符分类

本节说明用于对字符进行分类的库函数。例如， isalpha是测试字母字符的功能。它使用一个参数（要测试的字符），如果字符是字母，则返回非零整数，否则返回零。您可以这样使用它：

if (isalpha (c))
  printf ("The character `%c' is alphabetic.\n", c);

本节中的每个功能都会测试特定字符类别的成员资格；每个都有一个以'开头的名字is '。它们每个都接受一个参数（该参数是要测试的字符），然后返回int视为布尔值。字符参数作为int ，它可能是常数值EOF而不是真实角色。

任何给定字符的属性都可以在语言环境之间变化。有关语言环境的更多信息，请参见语言环境。

这些函数在头文件中声明ctype.h 。

函数： int islower （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是小写字母。该字母不必来自拉丁字母，任何可表示的字母都是有效的。

函数： int isupper （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是大写字母。该字母不必来自拉丁字母，任何可表示的字母都是有效的。

函数： int isalpha （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是字母字符（字母）。如果islower要么isupper对一个角色是真实的，那么isalpha也是如此。

在某些语言环境中，可能还会有其他字符isalpha是正确的-字母既不大写也不小写。但是在标准"C"语言环境，没有这样的附加字符。

函数： int isdigit （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是十进制数字（' 0 '通过' 9 '）。

函数： int isalnum （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是字母数字字符（字母或数字）；换句话说，如果有isalpha要么isdigit对一个角色是真实的，那么isalnum也是如此。

函数： int isxdigit （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是一个十六进制数字。十六进制数字包括普通的十进制数字' 0 '通过' 9 '和字母' A '通过' F '和' a '通过' f '。

函数： int ispunct （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是标点符号。这意味着不是字母数字或空格字符的任何打印字符。

函数： int isspace （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是空白字符。在标准"C"语言环境isspace仅对标准空白字符返回true：

' '

空间

'\f'

换页

'\n'

新队

'\r'

回车

'\t'

水平标签

'\v'

垂直标签

函数： int isblank （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是一个空白字符；即空格或制表符。该函数最初是GNU扩展，但已添加到ISO C99 。

函数： int isgraph （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是图形字符；也就是说，具有与之相关联的字形的字符。空格字符不视为图形字符。

函数： int isprint （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是印刷字符。打印字符包括所有图形字符，加上空格（“ '）字符。

函数： int iscntrl （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是控制字符（即不是打印字符的字符）。

函数： int isascii （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果返回true c是7位unsigned char符合美国/英国ASCII字符集的值。此功能是BSD扩展，也是SVID扩展。
