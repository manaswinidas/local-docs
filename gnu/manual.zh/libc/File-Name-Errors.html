11.2.3文件名错误

接受文件名参数的函数通常会检测到这些errno与文件名语法有关的错误条件或查找命名文件的麻烦。在本手册中，这些错误通常称为文件名错误 。

EACCES

该进程没有搜索文件名的目录组件的权限。

ENAMETOOLONG

当文件名的总长度大于或等于PATH_MAX ，或者单个文件名部分的长度大于NAME_MAX 。请参阅文件限制 。

在GNU / Hurd系统上，总文件名长度没有限制，但是某些文件系统可能会限制组件的长度。

ENOENT

当在文件名中被引用为目录组件的文件不存在，或者该组件是目标文件不存在的符号链接时，将报告此错误。请参阅符号链接 。

ENOTDIR

存在一个在文件名中被引用为目录组件的文件，但它不是目录。

ELOOP

尝试查找文件名时，解析了太多符号链接。系统对查找单个文件名时可以解析的符号链接数有任意限制，这是检测循环的原始方法。请参阅符号链接 。
