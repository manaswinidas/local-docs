25.3.6组合多个Argp解析器

的children在一个领域struct argp允许将其他argp解析器与引用一个argp解析器组合使用，以解析单个参数集。该字段应指向的向量struct argp_child ，该终止符由条目中值为零的条目终止argp领域。

当组合解析器之间出现冲突时，例如当两个解析器使用相同的名称指定选项时，解析器冲突将通过父argp解析器或子列表中较早的argp解析器解决。

数据类型： struct argp_child

子程序argp解析器列表中指向的条目children在一个领域struct argp 。字段如下：

const struct argp *argp

子argp解析器，或者为零到列表末尾。

int flags

这个孩子的标志。

const char *header

如果非零，这是一个可选的标题，将在子选项之前的帮助输出中打印。副作用是，非零值会强制将子选项组合在一起。要实现此效果而不实际打印标题字符串，请使用值"" 。与在选项条目中指定的标题字符串一样，最后一个字符的常规值是' : '。请参阅Argp选项向量 。

int group

这是相对于父argp解析器中其他“合并”选项对子选项进行分组的地方。值与group在struct argp_option 。请参阅Argp选项向量 。在特定的组级别，所有子组均遵循父选项。如果此字段和header为零，则子选项未组合在一起，它们在父选项组级别与父选项合并。
