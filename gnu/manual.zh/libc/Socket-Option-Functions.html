16.12.1套接字选项功能

这是检查和修改套接字选项的功能。它们在sys/socket.h 。

函数： int getsockopt （int socket ，int level ，int optname ，无效* optval ，socklen_t * optlen-ptr ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的getsockopt函数获取有关期权价值的信息optname在水平level用于插座socket 。

选项值存储在缓冲区中optval指着。通话之前，您应提供*optlen-ptr该缓冲区的大小；返回时，它包含实际存储在缓冲区中的信息字节数。

大多数选项解释optval作为一个缓冲区int值。

的实际返回值getsockopt是0成功与-1失败。下列errno错误条件已定义：

EBADF

的socket参数不是有效的文件描述符。

ENOTSOCK

描述符socket不是套接字。

ENOPROTOOPT

的optname对于给定没有意义level 。

函数： int setsockopt （int socket ，int level ，int optname ，const void * optval ，socklen_t optlen ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此功能用于设置套接字选项optname在水平level用于插座socket 。选项的值在缓冲区中传递optval大小optlen 。
