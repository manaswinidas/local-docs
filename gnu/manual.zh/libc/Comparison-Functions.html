9.1定义比较功能

为了使用排序的数组库函数，您必须描述如何比较数组的元素。

为此，您提供了一个比较函数来比较数组的两个元素。库将调用此函数，将指向要比较的两个数组元素的指针作为参数传递。您的比较函数应以以下方式返回值strcmp （请参阅字符串/数组比较 ）：如果第一个参数“小于”第二个，则为负；如果“等于”，则为零；如果第一个参数为“更大”，则为正。

这是一个比较函数的示例，可与类型为数字的数组一起使用double ：

int
compare_doubles (const void *a, const void *b)
{
  const double *da = (const double *) a;
  const double *db = (const double *) b;

  return (*da > *db) - (*da < *db);
}

头文件stdlib.h为比较函数的数据类型定义一个名称。此类型是GNU扩展。
