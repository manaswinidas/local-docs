16.12.2套接字级选项

常数： int SOL_SOCKET

将此常数用作level争论getsockopt要么setsockopt操作本节中描述的套接字级别的选项。

这是套接字级选项名称的表；所有这些都在头文件中定义sys/socket.h 。

SO_DEBUG

此选项在基础协议模块中切换调试信息的记录。值具有类型int ;非零值表示“是”。

SO_REUSEADDR

此选项控制是否bind （请参阅设置地址 ）应允许对此套接字重用本地地址。如果启用此选项，则实际上可以有两个具有相同Internet端口号的套接字。但是系统不允许您以混淆互联网的方式使用两个同名的套接字。使用此选项的原因是某些更高级别的Internet协议（包括FTP）要求您保持重复使用相同的端口号。

值具有类型int ;非零值表示“是”。

SO_KEEPALIVE

此选项控制基础协议是否应定期在连接的套接字上传输消息。如果对等方未能响应这些消息，则认为连接已断开。值具有类型int ;非零值表示“是”。

SO_DONTROUTE

此选项控制传出邮件是否绕过常规邮件路由功能。如果设置，则消息直接发送到网络接口。值具有类型int ;非零值表示“是”。

SO_LINGER

此选项指定在关闭时，保证可靠传递的类型的套接字仍然有未传输的消息时应发生的情况。请参见关闭套接字 。值具有类型struct linger 。

数据类型： struct linger

此结构类型具有以下成员：

int l_onoff

该字段被解释为布尔值。如果不是零， close块，直到数据传输或超时时间已过。

int l_linger

以秒为单位指定超时时间。

SO_BROADCAST

此选项控制是否可以从套接字广播数据报。值具有类型int ;非零值表示“是”。

SO_OOBINLINE

如果设置了此选项，则在套接字上接收的带外数据将放置在普通输入队列中。这允许使用read要么recv没有指定MSG_OOB旗。请参阅带外数据 。值具有类型int ;非零值表示“是”。

SO_SNDBUF

此选项获取或设置输出缓冲区的大小。该值是一个size_t ，以字节为单位的大小。

SO_RCVBUF

此选项获取或设置输入缓冲区的大小。该值是一个size_t ，以字节为单位的大小。

SO_STYLE
SO_TYPE

该选项可以与getsockopt只要。它用于获取套接字的通信样式。 SO_TYPE是历史名称，并且SO_STYLE是GNU中的首选名称。值具有类型int它的价值表示一种沟通方式；请参阅交流风格 。

SO_ERROR

该选项可以与getsockopt只要。用于重置套接字的错误状态。该值是int ，代表先前的错误状态。
