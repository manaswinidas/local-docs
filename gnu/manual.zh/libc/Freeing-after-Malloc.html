3.2.3.3释放分配给malloc

当您不再需要获得的障碍物时malloc ，使用功能free使该块可用于再次分配。该函数的原型在stdlib.h 。

功能： 无空隙 （void *的ptr ）

初步： MT安全| AS不安全锁| AC不安全锁fd mem |请参阅POSIX安全概念 。

的free函数取消分配由指向的内存块ptr 。

释放一个块会改变该块的内容。释放数据后，不要期望在该数据块中找到任何数据（例如，指向一个块链中的下一个块的指针）。 释放任何所需的内容后，再释放它！这是释放链中所有块及其指向的字符串的正确方法的示例：

struct chain
  {
    struct chain *next;
    char *name;
  }

void
free_chain (struct chain *chain)
{
  while (chain != 0)
    {
      struct chain *next = chain->next;
      free (chain->name);
      free (chain);
      chain = next;
    }
}

偶尔， free实际上可以将内存返回给操作系统，并使进程更小。通常，它所能做的就是允许以后调用malloc重用空间。同时，该空间作为程序内部使用的空闲列表的一部分保留在程序中malloc 。

在程序结束时释放块是没有意义的，因为当进程终止时，程序的所有空间都将返还给系统。
