19.6特殊功能

这些是一些更奇特的数学函数，有时很有用。当前，它们仅具有实值版本。

功能： 双 erf （双x ）
功能： 浮动 erff （float x ）
功能： 长双 erfl （长双x ）
函数： _FloatN erffN （_Float Nx ）
函数： _FloatNx erffNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

erf返回的错误函数x 。误差函数定义为

erf (x) = 2/sqrt(pi) * integral from 0 to x of exp(-t^2) dt
功能： 双 erfc （双x ）
功能： float erfcf （float x ）
功能： 长双 erfcl （长双x ）
函数： _FloatN erfcfN （_Float Nx ）
函数： _FloatNx erfcfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

erfc退货1.0 - erf(x) ，但以避免发生舍入误差的方式进行计算x大。

功能： 双 lgamma （双x ）
功能： 浮法 lgammaf （float x ）
功能： 长双 l （长双double x ）
函数： _FloatN lgammafN （_Float Nx ）
函数： _FloatNx lgammafNx （_Float N X x ）

初步： MT-不安全种族：signgam | AS不安全|交流安全请参阅POSIX安全概念 。

lgamma返回的伽玛函数绝对值的自然对数x 。伽玛函数定义为

gamma (x) = integral from 0 to &infin; of t^(x-1) e^-t dt

伽玛函数的符号存储在全局变量中signgam ，在中声明math.h 。它是1如果中间结果是正数或零，或者-1如果是负面的。

要计算真实的伽玛函数，您可以使用tgamma函数，也可以如下计算值：

lgam = lgamma(x);
gam  = signgam*exp(lgam);

伽玛函数在非正整数处具有奇点。 lgamma如果以奇异性进行评估，将引发零除异常。

函数： double lgamma_r （double x ，int * signp ）
函数： float lgammaf_r （float x ，int * signp ）
函数： long double lgammal_r （long double x ，int * signp ）
函数： _FloatN lgammafN_r （_Float Nx ，int * signp ）
函数： _FloatNx lgammafNx_r （_Float N X x ，int * signp ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

lgamma_r就像lgamma ，但是它将中间结果的符号存储在所指向的变量中signp而不是在signgam全球。这意味着它是可重入的。

的lgammafN_r和lgammafNx_r函数是GNU扩展。

功能： 双 伽玛 （双x ）
功能： 浮 伽玛 （float x ）
功能： 长双 gammal （长双x ）

初步： MT-不安全种族：signgam | AS不安全|交流安全请参阅POSIX安全概念 。

存在这些功能是出于兼容性原因。它们相当于lgamma等等最好用lgamma因为一个名字更好地反映了实际的计算，而且lgamma已在ISO C99 中标准化gamma不是。

功能： 双 tgamma （双x ）
功能： 浮法 tgammaf （float x ）
功能： 长双 tgammal （长双x ）
函数： _FloatN tgammafN （_Float Nx ）
函数： _FloatNx tgammafNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

tgamma将伽玛函数应用于x 。伽玛函数定义为

gamma (x) = integral from 0 to &infin; of t^(x-1) e^-t dt

此功能在ISO C99 引入。的_FloatN和_FloatNx在ISO / IEC TS 18661-3 中引入了各种变体。

功能： 双 j0 （双x ）
函数： float j0f （浮点x ）
