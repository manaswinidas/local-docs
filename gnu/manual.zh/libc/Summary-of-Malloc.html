3.2.3.11总结malloc相关功能

这是与之配合使用的功能的摘要malloc ：

void *malloc (size_t size)

分配一个size个字节。请参阅基本分配 。

void free (void *addr)

释放以前由分配的块malloc 。请参见在Malloc之后释放 。

void *realloc (void *addr, size_t size)

制作一个先前分配的块malloc更大或更小，可以通过将其复制到新位置来实现。请参阅更改块大小 。

void *reallocarray (void *ptr, size_t nmemb, size_t size)

更改先前分配给的块的大小malloc至nmemb * size字节与realloc 。请参阅更改块大小 。

void *calloc (size_t count, size_t eltsize)

分配一个count * eltsize字节使用malloc ，并将其内容设置为零。请参阅分配净空 。

void *valloc (size_t size)

分配一个size字节，从页面边界开始。请参阅对齐的内存块 。

void *aligned_alloc (size_t size, size_t alignment)

分配一个size字节，从一个整数倍的地址开始alignment 。请参阅对齐的内存块 。

int posix_memalign (void **memptr, size_t alignment, size_t size)

分配一个size字节，从一个整数倍的地址开始alignment 。请参阅对齐的内存块 。

void *memalign (size_t size, size_t boundary)

分配一个size字节，从一个整数倍的地址开始boundary 。请参阅对齐的内存块 。

int mallopt (int param, int value)

调整可调参数。请参见Malloc可调参数 。

int mcheck (void (*abortfn) (void))

告诉malloc对动态分配的内存执行偶尔的一致性检查，并调用abortfn发现不一致时。请参阅堆一致性检查 。

void *(*__malloc_hook) (size_t size, const void *caller)

指向函数的指针malloc在调用时使用。

void *(*__realloc_hook) (void *ptr, size_t size, const void *caller)

指向函数的指针realloc在调用时使用。

void (*__free_hook) (void *ptr, const void *caller)

指向函数的指针free在调用时使用。

void (*__memalign_hook) (size_t size, size_t alignment, const void *caller)

指向函数的指针aligned_alloc ， memalign ， posix_memalign和valloc每当调用它们时使用。

struct mallinfo mallinfo (void)

返回有关当前动态内存使用情况的信息。请参阅Malloc的统计信息 。
