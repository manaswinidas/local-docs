3.2.7可变大小的自动存储

功能alloca支持一种半动态分配，其中块是动态分配的，但会自动释放。

用以下方式分配块alloca是一个明确的动作；您可以根据需要分配任意数量的块，并在运行时计算大小。但是，当您退出该函数时，所有块都将被释放alloca被调用，就像它们是在该函数中声明的自动变量一样。无法显式释放空间。

的原型alloca在stdlib.h 。此功能是BSD扩展。

函数： void * alloca （size_t size ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的返回值alloca是一个块的地址size内存字节，在调用函数的堆栈帧中分配。

不使用alloca在函数调用的参数中，您将获得无法预测的结果，因为alloca将出现在函数参数空间中间的堆栈中。一个避免的例子是foo (x, alloca (4), y) 。
