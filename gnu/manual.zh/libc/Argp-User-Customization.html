25.3.12 Argp用户自定义

argp'的格式--help通过设置用户的输出，程序的用户可以在某种程度上控制输出ARGP_HELP_FMT环境变量，以逗号分隔的令牌列表。空格被忽略：

' dup-args '
' no-dup-args '

这些可以打开或关闭重复参数模式 。在重复参数模式下，如果接受参数的选项具有多个名称，则会为每个名称显示该参数。否则，仅针对第一个长选项显示。随后打印便笺，以便用户知道它也适用于其他名称。默认值为“ no-dup-args ”，虽然不太一致，但更漂亮。

' dup-args-note '
' no-dup-args-note '

这些将启用或禁用通知用户抑制选项参数重复的注释。默认值为“ dup-args-note '。

' short-opt-col=n '

这将打印列中的第一个短选项n 。预设值为2。

' long-opt-col=n '

这将打印列中的第一个long选项n 。预设值为6。

' doc-opt-col=n '

这将在列中打印“文档选项”（请参阅Argp选项标志 ） n 。预设值为2。

' opt-doc-col=n '

这将从列开始打印选项的文档n 。默认值为29。

' header-col=n '

这将缩进记录组选项列的组标题n 。预设值为1。

' usage-indent=n '

这将缩进' Usage: '讯息至专栏n 。预设值为12。

' rmargin=n '

这将在列或列之前自动换行帮助输出n 。默认值为79。
