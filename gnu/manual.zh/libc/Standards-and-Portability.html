1.2标准和可移植性

本节讨论GNU C库所基于的各种标准和其他来源。这些来源包括ISO C 和POSIX标准，以及System V和Berkeley Unix实现。

本手册的主要重点是告诉您如何有效利用GNU C库功能。但是，如果您担心使程序与这些标准兼容，或者可以移植到GNU以外的其他操作系统，则可能会影响您使用该库的方式。本节概述了这些标准，以便在手册的其他部分中提及它们时了解它们。

有关库提供的功能和其他符号的字母列表，请参见库摘要 。该列表还说明了每个功能或符号来自哪个标准。

• ISO C  C编程语言的国际标准。
• POSIX  操作系统的ISO / IEC 9945（aka IEEE 1003）标准。
• 伯克利Unix  BSD和SunOS。
• SVID  系统V接口说明。
• XPG  X / Open可移植性指南。
