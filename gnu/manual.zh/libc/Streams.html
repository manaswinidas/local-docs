12.1流

由于历史原因，表示流的C数据结构的类型称为FILE而不是“流”。由于大多数库函数都处理类型的对象FILE * ，有时术语文件指针也用于表示“流”。不幸的是，这导致许多有关C的书籍对术语的混淆。但是，本手册仅在技术意义上谨慎使用术语“文件”和“流”。

资料类型： FILE

这是用于表示流对象的数据类型。一种FILE对象包含有关与关联文件的连接的所有内部状态信息，包括文件位置指示符和缓冲信息。每个流还具有错误和文件结束状态指示符，可以使用ferror和feof功能;请参阅EOF和错误 。

FILE对象由输入/输出库函数在内部分配和管理。不要尝试创建自己的类型的对象FILE ;让图书馆去做。您的程序应仅处理指向这些对象的指针（即， FILE *值）而不是对象本身。
