3.2.3.10使用以下命令分配内存的统计信息malloc

您可以通过调用以下命令获取有关动态内存分配的信息mallinfo功能。此函数及其关联的数据类型在中声明malloc.h ;它们是标准SVID / XPG版本的扩展。

数据类型： struct mallinfo

此结构类型用于返回有关动态内存分配器的信息。它包含以下成员：

int arena

这是分配给的内存总大小sbrk通过malloc ，以字节为单位。

int ordblks

这是未使用的块数。（内存分配器在内部从操作系统中获取内存块，然后将其分块以满足个人需求。 malloc要求;参见GNU分配器 。）

int smblks

该字段未使用。

int hblks

这是分配给的块总数mmap 。

int hblkhd

这是分配给的内存总大小mmap ，以字节为单位。

int usmblks

该字段未使用，始终为0。

int fsmblks

该字段未使用。

int uordblks

这是由分发的块占用的内存总大小malloc 。

int fordblks

这是空闲（未使用）块所占用的内存总大小。

int keepcost

这是通常与堆的末端（即虚拟地址空间的数据段的高端）接壤的最顶部可释放块的大小。

功能： struct mallinfo mallinfo （无效）

初步： MT-Unsafe初始化const：mallopt | AS不安全初始化锁| AC不安全初始化锁|请参阅POSIX安全概念 。

该函数以类型结构返回有关当前动态内存使用情况的信息struct mallinfo 。
