4.2案例转换

本节说明用于执行转换的库函数，例如字符的大小写映射。例如， toupper如果可能，将任何字符转换为大写。如果字符无法转换， toupper保持不变。

这些函数采用一个类型的参数int ，即要转换的字符，然后将转换后的字符作为int 。如果该转换不适用于给定的参数，则该参数将按原样返回。

兼容性说明：在ISO C 方言中，当参数不适合转换时，这些函数可能会失败，而不是返回不变的参数。因此，为了可移植性，您可能需要编写islower(c) ? toupper(c) : c而不只是toupper(c) 。

这些函数在头文件中声明ctype.h 。

函数： int tolower （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果c是大写字母， tolower返回相应的小写字母。如果c不是大写字母， c返回不变。

函数： int toupper （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果c是小写字母， toupper返回相应的大写字母。除此以外c返回不变。

函数： int toascii （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

该函数转换c到7位unsigned char通过清除高阶位，使该值适合美国/英国ASCII字符集。此功能是BSD扩展，也是SVID扩展。

函数： int _tolower （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这等同于tolower ，并且为了与SVID兼容而提供。参见SVID 。

函数： int _toupper （int c ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这等同于toupper ，并且为了与SVID兼容而提供。
