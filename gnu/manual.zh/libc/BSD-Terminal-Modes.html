17.5 BSD终端模式

通常的方式来获得和设定终端模式是在所描述的功能的终端模式 。但是，在某些系统上，您可以使用本节中的BSD派生功能执行某些相同的操作。在许多系统上，这些功能不存在。即使使用GNU C库，这些功能也因errno = ENOSYS具有许多内核，包括Linux。

本节中使用的符号在sgtty.h 。

数据类型： struct sgttyb

此结构是输入或输出参数列表，用于gtty和stty 。

char sg_ispeed

输入线速度

char sg_ospeed

输出线速度

char sg_erase

擦除字符

char sg_kill

杀死角色

int sg_flags

各种标志

函数： int gtty （int filedes ，struct sgttyb * attributes ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此函数获取终端的属性。

gtty套* attributes描述使用文件描述符打开的终端的终端属性filedes 。

函数： int stty （int filedes ，const struct sgttyb * attributes ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此功能设置终端的属性。

stty设置使用文件描述符打开的终端的终端属性filedes * attributes 。
