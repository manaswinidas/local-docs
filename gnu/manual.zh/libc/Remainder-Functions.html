20.8.4剩余功能

本节中的函数计算两个浮点数除法的余数。每个都有一点不同；选择一个适合您的问题。

功能： 双 fmod （双numerator ，双denominator ）
函数： float fmodf （float numerator ，浮动denominator ）
功能： long double fmodl （long double numerator长双denominator ）
函数： _FloatN fmodfN （_Float Nnumerator ，_Float Ndenominator ）
函数： _FloatNx fmodfNx （_Float N X numerator ，_Float N X denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数通过除法计算余数numerator通过denominator 。具体来说，返回值为numerator - n * denominator ，在哪里n是...的商numerator除以denominator ，四舍五入为整数。从而， fmod (6.5, 2.3) 返回1.9 ，这是6.5减去4.6 。

结果的符号与numerator并且其幅度小于denominator 。

如果denominator是零fmod发出域错误信号。

功能： 双 余数 （双numerator ，双denominator ）
功能： 浮 余数 （float numerator ，浮动denominator ）
功能： 长双 残留 （长双numerator长双denominator ）
功能：_FloatN remainderfN（_Float Nnumerator ，_Float Ndenominator ）
功能：_FloatNx remainderfNx（_Float N X numerator ，_Float N X denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些功能就像fmod除了它们围绕内部商n到最接近的整数，而不是从零到整数。例如， remainder (6.5, 2.3)退货-0.4 ，这是6.5减去6.9 。

结果的绝对值小于或等于结果的绝对值的一半denominator 。和...之间的不同fmod (numerator, denominator)和remainder (numerator, denominator)总是denominator ，减去denominator ，或零。

如果denominator是零remainder发出域错误信号。

功能： 双 dr （双double numerator ，双denominator ）
功能： 浮子 dremf （float numerator ，浮动denominator ）
功能： 长双 dreml （长双numerator长双denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

该功能是remainder 。
