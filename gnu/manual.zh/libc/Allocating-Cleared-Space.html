3.2.3.5分配净空

功能calloc分配内存并将其清除为零。声明在stdlib.h 。

函数： void * calloc （size_t count ，size_t eltsize ）

初步： MT安全| AS不安全锁| AC不安全锁fd mem |请参阅POSIX安全概念 。

该函数分配一个足以包含向量的块count元素，每个大小eltsize 。其内容在清除之前为零calloc返回。

您可以定义calloc如下：

void *
calloc (size_t count, size_t eltsize)
{
  size_t size = count * eltsize;
  void *value = malloc (size);
  if (value != 0)
    memset (value, 0, size);
  return value;
}

但总的来说，不能保证calloc来电malloc内部。因此，如果应用程序提供了自己的malloc / realloc / free在C库之外，它应该始终定义calloc也一样
