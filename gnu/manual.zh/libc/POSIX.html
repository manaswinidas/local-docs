1.2.2 POSIX（便携式操作系统接口）

GNU C库还与ISO POSIX系列标准兼容，该系列更正式地称为计算机环境的便携式操作系统接口 （ISO / IEC 9945）。它们还以ANSI / IEEE Std 1003的形式发布。POSIX主要源自Unix操作系统的各种版本。

POSIX标准指定的库功能是ISO C 所要求的功能的超集； POSIX为ISO C 函数指定附加功能，并指定新的附加功能。通常，POSIX标准定义的其他要求和功能旨在为特定种类的操作系统环境提供低级支持，而不是可以在许多不同的操作系统环境中运行的通用编程语言支持。

GNU C库实现了ISO / IEC 9945-1：1996（POSIX系统应用程序接口 ，通常称为POSIX.1）中指定的所有功能。此标准指定的ISO C 功能的主要扩展包括文件系统接口原语（请参阅文件系统接口 ），特定于设备的终端控制功能（请参见低级终端接口 ）和过程控制功能（请参见过程 ）。

GNU C库中还实现了ISO / IEC 9945-2：1993，POSIX Shell and Utilities标准 （POSIX.2）中的某些功能。这些工具包括用于处理正则表达式的实用程序和其他模式匹配工具（请参见模式匹配 ）。
