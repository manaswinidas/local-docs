17.4.1终端模式数据类型

终端属性的整个集合存储在type结构中struct termios 。该结构与功能一起使用tcgetattr和tcsetattr读取并设置属性。

数据类型： struct termios

一种struct termios记录终端的所有I / O属性。该结构至少包括以下成员：

tcflag_t c_iflag

位掩码，指定输入模式的标志；请参阅输入模式 。

tcflag_t c_oflag

位掩码，指定输出模式的标志；请参阅输出模式 。

tcflag_t c_cflag

位掩码，用于指定控制模式的标志；请参阅控制模式 。

tcflag_t c_lflag

位掩码，指定本地模式的标志；请参阅本地模式 。

cc_t c_cc[NCCS]

一个数组，用于指定与各种控制功能关联的字符；参见特殊字符 。

的struct termios结构还包含对输入和输出传输速度进行编码的成员，但未指定表示形式。有关如何检查和存储速度值的信息，请参见线速度 。

以下各节介绍了struct termios结构体。

数据类型： tcflag_t

这是一种无符号整数类型，用于表示终端标志的各种位掩码。

数据类型： cc_t

这是一种无符号整数类型，用于表示与各种终端控制功能关联的字符。

巨集： int NCCS

此宏的值是c_cc数组。
