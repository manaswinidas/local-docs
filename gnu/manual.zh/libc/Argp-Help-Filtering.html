25.3.8自定义Argp帮助输出

的help_filter在一个领域struct argp是指向在显示帮助消息之前对其进行过滤的函数的指针。它们具有如下功能签名：

char *help-filter (int key, const char *text, void *input)

哪里key是选项的键，在这种情况下text是该选项的帮助文本。请参阅Argp选项向量 。或者，名称以'开头的特殊键之一ARGP_KEY_HELP_ '，可能会用到，说明其他哪些帮助文本text将包含。请参阅Argp帮助过滤器键 。

该函数应返回text如果它保持原样，或者使用malloc 。可以通过argp释放它或将其释放为零，这将不显示任何内容。的价值text在完成任何翻译后会提供，因此如果任何替换文本需要翻译，则将通过过滤器功能来完成。 input是提供给的输入argp_parse或为零，如果argp_help是由用户直接调用的。
