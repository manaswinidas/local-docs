这些函数提取了x并将其作为浮点值返回。如果FLT_RADIX是两个logb等于floor (log2 (x)) ，只是速度可能更快。

如果x被去规范化， logb返回指数x如果将其标准化将不会有。如果x是无穷大（正数或负数）， logb返回∞ 。如果x是零logb返回∞ 。它不发信号。

功能： int ilogb （双 精度 x ）
功能： int ilogbf （float x ）
功能： int ilogbl （长双 精度 x ）
函数： int ilogbfN （_Float Nx ）
函数： int ilogbfNx （_Float N X x ）
函数： long int llogb （双 精度 x ）
函数： long int llogbf （浮点型x ）
函数： long int llogbl （long double x ）
函数： long int llogbfN （_Float Nx ）
函数： long int llogbfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些功能相当于相应的logb函数，除了它们返回带符号的整数值。的ilogb ， ilogbf和ilogbl功能来自ISO C99；的llogb ， llogbf ， llogbl功能来自TS 18661-1：2014；的ilogbfN ， ilogbfNx ， llogbfN和llogbfNx功能来自TS 18661-3：2015。

由于整数不能表示无穷大和NaN， ilogb而是返回一个整数，该整数不能是普通浮点数的指数。 math.h定义常量，以便您可以进行检查。

巨集： int FP_ILOGB0

ilogb如果它的参数是0 。数值为INT_MIN要么-INT_MAX 。

此宏在ISO C99 定义。

宏： long int FP_LLOGB0

llogb如果它的参数是0 。数值为LONG_MIN要么-LONG_MAX 。

TS 18661-1：2014中定义了此宏。

巨集： int FP_ILOGBNAN

ilogb如果它的参数是NaN 。数值为INT_MIN要么INT_MAX 。

此宏在ISO C99 定义。

宏： long int FP_LLOGBNAN

llogb如果它的参数是NaN 。数值为LONG_MIN要么LONG_MAX 。

TS 18661-1：2014中定义了此宏。

这些值是系统特定的。他们甚至可能是相同的。测试结果的正确方法ilogb如下：

i = ilogb (f);
if (i == FP_ILOGB0 || i == FP_ILOGBNAN)
  {
    if (isnan (f))
      {
        /* Handle NaN.  */
      }
    else if (f  == 0.0)
      {
        /* Handle 0.0.  */
      }
    else
      {
        /* Some other value with large exponent,
           perhaps +Inf.  */
      }
  }
