19.3反三角函数

这些是通常的反正弦函数，反余弦函数和反正切函数，分别是正弦函数，余弦函数和正切函数的逆函数。

功能： 双重 阿辛 （双重x ）
函数： float asinf （float x ）
功能： 长双 asinl（长双x ）
函数： _FloatN asinfN （_Float Nx ）
函数： _FloatNx asinfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数计算x —即正弦值为x 。该值以弧度为单位。从数学上讲，有无数这样的值。实际返回的是介于-pi/2和pi/2 （包括的）。

反正弦函数仅在域上进行数学定义-1至1 。如果x在域之外， asin发出域错误信号。

功能： 双 ACOS （双x ）
功能： 浮子 acosf （float x ）
功能： 长双 阿斯科尔 （长双x ）
函数： _FloatN acosfN （_Float Nx ）
函数： _FloatNx acosfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数计算出的反余弦x —即余弦为x 。该值以弧度为单位。从数学上讲，有无数这样的值。实际返回的是介于0和pi （包括的）。

反余弦函数仅在域上进行数学定义-1至1 。如果x在域之外， acos发出域错误信号。

功能： 双 阿丹 （双x ）
功能： 浮法 atanf （浮法x ）
功能： 长双 金属环 （长双x ）
函数： _FloatN atanfN （_Float Nx ）
函数： _FloatNx atanfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数可计算的反正切x —即，其切线为的值x 。该值以弧度为单位。从数学上讲，有无数这样的值。实际返回的是介于-pi/2和pi/2 （包括的）。

功能： double atan2 （double y ，双x ）
函数： float atan2f （float y ，浮动x ）
功能： 长双 atan2l （长双y长双x ）
函数： _FloatN atan2fN （_Float Ny ，_Float Nx ）
函数： _FloatNx atan2fNx （_Float N X y ，_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此函数计算的反正切y / x ，但是两个参数的符号都用于确定结果的象限，并且x允许为零。返回值以弧度给出，范围为-pi至pi ， 包括的。

如果x和y是平面上一点的坐标， atan2返回从原点到该点的线与x轴之间的正负角。从而， atan2用于将笛卡尔坐标转换为极坐标。（要计算径向坐标，请使用hypot ;参见指数和对数 。）

如果两者x和y为零atan2返回零。

ISO C99 定义了反向触发函数的复杂版本。

功能： 复式双 棺 （复式双z ）
功能： 复杂浮点数 casinf （复杂浮点数z ）
功能： 复数长双 casinl （复数长双z ）
功能： 复杂_FloatN casinfN （复杂_Float Nz ）
功能： 复杂_FloatNx casinfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数计算出z —即正弦值为z 。返回的值以弧度为单位。

与实值函数不同， casin为的所有值定义z 。

功能： 复方双 可可 （复方z ）
功能： 复杂浮点 cacosf （复杂浮点z ）
功能： 复长双 可可 （复长双z ）
函数： complex _FloatN cacosfN （complex _Float Nz ）
函数： 复杂_FloatNx cacosfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数可以计算出z —即余弦为z 。返回的值以弧度为单位。

与实值函数不同， cacos为的所有值定义z 。

功能： 复式双 卡坦 （complex double z ）
功能： 复数浮 线 （复数浮线z ）
功能： 复长双 连环 （复长双z ）
功能： 复杂_FloatN catanfN （复杂_Float Nz ）
功能： 复杂_FloatNx catanfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数可计算的复杂反正切z —即，其切线为的值z 。该值以弧度为单位。
