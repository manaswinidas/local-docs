19.2三角函数

这些是熟悉的sin ， cos和tan功能。所有这些函数的参数都以弧度为单位；回想一下pi弧度等于180度。

数学库通常定义M_PI到一个double pi的近似值。如果要求严格遵守ISO和/或POSIX，则不定义此常数，但是您可以自己轻松定义它：

#define M_PI 3.14159265358979323846264338327

您也可以使用以下表达式计算pi的值acos (-1.0) 。

功能： 双 罪 （双 罪 x ）
函数： float sinf （float x ）
功能： 长双 尖齿 （长双 尖 x ）
函数： _FloatN sinfN （_Float Nx ）
函数： _FloatNx sinfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回正弦值x ，在哪里x以弧度表示。返回值在范围内-1至1 。

功能： 双 cos （双x ）
函数： float cosf （float x ）
功能： 长双 cosl （长双x ）
函数： _FloatN cosfN （_Float Nx ）
函数： _FloatNx cosfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的余弦值x ，在哪里x以弧度表示。返回值在范围内-1至1 。

功能： 双 棕褐色 （双x ）
功能： 浮法 油棕 （float x ）
功能： 长双 tanl（长双x ）
函数： _FloatN tanfN （_Float Nx ）
函数： _FloatNx tanfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回正切x ，在哪里x以弧度表示。

在数学上，切线函数在pi / 2的奇数倍处具有奇异性。如果论点x太接近这些奇异点之一， tan将发出信号溢出。

在许多应用中sin和cos如果使用，则同时需要相同角度的正弦和余弦。同时计算它们的效率更高，因此该库提供了执行此操作的功能。

功能： 虚空 sincos （双x ，双* sinx ，双* cosx ）
功能： 虚空 sincosf （浮动x ，浮动* sinx ，浮动* cosx ）
功能： 无效的 窦房结 （长双x长双* sinx长双* cosx ）
函数： _FloatN sincosfN （_Float Nx ，_Float N * sinx ，_Float N * cosx ）
函数： _FloatNx sincosfNx （_Float N X x ，_Float N X * sinx ，_Float N X * cosx ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回正弦值x在*sinx和的余弦x在*cosx ，在哪里x以弧度表示。这两个值*sinx和*cosx ，在的范围内-1至1 。

所有这些功能，包括_FloatN和_FloatNx变体是GNU扩展。可移植程序应做好准备以应对它们的缺失。

ISO C99 定义了对复数起作用的触发函数的变体。GNU C库提供了这些功能，但是仅当您的编译器支持该标准定义的新复杂类型时，它们才有用。（在撰写本文时，GCC支持复数，但实现中存在错误。）

功能： 复数双倍 csin （复数双倍z ）
功能： 复数浮点数 csinf （复数浮点数z ）
功能： 复数长双倍 csinl （复数长双倍z ）
函数： complex _FloatN csinfN （complex _Float Nz ）
函数： complex _FloatNx csinfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的复杂正弦值z 。复正弦的数学定义是

sin（z）= 1 /（2 * i）*（exp（z * i）-exp（-z * i）） 。

功能： 复杂双 CCOS （复杂双z ）
功能： 复杂浮点 ccosf （复杂浮点z ）
功能： 复数长双 ccosl （复数长双z ）
函数： complex _FloatN ccosfN （complex _Float Nz ）
函数： complex _FloatNx ccosfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的余弦值z 。复余弦的数学定义是

cos（z）= 1/2 *（exp（z * i）+ exp（-z * i））

功能： 复杂双 ctan （复杂双z ）
功能： 复杂浮点 ctanf （复杂浮点z ）
功能： 复数长双 标准 （复数长双标准） z ）
函数： complex _FloatN ctanfN （complex _Float Nz ）
函数： complex _FloatNx ctanfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的复杂切线z 。复切线的数学定义是

tan（z）= -i *（exp（z * i）-exp（-z * i））/（exp（z * i）+ exp（-z * i））

复数切线在pi / 2 + 2n处具有极点，其中n是整数。 ctan如果以下情况可能会发出溢出信号z太靠近杆子了。
