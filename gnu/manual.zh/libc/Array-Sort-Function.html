9.3数组排序功能

要使用任意比较函数对数组进行排序，请使用qsort功能。该函数的原型在stdlib.h 。

功能： void qsort （void * array ，size_t count ，size_t size ，comparison_fn_t compare ）

初步： MT安全| AS安全AC不安全的腐败|请参阅POSIX安全概念 。

的qsort函数对数组进行排序array 。数组包含count元素，每个元素的大小size 。

的compare函数用于对数组元素进行比较。该函数使用两个指针参数调用，并且应返回小于，等于或大于零的整数，该整数对应于是否将其第一个参数视为小于，等于或大于其第二个参数。

警告：如果两个对象比较相等，则排序后的顺序是不可预测的。也就是说，排序不稳定。当比较仅考虑部分元素时，这可能会有所不同。具有相同排序键的两个元素可能在其他方面有所不同。

尽管传递给比较函数的对象地址位于数组内，但它们不必与这些对象的原始位置相对应，因为排序算法可能会在进行一些比较之前在数组中的对象周围交换。执行稳定排序的唯一方法是qsort首先使用某种单调计数器来增强对象。

这是一个使用上面定义的比较函数按数字顺序对双精度数组进行排序的简单示例（请参见比较函数 ）：

{
  double *array;
  int size;
  …
  qsort (array, size, sizeof (double), compare_doubles);
}

的qsort函数的名称源于它最初是使用“快速排序”算法实现的事实。

实施qsort该库中的“内存”可能不是就地排序，因此可能会使用额外的内存来存储数组。
