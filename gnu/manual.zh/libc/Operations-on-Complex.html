20.10复数的投影，共轭和分解

ISO C99 还定义了对复数执行基本运算（例如分解和共轭）的函数。所有这些功能的原型都在complex.h 。所有功能都有三种变体可用，三种复杂类型各一种。

功能： 双 Creal （复杂双z ）
功能： 浮动 crealf（复数浮点z ）
功能： 长 双绉 （复合长双z ）
函数： _FloatN crealfN （复杂的_Float Nz ）
函数： _FloatNx crealfNx （复杂的_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数的实部z 。

功能： 双重 cimag （复杂双重z ）
功能： 浮点 cimagf （复数浮点z ）
功能： 长双 cimagl （复杂的长双z ）
函数： _FloatN cimagfN （复杂的_Float Nz ）
函数： _FloatNx cimagfNx （复杂的_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数的虚部z 。

功能： 双重复杂 连词 （复双z ）
功能： 复数浮点数 conjf （complex float z ）
功能： 复数长双 连 （复数长双z ）
函数： complex _FloatN conjfN （complex _Float Nz ）
函数： complex _FloatNx conjfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数的共轭值z 。复数的共轭具有相同的实部和虚部。换一种说法， ' conj(a + bi) = a + -bi '。

功能： 双 焦 （复杂双z ）
功能： 浮子 cargf （复数浮子z ）
功能： 长双 焦糖 （复杂长双z ）
函数： _FloatN cargfN （复杂的_Float Nz ）
函数： _FloatNx cargfNx （复杂的_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数的参数z 。复数的参数是正实轴与穿过零和该数的线之间的复平面中的角度。该角度以常规方式测量，范围为-π到π 。

carg沿负实轴有一个分支。

功能： 复数双 cproj （复数双z ）
功能： 复杂浮点数 cprojf （复杂浮点数z ）
功能： 复数长双倍 cprojl （复数长双倍z ）
函数： complex _FloatN cprojfN （complex _Float Nz ）
函数： complex _FloatNx cprojfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数值的投影z进入黎曼球面即使实部为NaN，虚部为无穷大的值也会在实轴上投影为正无穷大。如果实部是无限的，则结果等于

INFINITY + I * copysign (0.0, cimag (z))
