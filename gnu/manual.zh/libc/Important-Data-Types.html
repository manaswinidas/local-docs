A.4重要数据类型

C中减去两个指针的结果始终是整数，但是精确的数据类型因C编译器而异。同样，结果的数据类型sizeof编译器之间也有所不同。ISO C为这两种类型定义了标准别名，因此您可以以可移植的方式引用它们。它们在头文件中定义stddef.h 。

数据类型： ptrdiff_t

这是两个指针相减所得结果的有符号整数类型。例如，带有声明char *p1, *p2; ， 表达方式p2 - p1是类型ptrdiff_t 。这可能是标准的有符号整数类型之一（ short int ， int要么long int ），但可能是仅出于此目的而存在的非标准类型。

数据类型： size_t

这是用于表示对象大小的无符号整数类型。结果sizeof运算符是这种类型的，并且诸如malloc （请参阅无限制分配 ）和memcpy （请参阅复制字符串和数组 ）接受此类型的参数来指定对象大小。在使用GNU C库的系统上，这将是unsigned int 或unsigned long int 。

使用注意事项： size_t是声明任何保留对象大小的参数或变量的首选方法。

兼容性说明： ISO C 出现之前的C实现unsigned int用于表示对象的大小和int用于指针减法结果。他们并不一定要定义size_t要么ptrdiff_t 。Unix系统确实定义了size_t ，在sys/types.h ，但定义通常是带符号的类型。
