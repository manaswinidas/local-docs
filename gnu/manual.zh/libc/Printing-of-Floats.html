' strfrom '函数在stdlib.h 。

函数： int strfromd （char * restrict string ，size_t size ，const char *限制format ，双value ）
函数： int strfromf （char * restrict string ，size_t size ，const char *限制format ，浮动value ）
函数： int strfroml （char * restrict string ，size_t size ，const char *限制format长双value ）

初步： MT安全区域| AS不安全堆| AC不安全的内存|请参阅POSIX安全概念 。

功能strfromd （“双引号字符串”）， strfromf （“浮动字符串”），以及strfroml （“ string-from-long-double”）转换浮点数value到一个字符串，并将它们存储到由指向的区域中string 。转换最多写入size字符并遵守以下格式： format 。

格式字符串必须以字符“ % '。随后是一个可选的精度，以句号“ . '，然后可以跟一个十进制整数，代表精度。如果句点后未指定十进制整数，则精度为零。性格 ' * ' 不允许。最后，格式字符串以以下转换说明符之一结尾：' a '，' A '，' e '，' E '，' f '，' F '，' g ' 要么 ' G '（请参见输出转换表 ）。无效的格式字符串会导致未定义的行为。

这些函数返回将要写入的字符数string有size足够大，不计算终止的空字符。因此，当且仅当返回值小于时，才以零结尾的输出被完全写入。 size 。

这些功能由ISO / IEC TS 18661-1引入。

函数： int strfromfN （char * restrict string ，size_t size ，const char *限制format ，_Float Nvalue ）
函数： int strfromfNx （char * restrict string ，size_t size ，const char *限制format ，_Float N X value ）

初步： MT安全区域| AS不安全堆| AC不安全的内存|请参阅POSIX安全概念 。
