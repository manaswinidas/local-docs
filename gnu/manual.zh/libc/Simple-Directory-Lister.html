14.2.4列出目录的简单程序

这是一个简单的程序，可打印当前工作目录中的文件名：



#include <stdio.h>
#include <sys/types.h>
#include <dirent.h>


int
main (void)
{
  DIR *dp;
  struct dirent *ep;

  dp = opendir ("./");
  if (dp != NULL)
    {
      while (ep = readdir (dp))
        puts (ep->d_name);
      (void) closedir (dp);
    }
  else
    perror ("Couldn't open the directory");

  return 0;
}

文件在目录中出现的顺序往往是相当随机的。一个更有用的程序将在打印之前对条目进行排序（也许通过按字母顺序排列）。请参阅扫描目录内容和数组排序功能 。
