14.2.5目录流中的随机访问

本节介绍如何重新读取已从打开目录流中读取的目录部分。所有符号都在头文件中声明dirent.h 。

功能： void rewinddir （DIR * dirstream ）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

的rewinddir函数用于重新初始化目录流dirstream ，以便您致电readdir它再次返回有关目录中第一个条目的信息。此功能还会注意到自从打开目录以来是否已将文件添加或删除到目录中。 opendir 。（这些文件的条目可能会或可能不会由以下人员返回readdir自您上次致电以来是否添加或删除了它们opendir要么rewinddir ）

函数： long inttelldir （DIR * dirstream ）

初步： MT安全| AS不安全堆/ bsd锁/ bsd |交流电源不安全的mem / bsd锁/ bsd |请参阅POSIX安全概念 。

的telldir函数返回目录流的文件位置dirstream 。您可以使用此值seekdir将目录流还原到该位置。

函数： void seekdir （DIR * dirstream ，长整数pos ）

初步： MT安全| AS不安全堆/ bsd锁/ bsd |交流电源不安全的mem / bsd锁/ bsd |请参阅POSIX安全概念 。

的seekdir函数设置目录流的文件位置dirstream至pos 。价值pos必须是先前呼叫的结果telldir在这个特定的流上；关闭并重新打开目录可以使以下参数返回的值无效telldir 。
