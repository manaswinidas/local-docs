20.4浮点数分类函数

ISO C99 定义了宏，这些宏使您可以确定变量保存哪种浮点数。

宏： int fpclassify （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这是一个通用宏，适用于所有浮点类型，并返回type的值int 。可能的值为：

FP_NAN

浮点数x是“不是数字”（请参阅Infinity和NaN ）

FP_INFINITE

的价值x是正负无穷大（请参阅Infinity和NaN ）

FP_ZERO

的价值x是零。在像IEEE 754 这样的浮点格式中，可以对零进行签名，如果x为负零。

FP_SUBNORMAL

绝对值太小而无法以常规格式表示的数字将以替代的非规范化格式表示（请参阅浮点概念 ）。此格式不太精确，但可以表示接近零的值。 fpclassify返回此值的值为x以这种替代格式。

FP_NORMAL

对于其他所有其他值，返回此值x 。它表示该数字没有什么特别的。

fpclassify如果必须测试多个属性中的一个以上，则最有用。还有一些更具体的宏，它们一次只能测试一个属性。通常，这些宏的执行速度比fpclassify ，因为它们有特殊的硬件支持。因此，您应尽可能使用特定的宏。

宏： int iscanonical （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

在某些浮点格式中，某些值具有规范（首选）和非规范编码（对于IEEE互换二进制格式，所有编码均为规范编码）。如果此宏返回一个非零值x具有规范的编码。它来自TS 18661-1：2014。

请注意，某些格式具有相同值的多个编码值； iscanonical对于所有此类编码，返回非零值。同样，格式可能具有与该类型的任何有效值都不对应的编码。用ISO C的术语表示陷阱 ;在GNU C库中iscanonical对于此类编码，返回零。

宏： int isfinite （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是有限的：不是正负无穷大，不是NaN。相当于

(fpclassify (x) != FP_NAN && fpclassify (x) != FP_INFINITE)

isfinite被实现为接受任何浮点类型的宏。

宏： int isnormal （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是有限的和规范化的相当于

(fpclassify (x) == FP_NORMAL)
宏： int isnan （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是NaN。相当于

(fpclassify (x) == FP_NAN)
宏： int issignaling （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是信号NaN（sNaN）。它来自TS 18661-1：2014。

巨集： int 为非正规 （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是次普通的。它来自TS 18661-1：2014。

宏： int iszero （ 浮点型 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此宏返回一个非零值x是零。它来自TS 18661-1：2014。

BSD提供了另一组浮点分类功能。GNU C库也支持这些功能。但是，我们建议您在新代码中使用ISO C99宏。这些是标准配置，将在更广泛的范围内提供。另外，由于它们是宏，因此您不必担心其参数的类型。

函数： int isinf （双 精度 x ）
函数： int isinff （浮点型x ）
函数： int isinfl （长双 精度 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

该函数返回-1如果x代表负无穷大， 1如果x代表正无穷大，并且0除此以外。

函数： int isnan （双 精度 x ）
函数： int isnanf （float x ）
函数： int isnanl （长 整数 x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此函数返回一个非零值x是“非数字”值，否则为零。

注意： isnan ISO C99 定义的宏将覆盖BSD函数。通常这不是问题，因为这两个例程的行为相同。但是，如果由于某些原因您确实需要获得BSD功能，则可以编写

(isnan) (x)
函数： int 有限 （双精度x ）
函数： int finitef （float x ）
函数： int finitel （long double x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

如果此函数返回一个非零值x既不是无限也不是“非数字”值，否则为零。
