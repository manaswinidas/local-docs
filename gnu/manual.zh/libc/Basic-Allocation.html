3.2.3.1基本内存分配

要分配一块内存，请调用malloc 。该函数的原型在stdlib.h 。

函数： void * malloc （size_t size ）

初步： MT安全| AS不安全锁| AC不安全锁fd mem |请参阅POSIX安全概念 。

该函数返回一个指向新分配块的指针size字节长，如果无法分配该块，则为空指针。

该块的内容是不确定的。您必须自己初始化（或使用calloc代替;请参阅分配净空 ）。通常，您会将值强制转换为要存储在块中的对象的指针。这里我们展示了一个这样做的例子，并使用库函数将空间初始化为零memset （请参阅复制字符串和数组 ）：

struct foo *ptr;
…
ptr = (struct foo *) malloc (sizeof (struct foo));
if (ptr == 0) abort ();
memset (ptr, 0, sizeof (struct foo));

您可以存储的结果malloc转换为任何指针变量而无需转换，因为ISO C 自动转换类型void *必要时指向另一种类型的指针。但是，在除赋值运算符之外的其他上下文中，或者如果您希望代码在传统C语言中运行，则强制转换是必需的。

请记住，在为字符串分配空间时， malloc必须为一加上字符串的长度。这是因为字符串以空字符结尾，该空字符不计入字符串的“长度”，但确实需要空格。例如：

char *ptr;
…
ptr = (char *) malloc (length + 1);

有关更多信息，请参见字符串表示 。
