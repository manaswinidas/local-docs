3.2.6.11 Obstack函数汇总

这是与堆栈相关的所有功能的摘要。每个都取一个栈的地址（ struct obstack * ）作为第一个参数。

void obstack_init (struct obstack *obstack-ptr)

初始化使用堆栈。请参阅创建堆栈 。

void *obstack_alloc (struct obstack *obstack-ptr, int size)

分配一个对象size未初始化的字节。请参阅在堆栈中分配 。

void *obstack_copy (struct obstack *obstack-ptr, void *address, int size)

分配一个对象size字节，内容从address 。请参阅在堆栈中分配 。

void *obstack_copy0 (struct obstack *obstack-ptr, void *address, int size)

分配一个对象size +1个字节，带有size其中复制自address ，最后是一个空字符。请参阅在堆栈中分配 。

void obstack_free (struct obstack *obstack-ptr, void *object)

自由object （以及指定的堆栈中分配的所有内容比object ）。请参阅释放Obstack对象 。

void obstack_blank (struct obstack *obstack-ptr, int size)

加size未初始化的字节到增长的对象。请参阅生长对象 。

void obstack_grow (struct obstack *obstack-ptr, void *address, int size)

加size字节，从address ，到一个增长的对象。请参阅生长对象 。

void obstack_grow0 (struct obstack *obstack-ptr, void *address, int size)

加size字节，从address到一个增长的对象，然后添加另一个包含空字符的字节。请参阅生长对象 。

void obstack_1grow (struct obstack *obstack-ptr, char data-char)

加一个包含的字节data-char越来越大的对象。请参阅生长对象 。

void *obstack_finish (struct obstack *obstack-ptr)

完成正在增长的对象并返回其永久地址。请参阅生长对象 。

int obstack_object_size (struct obstack *obstack-ptr)

获取当前正在增长的对象的当前大小。请参阅生长对象 。

void obstack_blank_fast (struct obstack *obstack-ptr, int size)

加size未初始化的字节到增长的对象，而无需检查是否有足够的空间。请参阅“ 快速增长” 。

void obstack_1grow_fast (struct obstack *obstack-ptr, char data-char)

加一个包含的字节data-char到正在生长的物体上，而不检查是否有足够的空间。请参阅“ 快速增长” 。

int obstack_room (struct obstack *obstack-ptr)

获取现在可用于生长当前对象的空间量。请参阅“ 快速增长” 。

int obstack_alignment_mask (struct obstack *obstack-ptr)

用于对齐对象起点的蒙版。这是一个左值。请参阅对象数据对齐 。

int obstack_chunk_size (struct obstack *obstack-ptr)

分配块的大小。这是一个左值。请参阅Obstack块 。

void *obstack_base (struct obstack *obstack-ptr)

当前正在增长的对象的暂定起始地址。请参阅容器的状态 。

void *obstack_next_free (struct obstack *obstack-ptr)

在当前增长的对象结束之后进行寻址。请参阅容器的状态 。
