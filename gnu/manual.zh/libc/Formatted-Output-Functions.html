12.12.7格式化输出函数

本节介绍如何呼叫printf和相关功能。这些函数的原型在头文件中stdio.h 。由于这些函数采用可变数量的参数， 因此在使用它们之前必须为其声明原型。当然，确保您拥有所有正确原型的最简单方法是仅包含stdio.h 。

函数： int printf （const char * template ，…）

初步： MT安全区域| AS不安全的损坏堆| AC不安全的内存锁损坏|请参阅POSIX安全概念 。

的printf函数在模板字符串的控制下打印可选参数template到流stdout 。它返回打印的字符数，如果出现输出错误，则返回负值。

函数： int wprintf （const wchar_t * template ，…）

初步： MT安全区域| AS不安全的损坏堆| AC不安全的内存锁损坏|请参阅POSIX安全概念 。

的wprintf函数在宽模板字符串的控制下打印可选参数template到流stdout 。它返回打印的宽字符数，如果出现输出错误，则返回负值。

函数： int fprintf （FILE * stream ，const char * template ，…）

初步： MT安全区域| AS不安全的损坏堆| AC不安全的内存锁损坏|请参阅POSIX安全概念 。

这个功能就像printf ，除了将输出写入流中stream代替stdout 。

函数： int fwprintf （FILE * stream ，const wchar_t * template ，…）

初步： MT安全区域| AS不安全的损坏堆| AC不安全的内存锁损坏|请参阅POSIX安全概念 。

这个功能就像wprintf ，除了将输出写入流中stream代替stdout 。

函数： int sprintf （char * s ，const char * template ，…）

初步： MT安全区域| AS不安全堆| AC不安全的内存|请参阅POSIX安全概念 。

就像printf ，除了输出存储在字符数组中s而不是写入流中。空字符被写入以标记字符串的结尾。

的sprintf函数返回数组中存储的字符数s ，不包括终止的空字符。

如果在重叠的对象之间进行复制，则该函数的行为不确定（例如，如果s也作为要在' %s转换。请参见复制字符串和数组 。

警告： sprintf该函数可能很危险，因为它可能输出的字符数可能超出了字符串的分配大小s 。请记住，转换规范中给出的字段宽度仅是最小值 。

为避免此问题，您可以使用snprintf要么asprintf ， 如下面所描述的。

函数： int swprintf （wchar_t * ws ，size_t size ，const wchar_t * template ，…）

初步： MT安全区域| AS不安全堆| AC不安全的内存|请参阅POSIX安全概念 。

就像wprintf ，除了输出存储在宽字符数组中ws而不是写入流中。将写入一个空的宽字符以标记字符串的结尾。的size参数指定要产生的最大字符数。尾随的空字符计入此限制，因此您至少应分配size字符串的宽字符ws 。

返回值是为给定输入生成的字符数，不包括结尾的null。如果不是所有输出都适合提供的缓冲区，则返回负值。您应该使用更大的输出字符串再试一次。注意：这与snprintf处理这种情况。

注意，相应的窄流函数采用较少的参数。 swprintf实际上对应于snprintf功能。自从sprintf函数可能很危险，应避免使用ISO C 委员会拒绝再次犯同样的错误，并决定不定义与之完全对应的函数sprintf 。

函数： int snprintf （char * s ，size_t size ，const char * template ，…）

初步： MT安全区域| AS不安全堆| AC不安全的内存|请参阅POSIX安全概念 。

的snprintf功能类似于sprintf ，除了size参数指定要产生的最大字符数。尾随的空字符计入此限制，因此您至少应分配size字符串字符s 。如果size为零，将不写入任何内容，甚至不写入空字节，并且s可以是空指针。

返回值是将为给定输入生成的字符数，不包括结尾的null。如果此值大于或等于size ，并非结果中的所有字符都已存储在s 。您应该使用更大的输出字符串再试一次。这是执行此操作的示例：

/* Construct a message describing the value of a variable
   whose name is name and whose value is value. */
char *
make_message (char *name, char *value)
{
  /* Guess we need no more than 100 chars of space. */
  int size = 100;
  char *buffer = (char *) xmalloc (size);
  int nchars;

  if (buffer == NULL)
    return NULL;

 /* Try to print in the allocated space. */
  nchars = snprintf (buffer, size, "value of %s is %s",
		     name, value);

  if (nchars >= size)
    {
      /* Reallocate buffer now that we know
	 how much space is needed. */
      size = nchars + 1;
      buffer = (char *) xrealloc (buffer, size);

      if (buffer != NULL)
	/* Try again. */
	snprintf (buffer, size, "value of %s is %s",
		  name, value);
    }
  /* The last call worked, return the string. */
  return buffer;
}
