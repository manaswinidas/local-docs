19.8.1 ISO C随机数函数

本部分描述了ISO C 标准中的随机数函数。

要使用这些功能，您应该包括头文件stdlib.h在您的程序中。

巨集： int RAND_MAX

此宏的值是一个整数常量，表示最大值rand函数可以返回。在GNU C库中，它是2147483647 ，这是32位代表的最大有符号整数。在其他库中，它可能低至32767 。

函数： int rand （无效）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

的rand函数返回序列中的下一个伪随机数。取值范围为0至RAND_MAX 。

函数： 无效 srand （无符号int seed ）

初步： MT安全| AS不安全锁| AC不安全锁|请参阅POSIX安全概念 。

该功能建立seed作为一系列新的伪随机数的种子。如果你打电话rand在建立种子之前srand ，它使用值1作为默认种子。

要在每次运行程序时产生不同的伪随机序列，请执行srand (time (0)) 。

POSIX.1扩展了C标准功能，以支持多线程程序中可再现的随机数。但是，扩展名设计不当，不适合进行认真的工作。

函数： int rand_r （无符号int * seed ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

此函数返回范围为0至RAND_MAX就像rand做。但是，其所有状态都存储在seed论点。这意味着RNG的状态只能具有与类型一样多的位unsigned int拥有。这太少了，无法提供良好的RNG。

如果您的程序需要可重入RNG，我们建议您对SVID随机数生成器使用可重入GNU扩展。仅当GNU扩展不可用时才应使用POSIX.1接口。
