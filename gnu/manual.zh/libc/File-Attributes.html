14.9文件属性

当您发出“ ls -l在文件上执行shell命令，它会为您提供有关文件大小，拥有者，上次修改时间等信息。这些称为文件属性 ，并且与文件本身而不是其名称中的特定名称相关联。

本节包含有关如何查询和修改文件属性的信息。

• 属性含义  文件属性的名称及其值的含义。
• 阅读属性  如何读取文件的属性。
• 测试文件类型  区分普通文件，目录，链接...
• 文件所有者  如何确定新文件的所有权以及如何更改它。
• 权限位  如何存储有关文件访问模式的信息。
• 访问权限  系统如何决定谁可以访问文件。
• 设置权限  如何分配新文件的权限，以及如何更改它们。
• 测试文件访问  如何确定您的进程是否可以访问文件。
• 文件时间  关于文件的时间属性。
• 文件大小  手动更改文件的大小。
• 存储分配  分配文件的后备存储。
