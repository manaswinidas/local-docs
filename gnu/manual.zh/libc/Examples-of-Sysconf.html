32.4.3的例子sysconf

我们建议您首先测试感兴趣的参数的宏定义，然后调用sysconf仅在未定义宏的情况下。例如，下面是如何测试是否支持作业控制的方法：

int
have_job_control (void)
{
#ifdef _POSIX_JOB_CONTROL
  return 1;
#else
  int value = sysconf (_SC_JOB_CONTROL);
  if (value < 0)
    /* If the system is that badly wedged,
       there’s no use trying to go on.  */
    fatal (strerror (errno));
  return value;
#endif
}

这是获取数字限制值的方法：

int
get_child_max ()
{
#ifdef CHILD_MAX
  return CHILD_MAX;
#else
  int value = sysconf (_SC_CHILD_MAX);
  if (value < 0)
    fatal (strerror (errno));
  return value;
#endif
}
