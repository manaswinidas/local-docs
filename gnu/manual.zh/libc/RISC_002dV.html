E.2 RISC-V专用设施

特定于实现Linux ABI的RISC-V系统的缓存管理工具在sys/cachectl.h 。

功能： void __riscv_flush_icache （void * start ，无效* end ，unsigned long int flags ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

强制在存储和指令高速缓存获取之间进行排序。强制执行排序的地址范围由start和end 。的flags参数以默认行为控制此排序的程度（a flags值0）强制在当前进程中的所有线程上执行隔离。设置SYS_RISCV_FLUSH_ICACHE_LOCAL该位允许用户指示仅在当前线程上强制执行排序是必要的。所有其他标志位均保留。
