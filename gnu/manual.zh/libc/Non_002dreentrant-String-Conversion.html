6.4.2字符串的不可重入转换

为了方便起见，ISO C90 标准还定义了用于转换整个字符串而不是单个字符的函数。这些功能与修订1 到ISO C90 的可重入功能存在相同的问题；请参见转换字符串 。

函数： size_t mbstowcs （wchar_t * wstring ，const char * string ，size_t size ）

初步： MT安全| AS不安全损坏的堆锁dlopen | AC不安全损坏的锁内存fd |请参阅POSIX安全概念 。

的mbstowcs （“多字节字符串到宽字符串”）功能转换以空终止的多字节字符字符串string到宽字符代码的数组，存储不超过size宽字符从数组开始wstring 。终止的空字符计入大小，因此如果size小于实际产生的宽字符数string ，不存储任何终止的空字符。

从字符转换string开始于初始换档状态。

如果发现无效的多字节字符序列，则mbstowcs函数返回值-1 。否则，它返回数组中存储的宽字符数wstring 。该数字不包含终止空字符，如果该数字小于size 。

这是一个示例，显示了如何转换多字节字符的字符串，并为结果分配足够的空间。 

wchar_t *
mbstowcs_alloc (const char *string)
{
  size_t size = strlen (string) + 1;
  wchar_t *buf = xmalloc (size * sizeof (wchar_t));

  size = mbstowcs (buf, string, size);
  if (size == (size_t) -1)
    return NULL;
  buf = xrealloc (buf, (size + 1) * sizeof (wchar_t));
  return buf;
}
功能： size_t wcstombs （字符* string ，const wchar_t * wstring ，size_t size ）

初步： MT安全| AS不安全损坏的堆锁dlopen | AC不安全损坏的锁内存fd |请参阅POSIX安全概念 。

的wcstombs （“宽字符串到多字节字符串”）函数转换以空值终止的宽字符数组wstring转换成包含多字节字符的字符串，存储不超过size字节起始于string ，如果有空间，则后面跟一个终止的空字符。字符转换从初始移位状态开始。

终止的空字符计入大小，因此如果size小于或等于所需的字节数wstring ，不存储任何终止的空字符。

如果找到与有效的多字节字符不对应的代码，则wcstombs函数返回值-1 。否则，返回值是数组中存储的字节数string 。该数字不包含终止空字符，如果该数字小于size 。
