20.8.1绝对值

提供这些功能是为了获得数字的绝对值 （或大小 ）。实数的绝对值x是x如果x是肯定的- x如果x是负面的。对于复数z ，其真正的部分是x而其假想部分是y ，绝对值为sqrt (x*x + y*y) 。

的原型abs ， labs和llabs在stdlib.h ; imaxabs在中声明inttypes.h ;的fabs函数在math.h ;的cabs函数在complex.h 。

函数： int abs （int number ）
函数： long int 实验室 （long int number ）
函数： long long int llabs （long long int number ）
函数： intmax_t imaxabs （intmax_t number ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的绝对值number 。

大多数计算机使用二进制补码整数表示，其中的绝对值INT_MIN （最小int ）无法代表；从而， abs (INT_MIN)未定义 。

llabs和imaxdiv是ISO C99 新功能。

有关整数的说明，请参见整数 。 intmax_t类型。

功能： 双 晶圆厂 （双number ）
功能： 浮法 fabsf （float number ）
功能： 长双 fabsl （长双number ）
功能： _FloatN fabsfN （_Float Nnumber ）
功能： _FloatNx fabsfNx （_Float N X number ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

该函数返回浮点数的绝对值number 。

功能： 双人 驾驶室 （复式z ）
功能： 浮动 cabsf （复杂浮动z ）
功能： 长双 cabsl（复杂的长双z ）
函数： _FloatN cabsfN （复杂的_Float Nz ）
函数： _FloatNx cabsfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复数的绝对值z （请参阅复数 ）。复数的绝对值为：

sqrt (creal (z) * creal (z) + cimag (z) * cimag (z))

应始终使用此函数而不是直接公式，因为它要特别小心以避免丢失精度。它也可以利用此操作的硬件支持。看到hypot用指数和对数表示 。
