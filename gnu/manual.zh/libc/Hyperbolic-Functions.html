19.5双曲函数

本节中的函数与指数函数有关；参见指数和对数 。

功能： 双 正弦 （双x ）
函数： float sinhf （float x ）
功能： 长双 sinhl （长双x ）
函数： _FloatN sinhfN （_Float Nx ）
函数： _FloatNx sinhfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的双曲正弦值x ，在数学上定义为(exp (x) - exp (-x)) / 2 。如果出现以下情况，它们可能会发出溢出信号x太大。

功能： 双 cosh （双x ）
函数： float coshf （float x ）
功能： 长双倍 coshl （长双倍x ）
函数： _FloatN coshfN （_Float Nx ）
函数： _FloatNx coshfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的双曲余弦值x ，在数学上定义为(exp (x) + exp (-x)) / 2 。如果出现以下情况，它们可能会发出溢出信号x太大。

功能： 双 tanh （双x ）
功能： 浮法 tanhf （float x ）
功能： 长双 tanhl （长双x ）
函数： _FloatN tanhfN （_Float Nx ）
函数： _FloatNx tanhfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的双曲正切值x ，在数学上定义为sinh (x) / cosh (x) 。如果出现以下情况，它们可能会发出溢出信号x太大。

双曲函数也有对应的函数，它们带有复杂的参数。

功能： 复数双倍 csinh （复数双倍z ）
功能： 复杂浮点 csinhf （复杂浮点z ）
功能： 复数长双倍 csinhl （复数长双倍z ）
函数： complex _FloatN csinhfN （complex _Float Nz ）
函数： complex _FloatNx csinhfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的复双曲正弦值z ，在数学上定义为(exp (z) - exp (-z)) / 2 。

功能： 复杂双 ccosh （复杂双z ）
功能： 复杂浮点 ccoshf （复杂浮点z ）
功能： 复数长双倍 ccoshl （复数长双倍z ）
函数： complex _FloatN ccoshfN （complex _Float Nz ）
函数： complex _FloatNx ccoshfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回复杂的双曲余弦z ，在数学上定义为(exp (z) + exp (-z)) / 2 。

功能： 复式双 箱 （复式双 箱 z ）
功能： 复杂浮点 ctanhf （复杂浮点z ）
功能： 复数长双倍 ctanhl （复数长双倍z ）
函数： complex _FloatN ctanhfN （complex _Float Nz ）
函数： complex _FloatNx ctanhfNx （complex _Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的复杂双曲正切值z ，在数学上定义为csinh (z) / ccosh (z) 。

功能： double asinh （双x ）
函数： float asinhf （float x ）
功能： 长双倍 asinhl （长双倍x ）
函数： _FloatN asinhfN （_Float Nx ）
函数： _FloatNx asinhfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反双曲正弦值x -双曲正弦值为的值x 。

功能： 双 阿科什 （双x ）
功能： 浮动 acoshf （float x ）
功能： 长双 阿斯科尔 （长双x ）
函数： _FloatN acoshfN （_Float Nx ）
函数： _FloatNx acoshfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反双曲余弦值x -双曲余弦值为的值x 。如果x小于1 ， acosh发出域错误信号。

功能： 双 安塔 （双x ）
函数： float atanhf （float x ）
功能： 长双 阿坦 （长双x ）
函数： _FloatN atanhfN （_Float Nx ）
函数： _FloatNx atanhfNx （_Float N X x ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反双曲正切值x —双曲正切值为的值x 。如果绝对值x大于1 ， atanh发出域错误信号；如果等于1 atanh返回无穷大。

功能： 复式双 卡西娜 （complex double z ）
功能： 复杂浮点数 casinhf （复杂浮点数z ）
功能： 复数长双 casinhl （复数长双z ）
函数： complex _FloatN casinhfN （complex _Float Nz ）
功能： 复杂_FloatNx casinhfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反复双曲正弦值z -其复双曲正弦值为的值z 。

功能： 复加 可可 （复加z ）
功能： 复杂浮点 cacoshf （复杂浮点z ）
功能： 复数长双倍 cacoshl （复数长双倍z ）
函数： complex _FloatN cacoshfN （complex _Float Nz ）
函数： 复杂_FloatNx cacoshfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反复双曲余弦值z -复双曲余弦为的值z 。与实数值函数不同，对实数值没有任何限制z 。

功能： 双重复杂 catanh（复双z ）
功能： 复杂浮点 catanhf （复杂浮点z ）
功能： 复数长双 Catanhl （复数长双z ）
功能： 复杂_FloatN catanhfN （复杂_Float Nz ）
功能： 复杂_FloatNx catanhfNx （复杂_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回的反复双曲正切值z —复双曲正切为的值z 。与实数值函数不同，对实数值没有任何限制z 。
