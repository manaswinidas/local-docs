12.7通过字符或行简单输出

本节介绍用于执行面向字符和行的输出的功能。

这些窄流函数在头文件中声明stdio.h和广泛的流功能wchar.h 。

函数： int fputc （int c ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

的fputc函数转换字符c输入unsigned char ，并将其写入流stream 。 EOF如果发生写入错误，则返回；否则返回0。否则字符c返回。

函数： wint_t fputwc （wchar_t wc ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

的fputwc函数写宽字符wc到流stream 。 WEOF如果发生写入错误，则返回；否则返回0。否则字符wc返回。

函数： int fputc_unlocked （int c ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的fputc_unlocked功能相当于fputc函数，但它不会隐式锁定流。

功能： wint_t fputwc_unlocked （wchar_t wc ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的fputwc_unlocked功能相当于fputwc函数，但它不会隐式锁定流。

此函数是GNU扩展。

函数： int putc （int c ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

就像fputc ，但大多数系统将其实现为宏，从而使其运行更快。结果之一是它可能会评估stream参数不止一次，这是宏通用规则的一个例外。 putc通常是用于编写单个字符的最佳功能。

函数： wint_t putwc （wchar_t wc ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

就像fputwc ，但可以将其实现为宏，从而使其更快。结果之一是它可能会评估stream参数不止一次，这是宏通用规则的一个例外。 putwc通常是用于编写单个宽字符的最佳功能。

函数： int putc_unlocked （int c ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的putc_unlocked功能相当于putc函数，但它不会隐式锁定流。

功能： wint_t putwc_unlocked （wchar_t wc ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的putwc_unlocked功能相当于putwc函数，但它不会隐式锁定流。

此函数是GNU扩展。

函数： int putchar （int c ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

的putchar功能相当于putc与stdout作为的价值stream论点。

函数： wint_t putwchar （wchar_t wc ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

的putwchar功能相当于putwc与stdout作为的价值stream论点。

函数： int putchar_unlocked （int c ）

初步： MT不安全种族：标准输出|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的putchar_unlocked功能相当于putchar函数，但它不会隐式锁定流。

功能： wint_t putwchar_unlocked （wchar_t wc ）

初步： MT不安全种族：标准输出|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的putwchar_unlocked功能相当于putwchar函数，但它不会隐式锁定流。

此函数是GNU扩展。

函数： int fputs （const char * s ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

功能fputs写字符串s到流stream 。不写入终止空字符。此函数也不添加换行符。它仅输出字符串中的字符。

该函数返回EOF如果发生写错误，否则为非负值。

例如：

fputs ("Are ", stdout);
fputs ("you ", stdout);
fputs ("hungry?\n", stdout);

输出文字' Are you hungry? '，然后换行。

函数： int fputws （const wchar_t * ws ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全的损坏锁|请参阅POSIX安全概念 。

功能fputws写宽字符串ws到流stream 。不写入终止空字符。此函数也不添加换行符。它仅输出字符串中的字符。

该函数返回WEOF如果发生写错误，否则为非负值。

函数： int fputs_unlocked （const char * s ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的fputs_unlocked功能相当于fputs函数，但它不会隐式锁定流。

此函数是GNU扩展。

函数： int fputws_unlocked （const wchar_t * ws ，文件* stream ）

初步： MT-Safe竞赛：流|不安全AS损坏| AC不安全的腐败|请参阅POSIX安全概念 。

的fputws_unlocked功能相当于fputws函数，但它不会隐式锁定流。

此函数是GNU扩展。

函数： int puts （const char * s ）

初步： MT安全|不安全AS损坏| AC不安全锁损坏|请参阅POSIX安全概念 。

的puts函数写入字符串s到流stdout然后是换行符。不写入字符串的终止空字符。（注意fputs因为这个函数不写一个换行符。）

puts是打印简单消息的最方便的功能。例如：

puts ("This is a message.");

输出文字' This is a message. '，然后换行。

函数： int putw （int w ，文件* stream ）

初步： MT安全|不安全AS损坏| AC不安全锁损坏|请参阅POSIX安全概念 。

该函数写词w （即int ） 至stream 。提供它是为了与SVID兼容，但我们建议您使用fwrite相反（请参见块输入/输出 ）。
