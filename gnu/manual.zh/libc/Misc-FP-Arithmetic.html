20.8.7其他FP算术函数

本节中的函数执行其他但通用的操作，这些操作很难用C运算符表示。在某些处理器上，这些功能可以使用特殊的机器指令来比等效的C代码更快地执行这些操作。

功能： 双 fmin （双x ，双y ）
函数： float fminf （float x ，浮动y ）
功能： 长双 发 （长双发） x长双y ）
函数： _FloatN fminfN （_Float Nx ，_Float Ny ）
函数： _FloatNx fminfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的fmin函数返回两个值中的较小者x和y 。它类似于表达

((x) < (y) ? (x) : (y))

除了那个x和y仅被评估一次。

如果参数为NaN，则返回另一个参数。如果两个参数均为NaN，则返回NaN。

功能： 双倍 fmax （双倍x ，双y ）
函数： float fmaxf （float x ，浮动y ）
函数： long double fmaxl （long double x长双y ）
函数： _FloatN fmaxfN （_Float Nx ，_Float Ny ）
函数： _FloatNx fmaxfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的fmax函数返回两个值中的较大者x和y 。

如果参数为NaN，则返回另一个参数。如果两个参数均为NaN，则返回NaN。

功能： 双 fminmag （双x ，双y ）
功能： 浮点数 fminmagf （float x ，浮动y ）
功能： 长双 fminmagl （长双x长双y ）
函数： _FloatN fminmagfN （_Float Nx ，_Float Ny ）
函数： _FloatNx fminmagfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

TS 18661-1：2014和TS 18661-3：2015中的这些函数返回两个值中的任意一个x和y具有较小的绝对值。如果两者具有相同的绝对值，或者两者均为NaN，则它们的行为与fmin功能。

功能： 双 fmaxmag （双x ，双y ）
函数： float fmaxmagf （float x ，浮动y ）
功能： 长双倍 fmaxmagl （长双倍x长双y ）
函数： _FloatN fmaxmagfN （_Float Nx ，_Float Ny ）
函数： _FloatNx fmaxmagfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

TS 18661-1：2014中的这些函数返回两个值中的任意一个x和y具有更大的绝对值。如果两者具有相同的绝对值，或者两者均为NaN，则它们的行为与fmax功能。

功能： 双 fdim （双x ，双y ）
功能： 浮点数 fdimf （float x ，浮动y ）
功能： 长双 fdiml （长双x长双y ）
函数： _FloatN fdimfN （_Float Nx ，_Float Ny ）
函数： _FloatNx fdimfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的fdim函数返回之间的正差x和y 。积极的区别是x -- y如果x大于y ，否则为0 。

如果x ， y ，或两者均为NaN，则返回NaN。

功能： 双 fma （双x ，双y ，双z ）
功能： 浮点数 fmaf （float x ，浮动y ，浮动z ）
功能： 长 整型 （长 整型） x长双y长双z ）
函数： _FloatN fmafN （_Float Nx ，_Float Ny ，_Float Nz ）
函数： _FloatNx fmafNx （_Float N X x ，_Float N X y ，_Float N X z ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的fma函数执行浮点乘法加法。这是操作（ x · y ）+ z ，但中间结果不会四舍五入到目标类型。有时这可以提高计算的精度。

之所以引入此功能，是因为某些处理器具有执行乘法加法的特殊指令。C编译器无法直接使用它，因为表达式' x*y + z定义为舍入中间结果。 fma让您选择只想舍入一次的时间。

在未在硬件中实现乘法加法的处理器上， fma可能非常慢，因为它必须避免中间舍入。 math.h定义符号FP_FAST_FMA ， FP_FAST_FMAF和FP_FAST_FMAL当对应版本fma不比表达式' x*y + z '。在GNU C库中，这始终意味着该操作是在硬件中实现的。

功能： float faddl （长双x长双y ）
功能： 双 daddl （长双x长双y ）

函数： _FloatMx fMxaddfN （_Float Nx ，_Float Ny ）
函数： _FloatMx fMxaddfNx （_Float N X x ，_Float N X y ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

从TS 18661-1：2014和TS 18661-3：2015开始的这些功能返回x + y ，四舍五入到函数的返回类型一次，而没有中间舍入到参数的类型。

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。
