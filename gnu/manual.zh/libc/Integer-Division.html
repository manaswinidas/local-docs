20.2整数除法

本节介绍执行整数除法的功能。使用GNU CC时，这些功能是多余的，因为在GNU C中，“ / '运算符总是四舍五入到零。但是在其他C实现中， /否定论点可能会以不同的方式出现。 div和ldiv之所以有用，是因为它们指定了如何将商：舍入为零。其余部分与分子的符号相同。

指定这些函数以返回结果r这样的价值r.quot*denominator + r.rem等于numerator 。

要使用这些功能，您应该包括头文件stdlib.h在您的程序中。

资料类型： div_t

这是一种结构类型，用于保存由div功能。它具有以下成员：

int quot

来自除法的商。

int rem

其余部分来自该部门。

函数： div_t div （int numerator ，int denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

功能div通过除法计算商和余数numerator通过denominator ，以类型的结构返回结果div_t 。

如果结果无法表示（如被零除），则行为是不确定的。

这是一个示例，尽管它不是一个非常有用的示例。 

div_t result;
result = div (20, -6);

现在result.quot是-3和result.rem是2 。

数据类型： ldiv_t

这是一种结构类型，用于保存由ldiv功能。它具有以下成员：

long int quot

来自除法的商。

long int rem

其余部分来自该部门。

（这与div_t除了组件是类型long int而不是int ）

函数： ldiv_t ldiv （长整数numerator ，长整数denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的ldiv功能类似于div ，除了参数是类型long int并将结果作为类型的结构返回ldiv_t 。

数据类型： lldiv_t

这是一种结构类型，用于保存由lldiv功能。它具有以下成员：

long long int quot

来自除法的商。

long long int rem

其余部分来自该部门。

（这与div_t除了组件是类型long long int而不是int ）

函数： lldiv_t lldiv （long long int numerator ，long long int denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的lldiv功能就像div函数，但是参数是类型long long int并将结果作为类型的结构返回lldiv_t 。

的lldiv在ISO C99 添加了功能。

这是一种结构类型，用于保存由imaxdiv功能。它具有以下成员：

来自除法的商。

intmax_t rem

其余部分来自该部门。

（这与div_t除了组件是类型intmax_t而不是int ）

有关整数的说明，请参见整数 。 intmax_t类型。

函数： imaxdiv_t imaxdiv （intmax_t numerator ，intmax_t denominator ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

的imaxdiv功能就像div函数，但是参数是类型intmax_t并将结果作为类型的结构返回imaxdiv_t 。

有关整数的说明，请参见整数 。 intmax_t类型。
