1.4手册路线图

这是本手册其余各章内容的概述。

  • 错误报告 ，描述了如何报告库检测到的错误。
  • 内存 ，描述了GNU C库用于管理和使用虚拟内存和实际内存的功能，包括动态分配虚拟内存。如果您事先不知道程序需要多少内存，则可以动态分配它，并通过指针对其进行操作。
  • 字符处理 ，包含有关字符分类功能的信息（例如isspace ）和执行大小写转换的功能。
  • 字符串和数组实用程序 ，介绍了用于处理字符串（以空字符结尾的字符数组）和常规字节数组的函数，包括复制和比较之类的操作。
  • 字符集处理 ，包含有关使用比正常情况下大的字符集来处理字符和字符串的信息char数据类型。
  • 语言环境 ，描述选择特定国家或语言如何影响图书馆的行为。例如，语言环境会影响字符串的排序规则序列以及货币值的格式。
  • 搜索和排序 ，包含有关数组搜索和排序功能的信息。通过提供适当的比较功能，可以在任何类型的阵列上使用这些功能。
  • 模式匹配 ，提供了一些功能，用于匹配正则表达式和shell文件名模式，并像shell一样扩展单词。
  • I / O概述全面介绍了库中的输入和输出功能，并包含有关基本概念（如文件名）的信息。
  • 流上的I / O ，描述涉及流的I / O操作（或FILE * 对象）。这些是来自的正常C库函数stdio.h 。
  • 底层I / O ，包含有关文件描述符上I / O操作的信息。文件描述符是特定于Unix系列操作系统的较低级机制。
  • 文件系统接口 ，描述了对整个文件的操作，例如删除和重命名文件以及创建新目录的功能。本章还包含有关如何访问文件属性的信息，例如文件所有者和文件保护模式。
  • 管道和FIFO包含有关简单的进程间通信机制的信息。管道允许两个相关进程之间的通信（例如，父级和子级之间），而FIFO允许共享同一台机器上的公共文件系统的进程之间的通信。
  • 套接字 ，描述了一种更复杂的进程间通信机制，该机制允许在不同计算机上运行的进程通过网络进行通信。本章还包含有关Internet主机寻址以及如何使用系统网络数据库的信息。
  • 底层终端接口 ，描述如何更改终端设备的属性。例如，如果要禁用用户键入的字符的回显，请阅读本章。
  • 数学 ，包含有关数学库功能的信息。这些包括随机数生成器和整数的余数函数，以及浮点数的常用三角函数和指数函数。
  • 低级算术函数 ，描述用于简单算术，浮点值分析和从字符串中读取数字的函数。
  • 日期和时间 ，描述用于测量日历时间和CPU时间的功能，以及用于设置警报和计时器的功能。
  • 非本地出口 ，包含对setjmp和longjmp功能。这些功能为goto类似的跳转，可以从一个功能跳转到另一个功能。
  • 信号处理 ，告诉您所有有关信号的信息-它们是什么，如何建立在传递特定种类的信号时调用的处理程序，以及如何防止信号在程序的关键部分到达。
  • 程序基础知识 ，告诉您的程序如何访问其命令行参数和环境变量。
  • 流程 ，包含有关如何启动新流程和运行程序的信息。
  • 作业控制 ，描述用于处理过程组和控制终端的功能。仅当您正在编写专门处理作业控制的Shell或其他程序时，此材料才可能是您感兴趣的内容。
  • 名称服务开关 ，描述了可用于在系统数据库中查找名称的服务，如何确定将哪个服务用于哪个数据库以及如何实现这些服务，以便贡献者可以设计自己的服务。
  • User Database和Group Database告诉您如何访问系统用户数据库和组数据库。
  • 系统管理 ，描述用于控制和获取有关程序正在执行的硬件和软件配置的信息的功能。
  • 系统配置 ，告诉您如何获取有关各种操作系统限制的信息。提供大多数这些参数是为了与POSIX兼容。
  • 语言功能 ，包含有关C语言标准部分的库支持的信息，包括诸如sizeof运算符和符号常量NULL ，如何编写接受可变数量的参数的函数以及描述数值类型范围和其他属性的常量。还有一个简单的调试机制，可让您在代码中放入断言，并在测试失败时打印诊断消息。
  • 库摘要 ，提供了库中所有函数，变量和宏的摘要，以及完整的数据类型和函数原型，并说明了每种标准或系统的来源。
  • Installation ，说明如何在系统上构建和安装GNU C库，以及如何报告可能发现的错误。
  • 维护 ，说明如何添加新功能或将库移植到新系统。

如果您已经知道所需设施的名称，则可以在Library Summary中查找它。这为您提供了其语法的摘要以及指向可以在其中找到更详细说明的指针。例如，如果您只想验证函数参数的顺序和类型，则此附录特别有用。它还告诉您每个函数，变量或宏都源自什么标准或系统。
