20.8.3舍入函数

此处列出的功能执行诸如舍入和截断浮点值的操作。其中一些函数将浮点数转换为整数值。它们都在声明math.h 。

您也可以将浮点数转换为整数，只需将其转换为int 。这将丢弃小数部分，有效地舍入为零。但是，只有在结果可以实际表示为int —对于非常大的数字，这是不可能的。此处列出的函数将结果返回为double而是解决这个问题。

的fromfp函数使用TS 18661-1：2014中的以下宏来指定取整方向。这些对应于IEEE 754-2008中定义的舍入方向。

FP_INT_UPWARD

向+∞舍入 。

FP_INT_DOWNWARD

向-∞舍入 。

FP_INT_TOWARDZERO

向零舍入。

FP_INT_TONEARESTFROMZERO

四舍五入到最接近，关系从零开始舍入。

FP_INT_TONEAREST

四舍五入到最接近，领带四舍五入到偶数。

功能： 双 modf （双value ，双* integer-part ）
功能： float modff （float value ，浮动* integer-part ）
函数： long double modfl （长双value长双* integer-part ）

功能： _FloatNx modffNx （_Float N X value ，_Float N X * integer-part ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些功能打破了论点value分为整数部分和小数部分（介于-1和1 ，独家）。他们的总和等于value 。每个部分的符号都与value ，并且整数部分始终四舍五入为零。

modf将整数部分存储在*integer-part ，并返回小数部分。例如， modf (2.5, &intpart)退货0.5和商店2.0进入intpart 。
