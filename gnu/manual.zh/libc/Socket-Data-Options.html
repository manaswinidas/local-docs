16.9.5.3套接字数据选项

的flags争论send和recv是一个面具。您可以将以下宏的值按位进行“或”运算以获得该参数的值。全部在头文件中定义sys/socket.h 。

巨集： int MSG_OOB

发送或接收带外数据。请参阅带外数据 。

巨集： int MSG_PEEK

查看数据，但不要将其从输入队列中删除。这仅对输入功能有意义，例如recv ，不与send 。

巨集： int MSG_DONTROUTE

不要在邮件中包含路由信息。这仅对输出操作有意义，通常仅对诊断或路由程序有意义。我们不在这里解释。
