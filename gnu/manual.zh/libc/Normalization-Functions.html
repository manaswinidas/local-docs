20.8.2归一化函数

本节中描述的功能主要是作为一种方法来有效地对使用二进制基数在内部表示的浮点数执行某些低级操作；请参阅浮点概念 。即使表示形式不使用基数2，也要求这些函数具有相同的行为，但是在这些情况下，它们当然不太可能特别有效。

所有这些功能都在中声明math.h 。

功能： 双 frexp （双value ，int * exponent ）
函数： float frexpf （float value ，int * exponent ）
功能： 长双倍 frexpl （长双倍value ，int * exponent ）
函数： _FloatN frexpfN （_Float Nvalue ，int * exponent ）
函数： _FloatNx frexpfNx （_Float N X value ，int * exponent ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些功能用于拆分数字value归一化的分数和一个指数。

如果论点value不为零，返回值为value乘以2的幂，并且其大小始终在1/2（含）到1（不含）之间。相应的指数存储在*exponent ;返回值乘以2得出的指数等于原始数value 。

例如， frexp (12.8, &exponent)退货0.8和商店4在exponent 。

如果value为零，则返回值为零，零存储在*exponent 。

功能： 双 ldexp （双value ，int exponent ）
函数： float ldexpf （float value ，int exponent ）
功能： 长双 ldexpl （长双value ，int exponent ）
函数： _FloatN ldexpfN （_Float Nvalue ，int exponent ）
函数： _FloatNx ldexpfNx （_Float N X value ，int exponent ）

初步： MT安全| AS安全交流安全请参阅POSIX安全概念 。

这些函数返回浮点数相乘的结果value乘以2 exponent 。（它可用于重组被除以的浮点数frexp ）

例如， ldexp (0.8, 4)退货12.8 。
