14.2.7列出目录的简单程序，Mark II

这是上面找到的目录列表器的修订版（请参阅Simple Directory Lister ）。使用scandir函数我们可以避免直接与目录内容一起使用的函数。通话后，返回的条目可直接使用。 



#include <stdio.h>
#include <dirent.h>


static int
one (const struct dirent *unused)
{
  return 1;
}

int
main (void)
{
  struct dirent **eps;
  int n;

  n = scandir ("./", &eps, one, alphasort);
  if (n >= 0)
    {
      int cnt;
      for (cnt = 0; cnt < n; ++cnt)
        puts (eps[cnt]->d_name);
    }
  else
    perror ("Couldn't open the directory");

  return 0;
}

注意此示例中的简单选择器功能。因为我们要查看所有目录条目，所以我们总是返回1 。
