jetty-web.xml是Jetty配置文件，您可以将其与特定的Web应用程序捆绑在一起。格式jetty-web.xml与jetty.xml相同–它是Jetty API的XML映射。

本文档概述了如何使用jetty-web.xml配置文件。有关语法的更多信息，请参见Jetty XML语法 。

根元素

jetty-web.xml基于每个Web应用程序应用，并配置一个实例org.eclipse.jetty.webapp.WebAppContext 。

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE Configure PUBLIC "-//Jetty//Configure//EN" "http://www.eclipse.org/jetty/configure_9_3.dtd">

<Configure class="org.eclipse.jetty.webapp.WebAppContext">
 ..
</Configure>

警告

确保将配置应用到适当类的实例。 jetty-web.xml配置一个实例WebAppContext ; jetty.xml配置一个实例Server 。

使用jetty-web.xml

放置jetty-web.xml进入您的Web应用程序WEB-INF夹。当Jetty部署Web应用程序时，它将查找一个名为WEB-INF/jetty-web.xml要么WEB-INF/web-jetty.xml在Web应用程序（或WAR）中，并应用在那里找到的配置。意识到jetty-web.xml 在所有其他配置都已应用到Web应用程序之后调用。

重要

重要的是要注意jetty-web.xml文件使用webapp的类路径，而不是服务器的类路径。

jetty-web.xml示例

该发行版包含一个示例jetty-web.xml在的WEB-INF文件夹中test webapp WAR（ $JETTY_HOME/demo-base/webapps/test.war/WEB-INF/jetty-web.xml ）。
