自动Web应用程序部署

部署Web应用程序的最基本技术是将WAR文件或Exploded WAR目录放到${jetty.base}/webapps/目录，然后让Jetty的部署扫描程序找到它并在同名的Context路径下部署它。

Jetty将仅检测遵循Web应用程序布局的Web应用程序，并以这种方式进行部署。

分配给此自动部署的上下文路径基于WAR的文件名（或目录名称）。

文件或目录名称分配的上下文路径

/webapps/footrope.war

http：// host / footrope /

/webapps/baggywrinkle-1.0.war

http：//host/baggywrinkle-1.0/

/webapps/lazaret-2.1.3-SNAPSHOT.war

http：//host/lazaret-2.1.3-SNAPSHOT/

/webapps/belaying-pins/WEB-INF/web.xml

http：// host / belaying-pins /

/webapps/root.war （保留名称）

http：//主机/

/webapps/root/WEB-INF/web.xml （保留名称）

http：//主机/
