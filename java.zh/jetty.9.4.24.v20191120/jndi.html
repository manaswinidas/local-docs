Jetty支持java:comp/env在webapps中查找。这是一项可选功能，需要进行一些配置。

快速设置

如果使用的是Jetty的标准发行版，则必须启用JNDI 模块以获得Jetty的JNDI实现，并启用plus 模块提供与JNDI进行交互的类。由于plus模块取决于JNDI模块，因此只需启用plus模块即可同时启用两者。假设您已经安装了Jetty /opt/jetty和你做一个Jetty基地在/opt/jetty/my-base ，执行：

cd /opt/jetty
cd my-base
java -jar $JETTY_HOME/start.jar --add-to-start=plus

现在，您可以启动Jetty并在Web应用程序中使用JNDI。有关如何向Jetty可以在webapps中查找的JNDI环境添加条目的信息，请参阅使用JNDI。

如果您有与您的JNDI资源相关联的额外jar，例如数据库驱动程序jar，并且尚未为其创建自定义模块 ，则可以将jar放入您的{$jetty base}ext/目录。然后，您需要启用ext模块，以确保ext/目录位于类路径上。假设您已经安装了Jetty /opt/jetty和你做一个Jetty基地在/opt/jetty/my-base ，执行：

cd /opt/jetty
cd my-base
java -jar $JETTY_HOME/start.jar --add-to-start=ext
