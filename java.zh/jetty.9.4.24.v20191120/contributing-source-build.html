Jetty为它的工件使用了几个开发主干。它们通过http://github.com/eclipse反映在github上，或者您可以通过以下URL上的Eclipse设置浏览它们。

建立

Jetty需要使用Java 8，并且始终建议构建最新版本。

Jetty使用Apache Maven 3来管理其构建和主要项目元数据。

只需通过进入相关目录并执行以下命令来构建Jetty：

$ git clone https://github.com/eclipse/jetty.project.git
$ cd jetty.project
$ mvn install

所有相关的依赖项将自动下载到本地存储库中。

注意

Jetty在其构建过程中有很多测试用例。周期性地，我们发现一些测试用例比它们应有的时间更多地依赖于时间，这会导致间歇性的测试失败。您可以通过打开错误报告来帮助跟踪这些问题。

导入到Eclipse

码头是一个Maven项目。要在Eclipse中开发Jetty，请遵循以下指示：

安装M2E插件

  1. 从屏幕顶部的Eclipse菜单中，选择帮助> Eclipse Marketplace。
  2. 搜索m2e 。
  3. 安装Maven Integration for Eclipse

克隆git仓库

在命令行上使用egit插件或git（如上述构建部分中所述），获得码头资源。

导入Maven项目

  1. 从Eclipse菜单中，选择File> Import。
  2. 从Maven文件夹中，选择“ 现有Maven项目”。
  3. 单击下一步 。
  4. 在“导入Maven项目”窗格中，单击“ 浏览” ，然后选择码头源树的顶部文件夹。
  5. 单击下一步/完成以将所有jetty导入Eclipse。
  6. 等待Eclipse和m2e合并并设置项目。
