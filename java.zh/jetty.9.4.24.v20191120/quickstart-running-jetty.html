跑步Jetty

示范基地

下载完Jetty发行版本后，解压缩zip要么tar.gz文件到您具有读写访问权限的位置。Jetty没有GUI（图形用户界面），因此可以从命令行运行服务器并执行许多配置选项。

一旦可以访问系统的命令行，请导航至解压缩Jetty发行版副本的目录。要在默认端口8080上启动Jetty，请运行以下命令：

$ java -jar start.jar

您可以通过将浏览器指向http：// localhost：8080来访问此演示服务器，该浏览器现在将显示一个欢迎页面和几个演示/测试Web应用程序。

警告

演示Web应用程序不一定安全，因此不应部署在生产Web服务器中。

您可以看到demo-base通过使用以下命令：

> cd $JETTY_HOME/demo-base/
> java -jar $JETTY_HOME/start.jar --list-modules
...

> java -jar $JETTY_HOME/start.jar --list-config
...

的--list-modules命令将返回服务器可用和已启用模块的完整列表。它还将显示模块的位置，它们的实现方式和执行顺序，从属模块以及关联的jar文件。

的--list-config命令显示有关服务器的信息，包括Java和Jetty环境，配置顺序，任何JVM参数或系统属性集，常规服务器属性，Jetty服务器类路径的完整列表以及活动的Jetty XML文件。
