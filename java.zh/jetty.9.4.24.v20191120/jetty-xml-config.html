jetty.xml是Jetty的默认配置文件，通常位于$JETTY_HOME/etc/jetty.xml 。通常情况下jetty.xml配置：

  • Server类（或子类，如果已扩展）和全局选项。
  • ThreadPool（最小和最大线程）。
  • 连接器（端口，超时，缓冲区大小，协议）。
  • 处理程序结构（默认处理程序和/或contextHandlerCollections）。
  • 扫描并部署Web应用程序和上下文的Deployment Manager。
  • 提供身份验证检查的登录服务。
  • 请求日志。

并非所有Jetty功能都在jetty.xml 。有几个共享配置格式的可选配置文件jetty.xml并连接（如果指定）。这些配置文件也存储在$JETTY_HOME/etc/ ，其中的示例在Github存储库中 。使用哪个配置文件的选择由以下控制start.jar Jetty XML Usage中描述了合并配置的过程。

根元素

jetty.xml配置一个实例Jetty org.eclipse.jetty.server.Server.

<?xml version="1.0"?>
<!DOCTYPE Configure PUBLIC "-//Jetty//Configure//EN" "http://www.eclipse.org/jetty/configure_9_3.dtd">

<Configure id="Server" class="org.eclipse.jetty.server.Server">
 ...
</Configure>

例子

$JETTY_HOME/etc包含默认jetty.xml ，以及其他示例配置文件（ jetty-*.xml ），可以通过命令行将其传递给服务器。
