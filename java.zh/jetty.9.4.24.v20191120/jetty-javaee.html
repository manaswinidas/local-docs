Java EE 7 Web配置文件
Jetty EE 6网站资料

Jetty实现了Java EE规范的各个方面，主要是Servlet规范。Java EE平台的最新版本引入了一个Web Profile，它认识到许多开发人员只需要Java EE框架下的许多技术中的一部分即可。

尽管Jetty本身并未提供所有Web Profile技术，但Jetty体系结构使您可以插入第三方实现，以生产出满足您确切需求的容器。
