将消息发送到远程端点

会议最重要的功能是访问org.eclipse.jetty.websocket.api.RemoteEndpoint需要发送消息。

使用RemoteEndpoint，您可以选择发送TEXT或BINARY WebSocket消息，或WebSocket PING和PONG控制帧。

阻止发送消息

大多数调用本质上都是阻塞的，直到发送完成（或引发异常）后才返回。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a BINARY message to remote endpoint
ByteBuffer buf = ByteBuffer.wrap(new byte[] { 0x11, 0x22, 0x33, 0x44 });
try
{
    remote.sendBytes(buf);
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何使用RemoteEndpoint发送简单的二进制消息。这将阻塞，直到发送消息为止，如果无法发送消息，则可能抛出IOException。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a TEXT message to remote endpoint
try
{
    remote.sendString("Hello World");
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何使用RemoteEndpoint发送简单的文本消息。这将阻塞，直到发送消息为止，如果无法发送消息，则可能抛出IOException。

发送部分消息

如果您要发送一条大消息，并且想要分段发送，则可以使用RemoteEndpoint的部分消息发送方法。只要确保您完成发送信息（ isLast == true ）。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a BINARY message to remote endpoint
// Part 1
ByteBuffer buf1 = ByteBuffer.wrap(new byte[] { 0x11, 0x22 });
// Part 2 (last part)
ByteBuffer buf2 = ByteBuffer.wrap(new byte[] { 0x33, 0x44 });
try
{
    remote.sendPartialBytes(buf1,false);
    remote.sendPartialBytes(buf2,true); // isLast is true
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何使用RemoteEndpoint中的部分消息支持分两部分发送二进制消息。这将阻塞，直到发送消息的每个部分为止，如果无法发送部分消息，则可能抛出IOException。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a TEXT message to remote endpoint
String part1 = "Hello";
String part2 = " World";
try
{
    remote.sendPartialString(part1,false);
    remote.sendPartialString(part2,true); // last part
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何使用RemoteEndpoint中的部分消息支持分两部分发送文本消息。这将阻塞，直到发送消息的每个部分为止，如果无法发送部分消息，则可能抛出IOException。

发送乒乓球/乒乓球控制框

您也可以使用RemoteEndpoint发送Ping和Pong控制帧。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a PING to remote endpoint
String data = "You There?";
ByteBuffer payload = ByteBuffer.wrap(data.getBytes());
try
{
    remote.sendPing(payload);
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何发送有效载荷为Ping的Ping控制帧"You There?" （作为字节数组有效负载到达远程端点）。这将阻塞，直到发送消息为止，如果无法发送ping帧，则可能抛出IOException。

RemoteEndpoint remote = session.getRemote();

// Blocking Send of a PONG to remote endpoint
String data = "Yup, I'm here";
ByteBuffer payload = ByteBuffer.wrap(data.getBytes());
try
{
    remote.sendPong(payload);
}
catch (IOException e)
{
    e.printStackTrace(System.err);
}

如何发送有效载荷为Pong的Pong控制帧"Yup I'm here" （作为字节数组有效负载到达远程端点）。这将阻塞，直到发送消息为止，如果无法发送乒乓帧，则可能抛出IOException。

为了正确使用Pong帧，您应该返回在Ping帧中收到的相同字节数组数据。

异步发送消息

但是，还有2种异步发送消息方法可用：

  • RemoteEndpoint.sendBytesByFuture(ByteBuffer message)
  • RemoteEndpoint.sendStringByFuture(String message)

都返回一个Future可以用来测试使用标准发送消息的成功与否java.util.concurrent.Future行为。

RemoteEndpoint remote = session.getRemote();

// Async Send of a BINARY message to remote endpoint
ByteBuffer buf = ByteBuffer.wrap(new byte[] { 0x11, 0x22, 0x33, 0x44 });
remote.sendBytesByFuture(buf);

如何使用RemoteEndpoint发送简单的二进制消息。该消息将排入队列以进行传出写入，但是您将不知道消息是成功还是失败。

RemoteEndpoint remote = session.getRemote();

// Async Send of a BINARY message to remote endpoint
ByteBuffer buf = ByteBuffer.wrap(new byte[] { 0x11, 0x22, 0x33, 0x44 });
try
{
    Future<Void> fut = remote.sendBytesByFuture(buf);
    // wait for completion (forever)
    fut.get();
}
catch (ExecutionException | InterruptedException e)
{
    // Send failed
    e.printStackTrace();
}

如何使用RemoteEndpoint发送简单的二进制消息，跟踪Future知道发送成功还是失败。

RemoteEndpoint remote = session.getRemote();

// Async Send of a BINARY message to remote endpoint
ByteBuffer buf = ByteBuffer.wrap(new byte[] { 0x11, 0x22, 0x33, 0x44 });
Future<Void> fut = null;
try
{
    fut = remote.sendBytesByFuture(buf);
    // wait for completion (timeout)
    fut.get(2,TimeUnit.SECONDS);
}
catch (ExecutionException | InterruptedException e)
{
    // Send failed
    e.printStackTrace();
}
catch (TimeoutException e)
{
    // timeout
    e.printStackTrace();
    if (fut != null)
    {
        // cancel the message
        fut.cancel(true);
    }
}

如何使用RemoteEndpoint发送简单的二进制消息，跟踪Future并且仅等待规定的时间以完成发送，如果发生超时则取消消息。

RemoteEndpoint remote = session.getRemote();

// Async Send of a TEXT message to remote endpoint
remote.sendStringByFuture("Hello World");
