持久会话：L2会话数据缓存

MemcachedSessionDataMap

如果您选择的持久性技术很慢，则在本地缓存会话数据可能会有所帮助。的CachingSessionDataStore是一种特殊的SessionDataStore在本地缓存会话数据，从而使读取速度更快。它会直写您选择的类型SessionDataStore会话数据更改时。

MemcachedSessionDataMap

的MemcachedSessionDataMap用途memcached执行缓存。

要通过Jetty发行版启用它，请启用session-store-cache 模块 ，以及您选择的session-store-xxxx模块，以及可选的session-cache-hash要么session-cache-null模块。

启用后， $jetty.base/start.d/session-store-cache.ini文件将生成：

--module=session-store-cache


## Session Data Cache type: xmemcached
session-data-cache=xmemcached
#jetty.session.memcached.host=localhost
#jetty.session.memcached.port=11211
#jetty.session.memcached.expirySec=
#jetty.session.memcached.heartbeats=true

配置属性为：

jetty.session.memcached.host
默认值为localhost 。这是内存缓存服务器所在的主机。
jetty.session.memcached.port
默认值为11211 。这是内存缓存服务器正在侦听的端口。
jetty.session.memcached.expirySec
默认值0 。这是项目可以保留在内存缓存中的时间长度（以秒为单位），其中0表示无限期。
jetty.session.memcached.heartbeats
默认值true 。memcached系统是否应生成心跳。
